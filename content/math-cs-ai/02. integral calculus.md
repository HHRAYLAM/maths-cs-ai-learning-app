# 积分学

*积分学在区间上累积量，将局部变化率还原为总量。本章涵盖定积分和不定积分、微积分基本定理、积分技巧以及在 ML 中概率密度和期望值计算中的应用。*

- 微分告诉我们单点的变化率。积分则相反：它累积许多微小的片段来计算总量。

- 如果微分回答"有多快？"，积分回答"有多少？"

- 思考积分最简单的方式是作为**曲线下面积**。如果绘制函数 $f(x)$ 并将曲线和 x 轴之间从 $x = a$ 到 $x = b$ 的区域着色，积分给出该区域的有符号面积。

![积分通过求和薄矩形计算曲线下面积](../images/area_under_curve.svg)

- 为什么是"有符号"？x 轴上方的区域贡献正面积，下方的区域贡献负面积。这具有物理意义：如果 $f(x)$ 表示速度，积分给出净位移（向前减去向后），而不是总距离。

- 为了计算这个面积，想象将区域切片成 $n$ 个薄的垂直矩形，每个宽度为 $\Delta x$。每个矩形的高度是该切片中某点的函数值。将它们相加：

$$\text{面积} \approx \sum_{i=1}^{n} f(x_i^\ast) \, \Delta x$$

- 当矩形越来越薄时（$n \to \infty$，$\Delta x \to 0$），求和变得精确。这个极限过程定义了**定积分**：

$$\int_a^b f(x)\, dx = \lim_{n \to \infty} \sum_{i=1}^{n} f(x_i^\ast) \, \Delta x$$

- $\int$ 符号是拉长的"S"，代表"求和"。$dx$ 提醒我们是沿 x 轴求和无穷小的切片。

- **不定积分**（或**原函数**）是导数为 $f(x)$ 的函数 $F(x)$。我们写作：

$$\int f(x)\, dx = F(x) + C$$

- $+ C$ 是**积分常数**。由于任何常数的导数都是零，所以有无限多个原函数，它们只相差一个常数。例如，$\int 2x\, dx = x^2 + C$，因为 $x^2 + 7$ 或 $x^2 - 3$ 的导数仍然是 $2x$。

- **微积分基本定理**是连接微分和积分的桥梁。它有两个部分：

- **第一部分**：如果 $F(x)$ 是 $f(x)$ 的原函数，则定积分等于 $F$ 在端点的差值：

$$\int_a^b f(x)\, dx = F(b) - F(a)$$

- 这非常实用。与其计算求和的极限（很难），不如找到原函数并在两点求值（通常很容易）。

- **第二部分**：如果定义 $F(x) = \int_a^x f(t)\, dt$，则 $F'(x) = f(x)$。微分和积分是逆运算，它们相互抵消。

- 例如，计算 $\int_1^3 x^2\, dx$：$x^2$ 的原函数是 $\frac{x^3}{3}$。所以 $\int_1^3 x^2\, dx = \frac{27}{3} - \frac{1}{3} = \frac{26}{3} \approx 8.67$。

- 就像微分有法则一样，积分有相应的法则来逆转它们：

| 函数 | 积分 | 条件 |
|---|---|---|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ | $n \neq -1$ |
| $\frac{1}{x}$ | $\ln\|x\| + C$ | |
| $e^x$ | $e^x + C$ | |
| $a^x$ | $\frac{a^x}{\ln a} + C$ | |
| $\sin x$ | $-\cos x + C$ | |
| $\cos x$ | $\sin x + C$ | |
| $k$（常数） | $kx + C$ | |

- **和/差法则**延续：$\int [f(x) \pm g(x)]\, dx = \int f(x)\, dx \pm \int g(x)\, dx$。常数可以提出：$\int k\, f(x)\, dx = k \int f(x)\, dx$。

- 当函数过于复杂无法直接积分时，我们有技巧来简化它。

- **u 替换法**是链式法则的逆转。如果看到复合函数 $f(g(x))$ 乘以 $g'(x)$，替换 $u = g(x)$ 使得 $du = g'(x)\, dx$，积分简化。

- 例如：$\int 2x \cos(x^2)\, dx$。令 $u = x^2$，所以 $du = 2x\, dx$。积分变为 $\int \cos(u)\, du = \sin(u) + C = \sin(x^2) + C$。

- **分部积分法**是乘积法则的逆转。如果被积函数是两个函数的乘积：

$$\int u\, dv = uv - \int v\, du$$

- 策略性地选择 $u$ 和 $dv$，使得剩余的积分 $\int v\, du$ 比原来的更简单。选择 $u$ 的常用助记符是**LIATE**：对数（Logarithmic）、反三角（Inverse trig）、代数（Algebraic）、三角（Trigonometric）、指数（Exponential）（从较早类别中选择 $u$）。

- 例如：$\int x\, e^x\, dx$。令 $u = x$（代数）和 $dv = e^x\, dx$。则 $du = dx$ 和 $v = e^x$。所以：$\int x\, e^x\, dx = x\, e^x - \int e^x\, dx = x\, e^x - e^x + C = e^x(x - 1) + C$。

- 在 ML 中，积分出现在概率论中（通过积分密度函数计算概率）、期望值中（连续分布的加权平均）以及计算 ROC 曲线下面积中。虽然我们在实践中很少手工积分，但理解积分的含义有助于解释这些量。

## 编码任务（使用 CoLab 或 notebook）

1. 使用黎曼和近似 $\int_0^1 x^2\, dx$，使用越来越多的矩形。与精确答案 $\frac{1}{3}$ 比较。
```python
import jax.numpy as jnp

for n in [10, 100, 1000, 10000]:
    x = jnp.linspace(0, 1, n, endpoint=False)
    dx = 1.0 / n
    area = jnp.sum(x**2 * dx)
    print(f"n={n:5d}  近似：{area:.6f}  精确：{1/3:.6f}")
```

2. 数值验证微积分基本定理。定义 $F(x) = \int_0^x t^2\, dt = \frac{x^3}{3}$ 并检查其导数（通过 `jax.grad` 计算）等于 $x^2$。
```python
import jax
import jax.numpy as jnp

F = lambda x: x**3 / 3
dF = jax.grad(F)

for x in [0.5, 1.0, 2.0, 3.0]:
    print(f"x={x:.1f}  F'(x)={dF(x):.4f}  x^2={x**2:.4f}")
```

3. 可视化 $f(x) = \sin(x)$ 从 $0$ 到 $\pi$ 的下方面积。使用 `plt.fill_between` 着色区域并用黎曼和数值计算。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

x = jnp.linspace(0, jnp.pi, 500)
y = jnp.sin(x)

plt.plot(x, y, color="purple", linewidth=2)
plt.fill_between(x, y, alpha=0.2, color="purple")
plt.title(f"面积 = {jnp.sum(jnp.sin(x) * (jnp.pi / 500)):.4f}  (精确：2.0)")
plt.show()
```
