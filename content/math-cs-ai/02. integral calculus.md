## 📐 积分是什么？

积分是微分的逆运算！

- 微分：已知总量 → 求速率
- 积分：已知速率 → 求总量

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 直观理解

想象将时间段切成很多小片：

```
每片宽度 = Δt（很小）
每片高度 = 速率 f(t)
每片面面积 = f(t) × Δt = 这片生产的奶酪
```

把所有小片加起来：

$$\text{总量} \approx \sum_{i=1}^{n} f(t_i) \cdot \Delta t$$

当切片无限薄时（$n \to \infty$），得到精确值：

$$\int_a^b f(t)\, dt = \lim_{n \to \infty} \sum_{i=1}^{n} f(t_i) \cdot \Delta t$$

</div>
  <div class="lesson-visual-column">

```
积分 = 累积面积：

  速率 (kg/h)
    ↑
  30|     ┌──────┐
    |    /│      │
  20|   / │      │
    |  /  │      │
  10| /   │      │
    |/____│______│____
  0 +----+------+------+→ 时间 (h)
    0    2      4

阴影面积 = 总产量！
```

## 🔢 定积分 vs 不定积分

### 定积分 - 计算具体数值

$$\int_a^b f(x)\, dx$$

- $\int$ = 拉长的 "S"，代表 Sum（求和）
- $a$ = 下限（起点）
- $b$ = 上限（终点）
- $dx$ = 无限薄的切片宽度

结果是一个数字（面积值）

### 不定积分 - 寻找原函数

$$\int f(x)\, dx = F(x) + C$$

- $F(x)$ 是原函数（$F'(x) = f(x)$）
- $C$ 是积分常数

结果是一个函数

<div class="key-point">
  <div class="key-point-title">🎯 为什么需要 +C？</div>
  <ul class="key-point-list">
    <li>常数的导数是 0</li>
    <li>所以 $x^2 + 5$ 和 $x^2 + 100$ 的导数都是 $2x$</li>
    <li>不定积分有<strong>无限多个</strong>原函数</li>
    <li>它们只相差一个常数</li>
  </ul>
</div>

## 🌉 微积分基本定理 - 连接微分和积分的桥梁

微积分基本定理说：

$$\int_a^b f(x)\, dx = F(b) - F(a)$$

其中 $F(x)$ 是 $f(x)$ 的原函数（$F'(x) = f(x)$）

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 为什么伟大？

计算面积不需要无限求和了！

步骤：
1. 找到原函数 $F(x)$
2. 代入上限 $F(b)$
3. 代入下限 $F(a)$
4. 相减：$F(b) - F(a)$

### 例子

计算 $\int_1^3 x^2\, dx$：

1. 原函数：$F(x) = \frac{x^3}{3}$
2. $F(3) = \frac{27}{3} = 9$
3. $F(1) = \frac{1}{3}$
4. 答案：$9 - \frac{1}{3} = \frac{26}{3} \approx 8.67$

</div>
  <div class="lesson-visual-column">

```
微积分基本定理：

  定积分 = F(上限) - F(下限)
  
       F(3) = 9
         ↑
    ┌────┴────┐
    │  面积   │
    │         │
    └────┬────┘
         ↑
       F(1) = 1/3
  
  面积 = 9 - 1/3 = 26/3
```

## 📊 常见函数的积分表

| 函数 $f(x)$ | 积分 $\int f(x)\, dx$ | 备注 |
|------|------|------|
| $x^n$ | $\frac{x^{n+1}}{n+1} + C$ | $n \neq -1$ |
| $\frac{1}{x}$ | $\ln|x| + C$ | |
| $e^x$ | $e^x + C$ | 唯一等于自身的积分！ |
| $a^x$ | $\frac{a^x}{\ln a} + C$ | |
| $\sin x$ | $-\cos x + C$ | 注意负号！ |
| $\cos x$ | $\sin x + C$ | |
| $k$（常数） | $kx + C$ | |

## 🔧 积分技巧

### 1. 换元积分法（u-substitution）

链式法则的逆运算！

当看到复合函数 $f(g(x)) \cdot g'(x)$ 时：

1. 令 $u = g(x)$（内层函数）
2. 则 $du = g'(x)\, dx$
3. 积分变为 $\int f(u)\, du$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的换元理解</div>
  <div class="story-card-content">
    就像剥洋葱：
    <br><br>
    1. 把复杂的内层换成简单的 $u$
    <br>
    2. 积分简单函数
    <br>
    3. 把 $u$ 换回来
    <br><br>
    反向的链式法则！
  </div>
</div>

#### 例子

$$\int 2x \cos(x^2)\, dx$$

- 令 $u = x^2$，所以 $du = 2x\, dx$
- 积分变为：$\int \cos(u)\, du = \sin(u) + C$
- 换回来：$\sin(x^2) + C$ ✓

---

### 2. 分部积分法

乘积法则的逆运算！

$$\int u\, dv = uv - \int v\, du$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 如何选择 u 和 dv？

用 LIATE 法则（优先选为 u）：

1. Logarithmic（对数）
2. Inverse trig（反三角）
3. Algebraic（代数）
4. Trig（三角）
5. Exponential（指数）

### 例子

$$\int x e^x\, dx$$

- $u = x$（代数），$dv = e^x dx$
- $du = dx$，$v = e^x$
- $\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C$

</div>
  <div class="lesson-visual-column">

```
分部积分流程：

  ∫ u dv = uv - ∫ v du
  
  例：∫ x·e^x dx
  
  u = x      dv = e^x dx
  ↓          ↓
  du = dx    v = e^x
  
  = x·e^x - ∫ e^x dx
  = x·e^x - e^x + C
```

### 概率密度函数

$$P(a \leq X \leq b) = \int_a^b f(x)\, dx$$

概率 = 密度函数曲线下面积！

### 期望值

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\, dx$$

加权平均的连续版本！
