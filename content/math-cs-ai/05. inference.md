# 统计推断

*统计推断超越是非决策，用量化的不确定性估计总体参数。本文件涵盖置信区间、点和区间估计、最大似然估计、矩方法和回归分析，这是原始数据和 ML 预测模型之间的桥梁。*

- 假设检验给你一个是非决策：拒绝或不拒绝。但通常你想要更有 informative 的东西，一个你估计参数的合理值范围。这就是**置信区间**提供的。

- **点估计**是从样本计算的单个数字，如样本均值 $\bar{x}$。它是对总体参数的最佳猜测，但单独来看，它没有给出估计有多精确的概念。

- **置信区间**包裹点估计，范围反映不确定性。它采用形式：

$$\text{CI} = \bar{x} \pm \text{ME}$$

- **误差幅度（ME）**取决于三件事：你想要的置信度、数据中有多少变异性，以及样本有多大：

$$\text{ME} = z^\ast \cdot \frac{\sigma}{\sqrt{n}}$$

- 这里 $z^\ast$ 是匹配你所需置信水平的正态分布的临界值。对于 95% 置信度，$z^\ast = 1.96$。对于 99% 置信度，$z^\ast = 2.576$。

![置信区间：点估计两侧带误差幅度](../images/confidence_interval.svg)

- **95% 置信区间**意味着：如果你多次重复实验并每次构建区间，约 95% 的这些区间将包含真实总体参数。这并不意味着参数在这个特定区间内有 95% 概率。参数是固定的；区间是变化的。

- **示例**：你测量 50 人的身高并发现 $\bar{x} = 170$ cm，$\sigma = 8$ cm。构建 95% 置信区间。

$$\text{ME} = 1.96 \cdot \frac{8}{\sqrt{50}} = 1.96 \cdot 1.131 = 2.22 \text{ cm}$$

$$\text{CI} = [170 - 2.22, \; 170 + 2.22] = [167.78, \; 172.22]$$

- 你可以有 95% 置信度说真实平均身高在 167.78 和 172.22 cm 之间。

- 当 $\sigma$ 未知时（通常情况），使用样本标准差 $s$ 和 t 分布：

$$\text{CI} = \bar{x} \pm t^\ast_{n-1} \cdot \frac{s}{\sqrt{n}}$$

- 更宽的区间更自信但较不精确。更窄的区间更精确但较不自信。你可以在不失去置信度的情况下通过增加样本量来窄化区间。

- **功效分析**帮助你在运行实验前规划实验。问题是：我需要多大样本才能以指定功效检测给定大小的效应？

- 回忆前一个文件，功效 = $1 - \beta$，正确拒绝错误 $H_0$ 的概率。常见目标是 80% 功效。

- 检测差异 $\delta$ 的 z 检验所需样本量，显著性 $\alpha$ 和功效 $1-\beta$ 是：

$$n = \left(\frac{(z_{\alpha/2} + z_{\beta}) \cdot \sigma}{\delta}\right)^2$$

- 例如，检测平均身高 2 cm 差异（$\sigma = 8$），$\alpha = 0.05$ 和 80% 功效（$z_{0.025} = 1.96$，$z_{0.20} = 0.84$）：

$$n = \left(\frac{(1.96 + 0.84) \cdot 8}{2}\right)^2 = \left(\frac{22.4}{2}\right)^2 = 11.2^2 \approx 126$$

- 每组需要约 126 人。

- 功效分析防止两个常见错误：运行实验太小无法检测真实效应（功效不足），或浪费资源在远大于必要的实验上（功效过度）。

- **蒙特卡罗方法**使用随机采样解决难以或不可能解析解决的问题。核心思想：如果不能精确计算某些东西，多次模拟它并使用结果作为近似。

- 名称来自蒙特卡罗赌场，暗示随机性的作用。这些方法是 ML 中的主力，用于估计积分、评估模型不确定性和近似复杂分布等任务。

- 通用蒙特卡罗配方：
    - 定义可能输入的域
    - 从该域生成随机输入
    - 在每个输入上评估函数
    - 聚合结果（平均、计数等）

- 经典示例是估计 $\pi$。想象边长为 2 的正方形，中心在原点，内切半径为 1 的圆。正方形面积是 4，圆面积是 $\pi$。

![带内切圆的正方形，随机点按内部/外部着色](../images/monte_carlo_pi.svg)

- 在正方形内均匀丢弃随机点。落在圆内的比例近似 $\pi/4$：

$$\pi \approx 4 \times \frac{\text{圆内点数}}{\text{总点数}}$$

- 点 $(x, y)$ 在圆内如果 $x^2 + y^2 \le 1$。丢弃的点越多，估计越接近 $\pi$ 的真实值。

- 在 ML 中，蒙特卡罗方法出现在：
    - **蒙特卡罗 dropout**：用启用的 dropout 运行推理多次以估计预测不确定性
    - **MCMC**（马尔可夫链蒙特卡罗）：从贝叶斯模型中的复杂后验分布采样
    - **策略梯度方法**：通过采样轨迹在强化学习中估计梯度

- **因子分析**是发现隐藏（潜在）变量的技术，解释观测变量之间的相关性。如果 10 个人格调查问题可以用 3 个潜在特质（外向性、宜人性、尽责性）解释，因子分析找到这些特质。

- 模型假设每个观测变量 $x_i$ 是少数潜在因子 $f_j$ 加噪声的线性组合：

$$x_i = \lambda_{i1} f_1 + \lambda_{i2} f_2 + \ldots + \lambda_{ik} f_k + \epsilon_i$$

- $\lambda$ 值称为**因子载荷**，告诉你每个观测变量与每个因子的关系强度。这直接连接到第 2 章的矩阵分解；因子分析与特征值分解和 SVD 密切相关。

- **实验设计**是结构化实验以得出有效结论的艺术。糟糕的设计可以使大数据集无用。

- 设计良好的实验的关键组件：
    - **自变量（IV）**：你操纵什么（例如药物剂量、模型架构）
    - **因变量（DV）**：你测量什么（例如恢复时间、准确率）
    - **对照组**：不接受治疗（或安慰剂），提供比较基线
    - **随机分配**：参与者随机分配到组，平衡你未测量的混淆变量

- **常见实验设计**：
    - **完全随机设计**：受试者随机分配到治疗组。当组可比时简单有效。
    - **随机区组设计**：受试者首先分组到区组（例如按年龄），然后在每个区组内随机分配到治疗。这减少来自区组因子的变异性，类似于分层抽样的精神。
    - **析因设计**：同时测试多个自变量。$2 \times 3$ 析因设计有一个变量 2 个水平和另一个变量 3 个水平，给出 6 个治疗组合。这允许检测**交互作用**，其中一个变量的效应取决于另一个变量的水平。
    - **交叉设计**：每个受试者按顺序接受所有治疗（之间有洗脱期）。每个受试者作为自己的对照，减少个体差异的影响。

- 在 ML 实验中，这些原则至关重要。比较模型时，你应该控制随机种子、数据集分割和硬件。交叉验证是交叉设计的一种形式。消融研究（一次移除一个组件）遵循析因设计的逻辑。

1. 为身高示例构建 95% 置信区间，然后试验不同置信水平和样本量。
```python
import jax.numpy as jnp

x_bar = 170.0    # sample mean
sigma = 8.0      # population std (known)
n = 50           # sample size

# Critical values for common confidence levels
z_stars = {0.90: 1.645, 0.95: 1.960, 0.99: 2.576}

for conf, z_star in z_stars.items():
    me = z_star * (sigma / jnp.sqrt(n))
    lower, upper = x_bar - me, x_bar + me
    print(f"{conf*100:.0f}% CI: [{lower:.2f}, {upper:.2f}]  (ME = {me:.2f})")
```

2. 使用蒙特卡罗模拟估计 $\pi$。绘制估计如何随点数增加而收敛。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

key = jax.random.PRNGKey(42)

# Generate random points in [-1, 1] x [-1, 1]
n_points = 100_000
k1, k2 = jax.random.split(key)
x = jax.random.uniform(k1, shape=(n_points,), minval=-1, maxval=1)
y = jax.random.uniform(k2, shape=(n_points,), minval=-1, maxval=1)

# Check which points are inside the unit circle
inside = (x**2 + y**2) <= 1.0
cumulative_inside = jnp.cumsum(inside)
counts = jnp.arange(1, n_points + 1)
pi_estimates = 4.0 * cumulative_inside / counts

plt.figure(figsize=(10, 4))
plt.plot(pi_estimates, color="#3498db", alpha=0.7, linewidth=0.5)
plt.axhline(y=jnp.pi, color="#e74c3c", linestyle="--", label=f"π = {jnp.pi:.6f}")
plt.xlabel("Number of points")
plt.ylabel("Estimate of π")
plt.title("Monte Carlo estimation of π")
plt.legend()
plt.ylim(2.8, 3.5)
plt.show()

print(f"Final estimate: {pi_estimates[-1]:.6f}")
print(f"True value:     {jnp.pi:.6f}")
print(f"Error:          {abs(pi_estimates[-1] - jnp.pi):.6f}")
```

3. 执行简单功效分析：对于给定效应大小和标准差，计算所需样本量并通过模拟验证。
```python
import jax
import jax.numpy as jnp

# Parameters
delta = 2.0      # effect size (difference in means)
sigma = 8.0      # population std
alpha = 0.05
power_target = 0.80

# Analytical sample size
z_alpha = 1.96   # two-tailed, alpha=0.05
z_beta = 0.84    # power=0.80
n_required = ((z_alpha + z_beta) * sigma / delta) ** 2
print(f"Required n per group: {n_required:.0f}")

# Verify by simulation
key = jax.random.PRNGKey(7)
n = int(jnp.ceil(n_required))
n_sims = 5000
rejections = 0

for _ in range(n_sims):
    key, k1, k2 = jax.random.split(key, 3)
    group_a = jax.random.normal(k1, shape=(n,)) * sigma + 50
    group_b = jax.random.normal(k2, shape=(n,)) * sigma + 50 + delta
    pooled_se = jnp.sqrt(2 * sigma**2 / n)
    z = (group_b.mean() - group_a.mean()) / pooled_se
    p = 2 * (1 - __import__("jax").scipy.stats.norm.cdf(jnp.abs(z)))
    if p <= alpha:
        rejections += 1

print(f"Simulated power: {rejections/n_sims:.3f}")
print(f"Target power:    {power_target:.3f}")
```

4. 可视化置信区间宽度如何随样本量变化。这说明为什么收集更多数据给出更精确估计。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

sigma = 8.0
z_star = 1.96  # 95% confidence

sample_sizes = jnp.array([10, 20, 30, 50, 100, 200, 500, 1000], dtype=jnp.float32)
margins = z_star * sigma / jnp.sqrt(sample_sizes)

plt.figure(figsize=(8, 4))
plt.bar([str(int(n)) for n in sample_sizes], margins, color="#3498db", alpha=0.7)
plt.xlabel("Sample size")
plt.ylabel("Margin of error (cm)")
plt.title("95% CI margin of error shrinks with larger samples")
plt.show()
```
