# 文本转语音和语音

*文本转语音合成逆转 ASR 管道，从书面文本生成自然听起来音频。本文件涵盖 TTS 管道（文本归一化、G2P、声学模型、声码器）、Tacotron、WaveNet、HiFi-GAN、语音克隆、语音转换和语音活动检测（VAD）*

- 在文件 01 中，我们构建了信号处理工具箱：波形、频谱图、mel 滤波器组和 MFCC。在文件 02 中，我们将语音转换为文本。现在我们反转箭头：给定文本，合成自然语音。这是**文本转语音**（TTS），一个问题，也打开语音转换、语音克隆和语音活动检测的大门。

- 将 TTS 想象成舞台表演。剧本是文本输入。导演（声学模型）决定每行如何说话，其音高、时机、强调。然后乐团（声码器）执行乐谱，产生观众听到的实际声波。现代神经 TTS 用媲美人类演讲者的表演，取代基于规则系统的僵硬、机器人式传递。

![TTS 管道：文本归一化，转换为音素，由声学模型处理产生 mel 频谱图，然后通过声码器生成最终波形](../images/tts_pipeline.svg)

- **文本转语音管道**标准 TTS 管道有四个阶段：(1) 文本归一化，(2) 音素转换，(3) 声学模型，(4) 声码器。一些现代系统将阶段 3 和 4 合并为单个端到端模型，但概念分解仍然有用。

- **文本归一化**将原始文本转换为可发音形式。缩写展开（"Dr." 到 "Doctor"），数字变成词（"1984" 到 "nineteen eighty-four"），货币符号口语化（"$5" 到 "five dollars"），URL 或特殊字符处理。这个阶段通常是基于规则的，带语言特定语法，尽管存在神经归一化模型。这里的错误传播到每个下游阶段：如果 "St." 读作 "saint" 而不是 "street"，整个话语错误。

- **字素到音素**（G2P）映射归一化文本到音素序列。英语非常不规则（"though"、"through"、"tough" 都用 "ough" 不同），所以字典查找（CMU 发音词典）处理常见词，而神经序列到序列模型（第 06 章的编码器 - 解码器或第 07 章的变换器）处理词汇外词。浅正字法语言（西班牙语、芬兰语）需要更简单 G2P。输出通常是 IPA（国际音标）序列或等效内部音素集。

- **声学模型**消费音素序列，产生中间声学表示，几乎总是**mel 频谱图**（文件 01）。mel 频谱图在每个时间帧捕捉频谱包络，编码声码器重构波形所需的感知相关信息。声学模型必须决定时机（每个音素持续多久）、音高（基频 $F_0$）和能量（响度）。

- **声码器**采用 mel 频谱图，产生原始音频波形。这是病态反演问题：许多波形可以产生相同频谱图，因为相位信息被丢弃。经典声码器（Griffin-Lim、WORLD）使用迭代或信号模型方法，但神经声码器现在主导质量。

- **声码器：WaveNet**（van den Oord 等人，2016）是第一个产生几乎与人类录音无法区分的语音的神经声码器。它自回归建模波形，预测每个样本 $x_t$，以所有先前样本为条件：

$$P(x) = \prod_{t=1}^{T} P(x_t \mid x_1, \ldots, x_{t-1}, c)$$

- 其中 $c$ 是条件信号（mel 频谱图）。每个样本 16 位，所以 65536 值的天真 softmax 不切实际。WaveNet 使用**μ 律压缩**减少到 256 量化级别，或后来变体使用物流混合分布。

- WaveNet 的核心构建块是**膨胀因果卷积**。因果意味着滤波器权重只看过去样本（无未来泄漏）。膨胀意味着滤波器以指数增加间隙跳过样本：膨胀因子 $1, 2, 4, 8, \ldots, 512$。这给出指数大感受野，同时保持参数计数线性。

- 门控激活为：

$$z = \tanh(W_{f} \ast x) \odot \sigma(W_{g} \ast x)$$

- 其中 $W_f$ 和 $W_g$ 是滤波器和门卷积权重，$\ast$ 表示膨胀因果卷积，$\odot$ 是逐元素乘法。这个门控机制（来自第 06 章的 LSTM）允许网络控制信息流。

- WaveNet 产生卓越质量，但在推理时极慢：生成 1 秒 24 kHz 音频需要 24000 次顺序前向传播。这激励所有后续声码器研究。

- **WaveRNN**（Kalchbrenner 等人，2018）用单层循环网络替换 WaveNet 的深卷积栈。它将每个 16 位样本分裂为粗（上 8 位）和细（下 8 位）组件，用 GRU（第 06 章）预测每个。这个双 softmax 方法显著减少计算，同时保持高质量。WaveRNN 足够快，通过仔细内核优化在移动 CPU 上实时。

- **WaveGlow**（Prenger 等人，2019）是**基于流**的声码器，完全避免自回归生成。它使用一系列可逆变换（仿射耦合层，第 06 章的归一化流）将简单高斯分布映射到波形分布。训练使用变量变化公式最大化精确对数似然：

$$\log P(x) = \log P(z) + \sum_{i} \log \left| \det \frac{\partial f_i}{\partial f_{i-1}} \right|$$

- 其中 $z = f(x)$ 是通过流传递 $x$ 获得的潜在变量。在推理时，样本 $z \sim \mathcal{N}(0, I)$ 绘制，通过逆流在单个并行通过中推送。WaveGlow 用模型大小（耦合层的大网络）换取生成速度。

- **HiFi-GAN**（Kong 等人，2020）使用**生成对抗网络**从 mel 频谱图合成波形。生成器通过一系列转置卷积上采样 mel 频谱图，每个后跟**多感受野融合**（MRF）模块。MRF 模块并行应用多个带不同内核大小和膨胀率的残差块，然后求和输出。这允许生成器同时捕捉多个时间尺度的模式。

![HiFi-GAN 生成器架构：mel 频谱图输入通过转置卷积上采样层，每个后跟多感受野融合块，组合并行残差栈，带不同膨胀模式](../images/hifi_gan_generator.svg)

- HiFi-GAN 使用两种判别器类型。**多周期判别器**（MPD）通过以不同周期（2、3、5、7、11）折叠，将 1D 波形重塑为 2D，然后应用 2D 卷积。这捕捉不同基频的周期结构。**多尺度判别器**（MSD）在原始波形、2x 下采样和 4x 下采样版本上操作，捕捉不同时间分辨率的模式。

- 训练目标结合对抗损失、**mel 频谱图重构损失**（合成和真实音频的 mel 频谱图的 L1 距离）和**特征匹配损失**（中间判别器特征的 L1 距离）：

$$\mathcal{L}_G = \mathcal{L}_{\text{adv}}(G) + \lambda_{\text{mel}} \mathcal{L}_{\text{mel}}(G) + \lambda_{\text{fm}} \mathcal{L}_{\text{fm}}(G)$$

- HiFi-GAN 实现与 WaveNet 相当的合成质量，同时快 1000 倍以上，实现在单个 GPU 上实时生成。

- **神经源滤波器**（NSF）结合传统信号处理与神经网络。在经典源滤波器模型中，浊音语音由源激励（基频 $F_0$ 的周期脉冲串）通过声道滤波器（频谱包络）产生。NSF 模型用神经网络替换手工滤波器，同时保持显式源信号。输入 $F_0$ 轮廓提供精细音高控制，纯数据驱动声码器有时难以处理。

- **声学模型：Tacotron**（Wang 等人，2017）是第一个端到端神经 TTS 系统，直接将字符序列转换为 mel 频谱图。它使用带注意力（第 07 章）的编码器 - 解码器架构。编码器用卷积库、高速公路网络和双向 GRU 处理字符/音素序列。解码器是自回归 GRU，一次预测一个 mel 帧，使用前一个帧和注意力上下文作为输入。

- **Tacotron 2**（Shen 等人，2018）显著细化架构。编码器是 3 层 1D 卷积栈，后跟双向 LSTM（第 06 章）。解码器是 2 层 LSTM，带**位置敏感注意力**，它不仅以编码器输出和解码器状态为条件，还以前一步的累积注意力权重为条件。这防止注意力跳过或重复单词的常见失败模式。

![Tacotron 2 架构：字符/音素编码器带卷积层和 BiLSTM，位置敏感注意力对齐 mel 频谱图帧，自回归解码器带停止标记预测](../images/tacotron2_architecture.svg)

- 解码器位置 $j$ 在解码器步骤 $i$ 的位置敏感注意力能量为：

$$e_{i,j} = w^T \tanh(W_s s_{i-1} + W_h h_j + W_f f_{i,j} + b)$$

- 其中 $s_{i-1}$ 是前一个解码器状态，$h_j$ 是位置 $j$ 的编码器输出，$f_{i,j}$ 是通过用 1D 卷积滤波器卷积累积注意力权重$\sum_{k<i} \alpha_{k,j}$获得的位置特征。注意力权重是 $\alpha_{i,j} = \text{softmax}(e_{i,j})$。

- Tacotron 2 的解码器还在每个步骤预测**停止标记**概率，指示 mel 频谱图何时完成。输出 mel 频谱图然后通过声码器（最初 WaveNet，后来替换为 HiFi-GAN 或类似）。

- Tacotron 2 的自回归性质意味着合成速度受 mel 帧数限制。对于典型每秒 80 帧 mel 频谱图，5 秒话语需要 400 个顺序解码器步骤。

- **FastSpeech**（Ren 等人，2019）用**非自回归**声学模型解决速度问题。FastSpeech 不是顺序生成 mel 帧，而是并行生成所有帧。关键挑战是确定每个音素应该产生多少 mel 帧，FastSpeech 用**持续时间预测器**处理。

- 持续时间预测器是小型卷积网络，预测每个音素的整数持续时间（mel 帧数）。在训练期间，真实持续时间从预训练自回归教师模型（Tacotron 2）使用其注意力对齐提取。在推理期间，预测持续时间用于通过**长度调节器**扩展音素级隐藏序列到帧级，简单地重复每个音素的隐藏表示预测的帧数。

- **FastSpeech 2**（Ren 等人，2021）改进 FastSpeech，移除教师 - 学生蒸馏。它使用强制对齐（来自文件 02 的声学模型框架）直接提取真实持续时间，添加明确的**方差适配器**用于音高（$F_0$）和能量，除了持续时间。每个适配器是小型卷积预测器，其输出条件解码器：

```math
\begin{aligned}
\hat{d}_i &= \text{DurationPredictor}(h_i) \\
\hat{p}_i &= \text{PitchPredictor}(h_i) \\
\hat{e}_i &= \text{EnergyPredictor}(h_i)
\end{aligned}
```

- 其中 $h_i$ 是音素 $i$ 的编码器隐藏状态。在训练时，使用真实值；在推理时，预测值给出口音控制。这是 FastSpeech 2 的主要优势：调整音高、速度或能量就像缩放预测器输出一样简单。

- FastSpeech 2 在推理时通常比 Tacotron 2 快 10-20 倍，避免常见自回归失败模式，如单词跳过、重复和注意力崩溃。

- **VITS**（Kim 等人，2021）是**端到端**TTS 模型，直接从文本生成波形，消除单独的声码器阶段。VITS 结合条件变分自编码器（第 06 章）与归一化流和对抗训练。后验编码器将真实 mel 频谱图映射到潜在空间，先验编码器通过基于变换器的文本编码器和持续时间预测器将音素映射到相同潜在空间，解码器（基于 HiFi-GAN）从潜在样本生成波形。

- VITS 的训练目标结合：
    - **重构损失**：VAE 强制潜在分布编码声学信息
    - **KL 散度**：对齐文本条件先验与音频条件后验
    - **对抗损失**：判别器确保波形质量
    - **持续时间损失**：训练随机持续时间预测器

- VITS 产生比两阶段系统（FastSpeech 2 + HiFi-GAN）更高质量，因为声学模型和声码器联合优化，避免预测和真实 mel 频谱图之间的不匹配，降低两阶段系统质量。

- **VALL-E**（Wang 等人，2023）激进地将 TTS 重构为**语言建模问题**，在离散音频标记上。它使用神经音频编解码器（EnCodec）将语音表示为来自多个码本级别的离散代码序列。给定文本提示和 3 秒注册话语（也编码为离散标记），VALL-E 使用变换器语言模型自回归预测音频标记。

- VALL-E 使用两个模型：**自回归**（AR）模型逐个生成第一个码本级别标记，**非自回归**（NAR）模型并行预测剩余码本级别，以第一个级别和彼此为条件。这个编解码器语言模型方法实现非凡的零样本语音克隆：3 秒样本足以重现说话者的声音、音色甚至情感语调。

- **StyleTTS**（Li 等人，2022）和**StyleTTS 2**将语音解耦为内容和风格组件。风格编码器从参考音频提取风格向量，捕捉说话者身份、口音和录音条件。在推理期间，风格可以从学习先验分布采样，或从参考话语转移。StyleTTS 2 对风格先验使用扩散模型（第 08 章），生成多样和自然口音。

- **Kokoro**（2024）是轻量级、高质量开源 TTS 模型，以其小尺寸（约 8200 万参数）和令人印象深刻的自然度著称。它使用 StyleTTS 2 启发架构，带基于扩散的风格先验和微调 ISTFTNet 声码器，直接预测 STFT 系数（来自文件 01），而不是原始波形样本。尽管是 VALL-E 等模型尺寸的一小部分，Kokoro 为英语、日语、法语、韩语和中文实现接近人类的自然度，证明精心策划的训练数据和高效架构设计可以与暴力规模竞争。Kokoro 的小 footprint 使其适用于本地和边缘部署。

- **Orpheus**（Canopy Labs，2025）是开源 TTS 模型家族（1B 和 3B 参数），建立在**编解码器语言模型**范式上，由 VALL-E 开创。Orpheus 进一步发展这个想法，用 LLM 骨干（微调 Llama 3）直接生成 SNAC 音频编解码器标记。其突出特点是人类般情感表现力：它处理笑声、叹息、犹豫和情感口音，非凡自然。Orpheus 可以用输入文本中的标签如 `[laugh]` 或 `[sigh]` 提示，给副语言表达细粒度控制。

- **Dia**（Nari Labs，2025）是开源对话 TTS 模型，从单个文本生成真实多说话者对话。建立在 1.6B 参数编码器 - 解码器变换器上，Dia 处理轮流、特定说话者声音和非语言线索（笑声、停顿）在对话内。它还支持从短音频提示语音克隆，在对话上下文中实现零样本说话者生成。

- **Sesame CSM**（Conversational Speech Model，2025）专注于自然多轮对话语音。Sesame 不是优化朗读式 TTS，而是模拟真实对话的动态：后台通道（"uh huh"）、打断、说话者之间的节奏变化和情感响应。模型使用变换器骨干，以对话上下文（文本和音频历史）为条件，产生适应对话流程风格的语音。

- **Fish Speech**（Fish Audio，2024）是开源 TTS 系统，使用双自回归架构：大语言模型从文本生成语义标记，较小模型将这些转换为 VQGAN 声学标记，由声码器解码为波形。Fish Speech 支持从 10-15 秒参考零样本语音克隆，实现适合实时应用的低延迟。其模块化设计允许独立交换组件（如不同声码器）。

- **ChatTTS**（2024）是为对话应用（如聊天机器人和虚拟助手）设计的开源对话 TTS 模型。它生成自然、对话听起来语音，带口音特征的细粒度控制（笑声、停顿、填充词），使用嵌入文本输入的特殊标记。ChatTTS 支持中英混合合成和多说话者生成。

- **Bark**（Suno，2023）是基于变换器的开源模型，从文本提示生成语音、音乐和音效。它使用三阶段变换器模型管道（文本 → 语义标记 → 粗声学标记 → 细声学标记），支持语音克隆、多语言合成和非语音音频如音乐和环境声音。Bark 的通用性以可控性为代价——它不如专用 TTS 系统精确，但更灵活。

- **Parler-TTS**（Hugging Face，2024）采用**自然语言描述**方法进行语音控制：不是需要参考音频剪辑用于风格，用户提供文本描述，如"温暖、富有表现力的女性声音在安静房间"。Parler-TTS 在标注语音数据上训练，每个话语与说话风格的自然语言描述配对，实现直观控制，无需任何参考音频。

- **Neuphonic** 是基于 API 的 TTS 平台，针对超低延迟语音合成优化，面向实时语音代理和对话 AI 应用。它通过流式架构实现首次音频时间低于 100 毫秒，在完整输入文本可用之前开始生成音频。Neuphonic 专注于部署和延迟优化层，而不是新颖模型架构，为现代神经 TTS 提供生产级基础设施。

- **KittenTTS** 是紧凑、快速 TTS 模型，设计用于效率和低资源部署。它优先考虑最小延迟和小模型尺寸，用于边缘和嵌入式应用，以一些自然度换取 CPU 和移动设备上的实时性能。

- 现代 TTS 格局正分为两个范式：(1) **编解码器语言模型**（VALL-E、Orpheus、Fish Speech）将语音生成视为离散音频代码上的下一个标记预测，利用 LLM 的缩放定律；(2) **基于流/扩散的模型**（VITS、StyleTTS 2、Kokoro）通过迭代细化生成连续 mel 频谱图或波形。编解码器 LM 擅长零样本克隆和表现力；流/扩散模型往往更小更快。两者都迅速收敛到人类水平自然度。

- **口音建模**控制语音的"音乐"：音高、持续时间、能量、节奏和语调。没有好的口音，合成语音听起来平淡和机器人，即使单个音素清晰。想象口音是单调 GPS 声音和富有表现力有声书 narrator 之间的差异。

- **音高**（基频 $F_0$）是感知语音的高低。它在问题末尾上升，在陈述末尾下降，在情感语音中连续变化。$F_0$ 使用算法如 CREPE（神经音高跟踪器）或 YIN（基于自相关，来自文件 01）从音频提取。在 TTS 中，音高要么由声学模型预测（FastSpeech 2 的音高预测器），或隐式学习（Tacotron 2）。

- **持续时间**决定语速和节奏。重读音节更长，功能词缩短，停顿标记短语边界。持续时间建模在非自回归模型中明确（FastSpeech），在自回归模型中隐式（Tacotron 的注意力对齐确定持续时间）。

- **能量**（响度）携带强调。"我没有说 HE 偷了它"与"我没有说他 STOLE 它"有完全不同的含义，完全通过能量模式传达。

- **风格嵌入**捕捉更高级口音模式。**全局风格标记**（GST）框架（Wang 等人，2018）学习风格标记库（对学习嵌入集的软注意力），捕捉说话风格如"兴奋"、"悲伤"或"耳语"。风格嵌入从参考话语提取，添加到编码器输出，允许推理时风格转移。

- **语音转换**（VC）改变话语的说话者身份，同时保留语言内容。想象录制自己，输出听起来像特定目标说话者。VC 需要解耦说话者身份与内容。

![语音转换管道：源语音分解为内容表示和说话者嵌入，目标说话者嵌入替换源，解码器用目标声音重构语音](../images/voice_conversion_pipeline.svg)

- **说话者嵌入**（文件 04 详细）将说话者身份编码为固定维向量。这些可以来自预训练说话者验证模型（x-vectors、ECAPA-TDNN）。在 VC 中，源语音编码为与说话者无关的内容表示，然后用目标说话者嵌入解码。

- **解耦表示**将语音分离为独立因子：内容（音素）、说话者身份、音高和节奏。方法包括：
    - **信息瓶颈**：压缩内容表示如此紧密，以至于说话者信息丢失（AutoVC）
    - **对抗训练**：在内容表示上训练说话者分类器，使用梯度反转移除说话者信息
    - **向量量化**：VQ-VAE 强制内容通过离散瓶颈，自然剥离说话者身份（因为码本条目表示语音类别，不是说话者特征）

- **语音克隆**用目标说话者的声音合成语音。**多说话者 TTS**在来自许多说话者的数据上训练，以说话者嵌入为条件模型。在推理时，从注册音频提取新说话者的嵌入，用于条件生成。

- **少样本语音克隆**用少量数据（几分钟）适应新说话者。说话者编码器从注册音频提取嵌入，TTS 模型生成以这个嵌入为条件的语音。这是 SV2TTS（Jia 等人，2018）使用的方法：单独训练的说话者编码器、以说话者嵌入为条件的 Tacotron 2 合成器、WaveRNN 声码器。

- **零样本语音克隆**不需要适应：单个短话语（3-30 秒）足够。VALL-E 实现这个，通过将注册音频视为语言模型的提示。模型学习以相同声音继续生成，因为它在大规模多说话者数据上训练，其中话语内的声音一致性是统计规范。

- **语音活动检测**（VAD）回答每个时间帧的简单二元问题：有人在说话还是没有？尽管简单，VAD 是 ASR（文件 02）、说话者日记化（文件 04）和降噪（文件 05）的关键预处理步骤。好的 VAD 通过跳过沉默减少计算，通过防止噪声作为语音处理提高准确性。

- 经典 VAD 使用能量阈值（语音比沉默响）、过零率（语音有特征交叉模式）和频谱特征。这些在信噪比低的噪声环境中失败。

- **神经 VAD**模型将问题视为帧级二元分类。小 RNN 或 CNN 采用声学特征（来自文件 01 的对数 mel 能量），预测语音/非语音概率。

- **WebRTC VAD**（Google）是经典轻量级 VAD，使用简单频谱特征的基于 GMM 的分类器。它以四种激进级别（0-3）操作，极快，但在音乐、非语音发声和低 SNR 环境中挣扎。它仍然广泛用作基线，因其零依赖简单性。

- **Silero VAD**（Silero Team，2021）是生产中事实标准的神经 VAD。其架构是小型深度可分离 1D 卷积栈（第 08 章的 MobileNet 思想应用于音频），后跟单个 LSTM 层用于时间上下文，最终线性头每帧产生语音概率。整个模型小于 2MB（约 100 万参数），以 30-100 毫秒块处理音频。
    - **输入**：原始 16 kHz 音频（无手动特征提取——卷积前端直接从波形学习自己的特征）。
    - **窗口状态推理**：LSTM 隐藏状态在块之间延续，所以模型处理流式音频，无需重新处理完整历史。每次调用处理 30、60 或 100 毫秒块，返回 $[0, 1]$ 中的语音概率。
    - **自适应阈值**：不是单个固定阈值，Silero VAD 使用独立的开始和结束阈值，带最小语音/沉默持续时间，防止在噪声边界上快速切换。语音段必须在最小持续时间前超过开始阈值才确认，沉默必须在结束阈值下持续才关闭段。
    - **性能**：Silero VAD 在 CPU 上以 1-2% 实时因子运行（处理 1 秒音频需要约 10-20 毫秒），适用于边缘设备、手机和实时管道。它在噪声和音乐重的音频上显著优于 WebRTC VAD，同时仍然足够小，用于设备内部署。
    - Silero VAD 常用作 Whisper（文件 02）的前端，将长音频分割为话语级块，然后转录，用于说话者日记化管道（文件 04），在提取说话者嵌入之前识别语音区域。

- **声学活动检测**（AAD）将 VAD 推广到检测任何声学活动，不仅是语音。这对智能家居设备、安全系统和野生动物监测有用。AAD 模型检测事件如玻璃破碎、狗叫或警报，通常使用文件 04 中描述的音频分类框架。

- **TTS 评估指标**测量客观质量和主观自然度：
    - **平均意见分数**（MOS）：人类听众在 1-5 量表上评分自然度。金标准，但昂贵和慢。
    - **mel 倒谱失真**（MCD）：测量合成和参考 mel 倒谱之间的距离。越低越好，但并不总是与感知相关。
    - **PESQ / POLQA**：标准化感知评估指标，最初为电话设计。
    - **说话者相似度**：合成和参考音频的说话者嵌入之间的余弦相似度（与语音克隆相关）。
    - **可懂度**：通过 ASR 系统（文件 02）馈送合成音频，计算 WER。

- **任务 1：从 mel 频谱图的 Griffin-Lim 声码器**。实现 Griffin-Lim 迭代相位重构算法，将 mel 频谱图转换回波形。这演示声码器问题，为什么需要神经声码器。

```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 生成合成波形（谐波和模拟元音）
sr = 16000
duration = 1.0
t = jnp.linspace(0, duration, int(sr * duration))
f0 = 220.0  # 基频
waveform = (
    0.6 * jnp.sin(2 * jnp.pi * f0 * t) +
    0.3 * jnp.sin(2 * jnp.pi * 2 * f0 * t) +
    0.1 * jnp.sin(2 * jnp.pi * 3 * f0 * t)
)

# 计算 STFT
n_fft = 1024
hop_length = 256
window = jnp.hanning(n_fft)

def stft(signal, n_fft, hop_length, window):
    """计算短时傅里叶变换。"""
    n_frames = 1 + (len(signal) - n_fft) // hop_length
    frames = jnp.stack([
        signal[i * hop_length : i * hop_length + n_fft] * window
        for i in range(n_frames)
    ])
    return jnp.fft.rfft(frames, n=n_fft)

def istft(stft_matrix, hop_length, window, length):
    """计算逆 STFT 带重叠 - 相加。"""
    n_fft = (stft_matrix.shape[1] - 1) * 2
    n_frames = stft_matrix.shape[0]
    frames = jnp.fft.irfft(stft_matrix, n=n_fft)
    frames = frames * window[None, :]
    output = jnp.zeros(length)
    for i in range(n_frames):
        start = i * hop_length
        end = start + n_fft
        if end <= length:
            output = output.at[start:end].add(frames[i])
    return output

# 前向 STFT
S = stft(waveform, n_fft, hop_length, window)
magnitude = jnp.abs(S)

# Mel 滤波器组
n_mels = 80
mel_low = 0.0
mel_high = 2595 * jnp.log10(1 + (sr / 2) / 700)
mel_points = jnp.linspace(mel_low, mel_high, n_mels + 2)
hz_points = 700 * (10 ** (mel_points / 2595) - 1)
freq_bins = jnp.floor((n_fft + 1) * hz_points / sr).astype(int)

mel_filterbank = jnp.zeros((n_mels, n_fft // 2 + 1))
for m in range(n_mels):
    f_left = freq_bins[m]
    f_center = freq_bins[m + 1]
    f_right = freq_bins[m + 2]
    for k in range(f_left, f_center):
        mel_filterbank = mel_filterbank.at[m, k].set(
            (k - f_left) / max(f_center - f_left, 1)
        )
    for k in range(f_center, f_right):
        mel_filterbank = mel_filterbank.at[m, k].set(
            (f_right - k) / max(f_right - f_center, 1)
        )

# 到 mel 和回（伪逆）
mel_spec = magnitude @ mel_filterbank.T
magnitude_reconstructed = mel_spec @ jnp.linalg.pinv(mel_filterbank.T)
magnitude_reconstructed = jnp.maximum(magnitude_reconstructed, 1e-7)

# Griffin-Lim 算法
def griffin_lim(magnitude, n_iter, hop_length, window, signal_length):
    """迭代相位重构。"""
    n_fft = (magnitude.shape[1] - 1) * 2
    key = jax.random.PRNGKey(42)
    phase = jax.random.uniform(key, magnitude.shape, minval=-jnp.pi, maxval=jnp.pi)

    for _ in range(n_iter):
        complex_spec = magnitude * jnp.exp(1j * phase)
        signal = istft(complex_spec, hop_length, window, signal_length)
        reanalysis = stft(signal, n_fft, hop_length, window)
        phase = jnp.angle(reanalysis)

    complex_spec = magnitude * jnp.exp(1j * phase)
    return istft(complex_spec, hop_length, window, signal_length)

reconstructed = griffin_lim(magnitude_reconstructed, n_iter=60, hop_length=hop_length,
                            window=window, signal_length=len(waveform))

# 绘图比较
fig, axes = plt.subplots(3, 1, figsize=(12, 8))

axes[0].plot(t[:1000], waveform[:1000], color='#3498db', linewidth=0.8)
axes[0].set_title('原始波形')
axes[0].set_ylabel('幅度')

axes[1].imshow(jnp.log1p(mel_spec.T), aspect='auto', origin='lower', cmap='magma')
axes[1].set_title('Mel 频谱图（中间表示）')
axes[1].set_ylabel('Mel 箱')

axes[2].plot(t[:1000], reconstructed[:1000], color='#e74c3c', linewidth=0.8)
axes[2].set_title('Griffin-Lim 重构波形（60 次迭代）')
axes[2].set_xlabel('时间 (s)')
axes[2].set_ylabel('幅度')

plt.tight_layout()
plt.show()

# 测量重构误差
mse = jnp.mean((waveform[:len(reconstructed)] - reconstructed[:len(waveform)]) ** 2)
print(f"原始和重构之间的 MSE: {mse:.6f}")
print("注意：通过 mel 反演的相位信息丢失导致伪影。")
```

- **任务 2：持续时间预测器**（FastSpeech 风格）。训练小型卷积持续时间预测器，将音素嵌入映射到持续时间。这是实现非自回归 TTS 的核心组件。

```python
import jax
import jax.numpy as jnp
import jax.random as jr
import matplotlib.pyplot as plt

# 模拟带真实持续时间的音素序列
# 在真实 TTS 中，持续时间来自强制对齐或教师注意力
def generate_synthetic_data(key, n_samples=200, max_phonemes=30, embed_dim=64):
    """生成合声音素嵌入和持续时间。"""
    keys = jr.split(key, 4)
    lengths = jr.randint(keys[0], (n_samples,), 5, max_phonemes)

    all_embeddings = []
    all_durations = []
    all_masks = []

    for i in range(n_samples):
        L = int(lengths[i])
        emb = jr.normal(keys[1], (max_phonemes, embed_dim))
        # 持续时间：元音（偶数索引）更长，辅音更短
        base_dur = jnp.where(jnp.arange(max_phonemes) % 2 == 0, 8.0, 4.0)
        noise = jr.normal(jr.fold_in(keys[2], i), (max_phonemes,)) * 1.5
        dur = jnp.clip(base_dur + noise, 1.0, 20.0).astype(jnp.float32)
        mask = (jnp.arange(max_phonemes) < L).astype(jnp.float32)

        all_embeddings.append(emb)
        all_durations.append(dur * mask)
        all_masks.append(mask)

    return (jnp.stack(all_embeddings), jnp.stack(all_durations),
            jnp.stack(all_masks))

key = jr.PRNGKey(42)
embeddings, durations, masks = generate_synthetic_data(key)

# 持续时间预测器：2 层 1D 卷积 + 线性投影
def init_duration_predictor(key, embed_dim=64, hidden_dim=128, kernel_size=3):
    """初始化持续时间预测器权重。"""
    keys = jr.split(key, 4)
    scale1 = jnp.sqrt(2.0 / (embed_dim * kernel_size))
    scale2 = jnp.sqrt(2.0 / (hidden_dim * kernel_size))
    params = {
        'conv1_w': jr.normal(keys[0], (kernel_size, embed_dim, hidden_dim)) * scale1,
        'conv1_b': jnp.zeros(hidden_dim),
        'conv2_w': jr.normal(keys[1], (kernel_size, hidden_dim, hidden_dim)) * scale2,
        'conv2_b': jnp.zeros(hidden_dim),
        'linear_w': jr.normal(keys[2], (hidden_dim, 1)) * jnp.sqrt(2.0 / hidden_dim),
        'linear_b': jnp.zeros(1),
    }
    return params

def duration_predictor(params, x):
    """从音素嵌入预测对数持续时间。x: (batch, seq, embed)。"""
    # 卷积层 1 带 ReLU
    h = jax.lax.conv_general_dilated(
        x.transpose(0, 2, 1),  # (batch, embed, seq)
        params['conv1_w'].transpose(2, 1, 0),  # (out, in, kernel)
        window_strides=(1,), padding='SAME'
    ).transpose(0, 2, 1) + params['conv1_b']  # 回到 (batch, seq, hidden)
    h = jax.nn.relu(h)

    # 卷积层 2 带 ReLU
    h = jax.lax.conv_general_dilated(
        h.transpose(0, 2, 1),
        params['conv2_w'].transpose(2, 1, 0),
        window_strides=(1,), padding='SAME'
    ).transpose(0, 2, 1) + params['conv2_b']
    h = jax.nn.relu(h)

    # 线性投影到标量
    log_dur = (h @ params['linear_w'] + params['linear_b']).squeeze(-1)
    return log_dur

# 损失：对数持续时间的 MSE（FastSpeech 标准）
def loss_fn(params, embeddings, durations, masks):
    log_dur_pred = duration_predictor(params, embeddings)
    log_dur_true = jnp.log(jnp.clip(durations, 1.0, None))
    sq_err = (log_dur_pred - log_dur_true) ** 2 * masks
    return jnp.sum(sq_err) / jnp.sum(masks)

grad_fn = jax.jit(jax.value_and_grad(loss_fn))

# 训练循环
params = init_duration_predictor(jr.PRNGKey(0))
lr = 1e-3
losses = []

for epoch in range(300):
    loss_val, grads = grad_fn(params, embeddings, durations, masks)
    params = jax.tree.map(lambda p, g: p - lr * g, params, grads)
    losses.append(float(loss_val))

# 在样本上评估
log_dur_pred = duration_predictor(params, embeddings[:1])
dur_pred = jnp.exp(log_dur_pred[0])
dur_true = durations[0]
mask = masks[0]
valid_len = int(jnp.sum(mask))

fig, axes = plt.subplots(1, 2, figsize=(14, 5))

axes[0].plot(losses, color='#3498db', linewidth=1.5)
axes[0].set_xlabel('Epoch')
axes[0].set_ylabel('MSE 损失（对数持续时间）')
axes[0].set_title('持续时间预测器训练')
axes[0].set_yscale('log')

x_pos = jnp.arange(valid_len)
width = 0.35
axes[1].bar(x_pos - width/2, dur_true[:valid_len], width, color='#27ae60',
            label='真实', alpha=0.8)
axes[1].bar(x_pos + width/2, dur_pred[:valid_len], width, color='#e74c3c',
            label='预测', alpha=0.8)
axes[1].set_xlabel('音素索引')
axes[1].set_ylabel('持续时间（帧）')
axes[1].set_title('持续时间预测 vs 真实')
axes[1].legend()

plt.tight_layout()
plt.show()
```

- **任务 3：带上采样卷积的简单神经声码器**。构建最小 HiFi-GAN 风格生成器，使用转置卷积和残差块将 mel 频谱图上采样到波形。

```python
import jax
import jax.numpy as jnp
import jax.random as jr
import matplotlib.pyplot as plt

def init_residual_block(key, channels, kernel_size, dilation):
    """初始化膨胀残差卷积块。"""
    k1, k2 = jr.split(key)
    scale = jnp.sqrt(2.0 / (channels * kernel_size))
    return {
        'conv1_w': jr.normal(k1, (kernel_size, channels, channels)) * scale,
        'conv1_b': jnp.zeros(channels),
        'conv2_w': jr.normal(k2, (kernel_size, channels, channels)) * scale,
        'conv2_b': jnp.zeros(channels),
        'dilation': dilation
    }

def residual_block(params, x):
    """x: (batch, time, channels)。膨胀残差块带 LeakyReLU。"""
    h = jax.nn.leaky_relu(x, negative_slope=0.1)
    # 简化：使用标准卷积（膨胀概念上处理）
    h = jax.lax.conv_general_dilated(
        h.transpose(0, 2, 1),
        params['conv1_w'].transpose(2, 1, 0),
        window_strides=(1,),
        padding='SAME',
        rhs_dilation=(params['dilation'],)
    ).transpose(0, 2, 1) + params['conv1_b']
    h = jax.nn.leaky_relu(h, negative_slope=0.1)
    h = jax.lax.conv_general_dilated(
        h.transpose(0, 2, 1),
        params['conv2_w'].transpose(2, 1, 0),
        window_strides=(1,),
        padding='SAME'
    ).transpose(0, 2, 1) + params['conv2_b']
    return x + h

def init_generator(key, n_mels=80, upsample_rates=(8, 8, 4),
                   channels=128):
    """初始化最小 HiFi-GAN 风格生成器。"""
    keys = jr.split(key, 10)
    params = {}

    # 输入投影：mel 箱 -> 通道
    params['input_w'] = jr.normal(keys[0], (7, n_mels, channels)) * 0.02
    params['input_b'] = jnp.zeros(channels)

    # 上采样块（转置卷积）
    in_ch = channels
    for i, rate in enumerate(upsample_rates):
        k_size = rate * 2
        scale = jnp.sqrt(2.0 / (in_ch * k_size))
        out_ch = in_ch // 2
        params[f'up{i}_w'] = jr.normal(keys[i+1], (k_size, in_ch, out_ch)) * scale
        params[f'up{i}_b'] = jnp.zeros(out_ch)
        # 每个尺度的残差块
        params[f'res{i}_0'] = init_residual_block(jr.fold_in(keys[i+4], 0),
                                                    out_ch, 3, 1)
        params[f'res{i}_1'] = init_residual_block(jr.fold_in(keys[i+4], 1),
                                                    out_ch, 3, 3)
        in_ch = out_ch

    # 输出投影到单声道波形
    params['output_w'] = jr.normal(keys[8], (7, in_ch, 1)) * 0.02
    params['output_b'] = jnp.zeros(1)
    params['upsample_rates'] = upsample_rates

    return params

def generator_forward(params, mel):
    """mel: (batch, time, n_mels) -> waveform: (batch, time * prod(rates), 1)。"""
    # 输入投影
    h = jax.lax.conv_general_dilated(
        mel.transpose(0, 2, 1),
        params['input_w'].transpose(2, 1, 0),
        window_strides=(1,), padding='SAME'
    ).transpose(0, 2, 1) + params['input_b']

    for i, rate in enumerate(params['upsample_rates']):
        h = jax.nn.leaky_relu(h, negative_slope=0.1)
        # 通过转置卷积上采样
        k_size = rate * 2
        h = jax.lax.conv_transpose(
            h.transpose(0, 2, 1),
            params[f'up{i}_w'].transpose(2, 1, 0),
            strides=(rate,),
            padding='SAME'
        ).transpose(0, 2, 1) + params[f'up{i}_b']
        # 残差块
        h = residual_block(params[f'res{i}_0'], h)
        h = residual_block(params[f'res{i}_1'], h)

    h = jax.nn.leaky_relu(h, negative_slope=0.1)
    out = jax.lax.conv_general_dilated(
        h.transpose(0, 2, 1),
        params['output_w'].transpose(2, 1, 0),
        window_strides=(1,), padding='SAME'
    ).transpose(0, 2, 1) + params['output_b']

    return jnp.tanh(out)

# 创建合成 mel 频谱图（模拟元音）
```

- **任务 4：带简单 RNN 的语音活动检测**。训练小型基于 GRU 的 VAD 模型在合成音频特征上，将帧分类为语音或沉默。

```python
import jax
import jax.numpy as jnp
import jax.random as jr
import matplotlib.pyplot as plt

# 生成带语音/沉默标签的合成对数 mel 能量特征
def generate_vad_data(key, n_sequences=100, n_frames=200, n_features=40):
    """模拟对数 mel 特征：语音区域更高能量带结构。"""
    keys = jr.split(key, 5)
    all_features = []
    all_labels = []

    for i in range(n_sequences):
        k = jr.fold_in(keys[0], i)
        k1, k2, k3 = jr.split(k, 3)

        # 随机语音/沉默模式
        label = jnp.zeros(n_frames)
        n_segments = jr.randint(k1, (), 2, 6)
        for seg in range(int(n_segments)):
            start = jr.randint(jr.fold_in(k2, seg), (), 0, n_frames - 20)
            length = jr.randint(jr.fold_in(k3, seg), (), 10, 50)
            end = jnp.minimum(start + length, n_frames)
            label = label.at[int(start):int(end)].set(1.0)

        # 特征：语音帧更高能量 + 频谱结构
        noise = jr.normal(jr.fold_in(keys[1], i), (n_frames, n_features)) * 0.3
        speech_pattern = jnp.outer(label, jnp.exp(-jnp.arange(n_features) / 15.0))
        features = speech_pattern * 2.0 + noise + 0.1

        all_features.append(features)
        all_labels.append(label)

    return jnp.stack(all_features), jnp.stack(all_labels)

key = jr.PRNGKey(123)
features, labels = generate_vad_data(key)
train_features, train_labels = features[:80], labels[:80]
test_features, test_labels = features[80:], labels[80:]

# 简单基于 GRU 的 VAD 模型
def init_vad_model(key, input_dim=40, hidden_dim=64):
    keys = jr.split(key, 6)
    scale_ih = jnp.sqrt(2.0 / input_dim)
    scale_hh = jnp.sqrt(2.0 / hidden_dim)
    return {
        'W_z': jr.normal(keys[0], (input_dim, hidden_dim)) * scale_ih,
        'U_z': jr.normal(keys[1], (hidden_dim, hidden_dim)) * scale_hh,
        'b_z': jnp.zeros(hidden_dim),
        'W_r': jr.normal(keys[2], (input_dim, hidden_dim)) * scale_ih,
        'U_r': jr.normal(keys[3], (hidden_dim, hidden_dim)) * scale_hh,
        'b_r': jnp.zeros(hidden_dim),
        'W_h': jr.normal(keys[4], (input_dim, hidden_dim)) * scale_ih,
        'U_h': jr.normal(keys[5], (hidden_dim, hidden_dim)) * scale_hh,
        'b_h': jnp.zeros(hidden_dim),
        'W_out': jr.normal(jr.fold_in(keys[0], 99), (hidden_dim, 1)) * 0.1,
        'b_out': jnp.zeros(1),
    }

def gru_step(params, h, x):
    """单个 GRU 步骤。"""
    z = jax.nn.sigmoid(x @ params['W_z'] + h @ params['U_z'] + params['b_z'])
    r = jax.nn.sigmoid(x @ params['W_r'] + h @ params['U_r'] + params['b_r'])
    h_tilde = jnp.tanh(x @ params['W_h'] + (r * h) @ params['U_h'] + params['b_h'])
    h_new = (1 - z) * h + z * h_tilde
    return h_new

def vad_forward(params, x):
    """x: (batch, time, features) -> logits: (batch, time)。"""
    batch_size, n_frames, _ = x.shape
    hidden_dim = params['W_z'].shape[1]
    h = jnp.zeros((batch_size, hidden_dim))

    outputs = []
    for t in range(n_frames):
        h = gru_step(params, h, x[:, t, :])
        logit = (h @ params['W_out'] + params['b_out']).squeeze(-1)
        outputs.append(logit)

    return jnp.stack(outputs, axis=1)

def bce_loss(params, features, labels):
    """VAD 的二元交叉熵损失。"""
    logits = vad_forward(params, features)
    probs = jax.nn.sigmoid(logits)
    probs = jnp.clip(probs, 1e-7, 1 - 1e-7)
    loss = -(labels * jnp.log(probs) + (1 - labels) * jnp.log(1 - probs))
    return jnp.mean(loss)

grad_fn = jax.jit(jax.value_and_grad(bce_loss))

# 训练
params = init_vad_model(jr.PRNGKey(0))
lr = 5e-3
losses = []

for epoch in range(200):
    loss_val, grads = grad_fn(params, train_features, train_labels)
    params = jax.tree.map(lambda p, g: p - lr * g, params, grads)
    losses.append(float(loss_val))
    if epoch % 50 == 0:
        print(f"Epoch {epoch}: loss = {loss_val:.4f}")

# 在测试集上评估
test_logits = vad_forward(params, test_features)
test_preds = (jax.nn.sigmoid(test_logits) > 0.5).astype(jnp.float32)
accuracy = jnp.mean(test_preds == test_labels)
print(f"\n测试准确率：{accuracy:.4f}")

# 可视化测试示例
idx = 0
fig, axes = plt.subplots(3, 1, figsize=(14, 7))

axes[0].imshow(test_features[idx].T, aspect='auto', origin='lower', cmap='magma')
axes[0].set_title('对数 Mel 能量特征')
axes[0].set_ylabel('Mel 箱')

axes[1].fill_between(range(200), test_labels[idx], alpha=0.4, color='#27ae60',
                     label='真实')
axes[1].plot(jax.nn.sigmoid(test_logits[idx]), color='#e74c3c',
             linewidth=1.5, label='预测概率')
axes[1].axhline(0.5, color='gray', linestyle='--', linewidth=0.8)
axes[1].set_ylabel('语音概率')
axes[1].legend()
axes[1].set_title('VAD 预测')

axes[2].fill_between(range(200), test_labels[idx], alpha=0.4, color='#27ae60',
                     label='真实')
axes[2].fill_between(range(200), test_preds[idx], alpha=0.4, color='#f39c12',
                     label='预测 (阈值=0.5)')
axes[2].set_ylabel('语音 / 沉默')
axes[2].set_xlabel('帧')
axes[2].legend()
axes[2].set_title('VAD 二元决策')

plt.tight_layout()
plt.show()
```
