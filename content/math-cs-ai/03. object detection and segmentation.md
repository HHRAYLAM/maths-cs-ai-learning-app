# 目标检测和分割

*目标检测定位和分类图像中的每个对象；分割为每个像素分配标签。本文件涵盖 IoU、mAP、锚框、R-CNN 家族、YOLO、SSD、特征金字塔网络、语义/实例/全景分割（U-Net、Mask R-CNN、SAM）和基准测试的指标*

- 图像分类（文件 02）回答"这张图像中有什么？"目标检测提出更难的问题："这张图像中有什么对象，它们在哪里？"

- 分割更进一步："哪些像素属于哪个对象或类别？"这些任务形成越来越精确空间理解的层次结构。

- 目标检测模型输出一组边界框，每个由四个坐标（左上角 $x, y$、宽度、高度）和带置信度分数的类别标签定义。单个图像可能包含来自多个类别的零个、一个或数百个对象。

![输入图像，多个对象被彩色边界框包围，带类别标签和置信度分数](../images/detection_boxes.svg)

- 交并比（IoU）测量预测边界框与真实框的匹配程度。它是重叠面积除以并集面积：

$$\text{IoU} = \frac{\text{重叠面积}}{\text{并集面积}}$$

- IoU 为 1 表示完美重叠；IoU 为 0 表示完全没有重叠。"正确"检测的标准阈值是 IoU $\geq 0.5$，尽管也使用更严格的阈值（0.75、0.9）。

- 检测是真正例（TP），如果其 IoU 与真实框超过阈值且类别正确。

- 假正例（FP）是预测框，不与任何真实框匹配。

- 假负例（FN）是没有预测匹配的真实对象。这些与第 06 章中的精确率/召回率概念相同。

- 平均精度（AP）总结单个类别的检测质量。对于每个类别，按置信度分数对所有检测排名，计算每个排名的精确率和召回率，计算精确率 - 召回率曲线下的面积：

$$\text{AP} = \int_0^1 p(r) \, dr$$

- 在实践中，曲线插值：在每个召回级别，精确率设置为任何召回$\geq r$的最大精确率。这平滑曲线，使其单调递减。

- 平均精度均值（mAP）跨所有类别平均 AP。"mAP@0.5"使用 IoU 阈值 0.5。"mAP@[.5:.95]"（COCO 标准）在从 0.5 到 0.95 的十个 IoU 阈值上平均 mAP，步长 0.05，奖励检测和精确定位。

- 非极大值抑制（NMS）移除重复检测。当模型为同一个对象预测多个重叠框时，NMS 保留最高置信度框，移除所有其他与它重叠超过 IoU 阈值的框。这在模型产生原始预测后每类别应用。

- 两阶段检测器首先提出候选区域，然后分类和细化每个提议。

- R-CNN（Girshick 等人，2014）是第一个成功的深度学习检测器。它使用选择性搜索（经典算法）提出约 2,000 个候选区域，将每个区域扭曲为固定大小，通过 CNN 独立运行每个，用 SVM 分类（第 06 章）。R-CNN 准确但极慢：每个图像运行 CNN 2,000 次。

- Fast R-CNN（Girshick，2015）通过在整个图像上运行一次 CNN 产生共享特征图，然后使用RoI 池化（感兴趣区域池化）从该共享图为每个提议提取特征，解决冗余。

- RoI 池化采用可变大小的特征图区域，产生固定大小输出，通过将区域划分为网格并在每个单元格内最大池化。这快得多，因为昂贵的 CNN 计算只发生一次。

- Faster R-CNN（Ren 等人，2015）通过引入区域提议网络（RPN）消除外部区域提议算法，RPN 是在共享特征图上运行的小型 CNN，直接预测提议。RPN 在特征图上滑动小窗口，在每个位置预测 $k$ 个提议（每个锚框一个）。

![Faster R-CNN 管道：输入图像 → 骨干 CNN → 共享特征图 → RPN 生成提议 → RoI 池化 → 分类和框回归头](../images/faster_rcnn.svg)

- 锚框是特征图每个空间位置的预定义边界框，覆盖不同尺度和长宽比（例如，三个尺度×三个比率=每个位置 9 个锚）。RPN 为每个锚预测两件事：对象性分数（对象 vs 背景）和坐标偏移，将锚细化为更紧密的提议。这个参数化使回归问题更容易：网络不是预测绝对坐标，而是预测合理起始框的小调整。

- 锚偏移参数化为：

$$t_x = \frac{x - x_a}{w_a}, \quad t_y = \frac{y - y_a}{h_a}, \quad t_w = \log\frac{w}{w_a}, \quad t_h = \log\frac{h}{h_a}$$

- 其中 $(x, y, w, h)$ 是预测框中心和大小，$(x_a, y_a, w_a, h_a)$ 是锚。宽度和高度的对数变换确保预测框总是正的，使回归尺度不变。

- Faster R-CNN 用多任务损失训练：分类损失（来自第 05 章的交叉熵）用于类别标签，加上平滑 L1 损失用于框回归。平滑 L1 对异常值比 L2 不敏感：

```math
\text{smooth}_{L1}(x) = \begin{cases} 0.5x^2 & \text{if } |x| < 1 \\ |x| - 0.5 & \text{otherwise} \end{cases}
```

- 特征金字塔网络（FPN）（Lin 等人，2017）通过构建自上而下通路带横向连接，合并高级语义和低级空间细节，解决多尺度问题。骨干在多个尺度产生特征图（每个池化层减半分辨率）。FPN 添加自上而下通路，每个级别从上面级别接收上采样特征，通过横向 1x1 卷积与相应的自下而上级别合并。结果是特征图金字塔，每个都有强语义和好的空间分辨率。

- 小对象从金字塔的较高分辨率级别检测；大对象从较低分辨率级别检测。FPN 现在是大多数现代检测架构的标准组件。

- 单阶段检测器完全跳过提议步骤，在单次通过中预测类别标签和边界框。这更快，但在历史上不如两阶段检测器准确，直到焦点损失缩小差距。

- YOLO（You Only Look Once，Redmon 等人，2016）将图像划分为 $S \times S$ 网格。每个网格单元预测 $B$ 个边界框和 $C$ 个类别概率。如果对象的中心落在网格单元中，该单元负责检测它。YOLO 非常快，因为整个检测是单个前向传播，没有提议阶段。

- YOLOv2 添加锚框、批量归一化和多尺度训练。YOLOv3 使用特征金字塔网络，在三个尺度预测。YOLOv4-v8 继续改进，用更好的骨干、路径聚合网络和马赛克数据增强（在训练期间将四个图像拼接在一起，增加上下文多样性）。

- SSD（Single Shot MultiBox Detector，Liu 等人，2016）在骨干内的多个特征图尺度预测，在每个尺度使用锚框。早期（高分辨率）特征图检测小对象；后期（低分辨率）图检测大对象。SSD 比 Faster R-CNN 快，有竞争力的准确性。

- RetinaNet（Lin 等人，2017）识别单阶段检测器的核心问题：类别不平衡。绝大多数锚框对应背景，产生容易的负例，主导损失并压倒来自稀有正例的梯度。

- 焦点损失通过降低容易样本的权重解决这个问题：

$$\text{FL}(p_t) = -\alpha_t (1 - p_t)^\gamma \log(p_t)$$

- 其中 $p_t$ 是正确类别的预测概率。当模型自信且正确时（$p_t$ 高），$(1 - p_t)^\gamma$ 小，减少来自容易负例的损失贡献。超参数 $\gamma$（通常为 2）控制降低权重的强度。$\gamma = 0$ 时，焦点损失简化为标准交叉熵。使用焦点损失，RetinaNet 实现与两阶段检测器相当的准确性，以单阶段速度。

- 无锚检测完全消除锚框，减少超参数调优并简化管道。

- FCOS（Fully Convolutional One-Stage，Tian 等人，2019）在特征图的每个空间位置预测从该位置到最近边界框四边的距离（左、上、右、下）加上类别标签。中心度分数降低远离对象中心的预测权重，提高质量。FCOS 使用 FPN 处理多尺度。

- CenterNet（Zhou 等人，2019）将对象检测为点：它预测热图，峰值对应对象中心，然后在每个峰值回归宽度和高度。检测变成关键点估计。这优雅且无锚，但需要仔细的热图后处理。

- CornerNet 将对象检测为角点对（左上和右下）。它预测两个热图（每个角点类型一个），使用关联嵌入将对应角点匹配为边界框。这避免了对锚的需求，处理任意形状的对象。

- 语义分割为图像中的每个像素分配类别标签。与检测（输出框）不同，分割产生密集像素级图。街景可能将每个像素标记为道路、人行道、汽车、行人、建筑物、天空等。

![语义分割：输入街景和其像素级标签图，每个颜色代表一个类别](../images/semantic_segmentation.svg)

- 全卷积网络（FCN）（Long 等人，2015）通过将全连接层替换为卷积层，使分类 CNN 适应分割，允许网络输出空间图而不是单个类别。上采样（通过转置卷积或双线性插值）将输出恢复到输入分辨率。来自早期层的跳过连接添加回在下采样期间丢失的空间细节。

- 转置卷积（有时称为"反卷积"）是卷积的上采样对应物。在步幅卷积减少空间维度的地方，转置卷积增加它们。它在输入元素之间插入零，然后应用标准卷积，有效地学习如何上采样。

- U-Net（Ronneberger 等人，2015）引入对称编码器 - 解码器架构，每层都有跳过连接。编码器（收缩路径）减少空间分辨率，同时增加通道，完全像分类 CNN。解码器（扩展路径）上采样回完整分辨率。跳过连接在每层连接编码器特征图与解码器特征图，为解码器提供精细空间细节。高级语义和低级细节的这个组合产生清晰、准确的分割边界。

![U-Net 架构：左侧编码器路径带下采样，右侧解码器路径带上采样，跳过连接桥接对应级别](../images/unet_architecture.svg)

- U-Net 最初设计用于生物医学图像分割（训练数据稀缺），其架构已成为许多后续模型的基础，包括潜在扩散模型中的 U-Net（文件 04）。

- DeepLab（Chen 等人，2014-2018）为分割引入两个关键创新：

    - 空洞（膨胀）：标准卷积，在滤波器元素之间插入间隙，由膨胀率 $r$ 控制。带膨胀 $r$ 的 3x3 滤波器有$(2r + 1) \times (2r + 1)$ 的感受野，而只使用 9 个参数。这在不下采样的情况下捕捉多尺度上下文，保留空间分辨率。

    - 空洞空间金字塔池化（ASPP）：并行应用多个带不同膨胀率的空洞卷积（例如，率 1、6、12、18），连接结果，用 1x1 卷积融合。ASPP 同时捕捉多尺度上下文，类似于 Inception 模块（文件 02）的精神，但使用膨胀而不是不同的内核大小。

- DeepLab 还使用条件随机场（CRF）（第 05 章）作为后处理步骤，通过鼓励空间上相近、颜色相似的像素共享相同标签来细化分割边界。

- 实例分割结合检测和分割：它识别每个单独的对象实例，为每个产生像素级掩码。场景中的两辆车获得两个独立的掩码，而不仅仅是"车"。

- Mask R-CNN（He 等人，2017）扩展 Faster R-CNN，添加小型分割头，为每个检测对象预测二值掩码。架构是 Faster R-CNN + 掩码分支：掩码分支采用 RoI 池化特征，为每个类别输出 $m \times m$ 二值掩码。它使用RoIAlign而不是 RoI 池化：在精确采样点双线性插值，而不是量化网格单元，避免量化引起的空间不对齐。这个小改变显著提高掩码质量。

- Mask R-CNN 用多任务损失训练：分类损失 + 框回归损失 + 掩码损失（逐像素二值交叉熵）。掩码分支为每个类别独立预测掩码；只有对应预测类别的掩码被使用，这解耦掩码预测与分类，改善两者。

- 全景分割将语义和实例分割统一为单个任务。每个像素获得类别标签（语义）和实例 ID（实例，用于"东西"类别如汽车和人）。" stuff"类别（天空、道路、草地）只获得语义标签，因为它们是无形态区域，没有可计数的实例。

- 全景质量（PQ）指标评估这个，通过分解为分割质量（匹配段的平均 IoU）和识别质量（匹配段的 F1 分数）：

$$\text{PQ} = \underbrace{\frac{\sum_{(p,g) \in \text{TP}} \text{IoU}(p,g)}{|\text{TP}|}}_{\text{SQ}} \times \underbrace{\frac{|\text{TP}|}{|\text{TP}| + \frac{1}{2}|\text{FP}| + \frac{1}{2}|\text{FN}|}}_{\text{RQ}}$$

- 实时分割对自动驾驶和增强现实等应用至关重要，延迟预算紧张（通常每帧低于 30 毫秒）。

- BiSeNet（Bilateral Segmentation Network，Yu 等人，2018）使用两个并行路径：空间路径带宽、浅层，保留空间细节，上下文路径带深、窄层，捕捉语义。输出融合，给出速度及准确性。

- DDRNet（Deep Dual-Resolution Network，Hong 等人，2021）在整个网络中维护两个不同分辨率的分支，在它们之间重复信息交换。高分辨率分支保留空间细节，而低分辨率分支捕捉全局上下文。多个双侧融合模块双向合并信息。

- 实时分割的一般趋势是避免沉重的编码器 - 解码器模式，而是在整个网络中保持足够的空间分辨率，以一些准确性换取显著更低的延迟。

1. 从头实现 IoU 计算和非极大值抑制。将 NMS 应用于一组重叠边界框，可视化结果。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt
import matplotlib.patches as patches

def compute_iou(box1, box2):
    """计算两个框 [x1, y1, x2, y2] 之间的 IoU。"""
    x1 = jnp.maximum(box1[0], box2[0])
    y1 = jnp.maximum(box1[1], box2[1])
    x2 = jnp.minimum(box1[2], box2[2])
    y2 = jnp.minimum(box1[3], box2[3])

    intersection = jnp.maximum(0, x2 - x1) * jnp.maximum(0, y2 - y1)
    area1 = (box1[2] - box1[0]) * (box1[3] - box1[1])
    area2 = (box2[2] - box2[0]) * (box2[3] - box2[1])
    union = area1 + area2 - intersection

    return intersection / (union + 1e-6)

def nms(boxes, scores, iou_threshold=0.5):
    """非极大值抑制。"""
    order = jnp.argsort(-scores)  # 按置信度降序排序
    keep = []

    remaining = list(range(len(scores)))
    order_list = order.tolist()

    while order_list:
        idx = order_list[0]
        keep.append(idx)
        order_list = order_list[1:]

        new_order = []
        for j in order_list:
            iou = compute_iou(boxes[idx], boxes[j])
            if iou < iou_threshold:
                new_order.append(j)
        order_list = new_order

    return keep

# 示例：同一对象的重叠检测
boxes = jnp.array([
    [50, 60, 150, 160],   # 高置信度
    [55, 65, 155, 165],   # 重叠重复
    [52, 58, 148, 158],   # 重叠重复
    [200, 100, 300, 200], # 不同对象
    [205, 105, 305, 205], # 重叠重复
])
scores = jnp.array([0.95, 0.80, 0.70, 0.90, 0.60])

keep = nms(boxes, scores, iou_threshold=0.5)

fig, axes = plt.subplots(1, 2, figsize=(14, 5))
colors = ['#3498db', '#e74c3c', '#27ae60', '#9b59b6', '#f39c12']

for ax, title, indices in zip(axes, ['NMS 之前', 'NMS 之后'],
                               [range(len(boxes)), keep]):
    ax.set_xlim(0, 400); ax.set_ylim(0, 300)
    ax.set_aspect('equal'); ax.invert_yaxis()
    ax.set_title(title)
    for i in indices:
        b = boxes[i]
        rect = patches.Rectangle((b[0], b[1]), b[2]-b[0], b[3]-b[1],
                                  linewidth=2, edgecolor=colors[i],
                                  facecolor='none')
        ax.add_patch(rect)
        ax.text(b[0], b[1]-5, f'{scores[i]:.2f}', color=colors[i], fontsize=10)

plt.tight_layout(); plt.show()
print(f"NMS 后保留 {len(keep)}/{len(boxes)} 个框")
```

2. 实现简化的区域提议网络（RPN）。给定特征图，在多个尺度和长宽比生成锚框，预测对象性分数和框偏移。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt
import matplotlib.patches as patches

def generate_anchors(feature_h, feature_w, stride, scales, ratios):
    """为特征图每个位置生成锚框。"""
    anchors = []
    for y in range(feature_h):
        for x in range(feature_w):
            cx = (x + 0.5) * stride
            cy = (y + 0.5) * stride
            for s in scales:
                for r in ratios:
                    w = s * jnp.sqrt(r)
                    h = s / jnp.sqrt(r)
                    anchors.append([cx - w/2, cy - h/2, cx + w/2, cy + h/2])
    return jnp.array(anchors)

def rpn_forward(feature_map, params):
    """简化 RPN：为每个锚预测对象性和框偏移。"""
    H, W, C = feature_map.shape
    n_anchors = params['cls_w'].shape[1]

    # 在特征图上滑动 1x1 卷积（简化）
    cls_scores = feature_map.reshape(-1, C) @ params['cls_w']  # (H*W, n_anchors)
    box_offsets = feature_map.reshape(-1, C) @ params['reg_w']  # (H*W, n_anchors*4)

    cls_scores = jax.nn.sigmoid(cls_scores)
    return cls_scores.ravel(), box_offsets.reshape(-1, 4)

# 设置
feature_h, feature_w, channels = 4, 4, 16
stride = 16  # 每个特征图单元覆盖 16x16 像素
scales = [32, 64, 128]
ratios = [0.5, 1.0, 2.0]
n_anchors_per_pos = len(scales) * len(ratios)

key = jax.random.PRNGKey(42)
k1, k2, k3 = jax.random.split(key, 3)

feature_map = jax.random.normal(k1, (feature_h, feature_w, channels))
params = {
    'cls_w': jax.random.normal(k2, (channels, n_anchors_per_pos)) * 0.01,
    'reg_w': jax.random.normal(k3, (channels, n_anchors_per_pos * 4)) * 0.01,
}

anchors = generate_anchors(feature_h, feature_w, stride, scales, ratios)
scores, offsets = rpn_forward(feature_map, params)

print(f"特征图：{feature_h}x{feature_w}, stride={stride}")
print(f"每个位置锚数：{n_anchors_per_pos}")
print(f"总锚数：{len(anchors)}")
print(f"对象性分数形状：{scores.shape}")
print(f"框偏移形状：{offsets.shape}")

# 可视化一个位置的锚
fig, ax = plt.subplots(figsize=(6, 6))
img_size = feature_h * stride
ax.set_xlim(0, img_size); ax.set_ylim(0, img_size)
ax.invert_yaxis(); ax.set_aspect('equal')

pos_idx = feature_h // 2 * feature_w + feature_w // 2  # 中心位置
colors = ['#3498db', '#e74c3c', '#27ae60']
for i, s in enumerate(scales):
    for j, r in enumerate(ratios):
        idx = pos_idx * n_anchors_per_pos + i * len(ratios) + j
        a = anchors[idx]
        rect = patches.Rectangle((a[0], a[1]), a[2]-a[0], a[3]-a[1],
                                  linewidth=1.5, edgecolor=colors[i],
                                  facecolor='none', linestyle=['--', '-', ':'][j])
        ax.add_patch(rect)

ax.scatter([img_size/2], [img_size/2], c='red', s=50, zorder=5)
ax.set_title(f'中心位置的锚\n3 尺度 × 3 比率 = {n_anchors_per_pos}')
```

3. 实现带跳过连接的简化 U-Net 编码器 - 解码器，用于 1D 分割（1D 信号的二值标记）。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def conv1d_same(x, kernel):
    """带相同填充的 1D 卷积。"""
    k = len(kernel)
    pad = k // 2
    x_pad = jnp.pad(x, pad, mode='edge')
    n = len(x)
    out = jnp.zeros(n)
    for i in range(n):
        out = out.at[i].set(jnp.sum(x_pad[i:i+k] * kernel))
    return out

def downsample(x):
    return x[::2]

def upsample(x, target_len):
    return jnp.interp(jnp.linspace(0, 1, target_len), jnp.linspace(0, 1, len(x)), x)

def unet_1d(x, params):
    """带 2 个编码器/解码器级别的简化 1D U-Net。"""
    # 编码器
    e1 = jnp.maximum(0, conv1d_same(x, params['enc1']))
    e1_down = downsample(e1)

    e2 = jnp.maximum(0, conv1d_same(e1_down, params['enc2']))
    e2_down = downsample(e2)

    # 瓶颈
    bottleneck = jnp.maximum(0, conv1d_same(e2_down, params['bottleneck']))

    # 带跳过连接的解码器
    d2_up = upsample(bottleneck, len(e2))
    d2 = jnp.maximum(0, conv1d_same(d2_up + e2, params['dec2']))  # 跳过连接

    d1_up = upsample(d2, len(e1))
    d1 = conv1d_same(d1_up + e1, params['dec1'])  # 跳过连接

    return jax.nn.sigmoid(d1)

# 创建带标签区域的信号
n = 128
t = jnp.linspace(0, 4 * jnp.pi, n)
signal = jnp.sin(t) + 0.5 * jnp.sin(3 * t)
labels = (signal > 0.5).astype(jnp.float32)  # 二值分割目标

key = jax.random.PRNGKey(42)
keys = jax.random.split(key, 5)
params = {
    'enc1': jax.random.normal(keys[0], (5,)) * 0.3,
    'enc2': jax.random.normal(keys[1], (5,)) * 0.3,
    'bottleneck': jax.random.normal(keys[2], (3,)) * 0.3,
    'dec2': jax.random.normal(keys[3], (5,)) * 0.3,
    'dec1': jax.random.normal(keys[4], (5,)) * 0.3,
}

def loss_fn(params, signal, labels):
    pred = unet_1d(signal, params)
    return -jnp.mean(labels * jnp.log(pred + 1e-7) + (1 - labels) * jnp.log(1 - pred + 1e-7))

grad_fn = jax.jit(jax.grad(loss_fn))
lr = 0.05

for step in range(500):
    grads = grad_fn(params, signal, labels)
    params = {k: params[k] - lr * grads[k] for k in params}

pred = unet_1d(signal, params)

fig, axes = plt.subplots(3, 1, figsize=(12, 7), sharex=True)
axes[0].plot(t, signal, color='#3498db', linewidth=1.5)
axes[0].set_title('输入信号'); axes[0].set_ylabel('值')

axes[1].fill_between(t, 0, labels, alpha=0.3, color='#27ae60')
axes[1].set_title('真实标签'); axes[1].set_ylabel('标签')

axes[2].plot(t, pred, color='#e74c3c', linewidth=1.5)
axes[2].fill_between(t, 0, (pred > 0.5).astype(float), alpha=0.2, color='#e74c3c')
axes[2].set_title('U-Net 预测'); axes[2].set_ylabel('概率')
axes[2].set_xlabel('t')

plt.tight_layout(); plt.show()
print(f"最终损失：{loss_fn(params, signal, labels):.4f}")
print(f"像素准确率：{jnp.mean((pred > 0.5) == labels):.2%}")
```
