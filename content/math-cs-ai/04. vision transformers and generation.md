# 视觉 Transformer 和生成

*视觉 Transformer 将自注意力应用于图像片段，用数据驱动的空间学习挑战 CNN 的主导地位。本章涵盖 ViT、DeiT、Swin Transformer、使用 GAN（StyleGAN）的图像生成、VAE 和扩散模型（DDPM、Stable Diffusion），以及超分辨率和神经风格迁移。*

- CNN（文件 02）构建了强大的空间归纳偏置：局部连接、权重共享和平移等变。视觉 Transformer（ViT）提出了一个发人深省的问题：如果我们完全放弃这些偏置，让模型仅使用第 06 章的注意力机制从数据中学习空间结构，会发生什么？

- 视觉 Transformer（ViT）（Dosovitskiy 等人，2021）将标准 Transformer 编码器直接应用于图像。关键思想是将图像视为片段序列，就像 NLP 将文本视为 token 序列一样。

- 过程如下：
    1. 将图像（高度 $H$、宽度 $W$、通道 $C$）分割成 $P \times P$ 大小的非重叠片段网格。这产生 $N = HW / P^2$ 个片段。
    2. 将每个片段展平为长度 $P^2 \cdot C$ 的向量，并通过学习的线性嵌入（单个矩阵乘法，第 02 章）将其投影到模型维度 $D$。
    3. 前置可学习的 [CLS] token 嵌入（类似于 BERT 的 [CLS]，第 07 章）。这个 token 关注所有片段，其最终表示用于分类。
    4. 添加位置嵌入（每个位置一个可学习向量）以提供空间信息，因为注意力是排列等变的。
    5. 将 $(N + 1)$ 个 token 嵌入序列通过标准 Transformer 编码器（多头自注意力 + FFN，第 06 章）。
    6. [CLS] token 的最终表示通过分类头（小型 MLP）。

![ViT 流水线：图像分割成 16x16 片段，每个展平并线性投影，[CLS] token 前置，添加位置嵌入，然后通过 Transformer 编码器块处理](../images/vit_pipeline.svg)

- 片段嵌入 等价于内核大小 $P$ 和步幅 $P$ 的卷积（非重叠）。ViT 字面上将 2D 图像转换为 1D 序列，然后用与语言相同的架构处理它。

- ViT 比 CNN 具有更少的归纳偏置：它不强制局部连接或平移等变。这意味着它需要更多训练数据从头学习空间结构。在小数据集上，CNN 优于 ViT。但当在非常大数据集（JFT-300M，3 亿张图像）上训练时，ViT 匹配或超过最佳 CNN，表明 CNN 的归纳偏置对数据效率有帮助，但对最终性能不是必需的。

- ViT 自注意力在片段数量上是 $O(N^2)$。对于 224x224 图像和 16x16 片段，$N = 196$，这是可管理的。但对于更高分辨率图像或更小片段，二次成本变得过高。

- DeiT（数据高效图像 Transformer，Touvron 等人，2021）表明 ViT 可以仅在 ImageNet 上有效训练（无需庞大的 JFT 数据集），使用强数据增强、正则化（随机深度、标签平滑、dropout）和知识蒸馏：预训练 CNN 教师提供软标签，ViT 学生学习匹配。DeiT 在 [CLS] token 旁边添加了蒸馏 token，训练预测教师的输出。

- Swin Transformer（Liu 等人，2021）解决了 ViT 的两个主要限制：其随图像大小的二次成本和缺乏层次特征图（检测和分割需要）。

- Swin 引入移位窗口：不是在所有片段上的全局自注意力，注意力在局部窗口内计算（例如 7x7 片段）。这使成本在图像大小上呈线性：$O(N)$ 而非 $O(N^2)$。但仅局部窗口会阻止区域之间的信息流。

- 窗口移位 解决了这个问题：在交替层中，窗口分区移动半个窗口大小。这创建了跨窗口连接，允许信息在层之间在图像的所有部分之间流动，而无需全局注意力的成本。

![Swin Transformer：层 l 在规则窗口内计算注意力，层 l+1 将窗口分区移动一半，创建跨窗口连接](../images/swin_shifted_windows.svg)

- Swin 还通过跨阶段合并片段构建层次表示。每个阶段后，相邻 2x2 片段被拼接并投影以双倍通道维度并减半空间分辨率。这产生类似于 CNN 和 FPN（文件 03）的多尺度特征图，使 Swin 直接与检测头（如 Faster R-CNN）和分割头（如 U-Net）兼容。

- PVT（金字塔视觉 Transformer）采用类似的层次方法，具有空间减少注意力：在每个阶段，键和值在计算注意力之前空间下采样，减少二次成本，同时保持全局感受野。

- 自监督视觉学习 从无标签图像训练表示。标签收集昂贵，但图像丰富。目标是学习能很好转移到下游任务的特征，无需任何人工标注。

- 对比学习 训练模型识别同一图像的两个增强视图（"正对"）应该具有相似表示，而不同图像的视图（"负对"）应该具有不同表示。

- SimCLR（Chen 等人，2020）为批次中每个图像创建两个增强视图，用共享主干 + 投影头编码两者，并应用NT-Xent 损失（归一化温度缩放交叉熵）：

$$\ell_{i,j} = -\log \frac{\exp(\text{sim}(z_i, z_j) / \tau)}{\sum_{k \neq i} \exp(\text{sim}(z_i, z_k) / \tau)}$$

- 其中 $\text{sim}$ 是余弦相似度（第 01 章），$\tau$ 是温度参数。分子推动正对在一起；分母推开负对。SimCLR 需要大批次大小（4,096+）以提供足够负样本。

- MoCo（动量对比，He 等人，2020）通过维护动量更新队列的负嵌入来解决大批次要求。查询编码器通过梯度下降更新；键编码器更新为查询编码器的指数移动平均（EMA，第 04 章）：$\theta_k \leftarrow m \theta_k + (1 - m) \theta_q$，$m = 0.999$。队列存储最近键嵌入，提供大量一致的负样本，无需巨大批次。

- BYOL（自举你的潜在表示，Grill 等人，2020）完全消除负对。它使用两个网络："在线"网络和"目标"网络（在线的 EMA）。在线网络预测目标网络对不同增强视图的表示。没有负样本，BYOL 通过预测器头的不对称性和 EMA 目标避免崩溃问题（模型对所有内容输出相同向量）。

- DINO（无标签自蒸馏，Caron 等人，2021）将自蒸馏应用于 ViT。学生网络预测教师网络（学生的 EMA）在不同增强视图上的输出。教师使用更大裁剪；学生使用更小裁剪。DINO 产生的特征包含场景布局的显式信息：DINO 训练 ViT 的自注意力图自然分割对象，无需任何分割监督。

- 掩码图像建模 是 BERT 掩码语言建模（第 07 章）的视觉类似物。大部分输入片段被掩码，模型学习重建它们。

- MAE（掩码自编码器，He 等人，2022）掩码 75% 的片段，并训练 ViT 编码器 - 解码器重建缺失像素值。仅未掩码片段由编码器处理（在预训练期间节省 4 倍计算），轻量级解码器从编码的可见片段和可学习掩码 token 重建完整图像。

- BEiT（图像 Transformer 的 BERT 预训练，Bao 等人，2022）掩码片段并预测离散视觉 token（从预训练 dVAE token 化器获得），而非原始像素。这类似于 BERT 预测离散词 token，避免像素重建的低级细节。

- 图像生成 旨在生成训练集中不存在的新颖、真实图像。核心挑战是建模自然图像的高维概率分布。

- 生成对抗网络（GAN）（Goodfellow 等人，2014）使用两个竞争网络：生成器$G$ 从随机噪声创建假图像，判别器$D$ 试图区分真实图像和假图像。它们对抗训练：$G$ 试图欺骗 $D$，$D$ 试图抓住 $G$。

$$\min_G \max_D \; \mathbb{E}_{x \sim p_{\text{data}}}[\log D(x)] + \mathbb{E}_{z \sim p(z)}[\log(1 - D(G(z)))]$$

- 生成器采用随机潜在向量 $z$（从简单分布如高斯采样），并通过一系列转置卷积将其映射生成图像。判别器是标准 CNN 分类器。在平衡时，$G$ 产生与真实数据无法区分的图像，$D$ 对所有输入输出 0.5。

- 模式崩溃 是 GAN 的主要失败模式：生成器学习仅生成少数欺骗判别器的图像类型，忽略训练数据的多样性。生成器找到一小部分"安全"输出，而非覆盖完整分布。

- 稳定 GAN 的训练技巧包括：谱归一化（约束判别器的 Lipschitz 常数）、渐进式生长（首先低分辨率训练，然后逐渐增加）、特征匹配（匹配中间判别器特征的统计而非最终输出）以及使用 Wasserstein 距离而非原始 JS 散度目标。

- StyleGAN（Karras 等人，2019）是高质量图像合成最具影响力的 GAN 架构。其关键创新是基于风格的生成器：不是将潜在向量 $z$ 直接馈入生成器，它首先通过映射网络（8 层 MLP）生成风格向量 $w$。这个风格向量通过自适应实例归一化（AdaIN） 注入到生成器的每一层，调节特征图统计：

$$\text{AdaIN}(x, y) = y_{s} \cdot \frac{x - \mu(x)}{\sigma(x)} + y_{b}$$

- 其中 $y_s$ 和 $y_b$ 是从 $w$ 导出的缩放和偏置。不同层控制不同方面：早期层控制粗略特征（姿势、脸型），中间层控制中等特征（发型、眼睛），晚期层控制精细细节（雀斑、头发纹理）。StyleGAN 可以生成 1024x1024 分辨率的逼真面孔。

- 变分自编码器（VAE）（第 06 章）提供替代生成方法。与 GAN 不同，VAE 具有原则性的概率框架，具有清晰的训练目标（ELBO）。它们倾向于产生比 GAN 更模糊的图像，但提供更平滑、更结构化的潜在空间。VAE 是在潜在扩散模型中用于将图像压缩到和从潜在空间返回的编码器 - 解码器对。

- 扩散模型 已成为图像生成的主导范式，在质量和多样性上都超越 GAN。这个想法在概念上很简单：逐渐向数据添加噪声直到它变成纯高斯噪声（前向过程），然后学习逐步逆转这个过程（反向过程）。

- 前向过程 在 $T$ 个时间步添加高斯噪声：

$$q(x_t | x_{t-1}) = \mathcal{N}(x_t; \sqrt{1 - \beta_t} \, x_{t-1}, \beta_t I)$$

- 其中 $\beta_t$ 是随时间增加的噪声计划。足够步数后，$x_T$ 近似纯高斯噪声，无论原始图像 $x_0$ 如何。使用重参数化技巧（第 06 章）并设置 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{s=1}^{t} \alpha_s$，我们可以直接从 $x_0$ 采样 $x_t$：

$$x_t = \sqrt{\bar{\alpha}_t} \, x_0 + \sqrt{1 - \bar{\alpha}_t} \, \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

- 反向过程 学习去噪：从纯噪声 $x_T$ 开始，模型预测每个步骤添加的噪声 $\epsilon$ 并减去它以恢复 $x_{t-1}$。这由神经网络 $\epsilon_\theta$ 参数化（通常是 U-Net，来自文件 03），用简单 MSE 损失训练：

$$\mathcal{L} = \mathbb{E}_{t, x_0, \epsilon}\left[\|\epsilon - \epsilon_\theta(x_t, t)\|^2\right]$$

![扩散前向和反向过程：清晰图像在 T 步内逐渐被噪声损坏（前向），神经网络学习逆转每个步骤（反向），从纯噪声开始生成清晰图像](../images/diffusion_process.svg)

- DDPM（去噪扩散概率模型，Ho 等人，2020）建立了这个框架。采样需要迭代所有 $T$ 步（通常 1,000），这很慢。DDIM（去噪扩散隐式模型，Song 等人，2021）将采样过程重新参数化为确定性映射，允许大步长跳过（例如 50 步而非 1,000），质量损失最小。

- 基于分数的模型（Song 和 Ermon，2019）提供替代视角。模型不是预测噪声 $\epsilon$，而是估计分数函数$\nabla_{x_t} \log p(x_t)$，即关于含噪声图像的对数概率梯度。这个梯度指向数据分布的更高概率（更清洁）区域。采样使用 Langevin 动态遵循这个梯度。基于分数的模型和 DDPM 在随机微分方程（SDE） 框架中统一：前向过程是添加噪声的 SDE，反向过程是时间反转的 SDE。

- 无分类器引导（Ho 和 Salimans，2022）控制样本质量和多样性之间的权衡。模型既有条件训练（带文本提示或类别标签）又有无条件训练（随机丢弃条件）。在采样时，预测是加权组合：

$$\hat{\epsilon} = \epsilon_\theta(x_t, \varnothing) + s \cdot (\epsilon_\theta(x_t, c) - \epsilon_\theta(x_t, \varnothing))$$

- 其中 $c$ 是条件，$\varnothing$ 是空条件，$s > 1$ 是引导比例。更高 $s$ 产生更强匹配条件但多样性更低的图像。$s = 1$ 给出无引导模型；$s = 7.5$ 是常见默认值。

- 潜在扩散（Rombach 等人，2022；Stable Diffusion）将扩散过程从像素空间移动到学习的潜在空间。预训练 VAE 编码器将图像压缩到较低维潜在表示（通常 4 倍或 8 倍空间下采样），扩散在这个压缩空间中操作，VAE 解码器从去噪潜在重建像素。这明显更高效：在像素空间中扩散 512x512 图像意味着处理 $512 \times 512 \times 3$ 张量，但在潜在空间中仅 $64 \times 64 \times 4$ 张量。

- 潜在扩散中的去噪 U-Net 接收含噪声潜在、时间步（编码为正弦嵌入，类似于 transformer 中的位置编码）和条件信号（来自冻结 CLIP 或 T5 文本编码器的文本嵌入）。文本条件通过 U-Net 内的交叉注意力层进入：文本嵌入作为键和值，图像特征作为查询。这使模型在每个空间位置关注文本提示的相关部分。

- 流匹配 是扩散的新兴替代方案，学习噪声和数据之间的直接传输路径，而非 DDPM 的迭代去噪。

- 连续归一化流（CNF） 定义时间依赖速度场 $v_\theta(x, t)$，沿平滑轨迹将样本从简单分布 $p_0$（噪声）推到数据分布 $p_1$。变换遵循常微分方程（ODE）：

$$\frac{dx}{dt} = v_\theta(x, t), \quad t \in [0, 1]$$

- 从 $x_0 \sim \mathcal{N}(0, I)$ 开始，向前积分 ODE 到 $t = 1$ 产生来自数据分布的样本。速度场由神经网络参数化，并训练匹配目标条件流。

- 最优传输（OT）流匹配（Lipman 等人，2023）使用噪声和数据之间的直线路径作为目标流：从噪声样本 $x_0$ 到数据样本 $x_1$ 的条件路径简单是 $x_t = (1 - t) x_0 + t x_1$，目标速度是 $v = x_1 - x_0$。训练损失变为：

$$\mathcal{L} = \mathbb{E}_{t, x_0, x_1} \left[\|v_\theta(x_t, t) - (x_1 - x_0)\|^2\right]$$

- 重整化流（Liu 等人，2022）迭代地拉直学习到的流路径。初始训练后，模型用于通过模拟 ODE 生成（噪声，数据）对。这些对随机配对更紧密对齐，用于重新训练模型。重复这个过程产生越来越直的路径，可以用更少 ODE 步骤（甚至单步）遍历，实现极快生成。

- 流匹配相比扩散有几个优势：训练目标更简单（直接速度回归，无需噪声计划）、采样 ODE 更平滑（需要更少积分步骤）、与最优传输的连接提供理论基础。Stable Diffusion 3 和 Flux 使用流匹配而非传统 DDPM。

1. 从头实现 ViT 片段嵌入。将图像分割成片段，展平它们，投影到模型维度，添加位置嵌入，并前置 [CLS] token。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def create_patch_embedding(image, patch_size, d_model, params):
    """Convert an image into a sequence of patch embeddings."""
    H, W, C = image.shape
    n_patches_h = H // patch_size
    n_patches_w = W // patch_size
    n_patches = n_patches_h * n_patches_w

    # Extract patches
    patches = []
    for i in range(n_patches_h):
        for j in range(n_patches_w):
            patch = image[i*patch_size:(i+1)*patch_size,
                          j*patch_size:(j+1)*patch_size, :]
            patches.append(patch.ravel())
    patches = jnp.stack(patches)  # (N, P*P*C)

    # Linear projection to d_model
    embeddings = patches @ params['proj_w'] + params['proj_b']  # (N, d_model)

    # Prepend CLS token
    cls_token = params['cls_token']  # (1, d_model)
    embeddings = jnp.concatenate([cls_token, embeddings], axis=0)  # (N+1, d_model)

    # Add position embeddings
    embeddings = embeddings + params['pos_embed']  # (N+1, d_model)

    return embeddings, patches

# Setup

patch_size = 8
d_model = 64

image = image.at[:16, :16, 0].set(1.0)   # red top-left
image = image.at[:16, 16:, 1].set(1.0)   # green top-right
```

2. 实现简单 GAN 训练循环。在 2D 数据上训练生成器和判别器，并可视化生成分布收敛到真实分布。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def generator(z, params):
    h = jnp.tanh(z @ params['g_w1'] + params['g_b1'])
    h = jnp.tanh(h @ params['g_w2'] + params['g_b2'])
    return h @ params['g_w3'] + params['g_b3']

def discriminator(x, params):
    h = jax.nn.leaky_relu(x @ params['d_w1'] + params['d_b1'], 0.2)
    h = jax.nn.leaky_relu(h @ params['d_w2'] + params['d_b2'], 0.2)
    return jax.nn.sigmoid(h @ params['d_w3'] + params['d_b3'])

def init_params(key):
    keys = jax.random.split(key, 6)
    z_dim, h_dim, data_dim = 2, 32, 2
    scale = 0.1
    return {
        'g_w1': jax.random.normal(keys[0], (z_dim, h_dim)) * scale,
        'g_b1': jnp.zeros(h_dim),
        'g_w2': jax.random.normal(keys[1], (h_dim, h_dim)) * scale,
        'g_b2': jnp.zeros(h_dim),
        'g_w3': jax.random.normal(keys[2], (h_dim, data_dim)) * scale,
        'g_b3': jnp.zeros(data_dim),
        'd_w1': jax.random.normal(keys[3], (data_dim, h_dim)) * scale,
        'd_b1': jnp.zeros(h_dim),
        'd_w2': jax.random.normal(keys[4], (h_dim, h_dim)) * scale,
        'd_b2': jnp.zeros(h_dim),
        'd_w3': jax.random.normal(keys[5], (h_dim, 1)) * scale,
        'd_b3': jnp.zeros(1),
    }

def d_loss(params, real_data, fake_data):
    real_score = discriminator(real_data, params)
    fake_score = discriminator(fake_data, params)
    return -jnp.mean(jnp.log(real_score + 1e-7) + jnp.log(1 - fake_score + 1e-7))

def g_loss(params, fake_data):
    fake_score = discriminator(fake_data, params)
    return -jnp.mean(jnp.log(fake_score + 1e-7))

# Real data: ring distribution
key = jax.random.PRNGKey(42)
theta = jax.random.uniform(key, (512,)) * 2 * jnp.pi
real_data = jnp.stack([jnp.cos(theta), jnp.sin(theta)], axis=1)
real_data = real_data + jax.random.normal(key, real_data.shape) * 0.05

params = init_params(jax.random.PRNGKey(0))
d_grad = jax.grad(d_loss)
g_grad = jax.grad(g_loss)
lr = 0.001

snapshots = []
for step in range(3000):
    key, k1 = jax.random.split(key)
    z = jax.random.normal(k1, (512, 2))
    fake_data = generator(z, params)

    # Update discriminator
    grads = d_grad(params, real_data, fake_data)
    for k in ['d_w1', 'd_b1', 'd_w2', 'd_b2', 'd_w3', 'd_b3']:
        params[k] = params[k] - lr * grads[k]

    # Update generator
    fake_data = generator(z, params)
    grads = g_grad(params, fake_data)
    for k in ['g_w1', 'g_b1', 'g_w2', 'g_b2', 'g_w3', 'g_b3']:
        params[k] = params[k] - lr * grads[k]

    if step in [0, 500, 1500, 2999]:
        snapshots.append((step, fake_data.copy()))

fig, axes = plt.subplots(1, 4, figsize=(16, 4))
for ax, (step, fake) in zip(axes, snapshots):
    ax.scatter(real_data[:, 0], real_data[:, 1], s=5, alpha=0.3, c='#3498db', label='Real')
    ax.scatter(fake[:, 0], fake[:, 1], s=5, alpha=0.3, c='#e74c3c', label='Generated')
    ax.set_title(f'Step {step}'); ax.set_xlim(-2, 2); ax.set_ylim(-2, 2)
    ax.set_aspect('equal'); ax.legend(markerscale=3)
plt.suptitle('GAN Training: Generator Learns the Ring Distribution')
plt.tight_layout(); plt.show()
```

3. 实现扩散前向过程：在增加的时间步向图像添加噪声并可视化渐进损坏。然后实现单个去噪步骤。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def noise_schedule(T, beta_start=0.0001, beta_end=0.02):
    """Linear noise schedule."""
    betas = jnp.linspace(beta_start, beta_end, T)
    alphas = 1.0 - betas
    alpha_bars = jnp.cumprod(alphas)
    return betas, alphas, alpha_bars

def forward_diffusion(x0, t, alpha_bars, key):
    """Add noise to x0 at timestep t."""
    alpha_bar_t = alpha_bars[t]
    noise = jax.random.normal(key, x0.shape)
    xt = jnp.sqrt(alpha_bar_t) * x0 + jnp.sqrt(1 - alpha_bar_t) * noise
    return xt, noise

# Create a simple 2D "image" (checkerboard)
img = jnp.zeros((32, 32))
for i in range(4):
    for j in range(4):
        if (i + j) % 2 == 0:
            img = img.at[i*8:(i+1)*8, j*8:(j+1)*8].set(1.0)

T = 1000
betas, alphas, alpha_bars = noise_schedule(T)

# Visualise forward process
timesteps = [0, 50, 200, 500, 999]
key = jax.random.PRNGKey(42)

fig, axes = plt.subplots(1, len(timesteps), figsize=(16, 3.5))
for ax, t in zip(axes, timesteps):
    key, subkey = jax.random.split(key)
    xt, noise = forward_diffusion(img, t, alpha_bars, subkey)
    ax.imshow(xt, cmap='gray', vmin=-2, vmax=2)
    ax.set_title(f't={t}\n$\\bar{{\\alpha}}$={alpha_bars[t]:.3f}')
    ax.axis('off')
plt.suptitle('Diffusion Forward Process: Progressive Noise Addition')
plt.tight_layout(); plt.show()

# Simple denoising: train a tiny network to predict noise at t=200
t_denoise = 200
key, k1 = jax.random.split(key)
xt, true_noise = forward_diffusion(img, t_denoise, alpha_bars, k1)

# Tiny "denoiser": just learn a constant noise estimate (for illustration)
noise_estimate = jnp.zeros_like(img)
lr = 0.01
for step in range(100):
    residual = noise_estimate - true_noise
    noise_estimate = noise_estimate - lr * residual

# Reverse one step
alpha_bar_t = alpha_bars[t_denoise]
x_denoised = (xt - jnp.sqrt(1 - alpha_bar_t) * noise_estimate) / jnp.sqrt(alpha_bar_t)

fig, axes = plt.subplots(1, 3, figsize=(12, 4))
axes[0].imshow(img, cmap='gray'); axes[0].set_title('Original $x_0$'); axes[0].axis('off')
axes[1].imshow(xt, cmap='gray', vmin=-2, vmax=2)
axes[1].set_title(f'Noisy $x_{{200}}$'); axes[1].axis('off')
axes[2].imshow(x_denoised, cmap='gray')
axes[2].set_title('Denoised (one step)'); axes[2].axis('off')
plt.tight_layout(); plt.show()

mse = jnp.mean((x_denoised - img)2)
print(f"Denoising MSE: {mse:.4f}")
```
