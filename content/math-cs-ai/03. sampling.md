# 抽样

*抽样决定了我们如何收集数据，并直接控制我们得出的每个结论的质量。本章涵盖随机抽样、分层抽样、整群抽样和系统抽样、抽样分布、大数定律和自助法，这些方法对于 ML 中的训练/测试分割和数据集管理至关重要。*

- 在理想世界中，你会测量你关心的群体中的每一个成员。在实践中，这几乎从不可能。你不能调查每个选民、测试每个灯泡或扫描每个病人。所以你抽取一个**样本**并用它来了解整体。

- **总体**是你想要研究的个体或项目的完整集合。**样本**是你实际观察的子集。

- **参数**是描述总体的数字（例如一个国家所有成年人的真实平均身高）。

- **统计量**是从你的样本计算的数字（例如你测量的 500 人的平均身高）。统计量用于估计参数。

- 你结论的质量完全取决于你如何选择样本。有偏的样本导致有偏的结论，无论你的分析多么复杂。

- **抽样框**是你实际抽取样本的所有个体的列表。理想情况下这应该与总体完全匹配，但在实践中存在差距。

- 例如，如果你通过电话调查人们，你会错过所有没有电话的人。框和总体之间的差异称为**覆盖误差**。

- **抽样误差**是样本统计量和总体参数之间的自然差异。

- 即使是完全随机的样本也不会与总体完全匹配。更大的样本减少抽样误差。

- 有两大类的抽样：概率抽样和非概率抽样。

- **概率抽样**意味着总体中的每个成员都有已知的、非零的被选中机会。这让你能量化不确定性并推广结果。

- **简单随机抽样**：每个个体都有相等的被选中机会，并且每个大小为 $n$ 的可能样本都是等可能的。想象把每个名字放在帽子里并盲抽。

- **分层抽样**：根据共同特征（如年龄组、地区）将总体划分为不重叠的组（层），然后从每层随机抽样。这保证每个组都有代表，并在层之间相互差异时减少方差。

- **整群抽样**：将总体划分为组（群），随机选择一些群，然后包含所选群中的所有人。当总体在地理上分散时，这很实用，比如抽取整个学校而不是跨地区的个别学生。

- **系统抽样**：选择一个随机起点，然后从列表中选择每第 $k$ 个个体。例如，从第 7 个人开始，然后每 10 个人选择一个（7、17、27、...）。实现简单，但如果列表有隐藏模式，可能会引入偏差。

![三种概率抽样方法并排：简单随机、分层和整群](../images/sampling_methods.svg)

- **非概率抽样** 不给每个成员已知的被选中机会。结果不能严格推广，但这些方法通常更快、更便宜。

- **方便抽样**：选择最容易接触到的人。在购物中心调查人们很方便，但错过了不在那里购物的人。

- **定额抽样**：像分层抽样，但没有随机性。研究人员通过从每组选择可接触的个体来填充定额（例如 50 名男性和 50 名女性）。

- **滚雪球抽样**：从少数参与者开始，请他们招募其他人。对于难以接触的人群很有用（例如研究罕见疾病），但严重偏向于有联系的个体。

- 一旦你有了抽样方法，一个自然的问题就会出现：如果我抽取不同的样本，我会得到不同的统计量吗？几乎肯定是。抽样分布是同一大小的所有可能样本中统计量（如样本均值）的分布。

- 想象抽取 1000 个不同的 30 人样本并计算每个人的平均身高。这 1000 个均值形成一个分布。有些会略高于真实总体均值，有些略低，大多数会聚集在真实值周围。

- 这个抽样分布的标准差称为**标准误**：

$$SE = \frac{\sigma}{\sqrt{n}}$$

- 注意标准误随着 $n$ 增长而缩小。更大的样本给出更精确的估计。样本量翻两番，标准误减半。

- 统计学中最重要的结果是**中心极限定理（CLT）**。它说：无论原始总体的形状如何，随着样本量增加，样本均值的分布趋近于正态分布。

![中心极限定理：偏态总体产生正态分布的样本均值](../images/central_limit_theorem.svg)

- 更准确地说，如果 $X_1, X_2, \ldots, X_n$ 是来自任何分布的独立观测值，具有均值 $\mu$ 和有限方差 $\sigma^2$，那么当 $n$ 增长时：

$$\bar{X} \approx \text{Normal}\!\left(\mu, \frac{\sigma^2}{n}\right)$$

- CLT 使大多数推断统计成为可能。它让我们可以使用正态分布作为近似，即使底层数据不是正态的，只要样本足够大。

- "足够大"是多大？常见的经验法则是 $n \ge 30$，但这取决于总体的非正态程度。对于高度偏态分布，你可能需要更多。对于大致对称的总体，即使 $n = 10$ 也足够了。

- CLT 有三个关键条件：
    - **独立性**：每个观测值不能影响其他观测值
    - **有限方差**：总体方差必须存在（排除一些特殊分布）
    - **同分布**：所有观测值来自同一分布

1. 直观演示 CLT：从高度偏态分布中抽取样本，计算样本均值，观察均值的直方图变成钟形。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

key = jax.random.PRNGKey(0)

# 指数分布（非常偏态）
population = jax.random.exponential(key, shape=(100_000,))

fig, axes = plt.subplots(1, 4, figsize=(14, 3))
sample_sizes = [1, 5, 30, 100]

for ax, n in zip(axes, sample_sizes):
    keys = jax.random.split(key, 2000)
    means = jnp.array([jax.random.choice(k, population, shape=(n,)).mean() for k in keys])
    ax.hist(means, bins=40, color="#3498db", alpha=0.7, density=True)
    ax.set_title(f"n = {n}")
    ax.set_xlim(0, 4)

fig.suptitle("CLT：样本均值随 n 增加趋近正态分布", fontsize=13)
plt.tight_layout()
plt.show()
```

2. 比较简单随机抽样和分层抽样。创建一个具有不同组的总体，并显示分层抽样在估计中给出更低的方差。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(42)

# 总体：两个不同的组
group_a = jax.random.normal(key, shape=(500,)) + 10   # 均值 ~10
key, subkey = jax.random.split(key)
group_b = jax.random.normal(subkey, shape=(500,)) + 20  # 均值 ~20
population = jnp.concatenate([group_a, group_b])

# 简单随机抽样：1000 次试验，样本量 20
srs_means = []
for i in range(1000):
    key, subkey = jax.random.split(key)
    sample = jax.random.choice(subkey, population, shape=(20,), replace=False)
    srs_means.append(sample.mean())
srs_means = jnp.array(srs_means)

# 分层抽样：每组 10 个
strat_means = []
for i in range(1000):
    key, k1, k2 = jax.random.split(key, 3)
    s_a = jax.random.choice(k1, group_a, shape=(10,), replace=False)
    s_b = jax.random.choice(k2, group_b, shape=(10,), replace=False)
    strat_means.append(jnp.concatenate([s_a, s_b]).mean())
strat_means = jnp.array(strat_means)

print(f"简单随机 - 均值：{srs_means.mean():.3f}, 标准差：{srs_means.std():.3f}")
print(f"分层抽样   - 均值：{strat_means.mean():.3f}, 标准差：{strat_means.std():.3f}")
print(f"分层抽样将方差减少了 {(1 - strat_means.var()/srs_means.var())*100:.1f}%")
```

3. 探索样本量如何影响标准误。绘制标准误与样本量的关系图，并确认 $1/\sqrt{n}$ 关系。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

key = jax.random.PRNGKey(7)
population = jax.random.normal(key, shape=(50_000,)) * 10 + 50

sample_sizes = [5, 10, 20, 50, 100, 200, 500, 1000]
std_errors = []

for n in sample_sizes:
    means = []
    for _ in range(500):
        key, subkey = jax.random.split(key)
        sample = jax.random.choice(subkey, population, shape=(n,))
        means.append(sample.mean())
    std_errors.append(jnp.array(means).std())

plt.figure(figsize=(8, 4))
plt.plot(sample_sizes, std_errors, "o-", color="#e74c3c", label="观测标准误")
theoretical = population.std() / jnp.sqrt(jnp.array(sample_sizes, dtype=jnp.float32))
plt.plot(sample_sizes, theoretical, "--", color="#3498db", label="σ/√n (理论)")
plt.xlabel("样本量 (n)")
plt.ylabel("标准误")
plt.legend()
plt.title("标准误随样本量增大而缩小")
plt.show()
```
