## 📦 基本概念

### 总体 vs 样本

| 概念 | 定义 | 例子 |
|------|------|------|
| 总体 | 完整集合 | 10000 块奶酪 |
| 样本 | 子集 | 抽检的 100 块 |
| 参数 | 总体的数字 | 真实平均重量 μ |
| 统计量 | 样本的数字 | 样本平均重量 $\bar{x}$ |

### 抽样框

抽样框是你实际抽取样本的列表。

理想：抽样框 = 总体

现实：会有差距 → 覆盖误差

## 🎲 概率抽样 - 每个成员有已知机会

概率抽样：总体中每个成员都有已知的、非零的被选中机会。

### 1. 简单随机抽样

每个个体都有相等的被选中机会。

```
总体：①②③④⑤⑥⑦⑧⑨⑩
           ↓
样本：②⑤⑦⑩  （随机抽取）
```

优点：简单、无偏

缺点：需要完整抽样框

---

### 2. 分层抽样

先分组（层），再从每组随机抽样。

<div class="lesson-two-column">
  <div class="lesson-text-column">

步骤：
1. 将总体分成不重叠的组（层）
2. 从每层随机抽样

例子：
- 层 1：A 级奶酪（60%）→ 抽 60 块
- 层 2：B 级奶酪（40%）→ 抽 40 块

优点：
- 保证每组都有代表
- 层间差异大时减少方差

</div>
  <div class="lesson-visual-column">

```
分层抽样：

总体：AAAAAABBBBBB
        ↓ 分层
A 层：AAAAAA  → 抽 AAA
B 层：BBBBBB  → 抽 BBB

每组都有代表！
```

<div class="visual-caption">图 1：分层抽样保证每组代表性</div>

</div>
</div>

### 3. 整群抽样

先分群，随机选几个群，包含群内所有人。

```
总体：[群 1][群 2][群 3][群 4]
            ↓
样本：[群 2][群 4]  （选中整个群）
```

适用场景：地理上分散的总体

例子：抽取整个学校，而不是跨地区的个别学生

优点：实用、成本低

缺点：相同样本量下方差更大

---

### 4. 系统抽样

随机起点，然后每第 k 个选择一个。

```
总体：①②③④⑤⑥⑦⑧⑨⑩⑪⑫
              ↑        ↑
样本：        ④        ⑩   （每 6 个选 1 个）
```

优点：实现简单

风险：如果列表有隐藏模式，可能引入偏差

---

## ❌ 非概率抽样 - 不能推广结果

非概率抽样不给每个成员已知的被选中机会。

### 1. 方便抽样

选择最容易接触到的人。

例子：在购物中心调查路人

问题： misses 不在那里的人 → 有偏

---

### 2. 定额抽样

像分层抽样，但没有随机性。

```
目标：50 名男性 + 50 名女性
      ↓
     找"容易找到的"50 男 50 女
```

问题：研究人员的选择可能引入偏差

---

### 3. 滚雪球抽样

从少数参与者开始，请他们招募其他人。

适用：难以接触的人群（如罕见疾病患者）

问题：严重偏向有联系的个体

<div class="key-point">
  <div class="key-point-title">🎯 概率 vs 非概率</div>
  <ul class="key-point-list">
    <li><strong>概率抽样</strong>：可量化不确定性，可推广结果</li>
    <li><strong>非概率抽样</strong>：更快更便宜，但不能严格推广</li>
    <li><strong>ML 中</strong>：训练集/测试集分割必须用随机抽样！</li>
  </ul>

## 📊 抽样分布 - 统计量的分布

如果你抽取不同的样本，统计量会不同吗？几乎肯定会。

抽样分布：同一大小的所有可能样本中，统计量（如样本均值）的分布。

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子

想象抽取 1000 个不同的 30 人样本，计算每个人的平均身高：

```
样本 1 均值：172.3cm
样本 2 均值：171.8cm
样本 3 均值：173.1cm
...
样本 1000 均值：172.0cm
```

这 1000 个均值形成一个分布！

### 标准误 (SE)

抽样分布的标准差：

$$SE = \frac{\sigma}{\sqrt{n}}$$

- $\sigma$ = 总体标准差
- $n$ = 样本量

</div>
  <div class="lesson-visual-column">

```
标准误的意义：

小样本 (n=10):  ━━━━ 宽分布
大样本 (n=100): ━ 窄分布

样本量越大 → 估计越精确！

n 翻 4 倍 → SE 减半
```

<div class="visual-caption">图 2：样本量增大，标准误减小</div>

</div>

## 🌟 中心极限定理 (CLT) - 统计学的基石

中心极限定理说：

> 无论原始总体的形状如何，随着样本量增加，样本均值的分布趋近于正态分布！

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的 CLT 理解</div>
  <div class="story-card-content">
    原始总体可以是任何形状：
    <br>
    - 偏态分布 ✓
    <br>
    - 均匀分布 ✓
    <br>
    - 甚至奇形怪状 ✓
    <br><br>
    只要样本量够大，<strong>样本均值的分布</strong>就会变成正态分布！
    <br><br>
    这就是为什么正态分布如此重要！
  </div>
</div>

### 数学表述

如果 $X_1, X_2, \ldots, X_n$ 是来自任何分布的独立观测值，具有均值 $\mu$ 和方差 $\sigma^2$，那么：

$$\bar{X} \approx \text{Normal}\left(\mu, \frac{\sigma^2}{n}\right)$$

---

### CLT 的三个条件

| 条件 | 说明 |
|------|------|
| 独立性 | 每个观测值不能影响其他 |
| 有限方差 | 总体方差必须存在 |
| 同分布 | 所有观测值来自同一分布 |

### "足够大"是多大？

| 总体形状 | 所需样本量 |
|---------|-----------|
| 大致对称 | $n \ge 10$ |
| 中等偏态 | $n \ge 30$（经验法则） |
| 高度偏态 | $n \ge 50$ 或更多 |

  <div class="lesson-text-column">

```
   ╱│╲
  ╱ │ ╲

```

</div>

```
  ↑
  │╱
```
