# 向量空间

*向量空间是 ML/AI 的数学基础。本章涵盖向量加法、标量乘法、闭包公理、子空间，以及为什么 AI 中的几乎所有内容都表示为向量。*

- 想象一个**向量空间**就是一个特定的"游乐场"，数学对象生活在这里，每个对象叫做**向量**。

- 在机器学习的几何直觉中，我们总是把向量看作欧几里得空间中的一个点，用坐标表示。

- 向量 $\mathbf{a}$（数学上用粗体小写字母表示）有 $n$ 个坐标，每个坐标代表一个轴上的位置：

$$\mathbf{a} = [a_1, a_2, a_3]$$

- 例如，一个三维向量 $\mathbf{a} = (3, 2, 4)$ 可以在 3D 空间中表示为一个箭头。

## 向量的基本概念

### 向量的几何意义

- 向量可以理解为**移动指令**：
  - "向右走 3 步，向前走 2 步，向上走 4 步"
  - 这就是向量 $(3, 2, 4)$ 的含义

- 向量的**维度**就是独立方向的数量：
  - $\mathbb{R}^2$ 是 2 维的（需要 2 个坐标）
  - $\mathbb{R}^3$ 是 3 维的
  - $\mathbb{R}^n$ 是 $n$ 维的

### 实际应用示例

我们可以用向量表示任何对象。例如，表示一个人：

$$\mathbf{h} = [\text{身高}, \text{体重}, \text{年龄}] = [185, 75, 30]$$

- 这就是一个**特征向量**
- 可以添加更多特征：收入、教育年限、所在城市...
- 特征越多，表示越丰富

## 向量空间的规则

向量空间中的向量必须遵循一套不可破坏的规则：

### 1. 向量加法（组合）

```
向量 a = "向前走 3 步"
向量 b = "向右走 2 步"
a + b = "向前走 3 步 且 向右走 2 步"
```

对于两个向量 $\mathbf{a} = (a_1, a_2)$ 和 $\mathbf{b} = (b_1, b_2)$：

$$\mathbf{a} + \mathbf{b} = (a_1 + b_1, a_2 + b_2)$$

### 2. 标量乘法（缩放）

```
向量 v = "向前走 3 步"
2v = "向前走 6 步"（拉伸）
-v = "向后走 3 步"（反向）
0.5v = "向前走 1.5 步"（缩小）
```

对于标量 $c$ 和向量 $\mathbf{v} = (v_1, v_2)$：

$$c\mathbf{v} = (cv_1, cv_2)$$

## 向量空间公理

以下是向量空间必须满足的 8 条公理：

### 加法公理

1. **封闭性**：$\mathbf{u} + \mathbf{v} \in V$
2. **交换律**：$\mathbf{u} + \mathbf{v} = \mathbf{v} + \mathbf{u}$
3. **结合律**：$(\mathbf{u} + \mathbf{v}) + \mathbf{w} = \mathbf{u} + (\mathbf{v} + \mathbf{w})$
4. **零向量**：存在 $\mathbf{0}$ 使得 $\mathbf{v} + \mathbf{0} = \mathbf{v}$
5. **加法逆元**：对每个 $\mathbf{v}$，存在 $-\mathbf{v}$ 使得 $\mathbf{v} + (-\mathbf{v}) = \mathbf{0}$

### 标量乘法公理

6. **封闭性**：$c\mathbf{v} \in V$
7. **分配律 1**：$c(\mathbf{u} + \mathbf{v}) = c\mathbf{u} + c\mathbf{v}$
8. **分配律 2**：$(c + d)\mathbf{v} = c\mathbf{v} + d\mathbf{v}$
9. **结合律**：$(cd)\mathbf{v} = c(d\mathbf{v})$
10. **单位元**：$1\mathbf{v} = \mathbf{v}$

## 子空间

**子空间**是向量空间中的"小游乐场"。

- 想象 3D 空间是一个房间
- 一张通过房间中心的纸是一个子空间
- 一根通过中心的线也是一个子空间

**关键要求**：子空间必须通过原点（包含零向量）。

## 为什么在 AI 中如此重要？

在 AI/ML 中，几乎所有东西都表示为向量：

| 对象 | 向量表示 |
|------|----------|
| 图片 | 像素值数组 |
| 文本 | 词嵌入向量 |
| 用户 | 特征向量（年龄、兴趣...） |
| 音频 | 频谱图向量 |
| 模型参数 | 权重向量 |

**线性代数**就是研究向量和向量空间的数学分支，是 AI/ML 的基础。
