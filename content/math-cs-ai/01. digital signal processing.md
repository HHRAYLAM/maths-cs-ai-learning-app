# 数字信号处理

*数字信号处理将原始音频波形转换为机器学习模型可以学习的结构化表示。本文件涵盖声音物理学、采样和量化、傅里叶变换（DFT、FFT）、语谱图、mel 滤波器组、MFCC 以及加窗，这是所有语音和音频 AI 的特征提取流水线。*

- 声音是一种在介质（空气、水、固体）中传播的压力波。振动物体（声带、吉他弦、扬声器振膜）推拉空气分子，产生交替的高压区（压缩）和低压区（稀疏）。

- 这些压力变化以大约 343 m/s 的速度在空气中向外传播并到达你的耳朵，在那里它们振动鼓膜并被转换为神经信号。

- 想象声音就像向平静的池塘中扔石头：石头是振动源，涟漪是压力波，漂浮在水面上的软木塞是响应波到达的麦克风或鼓膜。

- 软木塞上下摆动的高度是振幅，每秒摆动的次数是频率，波到达时它从摆动顶部还是底部开始是相位。

- 波形是压力（或电压，在麦克风将声音转换为电信号后）随时间变化的图。最简单的波形是纯音，单个正弦波：

$$x(t) = A \sin(2\pi f t + \phi)$$

- 其中：
    - $A$ 是振幅（偏离零的峰值，决定响度），
    - $f$ 是频率，单位为 Hz（每秒周期数，决定音高），
    - $\phi$ 是相位，单位为弧度（波的时间偏移）。

- 周期是 $T = 1/f$，一个完整周期的持续时间。

![标注了振幅、周期、频率和相移的正弦波](../images/audio_waveform.svg)

- 振幅决定感知的响度。将振幅加倍会使功率变为四倍（因为功率与振幅的平方成正比）。

- 人耳跨越巨大的振幅范围，因此我们使用对数标度：分贝（dB）。声压级为：

$$L = 20 \log_{10}\left(\frac{A}{A_\text{ref}}\right) \text{ dB}$$

- 其中 $A_\text{ref}$ 是参考振幅（通常是听阈，$20 \mu\text{Pa}$）。耳语约为 30 dB，正常对话 60 dB，摇滚音乐会 110 dB。每增加 6 dB 大致使振幅加倍；每增加 10 dB 大致使感知响度加倍。这里的对数与第 03 章中的函数相同。

- 频率决定音高。低频（20-250 Hz）听起来是低音；高频（2000-20000 Hz）听起来是高音。人耳听力范围大约为 20 Hz 至 20 kHz。标准音 A 是 440 Hz。将频率加倍会将音高提高一个八度。

- 大多数自然声音不是纯音，而是许多频率的复杂混合，这就是为什么钢琴和小提琴演奏相同的音符听起来不同：它们共享相同的基频，但它们的泛音（基频的整数倍）及其相对振幅（音色）不同。

- 相位决定波在其周期中的起始位置。两个具有相同振幅和频率但不同相位的波可以相长干涉（相位对齐，振幅相加）或相消干涉（相位相反，振幅抵消）。

- 相位在立体声音频和波束形成中至关重要，但在许多语音处理流水线中基本上被丢弃，因为人类的音高和音色感知主要是相位不变的。

- 现实世界的音频信号是时间的连续函数，但计算机使用离散数字工作。采样通过定期测量信号的值将连续信号转换为离散序列。

- 采样率 $f_s$ 是每秒测量的次数。CD 音频使用 $f_s = 44{,}100$ Hz；电话使用 8000 Hz；现代语音模型通常使用 16000 Hz。

- 奈奎斯特 - 香农采样定理指出，当且仅当采样率至少是信号中存在的最高频率的两倍时，可以从其样本中完美重建连续信号：

$$f_s \geq 2 f_\text{max}$$

- 频率 $f_s / 2$ 称为奈奎斯特频率。如果信号包含高于奈奎斯特频率的频率，这些频率会折回有效范围并显示为虚假的低频分量。这种现象称为混叠。混叠是不可逆的：一旦发生，原始信号无法从样本中恢复。

- 混叠的日常类比是电影中的马车轮效应：刚好高于帧率旋转的轮子看起来缓慢向后旋转，因为相机对旋转欠采样。在音频中，以 16 kHz 采样（$f_\text{Nyquist} = 8$ kHz）的 15 kHz 音调会混叠到 $16 - 15 = 1$ kHz，一个完全不同的音高。

![信号的正确采样与采样率过低的混叠采样](../images/sampling_aliasing.svg)

- 为了防止混叠，抗混叠滤波器（低通滤波器）在采样前去除所有高于 $f_s/2$ 的频率。这由模数转换器（ADC）硬件在信号数字化之前应用。

- 量化将每个连续值的样本映射到有限级别集中的最近值。$n$ 位量化器有 $2^n$ 个级别。CD 音频使用 16 位量化（$2^{16} = 65{,}536$ 个级别）；电话通常使用 8 位和 $\mu$ 律或 A 律压扩（非线性映射，将更多级别用于小振幅，匹配人类感知）。量化引入量化噪声，一种舍入误差形式，方差为 $\Delta^2/12$，其中 $\Delta$ 是级别之间的步长。

- 时域分析直接从波形提取特征，而不将其转换到另一个域。这些特征简单、计算快速，并捕获基本信号属性。

- 帧的能量（$N$ 个样本）测量整体响度：

$$E = \sum_{n=0}^{N-1} x[n]^2$$

- 语音段具有高能量；静音具有低能量。能量是第 01 章中的 $\ell_2$ 范数平方应用于信号向量。

- 过零率（ZCR）计算信号在帧中改变符号的频率：

$$\text{ZCR} = \frac{1}{2(N-1)} \sum_{n=1}^{N-1} |\text{sign}(x[n]) - \text{sign}(x[n-1])|$$

- 高 ZCR 表示高频内容或噪声；低 ZCR 表示低频或浊音语音（声带周期性振动）。ZCR 是一个粗略的频率估计器：$f$ Hz 的纯音每秒过零 $2f$ 次。

- 自相关测量信号与其延迟副本的相似程度：

$$R[k] = \sum_{n=0}^{N-1-k} x[n] \cdot x[n+k]$$

- 在滞后 $k = 0$ 处，自相关等于能量。对于周期信号，自相关在等于周期及其倍数的滞后处有峰值。这是音高检测的标准技术：在 $k=0$ 后找到 $R[k]$ 中第一个显著峰值，音高为 $f_s / k_\text{peak}$。自相关与第 01 章中的点积相关：$R[k]$ 是信号与其 $k$ 偏移版本的点积。

- 频域分析揭示信号的频谱内容，这些信息在波形中是不可见的。关键工具是离散傅里叶变换（DFT），它将 $N$ 个样本的信号分解为 $N$ 个复数值频率分量：

$$X[k] = \sum_{n=0}^{N-1} x[n] \cdot e^{-j 2\pi k n / N}, \quad k = 0, 1, \ldots, N-1$$

- 每个 $X[k]$ 是一个复数，其幅值 $|X[k]|$ 给出频率 $f_k = k \cdot f_s / N$ Hz 的分量振幅，其相位 $\angle X[k]$ 给出相位偏移。DFT 是从时域基（单位脉冲）到频域基（复指数）的基变换，这是第 02 章中基概念的直接应用。DFT 可以写为矩阵乘法 $\mathbf{X} = W \mathbf{x}$，其中 $W$ 是 $N \times N$ DFT 矩阵，条目为 $W_{kn} = e^{-j2\pi kn/N}$。

- 快速傅里叶变换（FFT）是一种算法，通过递归地将问题拆分为偶数索引和奇数索引子问题（Cooley-Tukey 算法），以 $O(N \log N)$ 次操作而不是朴素的 $O(N^2)$ 计算 DFT。这种加速使得实时频谱分析成为可能。FFT 是整个计算领域最重要的算法之一。

- 功率谱 $|X[k]|^2$ 显示能量在频率上的分布。幅值谱 $|X[k]|$ 显示振幅。绘制这些揭示哪些频率在信号中占主导地位：元音在基频的整数倍处有强泛音；擦音（如"s"）具有宽的高频能量。

- 语谱图是信号频率内容如何随时间变化的视觉表示。它通过将信号切成短的 overlapping 帧，计算每帧的 FFT，并将得到的幅值谱并排堆叠来计算。横轴是时间，纵轴是频率，每个点的颜色（或亮度）表示幅值。语谱图是音频处理中最重要的单一可视化。

![语谱图显示水平轴为时间，垂直轴为频率，强度为颜色](../images/spectrogram_stft.svg)

- mel 标度是一种感知频率标度，反映人类如何感知音高。人类感知相等的频率比为相等的音高间隔（就像我们感知相等的强度比为相等的响度间隔）。在大约 1000 Hz 以下，mel 标度大致是线性的；在 1000 Hz 以上，它大致是对数的：

$$m = 2595 \log_{10}\left(1 + \frac{f}{700}\right)$$

- 逆函数为 $f = 700(10^{m/2595} - 1)$。mel 标度是音乐半音在对数频率轴上等距的原因：A4 (440 Hz) 到 A5 (880 Hz) 和 A5 到 A6 (1760 Hz) 都听起来像"一个八度"，即使 Hz 差距分别为 440 和 880。

- mel 滤波器组是一组在 mel 标度上均匀间隔的三角形带通滤波器。每个滤波器覆盖一个频带并对该频带内的谱能量求和，产生一个数字。典型的语音系统使用 40-80 个 mel 滤波器。低频滤波器窄（在我们感知敏感的高频率分辨率区域），高频滤波器宽（在我们不太敏感的区域分辨率低）。这模拟了人耳耳蜗的频率分辨率。

![在对数频率轴上叠加的三角形 mel 标度滤波器组，显示低频处的窄滤波器和高频处的宽滤波器](../images/mel_filterbank.svg)

- 梅尔频率倒谱系数（MFCCs）是语音和音频的经典特征表示。它们将 mel 谱压缩为少量去相关的系数，捕获谱包络的形状（编码声道配置，因此是语音身份），同时丢弃精细谱细节（编码音高和相位）。

- MFCC 流水线：
    1. 预加重：应用一阶高通滤波器 $y[n] = x[n] - \alpha x[n-1]$（通常 $\alpha = 0.97$）以提升被声道衰减的高频。
    2. 分帧：将信号切成 overlapping 帧（通常 25 ms 长，10 ms 跳步）。
    3. 加窗：将每帧乘以窗函数（Hamming）以减少谱泄漏（见下文）。
    4. FFT：计算每个加窗帧的功率谱。
    5. mel 滤波器组：将三角形 mel 滤波器组应用于功率谱，产生 mel 频带能量。
    6. 对数：取 mel 频带能量的对数。对数压缩动态范围并将乘法（谱分量）转换为加法，匹配人类响度感知。
    7. DCT：将对数 mel 能量应用于离散余弦变换。DCT 去相关 mel 频带（因为相邻频带高度相关）并将能量压缩到前几个系数中。保留前 13 个系数（MFCC-0 到 MFCC-12）。

![MFCC 流水线从原始音频通过加窗帧、FFT、mel 滤波器组、对数压缩和 DCT 到最终 MFCC 特征](../images/mfcc_pipeline.svg)

- 步骤 7 中的 DCT 本质上是"谱的傅里叶变换"（因此命名为倒谱 = spectrum 的变位词）。低阶倒谱系数捕获 broad 谱形状（声道共振，称为共振峰），而高阶系数捕获精细谱细节（音高泛音）。通过仅保留前 13 个，我们保留共振峰信息并丢弃音高细节。

- Delta和delta-delta MFCCs（MFCCs 的一阶和二阶时间导数，通过相邻帧的有限差分计算）捕获谱形状的动态，添加时间上下文。完整的 MFCC 特征向量通常是 39 维：13 个静态 + 13 个 delta + 13 个 delta-delta。

- 现代神经网络模型（第 06 章）在很大程度上用学习到的特征取代了 MFCCs：对数 mel 语谱图（步骤 6 的输出，跳过 DCT）是深度学习 ASR 和音频分类的标准输入。模型学习自己的去相关。尽管如此，MFCCs 对于资源有限的环境、经典 ML 流水线和理解信号处理基础仍然很重要。

- 加窗是在计算 FFT 之前将信号帧乘以平滑窗函数的过程。如果不加窗，FFT 假设帧无限重复；帧的突然开始和结束会产生人为的不连续性，将能量扩散到所有频率，这种伪影称为谱泄漏。

- 矩形窗 $w[n] = 1$ 对所有 $n$：无 tapering，最大泄漏，但主瓣最宽（对于给定帧长的最佳频率分辨率）。实践中很少使用。

- Hamming 窗：$w[n] = 0.54 - 0.46 \cos(2\pi n / (N-1))$。在边缘 tapering 到接近零，大大减少泄漏。语音处理的标准选择。

- Hann 窗（也称为 Hanning）：$w[n] = 0.5 - 0.5 \cos(2\pi n / (N-1))$。在边缘 tapering 到恰好为零。与 Hamming 非常相似，但具有稍好的旁瓣抑制。

- Blackman 窗：$w[n] = 0.42 - 0.5 \cos(2\pi n / (N-1)) + 0.08 \cos(4\pi n / (N-1))$。更好的旁瓣抑制但更宽的主瓣（更差的频率分辨率）。当旁瓣伪影特别成问题时使用。

- 存在一个基本的权衡：泄漏较少的窗具有更宽的主瓣，意味着它们无法分辨两个紧密间隔的频率。这是谱分辨率与泄漏的权衡，第 03 章不确定性原理的结果。

- 重叠相加（OLA）是一种从加窗、处理后的帧重建信号的技术。帧 overlapping（通常 50-75%），处理后，加窗输出相加。如果窗和重叠选择正确（例如，Hann 窗带 50% 重叠），overlapping 窗相加为常数，给出完美重建。这对于任何基于帧的音频修改（降噪、音高移位、时间拉伸）至关重要。

- 短时傅里叶变换（STFT）是语谱图背后的形式框架。它将 DFT 应用于信号的每个加窗帧：

```math
\text{STFT}\{x[n]\}(m, k) = \sum_{n=0}^{N-1} x[n + mH] \cdot w[n] \cdot e^{-j 2\pi k n / N}
```

- 其中 $m$ 是帧索引，$H$ 是跳步大小（连续帧之间的样本数），$w[n]$ 是窗函数，$N$ 是 FFT 大小。输出是一个 2D 复数值矩阵：信号的时频表示。

- STFT 体现了一个基本的时频权衡：
    - 长帧（大 $N$）：高频率分辨率（可以区分紧密间隔的频率）但差的时间分辨率（无法准确定位频率变化的时间）。
    - 短帧（小 $N$）：高时间分辨率但差的频率分辨率。
    - 时间分辨率和频率分辨率的乘积有下界：$\Delta t \cdot \Delta f \geq \frac{1}{4\pi}$。这是Gabor 极限，物理学中海森堡不确定性原理的信号处理类比。

- 典型语音 STFT 参数：25 ms 帧长（16 kHz 下 $N = 400$），10 ms 跳步（$H = 160$），Hamming 窗，512 点 FFT（从 400 零填充以提高效率和更平滑的谱插值）。

- 滤波通过放大某些频率和衰减其他频率来修改信号的频率内容。滤波器是一个接受输入信号并产生输出信号的系统。滤波器通过其频率响应 $H(f)$ 表征，描述应用于每个频率的增益和相移。

- 低通滤波器：通过低于截止频率 $f_c$ 的频率并衰减高于它的频率。去除高频噪声和细节。采样前的抗混叠滤波器是低通滤波器。

- 高通滤波器：通过高于 $f_c$ 的频率并衰减低于它的频率。去除低频隆隆声和直流偏移。MFCC 提取中的预加重滤波器（$y[n] = x[n] - 0.97 x[n-1]$）是一个简单的高通滤波器。

- 带通滤波器：通过 $[f_1, f_2]$ 范围内的频率并衰减范围外的频率。mel 滤波器组中的每个三角形是一个带通滤波器。

- 带阻（陷波）滤波器：衰减特定窄频率范围。用于去除特定干扰（例如，50/60 Hz 电源嗡嗡声）。

- 有限脉冲响应（FIR）滤波器将每个输出样本计算为当前和过去输入样本的加权和：

$$y[n] = \sum_{k=0}^{M} b_k \cdot x[n-k]$$

- 权重 $b_k$ 是滤波器系数（也称为抽头）。滤波器阶数为 $M$。FIR 滤波器总是稳定的（输出永远不会发散）并且可以设计为具有完全线性相位（所有频率延迟相同的量，保持波形形状）。它们的缺点是 achieving 锐利截止需要许多抽头（高 $M$），增加计算。输出是输入与系数向量的卷积，正是第 06 章中的一维卷积操作。

- 无限脉冲响应（IIR）滤波器使用反馈：输出取决于过去输入和过去输出：

```math
y[n] = \sum_{k=0}^{M} b_k \cdot x[n-k] - \sum_{k=1}^{L} a_k \cdot y[n-k]
```

- 反馈项 $a_k$ 创建一个递归结构，其脉冲响应在理论上持续无限时间。IIR 滤波器用更少的系数实现锐利截止，但它们可能不稳定（如果传递函数的极点位于单位圆外，输出会无界增长，这是 z 变换的概念）。它们还具有非线性相位，可能扭曲波形形状。经典滤波器设计（Butterworth、Chebyshev、椭圆）是 IIR。

- 离散时间滤波器的传递函数通过 z 变换获得：

$$H(z) = \frac{\sum_{k=0}^{M} b_k z^{-k}}{1 + \sum_{k=1}^{L} a_k z^{-k}}$$

- 分子的根称为零点，分母的根称为极点。极点 - 零点图完全表征滤波器的行为。靠近单位圆的极点放大附近频率；靠近单位圆的零点衰减它们。FIR 滤波器只有零点（分母为 1）。这与第 02 章和第 03 章中的特征值和求根概念相关。

- 卷积定理：时域中的卷积等于频域中的逐元素乘法。这意味着滤波可以通过直接将信号与滤波器脉冲响应卷积，或通过乘以它们的傅里叶变换并对结果进行逆变换来执行。对于长滤波器，频域方法（使用 FFT）更快：$O(N \log N)$ 对比 $O(NM)$。

- 逆 STFT（iSTFT）从 STFT 表示重建时域信号。这对于任何在频域中修改音频的系统（降噪、源分离、语音转换）至关重要。重建使用重叠相加：

```math
x[n] = \frac{\sum_{m} w[n - mH] \cdot \text{IDFT}\{X(m, k)\}[n - mH]}{\sum_{m} w[n - mH]^2}
```

- 分母对窗重叠进行归一化，确保当合成窗与分析窗匹配且重叠足够时完美重建。

- 语音 DSP 链总结：原始音频以 16 kHz 采样，预加重，切成 25 ms Hamming 加窗帧，跳步 10 ms，每帧进行 FFT 变换，通过 mel 滤波器组，对数压缩，并保留为对数 mel 特征（用于神经网络模型）或 DCT 变换产生 MFCCs（用于经典模型）。整个链将一维时域信号转换为适合下游机器学习的二维时频表示，这将是文件 02 的主题。

3. 从头开始在 JAX 中构建完整的 MFCC 流水线：预加重、分帧、加窗、FFT、mel 滤波器组、对数、DCT。可视化 mel 滤波器组和生成的 MFCCs 作为热图。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# --- 生成合成语音样信号 ---
key = jax.random.PRNGKey(42)
fs = 16000
duration = 1.0
t = jnp.arange(0, duration, 1.0 / fs)

# 模拟浊音语音：基频 + 泛音，振幅衰减
f0 = 150.0  # 基频
x = sum(jnp.sin(2 * jnp.pi * f0 * k * t) / k for k in range(1, 8))
# 添加一些噪声
x = x + 0.1 * jax.random.normal(key, t.shape)
x = x / jnp.max(jnp.abs(x))  # 归一化

# --- 步骤 1：预加重 ---
alpha = 0.97
x_pre = jnp.concatenate([x[:1], x[1:] - alpha * x[:-1]])

# --- 步骤 2：分帧 ---
frame_len = int(0.025 * fs)   # 25 ms = 400 样本
hop_len = int(0.010 * fs)     # 10 ms = 160 样本
n_frames = (len(x_pre) - frame_len) // hop_len + 1
frames = jnp.stack([x_pre[i * hop_len : i * hop_len + frame_len]
                     for i in range(n_frames)])

# --- 步骤 3：Hamming 窗 ---
hamming = 0.54 - 0.46 * jnp.cos(2 * jnp.pi * jnp.arange(frame_len) / (frame_len - 1))
windowed = frames * hamming

# --- 步骤 4：FFT ---
n_fft = 512
spectra = jnp.fft.rfft(windowed, n=n_fft)
power_spectra = jnp.abs(spectra)  2 / n_fft

# --- 步骤 5：Mel 滤波器组 ---
n_mels = 40
f_min, f_max = 0.0, fs / 2.0

def hz_to_mel(f):
    return 2595 * jnp.log10(1 + f / 700)

def mel_to_hz(m):
    return 700 * (10  (m / 2595) - 1)

mel_min = hz_to_mel(f_min)
mel_max = hz_to_mel(f_max)
mel_points = jnp.linspace(mel_min, mel_max, n_mels + 2)
hz_points = mel_to_hz(mel_points)

freq_bins = jnp.floor((n_fft + 1) * hz_points / fs).astype(jnp.int32)
n_freqs = n_fft // 2 + 1
filterbank = jnp.zeros((n_mels, n_freqs))

for m in range(n_mels):
    f_left = freq_bins[m]
    f_center = freq_bins[m + 1]
    f_right = freq_bins[m + 2]
    # 上升坡
    for k in range(int(f_left), int(f_center)):
        if f_center != f_left:
            filterbank = filterbank.at[m, k].set((k - f_left) / (f_center - f_left))
    # 下降坡
    for k in range(int(f_center), int(f_right)):
        if f_right != f_center:
            filterbank = filterbank.at[m, k].set((f_right - k) / (f_right - f_center))

# 应用滤波器组
mel_spectra = jnp.dot(power_spectra, filterbank.T)

# --- 步骤 6：对数 ---
log_mel = jnp.log(mel_spectra + 1e-10)

# --- 步骤 7：DCT (type-II) ---
n_mfcc = 13
n_mel_channels = log_mel.shape[1]
dct_matrix = jnp.zeros((n_mfcc, n_mel_channels))
for i in range(n_mfcc):
    for j in range(n_mel_channels):
        dct_matrix = dct_matrix.at[i, j].set(
            jnp.cos(jnp.pi * i * (j + 0.5) / n_mel_channels)
        )
mfccs = jnp.dot(log_mel, dct_matrix.T)

# --- 可视化 ---
fig, axes = plt.subplots(3, 1, figsize=(14, 11))

# Mel 滤波器组
freq_axis = jnp.linspace(0, fs / 2, n_freqs)
for m in range(n_mels):
    color = '#3498db' if m % 2 == 0 else '#e74c3c'
    axes[0].plot(freq_axis, filterbank[m], color=color, alpha=0.6, linewidth=0.8)
axes[0].set_title(f'Mel 滤波器组 ({n_mels} 个滤波器)')
axes[0].set_xlabel('频率 (Hz)'); axes[0].set_ylabel('权重')
axes[0].grid(True, alpha=0.3)

# 对数 mel 语谱图
im1 = axes[1].imshow(log_mel.T, aspect='auto', origin='lower',
                      extent=[0, duration, 0, n_mels], cmap='viridis')
axes[1].set_title('对数 Mel 语谱图')
axes[1].set_xlabel('时间 (s)'); axes[1].set_ylabel('Mel 频带')
plt.colorbar(im1, ax=axes[1], label='对数能量')

# MFCCs
im2 = axes[2].imshow(mfccs.T, aspect='auto', origin='lower',
                      extent=[0, duration, 0, n_mfcc], cmap='coolwarm')
axes[2].set_title(f'MFCCs (前 {n_mfcc} 个系数)')
axes[2].set_xlabel('时间 (s)'); axes[2].set_ylabel('MFCC 索引')
plt.colorbar(im2, ax=axes[2], label='系数值')

plt.tight_layout(); plt.show()
```

4. 实现 FIR 低通和高通滤波器，并可视化它们对包含低频和高频分量的信号的影响。显示时域和频域视图。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 创建具有低频 (100 Hz) 和高频 (2000 Hz) 分量的信号
fs = 8000
duration = 0.05  # 50 ms 用于清晰可视化
t = jnp.arange(0, duration, 1.0 / fs)

x_low = jnp.sin(2 * jnp.pi * 100 * t)
x_high = 0.5 * jnp.sin(2 * jnp.pi * 2000 * t)
x = x_low + x_high

# 使用加窗 sinc 设计简单的 FIR 低通滤波器
def fir_lowpass(cutoff_hz, fs, n_taps=51):
    """使用加窗 sinc 方法设计 FIR 低通滤波器。"""
    fc = cutoff_hz / fs  # 归一化截止
    n = jnp.arange(n_taps)
    mid = (n_taps - 1) / 2.0
    # Sinc 函数（理想低通脉冲响应）
    h = jnp.where(n == mid, 2 * fc,
                  jnp.sin(2 * jnp.pi * fc * (n - mid)) / (jnp.pi * (n - mid)))
    # 应用 Hamming 窗
    window = 0.54 - 0.46 * jnp.cos(2 * jnp.pi * n / (n_taps - 1))
    h = h * window
    h = h / jnp.sum(h)  # 归一化为 DC 单位增益
    return h

def apply_filter(x, h):
    """通过卷积应用 FIR 滤波器。"""
    return jnp.convolve(x, h, mode='same')

# 500 Hz 低通滤波器（通过 100 Hz，阻止 2000 Hz）
h_lp = fir_lowpass(500, fs, n_taps=51)
x_lp = apply_filter(x, h_lp)

# 高通 = delta - 低通（谱反转）
delta = jnp.zeros(51)
delta = delta.at[25].set(1.0)
h_hp = delta - h_lp
x_hp = apply_filter(x, h_hp)

# 计算所有信号的频谱
def compute_spectrum(signal, fs):
    X = jnp.fft.rfft(signal)
    freqs = jnp.fft.rfftfreq(len(signal), d=1.0 / fs)
    mag = jnp.abs(X) / len(signal) * 2
    return freqs, mag

fig, axes = plt.subplots(3, 2, figsize=(14, 10))

# 时域图
for i, (sig, title, color) in enumerate([
    (x, '原始 (100 Hz + 2000 Hz)', '#3498db'),
    (x_lp, '低通滤波 (< 500 Hz)', '#27ae60'),
    (x_hp, '高通滤波 (> 500 Hz)', '#e74c3c')
]):
    axes[i, 0].plot(t * 1000, sig[:len(t)], color=color, linewidth=1)
    axes[i, 0].set_title(f'时域：{title}')
    axes[i, 0].set_xlabel('时间 (ms)'); axes[i, 0].set_ylabel('振幅')
    axes[i, 0].grid(True, alpha=0.3)

# 频域图
for i, (sig, title, color) in enumerate([
    (x, '原始', '#3498db'),
    (x_lp, '低通', '#27ae60'),
    (x_hp, '高通', '#e74c3c')
]):
    freqs, mag = compute_spectrum(sig, fs)
    axes[i, 1].plot(freqs, mag, color=color, linewidth=1.5)
    axes[i, 1].set_title(f'频谱：{title}')
    axes[i, 1].set_xlabel('频率 (Hz)'); axes[i, 1].set_ylabel('幅值')
    axes[i, 1].set_xlim(0, 3000)
    axes[i, 1].axvline(x=500, color='#f39c12', linestyle='--', alpha=0.7,
                        label='截止 (500 Hz)')
    axes[i, 1].legend(); axes[i, 1].grid(True, alpha=0.3)

plt.tight_layout(); plt.show()
```
