# 计数

*计数是计算概率的前提，在分配可能性之前必须知道有多少结果存在。本章涵盖乘法和加法法则、阶乘、排列、组合和二项式系数，这些组合工具是 ML 中采样、哈希和概率分析的基础。*

- 在计算概率之前，我们需要计数结果。如果想知道抽到扑克赢牌的概率，首先需要知道有多少可能的手牌以及其中有多少是赢牌。计数使概率精确化。

- 最简单的计数法则是**乘法法则**。如果一个决策有 $m$ 个选项，第二个独立决策有 $n$ 个选项，则组合结果的总数是 $m \times n$。

- 想象早上穿衣服。你有 3 件衬衫和 4 条裤子。每件衬衫可以与每条裤子搭配，给你 $3 \times 4 = 12$ 种可能的服装。

![树状图显示 3 件衬衫乘以 4 条裤子等于 12 种服装](../images/counting_outfits.svg)

- 乘法法则可以扩展到任意数量的选择。如果还有 2 双鞋，总服装数变为 $3 \times 4 \times 2 = 24$。每个新的独立选择都会相乘。

- **加法法则**处理"或"场景。如果事件 A 可以以 $m$ 种方式发生，事件 B 可以以 $n$ 种方式发生，并且它们不能同时发生（互斥），则总方式数是 $m + n$。

- 假设可以开车（3 条路线）或坐火车（2 条路线）从城市 X 到城市 Y。不能同时乘坐两者，所以总选项是 $3 + 2 = 5$。

- 当事件重叠时，需要减去重复计数的结果。如果 $A$ 和 $B$ 可以同时发生，则计数是 $|A \cup B| = |A| + |B| - |A \cap B|$。这是容斥原理，它将在讨论概率加法法则时重新出现。

- 非负整数 $n$ 的**阶乘**是所有正整数直到 $n$ 的乘积：

$$n! = n \times (n-1) \times (n-2) \times \cdots \times 2 \times 1$$

- 将阶乘视为回答：有多少种方法将 $n$ 个不同的物体排成一行？书架上的三本书可以有 $3! = 3 \times 2 \times 1 = 6$ 种排列方式。按照惯例，$0! = 1$。

- 阶乘增长极快。$10! = 3{,}628{,}800$，$20!$ 已经超过 $2.4 \times 10^{18}$。这种爆炸性增长解释了为什么暴力搜索在组合问题中变得不切实际。

- **排列**是物体的有序排列。当从 $n$ 个不同物体中挑选 $r$ 个且顺序重要时，排列数是：

$$P(n, r) = \frac{n!}{(n - r)!}$$

- 想象从 10 人的俱乐部中挑选主席、副主席和财务。第一个角色有 10 个候选人，第二个剩下 9 个，第三个剩下 8 个。这给出 $P(10, 3) = 10 \times 9 \times 8 = 720$。公式确认了这一点：$\frac{10!}{7!} = 720$。

- **组合**是无序选择。当从 $n$ 中挑选 $r$ 个且顺序不重要时，我们除掉冗余的排序：

$$C(n, r) = \binom{n}{r} = \frac{n!}{r!(n - r)!}$$

- 符号 $\binom{n}{r}$ 读作"n 选 r"。关键洞察是每个组合对应 $r!$ 个排列（选中的 $r$ 个物体可以以 $r!$ 种方式重新排列），所以我们将排列计数除以 $r!$。

![并排比较：排列计数所有排序，组合合并相同的组](../images/permutation_vs_combination.svg)

- 示例：从 10 人群体中，有多少种方法组成 3 人委员会？顺序不重要（没有主席或副主席，只有成员），所以我们使用组合：

$$\binom{10}{3} = \frac{10!}{3! \cdot 7!} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120$$

- 同样的 10 人产生 720 个排列但只有 120 个组合，因为每组 3 人有 $3! = 6$ 个内部排序。

- 组合是概率的核心。二项式系数 $\binom{n}{r}$ 计数在 $n$ 次试验中获得恰好 $r$ 次成功的方式数，这是二项分布的核心（在文件 03 中讨论）。

- 让我们解决一个结合多个计数想法的经典委员会问题。

- **问题**：一个俱乐部有 8 名男性和 6 名女性。有多少种方法组成包含恰好 3 名男性和 2 名女性的 5 人委员会？

- **步骤 1**：从 8 人中挑选 3 名男性。

$$\binom{8}{3} = \frac{8!}{3! \cdot 5!} = \frac{8 \times 7 \times 6}{3 \times 2 \times 1} = 56$$

- **步骤 2**：从 6 人中挑选 2 名女性。

$$\binom{6}{2} = \frac{6!}{2! \cdot 4!} = \frac{6 \times 5}{2 \times 1} = 15$$

- **步骤 3**：应用乘法法则。每种男性选择可以与每种女性选择配对：

$$56 \times 15 = 840 \text{ 个委员会}$$

- 这种模式，将复杂计数问题分解为独立子选择并相乘，是组合学中的标准方法。

- 还有**可重复排列**。当物体可以重复时，从 $n$ 种类型中选择 $r$ 个物体给出 $n^r$ 个结果。使用数字 0-9 的 4 位 PIN 码有 $10^4 = 10{,}000$ 种可能。每个位置有 10 个选项，乘法法则处理其余部分。

- **可重复组合**（也称为"星条"）计数当允许重复且顺序不重要时，从 $n$ 种类型中选择 $r$ 个物体的方式数：

$$\binom{n + r - 1}{r} = \frac{(n + r - 1)!}{r!(n - 1)!}$$

- 示例：从 4 种冰淇淋口味中选择 3 勺（允许重复）给出 $\binom{4 + 3 - 1}{3} = \binom{6}{3} = 20$ 个选项。

- 总结计数工具包：

| 场景 | 公式 |
|---|---|
| 有序，无重复（排列） | $P(n,r) = \frac{n!}{(n-r)!}$ |
| 无序，无重复（组合） | $\binom{n}{r} = \frac{n!}{r!(n-r)!}$ |
| 有序，可重复 | $n^r$ |
| 无序，可重复 | $\binom{n+r-1}{r}$ |

- 涉及等可能结果的每个概率计算都使用公式 $P(\text{事件}) = \frac{\text{有利结果}}{\text{总结果}}$。计数给我们两个数字。有了这个基础，我们准备在下一个文件中正式化概率本身。

1. 使用阶乘公式和直接计算计算 $P(10, 3)$ 和 $\binom{10}{3}$。验证排列计数总是组合计数的 $r!$ 倍。
```python
import jax.numpy as jnp
from math import factorial

n, r = 10, 3

perm = factorial(n) // factorial(n - r)
comb = factorial(n) // (factorial(r) * factorial(n - r))

print(f"P({n},{r}) = {perm}")
print(f"C({n},{r}) = {comb}")
print(f"P / C = {perm // comb} (应该等于 {r}! = {factorial(r)})")
```

2. 编程解决委员会问题（从 8 人中选 3 名男性，从 6 人中选 2 名女性），并通过枚举所有有效委员会验证。
```python
from itertools import combinations
from math import factorial

def comb_count(n, r):
    return factorial(n) // (factorial(r) * factorial(n - r))

# 公式方法
men_ways = comb_count(8, 3)
women_ways = comb_count(6, 2)
print(f"公式：{men_ways} × {women_ways} = {men_ways * women_ways}")

# 枚举方法
men = [f"M{i}" for i in range(1, 9)]
women = [f"W{i}" for i in range(1, 7)]
count = sum(1 for _ in combinations(men, 3) for _ in combinations(women, 2))
print(f"枚举：{count}")
```

3. 计算可以用 26 个小写字母制作多少种 4 字符密码（允许重复）。然后计算有多少种不含重复字母。
```python
from math import factorial

n = 26
r = 4

with_rep = n ** r
without_rep = factorial(n) // factorial(n - r)

print(f"允许重复：   {with_rep:>10,}")
print(f"不允许重复： {without_rep:>10,}")
print(f"有重复的比例：{1 - without_rep/with_rep:.2%}")
```

4. 模拟生日问题：在 $k$ 人群体中，至少两人共享生日的概率是多少？绘制 $k = 1$ 到 $60$ 的概率并找到超过 50% 的位置。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def birthday_prob_exact(k):
    """k 人群体中至少一个共享生日的概率。"""
    p_no_match = 1.0
    for i in range(k):
        p_no_match *= (365 - i) / 365
    return 1 - p_no_match

ks = list(range(1, 61))
probs = [birthday_prob_exact(k) for k in ks]

plt.figure(figsize=(8, 4))
plt.plot(ks, probs, color="#3498db", linewidth=2)
plt.axhline(y=0.5, color="#e74c3c", linestyle="--", alpha=0.7, label="50%")
cross = next(k for k, p in zip(ks, probs) if p >= 0.5)
plt.axvline(x=cross, color="#e74c3c", linestyle="--", alpha=0.7)
plt.xlabel("群体大小 (k)")
plt.ylabel("P(至少一个共享生日)")
plt.title(f"生日问题（在 k={cross} 处超过 50%）")
plt.legend()
plt.grid(alpha=0.3)
plt.show()
```
