## 📝 形态学 - 单词的内部结构

形态学研究单词如何由更小的有意义单位构成：

### 语素（Morpheme）

语素是最小的有意义单位：

```
单词 "unhappiness" 分解：

un-    +   happy   +   -ness
前缀       词根       后缀
"不"      "快乐"      "名词化"

三个语素组合成一个单词！
```

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 语素类型

| 类型 | 说明 | 例子 |
|------|------|------|
| 词根 | 核心意义 | happy, run, compute |
| 前缀 | 词根前 | un-, re-, pre- |
| 后缀 | 词根后 | -ing, -ed, -tion |
| 中缀 | 插入词根内 | （某些语言有） |

### 形态学过程

屈折变化：改变语法属性，不改变核心意义

| 原词 | 屈折形式 | 变化 |
|------|---------|------|
| run | runs/running/ran | 时态/人称 |
| cat | cats | 复数 |

派生：创造新单词，常改变词性

| 原词 | 派生形式 | 变化 |
|------|---------|------|
| happy (adj) | happiness (n) | 形容词→名词 |
| compute (v) | computation (n) | 动词→名词 |

</div>

```
语素树可视化：

      unhappiness
         / | \
       /   |   \
     un-  happy  -ness
    前缀  词根   后缀
   "不"  "快乐" "性质"

每个语素都贡献意义！
```

### 世界语言的形态复杂性

| 语言类型 | 特点 | 例子 |
|---------|------|------|
| 分析语 | 每词语素少，依赖词序 | 英语、中文 |
| 黏着语 | 词语可串很多语素 | 土耳其语、芬兰语 |
| 屈折语 | 语素融合，难分边界 | 拉丁语、俄语 |
| 模板语 | 辅音词根 + 元音模式 | 阿拉伯语、希伯来语 |

```
阿拉伯语模板例子：

词根 k-t-b（写）：
kitab = 书
kataba = 他写了
maktub = 已写的
katib = 作家

相同词根，不同元音模式！
```

## 🌳 句法学 - 单词如何组成句子

句法学研究单词如何组合成短语和句子：

```
合乎语法： "The cat sat on the mat" ✓
不合语法： "Mat the on sat cat the" ✗

每种语言都有词序规则！
```

### 两大句法框架

| 框架 | 核心思想 | 表示 |
|------|---------|------|
| 短语结构语法 | 短语嵌套成句子 | Constituency 树 |
| 依存语法 | 单词间直接关系 | Dependency 树 |

---

### 短语结构语法（Constituency）

短语结构认为句子由嵌套短语构成：

```
句子 (S) → 名词短语 (NP) + 动词短语 (VP)

NP → 限定词 (Det) + 名词 (N)
VP → 动词 (V) + 介词短语 (PP)
PP → 介词 (P) + 名词短语 (NP)
```

<div class="lesson-two-column">
  <div class="lesson-text-column">

### Constituency 树

```
        S
       / \
      NP  VP
     / \  / \
   Det N V  PP
    │  │ │  / \
   the cat sat P  NP
              / \
            Det  N
             │   │
            on the mat

每个内部节点是短语类型
每个叶子是单词
```

### 上下文无关文法（CFG）

```
S  → NP VP
NP → Det N
NP → Det N PP
VP → V NP
VP → V PP
PP → P NP
Det → "the" | "a"
N  → "cat" | "mat"
V  → "sat" | "chased"
P  → "on" | "under"
```

从 S 开始应用规则，生成所有合法句子！

</div>
  <div class="lesson-visual-column">

```
句法歧义例子：

"I saw the man with the telescope"

解析 1：我用望远镜看那个人
        saw
       / | \
     I  the man
           \
        with the telescope (修饰 saw)

解析 2：我看到一个拿着望远镜的人
        saw
       / | \
     I  the man
            \
         with the telescope (修饰 man)

同一句子，两种结构！
```

### 依存语法（Dependency）

依存语法描述单词间的直接关系：

```
"The cat sat on the mat" 的依存树：

        sat (根)
       /  \
   nsubj  prep
     │      │
    cat     on
     │       \
    det      pobj
     │        │
    the      mat
              │
             det
              │
             the

每个词依赖一个中心词
箭头标记语法关系
```

### 配价（Valency）

配价描述动词需要多少个论元：

| 配价 | 论元数 | 例子 |
|------|--------|------|
| 不及物 | 1 | sleep（某人睡觉） |
| 及物 | 2 | eat（某人吃某物） |
| 双及物 | 3 | give（某人给某物给某人） |

```
配价决定合法句子：

✓ "I sleep"
✗ "I sleep something"（sleep 不能有宾语）

✓ "I eat an apple"
✗ "I eat"（不完整，虽然口语可用）

✓ "I give you a book"
✗ "I give"（不完整）
```

## 📖 语义学 - 意义是什么

语义学研究单词和句子的意义：

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 词汇语义学

单词以系统方式关联：

| 关系 | 说明 | 例子 |
|------|------|------|
| 同义关系 | 意义相近 | big/large |
| 反义关系 | 意义相反 | hot/cold |
| 上下义关系 | is-a 关系 | dog 是 animal 的一种 |
| 部分整体关系 | part-of | wheel 是 car 的部分 |
| 多义性 | 一词多义 | bank（银行/河岸） |

### 词义消歧（WSD）

```
"I deposited money at the bank" → 金融机构
"We sat by the river bank" → 河岸

上下文决定哪个意义！

现代方法（BERT 等）：
为同一词的不同用法产生不同向量
自动解决歧义
```

```
语义关系网络：

         animal (上位词)
           │
         dog (下位词)
        / | \
     同义 反义 部分
      │   │   │
   puppy cat wheel
   
单词不是孤立的，
而是在语义网络中！
```

### 组合语义学

组合性原则（Frege）：

> 复杂表达式的意义 = 部分意义 + 组合规则

```
"The cat chased the dog" ≠ "The dog chased the cat"

单词相同，顺序不同 → 意义完全不同

句法结构决定：
- 谁是主语（chaser）
- 谁是宾语（chased）
```

习语挑战：

```
"kick the bucket" ≠ 踢桶
而是 = 死亡

习语意义无法从部分推导！
这是组合方法的挑战
```

---

### 分布语义学

分布假设（Firth, 1957）：

> "你应通过它所处的环境来认识一个单词"

```
出现在相似上下文的单词 → 意义相似

例子：
"cat" 常与 "pet", "fur", "meow" 共现
"dog" 常与 "pet", "fur", "bark" 共现

所以 cat 和 dog 语义相似！

这是 Word2Vec、GloVe 等词嵌入的基础！
```

## 💬 语用学 - 上下文如何影响意义

语用学研究上下文如何塑造解释：

```
"Can you pass the salt?"

句法上：是/否 能力问题
语用上：礼貌的请求

你不会回答 "Yes, I can" 然后坐着不动！
```

### 言语行为理论（Austin, Searle）

| 行为类型 | 说明 | 例子 |
|---------|------|------|
| 言内行为 | 说出话语本身 | "门开着" |
| 言外行为 | 说话时的意图 | 请求关门 |
| 言后行为 | 话语的效果 | 听话人去关门 |

---

### 语用学现象

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 指示词

依赖上下文才能理解的词：

| 指示词 | 依赖 |
|--------|------|
| 我、你、他 | 说话人/听话人 |
| 这里、那里 | 地点 |
| 现在、那时 | 时间 |
| 这个、那个 | 距离/显著性 |

"I am here now" 意义完全取决于谁说、在哪、何时！

### 预设

说话人假设听话人已知的信息：

```
"John stopped smoking"
预设：John 之前抽烟

预设触发语：
- 停止/开始
- 再次
- 知道/后悔
```

</div>
  <div class="lesson-visual-column">

### 会话含义

```
Grice 合作原则：

1. 量的准则：信息充分但不冗余
2. 质的准则：说真话
3. 关系准则：相关
4. 方式准则：清晰

A: "你觉得我的新发型怎么样？"
B: "嗯...很有...勇气"

B 违反了量的准则 → 含义：不好看！
```
