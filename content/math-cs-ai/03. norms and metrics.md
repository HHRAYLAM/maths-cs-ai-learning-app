# 🐱 第 3 课：范数与度量 - Tom 如何测量距离？

> **本课故事**：Tom 猫想要知道它离 Jerry 有多远。但是"距离"有不同的测量方式！是直线距离最短？还是必须沿着走廊走？或者是看方向有多相似？这就是范数和度量的奥秘！

## 📏 范数 - 向量有多大？

**范数（Norm）** 是测量向量"大小"的函数。就像用不同的尺子量东西。

<div class="lesson-two-column">
  <div class="lesson-text-column">

### L2 范数 - 欧几里得距离

这是我们最熟悉的"直线距离"：

$$\|\vec{v}\|_2 = \sqrt{v_x^2 + v_y^2}$$

**例子**：
- $\vec{v} = (3, 4)$
- $\|\vec{v}\|_2 = \sqrt{3^2 + 4^2} = \sqrt{25} = \mathbf{5}$

这就是 Tom 穿过房间的**直线距离**！

</div>
  <div class="lesson-visual-column">

```
    ↑
  4 |     Jerry
    |     🐭
    |    /
    |   /  L2 = 5
    |  /  (直线)
    | /
  0 |/_______→
    0     3
   Tom

直线距离 = 5 步
```

<div class="visual-caption">图 1：L2 范数 = 直线距离</div>

</div>

## 🏙️ L1 范数 - 曼哈顿距离

想象 Tom 不能穿墙，只能沿着走廊和房间边缘走。这就是**曼哈顿距离**（城市街区距离）：

$$\|\vec{v}\|_1 = |v_x| + |v_y|$$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的逃跑路线</div>
  <div class="story-card-content">
    Jerry 不能飞，只能沿着墙跑：
    <br><br>
    <strong>第一步</strong>：向右跑 3 步
    <br>
    <strong>第二步</strong>：向上跑 4 步
    <br>
    <strong>总距离</strong>：3 + 4 = <strong>7 步</strong>
    <br><br>
    这比直线距离（5 步）要远！
  </div>
</div>

### L1 vs L2 对比

<div class="lesson-two-column">
  <div class="lesson-text-column">

**L1 范数（曼哈顿）**：
- $\|\vec{v}\|_1 = |3| + |4| = 7$
- 沿着轴走
- 不能穿墙

**L2 范数（欧几里得）**：
- $\|\vec{v}\|_2 = \sqrt{3^2 + 4^2} = 5$
- 直线穿越
- 可以穿墙

**关键**：L1 ≥ L2（绕路永远不比直线短）

</div>
  <div class="lesson-visual-column">

```
L1 路径（曼哈顿）：

    🐭 Jerry
    ↑
    | 向上 4 步
    |
    |
    +——→ 向右 3 步
   🐱 Tom

总距离 = 3 + 4 = 7 步
```

## 🔝 L∞ 范数 - 最大范数

**最大范数**只看**最大的那个分量**，忽略其他的：

$$\|\vec{v}\|_\infty = \max(|v_x|, |v_y|)$$

<div class="key-point">
  <div class="key-point-title">🐾 L∞ 范数的意义</div>
  <ul class="key-point-list">
    <li><strong>只看最远的方向</strong>：不管其他方向走了多少</li>
    <li><strong>例子</strong>：$\vec{v} = (3, 4)$，L∞ = max(3, 4) = 4</li>
    <li><strong>应用</strong>：当你只关心"最坏情况"时很有用</li>
  </ul>
</div>

### 三种范数对比

| 范数 | 公式 | $\vec{v}=(3,4)$ 的结果 | 比喻 |
|------|------|---------------------|------|
| **L1** | $|v_x| + |v_y|$ | 7 | 沿着走廊走 |
| **L2** | $\sqrt{v_x^2 + v_y^2}$ | 5 | 直线穿越 |
| **L∞** | $\max(|v_x|, |v_y|)$ | 4 | 只看最远的 |

---

## 📐 一般 Lp 范数 - 万能公式

所有范数都是一个通用公式的特例：

$$\|\vec{v}\|_p = (|v_x|^p + |v_y|^p)^{1/p}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 不同的 p 值

- **p = 1**：L1 范数（曼哈顿）
- **p = 2**：L2 范数（欧几里得）
- **p → ∞**：L∞ 范数（最大）

**试试改变 p**：
- p 越大，最大分量越主导
- p 越小，所有分量平等对待

</div>
  <div class="lesson-visual-column">

```
p 值变化的效果：

p=1:  |3| + |4| = 7
       (平等对待)

p=2:  √(3² + 4²) = 5
       (平方后开方)

p=10: (3¹⁰ + 4¹⁰)^(1/10) ≈ 4.004
       (4 主导结果)

p→∞:  max(3, 4) = 4
       (只看最大)
```

<div class="visual-caption">图 3：p 越大，最大分量越重要</div>

</div>
</div>

## ✨ 范数的三条规则

要让一个函数成为合法的**范数**，必须满足三条规则：

<div class="key-point">
  <div class="key-point-title">📐 范数公理</div>
  <ul class="key-point-list">
    <li><strong>非负性</strong>：$\|\vec{v}\| \geq 0$，且只有零向量大小为 0</li>
    <li><strong>缩放性</strong>：$\|c \cdot \vec{v}\| = |c| \cdot \|\vec{v}\|$（放大 2 倍，大小也放大 2 倍）</li>
    <li><strong>三角不等式</strong>：$\|\vec{u} + \vec{v}\| \leq \|\vec{u}\| + \|\vec{v}\|$（捷径不比绕路长）</li>
  </ul>
</div>

<div class="story-card">
  <div class="story-card-title">🐱 Tom 理解三角不等式</div>
  <div class="story-card-content">
    Tom 想从 A 到 C：
    <br><br>
    <strong>直接路线</strong>：A → C（距离 = 5）
    <br>
    <strong>绕路</strong>：A → B → C（距离 = 3 + 4 = 7）
    <br><br>
    <strong>结论</strong>：直接走永远更近或相等！
    <br>
    这就是三角不等式：$\|A \to C\| \leq \|A \to B\| + \|B \to C\|$
  </div>
</div>

## 📍 度量 - 两个点相距多远？

**度量（Metric）** 测量**两个向量之间**的距离。

### 从范数得到度量

最简单的方法：测量两个向量的**差**的范数：

$$d(\vec{u}, \vec{v}) = \|\vec{u} - \vec{v}\|$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子

Tom 在 $\vec{u} = (1, 2)$，Jerry 在 $\vec{v} = (4, 6)$

**差向量**：
$$\vec{v} - \vec{u} = (4-1, 6-2) = (3, 4)$$

**欧几里得距离**：
$$d(\vec{u}, \vec{v}) = \|(3, 4)\|_2 = \sqrt{3^2 + 4^2} = 5$$

Tom 离 Jerry 有 5 步远！

</div>
  <div class="lesson-visual-column">

```
    ↑
  6 |     🐭 Jerry (4,6)
  5 |    /
  4 |   /
  3 |  /
  2 | 🐱 Tom (1,2)
  1 |
  0 +——————→
    0 1 2 3 4

距离 = √((4-1)² + (6-2)²) = 5
```

<div class="visual-caption">图 4：度量 = 两点间距离</div>

</div>
</div>

## 📊 常见度量 - 不同的距离定义

### 1. 欧几里得距离（L2 度量）

$$d(\vec{u}, \vec{v}) = \sqrt{(u_x - v_x)^2 + (u_y - v_y)^2}$$

- **特点**：直线距离
- **应用**：最常见的距离定义
- **对大差异敏感**：一个很大的差异会主导结果

### 2. 曼哈顿距离（L1 度量）

$$d(\vec{u}, \vec{v}) = |u_x - v_x| + |u_y - v_y|$$

- **特点**：沿轴走的距离
- **应用**：网格布局、城市街道
- **对所有差异平等对待**

### 3. 切比雪夫距离（L∞ 度量）

$$d(\vec{u}, \vec{v}) = \max(|u_x - v_x|, |u_y - v_y|)$$

- **特点**：只看最大的差异
- **应用**：国际象棋中国王的移动
- **最坏情况距离**

## 📐 度量的四条规则

要让一个函数成为合法的**度量**，必须满足四条规则：

<div class="key-point">
  <div class="key-point-title">📐 度量公理</div>
  <ul class="key-point-list">
    <li><strong>非负性</strong>：$d(\vec{u}, \vec{v}) \geq 0$（距离不能是负数）</li>
    <li><strong>同一性</strong>：$d(\vec{u}, \vec{v}) = 0$ 当且仅当 $\vec{u} = \vec{v}$（零距离 = 同一点）</li>
    <li><strong>对称性</strong>：$d(\vec{u}, \vec{v}) = d(\vec{v}, \vec{u})$（A 到 B = B 到 A）</li>
    <li><strong>三角不等式</strong>：$d(\vec{u}, \vec{w}) \leq d(\vec{u}, \vec{v}) + d(\vec{v}, \vec{w})$（直接走更近）</li>
  </ul>
</div>

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 理解对称性</div>
  <div class="story-card-content">
    <strong>Tom 到 Jerry 的距离</strong>：5 步
    <br>
    <strong>Jerry 到 Tom 的距离</strong>：也是 5 步！
    <br><br>
    这就是对称性：距离与方向无关。
  </div>
</div>

    <summary>点击查看答案</summary>

  <details>
    <summary>点击查看答案</summary>
    <p><strong>答案</strong>：$|4-1| + |5-1| = 3 + 4 = \mathbf{7}$</p>

  **问题 4**：以下哪个范数对大分量最敏感？

  <details>
    <summary>点击查看答案</summary>
    <p><strong>答案</strong>：L∞ 范数</p>

  </details>

  **问题 5**：三角不等式说明什么？

## 📝 本课总结

<div class="key-point">
  <div class="key-point-title">🎓 知识回顾</div>
  <ul class="key-point-list">
    <li><strong>范数</strong> = 测量单个向量的大小</li>
    <li><strong>度量</strong> = 测量两个向量之间的距离</li>
    <li><strong>L1 范数</strong> = 曼哈顿距离（沿轴走）</li>
    <li><strong>L2 范数</strong> = 欧几里得距离（直线）</li>
    <li><strong>L∞ 范数</strong> = 最大范数（只看最远）</li>
    <li><strong>Lp 范数</strong> = 通用公式，不同 p 值给出不同范数</li>
    <li><strong>范数三规则</strong>：非负、缩放、三角不等式</li>
    <li><strong>度量四规则</strong>：非负、同一、对称、三角不等式</li>
  </ul>
</div>
