# 范数与度量

*范数测量向量的大小；度量测量两个向量之间的距离。本章涵盖 L1、L2 和 L 无穷范数、欧几里得距离和余弦距离，以及为什么在机器学习的 kNN、聚类和检索中选择正确的距离函数至关重要。*

- 我们知道向量有大小和方向。但我们如何实际测量单个向量"有多大"，或两个向量"相距多远"？这就是**范数**和**度量**发挥作用的地方。

- 对于标量，我们知道 10 > 5，因为它们的值量化了它们，但我们如何量化一个向量？这就是**范数**，它测量单个向量的大小。

- 最熟悉的范数是**欧几里得范数**（L2），就是我们已知的大小公式：

$$\|\mathbf{v}\|_2 = \sqrt{v_1^2 + v_2^2 + \cdots + v_n^2}$$

- 但还有其他测量大小的方法。想象你在一个有网格街道的城市中。你不能穿过建筑物走对角线，所以你的旅程"长度"是沿着每条街道走过的街区总数。这就是**曼哈顿范数**（L1）：

$$\|\mathbf{v}\|_1 = |v_1| + |v_2| + \cdots + |v_n|$$

- 或者你可能只关心单个最大的分量，忽略其余的。这就是**最大范数**（L 无穷）：

$$\|\mathbf{v}\|_\infty = \max(|v_1|, |v_2|, \ldots, |v_n|)$$

- 所有这三种都是**一般 Lp 范数**的特殊情况：

$$\|\mathbf{v}\|_p = (|v_1|^p + |v_2|^p + \cdots + |v_n|^p)^{1/p}$$

- 设置 $p = 2$ 得到欧几里得范数，$p = 1$ 得到曼哈顿范数，当 $p \to \infty$ 时得到最大范数。随着 $p$ 增长，最大的分量贡献越来越多，直到最终只有它起作用。

- 每个范数必须遵守三条规则：

    - **非负性**：$\|\mathbf{v}\| \geq 0$，并且仅当 $\mathbf{v} = \mathbf{0}$ 时 $\|\mathbf{v}\| = 0$。大小永远不会是负数，只有零向量的大小为零。

    - **缩放**：$\|c\mathbf{v}\| = |c| \cdot \|\mathbf{v}\|$。将向量加倍会使其大小加倍。

    - **三角不等式**：$\|\mathbf{u} + \mathbf{v}\| \leq \|\mathbf{u}\| + \|\mathbf{v}\|$。捷径永远不会比绕远路更长。

- 现在，**度量**测量两个向量*之间*的距离。可以这样想："这两个点相距多远？"

- 获得度量最简单的方法是对差值使用范数：$d(\mathbf{u}, \mathbf{v}) = \|\mathbf{u} - \mathbf{v}\|$。减去两个向量，然后测量剩余部分的大小。

- 使用欧几里得范数，这给了我们熟悉的**欧几里得距离**：

$$d(\mathbf{u}, \mathbf{v}) = \sqrt{(u_1 - v_1)^2 + (u_2 - v_2)^2 + \cdots + (u_n - v_n)^2}$$

- 使用曼哈顿范数给出**曼哈顿距离**，即沿每个轴的总差值，就像计算两个位置之间的城市街区。

- 每个度量必须遵守四条规则：

    - **非负性**：$d(\mathbf{u}, \mathbf{v}) \geq 0$。距离永远不会是负数。

    - **同一性**：$d(\mathbf{u}, \mathbf{v}) = 0$ 当且仅当 $\mathbf{u} = \mathbf{v}$。零距离意味着同一点。

    - **对称性**：$d(\mathbf{u}, \mathbf{v}) = d(\mathbf{v}, \mathbf{u})$。从 A 到 B 的距离与从 B 到 A 的距离相同。

    - **三角不等式**：$d(\mathbf{u}, \mathbf{w}) \leq d(\mathbf{u}, \mathbf{v}) + d(\mathbf{v}, \mathbf{w})$。直接走永远不会比绕道更远。

- 那么两者之间的关系是什么？范数测量一个向量，度量测量两个向量之间的差距。每个范数自然创建一个度量（通过测量差值），但并非每个度量都来自范数。

- 例如，**汉明距离**计算两个向量不同位置的数量。它是一个有效的度量，但不来自任何范数。

- 在机器学习中，选择正确的范数或度量很重要。

- L2 距离在求和前对每个差值进行平方，因此单个大差值主导结果。

- L1 距离对绝对差值求和，平等对待每个差值。与 L2 相比，单个大差值的影响较小。

## 编码任务（使用 CoLab 或 notebook）

1. 计算同一向量的 L1 和 L2 范数。尝试改变值并注意哪个范数对大分量最敏感 vs 许多小分量。然后尝试为递增的 p 值（例如 1、2、5、10、50、100）计算 Lp 范数，并观察它收敛到 L 无穷值。
```python
import jax.numpy as jnp

v = jnp.array([3.0, -4.0, 1.0])

l1 = jnp.sum(jnp.abs(v))
l2 = jnp.sqrt(jnp.sum(v ** 2))

print(f"L1: {l1}, L2: {l2:.2f}")
```

2. 计算两个向量之间的欧几里得距离和曼哈顿距离。尝试移动向量使其更近或更远，并观察每个距离如何不同地响应。
```python
import jax.numpy as jnp

u = jnp.array([1.0, 2.0, 3.0])
v = jnp.array([4.0, 0.0, 1.0])

euclidean = jnp.sqrt(jnp.sum((u - v) ** 2))
manhattan = jnp.sum(jnp.abs(u - v))

print(f"Euclidean: {euclidean:.2f}, Manhattan: {manhattan}")
```
