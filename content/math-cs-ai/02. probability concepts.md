# 概率概念

*概率论形式化了不确定性并提供了在其下进行推理的规则。本章涵盖样本空间、事件、概率公理、条件概率、独立性、贝叶斯定理以及频率学派与贝叶斯学派的解释，这是 ML 中每个生成式和判别式模型背后的数学框架。*

- 概率为事件分配一个介于 0 和 1 之间的数字，度量它发生的可能性有多大。

- 概率为 0 意味着不可能，为 1 意味着确定，为 0.5 意味着抛硬币。

- 有两种主要解释。**频率学派**观点认为概率是长期相对频率：抛一枚均匀硬币 10,000 次，正面大约出现 50% 的时间。

- **贝叶斯学派**观点认为概率是信念程度：你可能会说明天有 70% 的机率下雨，即使明天只发生一次。

- 两种解释使用相同的数学规则。区别是哲学性的，但它在 ML 中很重要。频率学派方法给你点估计。贝叶斯学派方法给你参数的完整分布。

- **样本空间** $S$ 是实验所有可能结果的集合。抛硬币：$S = \{H, T\}$。掷骰子：$S = \{1, 2, 3, 4, 5, 6\}$。

- **事件**是样本空间的任何子集。"掷出偶数"是事件 $A = \{2, 4, 6\}$，它是 $S$ 的子集。

- 当所有结果等可能时，事件的概率只是计数（来自文件 01）：

$$P(A) = \frac{|A|}{|S|} = \frac{\text{有利结果}}{\text{总结果}}$$

- 对于偶数示例：$P(\text{偶数}) = \frac{3}{6} = 0.5$。

![Venn 图显示样本空间 S 内的事件 A 和 B，带有交集和补集](../images/venn_diagram.svg)

- 事件 $A$ 的**补集**，写作 $A'$ 或 $A^c$，是 $S$ 中不属于 $A$ 的一切。由于每个结果要么在 $A$ 中要么不在：

$$P(A') = 1 - P(A)$$

- 补集通常是更简单的途径。与其计算 5 次抛硬币中至少一次正面的所有方式，不如计算没有正面的唯一方式然后减去：$P(\text{至少一次正面}) = 1 - P(\text{全部反面}) = 1 - (0.5)^5 = 0.969$。

- 两个事件是**互斥的**（不相交的），如果它们不能同时发生：$A \cap B = \emptyset$。在单个骰子上掷出 2 点和掷出 5 点是互斥的。

- **互斥事件的加法法则**很简单：

$$P(A \cup B) = P(A) + P(B) \quad \text{(如果 } A \cap B = \emptyset\text{)}$$

- 当事件可以重叠时，你需要**一般加法法则**来避免重复计算交集：

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

- 这反映了计数的容斥原理。上面的 Venn 图显示了原因：紫色区域（交集）在 $P(A)$ 中计数一次，在 $P(B)$ 中又计数一次，所以我们减去一次。

- **联合概率** $P(A \cap B)$ 是 $A$ 和 $B$ 都发生的概率。在一副牌中，$P(\text{红色} \cap \text{K}) = \frac{2}{52}$，因为有 2 张红色 K。

- **边缘概率**是单个事件的概率，不考虑其他事件。$P(\text{红色}) = \frac{26}{52} = 0.5$ 是边缘概率。如果你有两个变量的联合分布，通过对另一个变量求和（或积分）获得边缘概率。

- **条件概率**回答：给定 $B$ 已经发生，$A$ 的概率是多少？我们将样本空间从 $S$ 缩小到 $B$，并询问 $B$ 的多少分数也属于 $A$：

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

![条件概率作为将样本空间从 S 缩小到 B](../images/conditional_probability.svg)

- 示例：你抽一张牌，有人告诉你它是红色的。它是 K 的概率是多少？有 26 张红色牌，其中 2 张是 K，所以 $P(\text{K} | \text{红色}) = \frac{2}{26} = \frac{1}{13}$。使用公式：$P(\text{K} \cap \text{红色}) / P(\text{红色}) = \frac{2/52}{26/52} = \frac{1}{13}$。

- 两个事件是**独立的**，如果知道一个发生不能告诉你关于另一个的任何信息。形式上：

$$P(A \cap B) = P(A) \cdot P(B)$$

- 等价地，$P(A | B) = P(A)$。抛两枚不同的硬币是独立事件。无放回地抽两张牌不是独立的（第一次抽牌改变了剩余的内容）。

- 独立性是一个巨大的简化器。对于独立事件，联合概率分解为乘积，这使得计算可处理。许多 ML 模型假设特征之间独立（如朴素贝叶斯）正是由于这种简化。

- 任何两个事件的**乘法法则**重新排列条件概率公式：

$$P(A \cap B) = P(A | B) \cdot P(B) = P(B | A) \cdot P(A)$$

- 对于独立事件，这简化为 $P(A \cap B) = P(A) \cdot P(B)$，因为条件概率等于边缘概率。

- **贝叶斯定理**是概率中最重要的结果之一，也是贝叶斯 ML 的基础。它让你反转条件概率的方向：

$$P(A | B) = \frac{P(B | A) \cdot P(A)}{P(B)}$$

- 该定理直接来自用两种方式书写 $P(A \cap B)$：$P(B|A) \cdot P(A) = P(A|B) \cdot P(B)$，然后求解 $P(A|B)$。

![贝叶斯定理组件：后验、似然、先验和证据](../images/bayes_components.svg)

- 每个组件都有一个名称：
    - **先验** $P(A)$：看到证据之前的初始信念
    - **似然** $P(B|A)$：假设 $A$ 为真时证据有多可能
    - **证据** $P(B)$：看到证据的总概率，作为归一化因子
    - **后验** $P(A|B)$：看到证据后更新的信念

- 让我们解决经典的医疗诊断示例。假设一种疾病影响 1% 的人口。一种检测该疾病的测试有 95% 的准确率：它正确识别 95% 的患病者（灵敏度），正确识别 90% 的健康人（特异度）。

- 你检测呈阳性。你实际患病的概率是多少？

- 令 $D$ = 患病，$+$ = 检测呈阳性。
    - 先验：$P(D) = 0.01$
    - 似然：$P(+ | D) = 0.95$
    - 假阳性率：$P(+ | D') = 0.10$

- 我们需要 $P(+)$。根据全概率公式：

$$P(+) = P(+ | D) \cdot P(D) + P(+ | D') \cdot P(D')$$
$$= 0.95 \times 0.01 + 0.10 \times 0.99 = 0.0095 + 0.099 = 0.1085$$

- 现在应用贝叶斯定理：

$$P(D | +) = \frac{P(+ | D) \cdot P(D)}{P(+)} = \frac{0.95 \times 0.01}{0.1085} \approx 0.088$$

- 尽管测试"95% 准确"，阳性结果只给你约 8.8% 的机率患病。先验非常重要。由于疾病罕见，大多数阳性结果是假阳性。这是 ML 中任何分类问题的关键洞察：当类别不平衡时，仅凭准确率会产生误导。

- **全概率公式**将样本空间划分为互斥、穷举的事件 $B_1, B_2, \ldots, B_n$，并将任何事件 $A$ 表达为：

$$P(A) = \sum_{i=1}^{n} P(A | B_i) \cdot P(B_i)$$

- 这正是我们在医疗示例中计算 $P(+)$ 时使用的：我们将人口分为"患病"和"未患病"。

- **概率链式法则**将乘法法则推广到任意数量的事件：

$$P(A_1 \cap A_2 \cap \cdots \cap A_n) = P(A_1) \cdot P(A_2 | A_1) \cdot P(A_3 | A_1 \cap A_2) \cdots P(A_n | A_1 \cap \cdots \cap A_{n-1})$$

- 每个因子以之前的一切为条件。这是自回归语言模型的支柱：句子的概率是每个词在给定之前所有词的条件下的概率的乘积。

- **条件独立**意味着两个事件给定第三个事件时是独立的。$A$ 和 $B$ 给定 $C$ 条件独立，如果：

$$P(A \cap B | C) = P(A | C) \cdot P(B | C)$$

- 事件可以边缘依赖但条件独立，反之亦然。例如，两个学生的考试成绩可能相关（都取决于考试难度），但给定考试难度，他们的分数是独立的。

- 条件独立是贝叶斯网络等图模型背后的关键假设。它让你将复杂的联合分布分解为可管理的片段，使推断在计算上可行。

1. 模拟医疗诊断问题。生成 100,000 人的群体，应用疾病流行率和测试准确率，并验证贝叶斯定理给出正确的后验。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(42)
n = 100_000

# 生成群体
k1, k2 = jax.random.split(key)
has_disease = jax.random.bernoulli(k1, p=0.01, shape=(n,))

# 生成测试结果
k3, k4 = jax.random.split(k2)
# 灵敏度：P(+|D) = 0.95, 特异度：P(-|D') = 0.90
test_positive = jnp.where(
    has_disease,
    jax.random.bernoulli(k3, p=0.95, shape=(n,)),
    jax.random.bernoulli(k4, p=0.10, shape=(n,))
)

# 在检测呈阳性的人中，实际患病的比例是多少？
positives = test_positive.astype(bool)
true_positives = (has_disease & positives).sum()
total_positives = positives.sum()

print(f"总阳性检测：{total_positives}")
print(f"真阳性：{true_positives}")
print(f"P(患病 | 阳性) = {true_positives / total_positives:.4f}")
print(f"贝叶斯公式：    {0.95 * 0.01 / 0.1085:.4f}")
```

2. 通过模拟验证加法法则。生成具有已知概率和重叠的随机事件 A 和 B，然后检查 $P(A \cup B) = P(A) + P(B) - P(A \cap B)$。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(0)
n = 200_000
k1, k2 = jax.random.split(key)

# 事件：A = 值 < 0.4, B = 值 < 0.6（在 < 0.4 处重叠）
vals_a = jax.random.uniform(k1, shape=(n,))
vals_b = jax.random.uniform(k2, shape=(n,))

A = vals_a < 0.4
B = vals_b < 0.6

p_a = A.mean()
p_b = B.mean()
p_a_and_b = (A & B).mean()
p_a_or_b = (A | B).mean()

print(f"P(A) = {p_a:.4f}")
print(f"P(B) = {p_b:.4f}")
print(f"P(A ∩ B) = {p_a_and_b:.4f}")
print(f"P(A ∪ B) 模拟 = {p_a_or_b:.4f}")
print(f"P(A) + P(B) - P(A∩B) = {p_a + p_b - p_a_and_b:.4f}")
```

3. 演示条件概率随证据变化。模拟掷两个骰子并计算 $P(\text{和} = 7)$，然后计算 $P(\text{和} = 7 | \text{第一个骰子} = 3)$。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(1)
n = 500_000
k1, k2 = jax.random.split(key)

d1 = jax.random.randint(k1, shape=(n,), minval=1, maxval=7)
d2 = jax.random.randint(k2, shape=(n,), minval=1, maxval=7)
total = d1 + d2

# 无条件
p_sum7 = (total == 7).mean()
print(f"P(和=7) = {p_sum7:.4f} (精确：{6/36:.4f})")

# 条件：第一个骰子 = 3
mask = d1 == 3
p_sum7_given_d1_3 = (total[mask] == 7).mean()
print(f"P(和=7 | d1=3) = {p_sum7_given_d1_3:.4f} (精确：{1/6:.4f})")
```

4. 将贝叶斯定理实现为函数，并使用它迭代更新信念。从硬币偏置的均匀先验开始，并在每次观察抛掷后更新。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

def bayes_update(prior, likelihood):
    """先验乘以似然并归一化。"""
    posterior = prior * likelihood
    return posterior / posterior.sum()

# 离散化可能的偏置值
theta = jnp.linspace(0, 1, 200)
prior = jnp.ones_like(theta)  # 均匀先验
prior = prior / prior.sum()

# 观察到的抛掷：1=正面，0=反面
flips = [1, 1, 0, 1, 1, 1, 0, 1, 0, 1]

plt.figure(figsize=(10, 5))
plt.plot(theta, prior, "--", color="#999", label="先验")

for i, flip in enumerate(flips):
    likelihood = theta if flip == 1 else (1 - theta)
    prior = bayes_update(prior, likelihood)
    if i in [0, 2, 4, 9]:
        plt.plot(theta, prior, label=f"{i+1}次抛掷后", linewidth=2)

plt.xlabel("硬币偏置 θ")
plt.ylabel("信念（归一化）")
plt.title("贝叶斯更新：关于硬币偏置的信念")
plt.legend()
plt.grid(alpha=0.3)
plt.show()
```
