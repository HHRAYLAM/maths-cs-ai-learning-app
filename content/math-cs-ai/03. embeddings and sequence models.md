# 嵌入和序列模型

*词嵌入将稀疏符号文本压缩为稠密向量空间，其中语义相似性变为几何邻近性。本文件涵盖 Word2Vec（CBOW、Skip-gram）、GloVe、FastText、RNN、LSTM、GRU、带注意力的 seq2seq 和编码器 - 解码器范式，从词袋到上下文表示的进展。*

- 在文件 01 中，我们介绍了分布假设：出现在相似上下文中的词往往有相似含义。在文件 02 中，我们用稀疏手工构造特征（如 TF-IDF 向量）表示文本。这些向量生活在非常高维空间（每个词汇词一个维度）且大多是零。**词嵌入**将这些信息压缩为捕获语义关系的稠密低维向量，它们直接从数据学习。

- **Word2Vec** (Mikolov et al., 2013) 通过在简单预测任务上训练浅层神经网络学习词嵌入。有两种架构。

- **连续词袋 (CBOW)** 模型从其周围上下文词预测目标词。给定上下文词窗口（例如，"the cat ___ on the"），模型平均它们的嵌入向量并通过线性层传递结果以预测缺失词（"sat"）。训练目标最大化：

$$P(w_t \mid w_{t-k}, \ldots, w_{t-1}, w_{t+1}, \ldots, w_{t+k})$$

- **Skip-gram** 模型做相反的事：给定目标词，预测周围上下文词。对目标词"sat"，模型尝试在单独预测中预测"the"、"cat"、"on"、"the"。目标最大化：

$$P(w_{t+j} \mid w_t) \quad \text{对每个 } j \in [-k, k], \; j \neq 0$$

![Skip-gram 和 CBOW 架构并排：CBOW 平均上下文嵌入预测中心词，skip-gram 用中心词嵌入预测每个上下文词](../images/word2vec_architectures.svg)

- Skip-gram 对稀有词往往工作更好，因为每个词生成多个训练示例（每个上下文位置一个）。CBOW 更快且对频繁词稍好，因为它平均多个上下文信号。

- 在完整词汇表上训练昂贵，因为 softmax 分母对所有 $V$ 个词求和。**负采样**通过将其转化为二分类问题来近似：区分真实上下文词（正样本）与随机采样噪声词（负样本）。不是计算完整 softmax，模型仅更新目标、真实上下文词和少数负数的嵌入：

$$\mathcal{L} = \log \sigma(v_{w_O}^T v_{w_I}) + \sum_{i=1}^{k} \mathbb{E}_{w_i \sim P_n} [\log \sigma(-v_{w_i}^T v_{w_I})]$$

- 这里 $v_{w_I}$ 是输入词嵌入，$v_{w_O}$ 是输出（上下文）词嵌入，$P_n$ 是噪声分布，通常是提升到 3/4 幂的单 gram 频率（这降低非常频繁词如"the"的权重）。

- 为什么这个简单目标产生有意义的嵌入？Levy 和 Goldberg (2014) 表明带负采样的 skip-gram 隐式分解**移位逐点互信息 (PMI)** 矩阵。在收敛时，两个词向量的点积近似：

$$v_w^T v_c \approx \text{PMI}(w, c) - \log k$$

- 其中 $\text{PMI}(w, c) = \log \frac{P(w, c)}{P(w) P(c)}$ 测量词 $w$ 和 $c$ 共现比偶然预期多多少（第 05 章信息论），$k$ 是负样本数。共现比偶然多得多的词有高 PMI 因此高点积（相似嵌入）。共现比预期少的词有负 PMI 和不相似嵌入。这揭示 Word2Vec 在做与经典分布语义方法（如潜在语义分析，对共现矩阵 SVD）相同的事，但以更可扩展、在线方式。

- Word2Vec 嵌入最令人惊讶的属性是它们通过向量算术捕获**类比**。向量 $v_{\text{king}} - v_{\text{man}} + v_{\text{woman}}$ 最接近 $v_{\text{queen}}。这有效因为嵌入空间编码语义关系为近似线性方向："royalty"方向大致是 $v_{\text{king}} - v_{\text{man}}$，将其加到 $v_{\text{woman}}$ 落在 $v_{\text{queen}}$ 附近。这连接到第 01 章的线性代数：语义关系是向量平移。

- **GloVe**（Global Vectors for Word Representation, Pennington et al., 2014）采用不同方法。它不一次从一个局部上下文窗口学习，而是构建全局词共现矩阵 $X$，其中 $X_{ij}$ 计数在整个语料中词 $j$ 出现在词 $i$ 上下文的频率。模型然后学习其点积近似对数共现的嵌入：

$$w_i^T \tilde{w}_j + b_i + \tilde{b}_j = \log X_{ij}$$

- 损失函数用截断函数 $f(X_{ij})$ 加权每个对，防止非常频繁共现主导：

$$\mathcal{L} = \sum_{i,j=1}^{V} f(X_{ij}) \left(w_i^T \tilde{w}_j + b_i + \tilde{b}_j - \log X_{ij}\right)^2$$

- GloVe 结合全局矩阵分解（如潜在语义分析）的优势与 Word2Vec 的局部上下文学习。在实践中，GloVe 和 Word2Vec 产生相似质量的嵌入。

- **FastText** (Bojanowski et al., 2017) 通过将每个词表示为字符 n-gram 袋扩展 skip-gram。词"where"带 $n = 3$ 变为："<wh"、"whe"、"her"、"ere"、"re>"，加上整词令牌"<where>"。词的嵌入是其所有 n-gram 嵌入的和。

- 这有关键优势：FastText 可以为训练期间从未见过的词生成嵌入。词"whereabouts"与"where"共享 n-gram，所以即使"whereabouts"从未出现在训练数据中，其嵌入也将合理。这对形态丰富语言（文件 01）特别有用，其中词有许多屈折形式。

- **嵌入评估**通常使用两类基准。**类比任务**测试是否 $v_a - v_b + v_c \approx v_d$（例如，"Paris" $-$ "France" $+$ "Italy" $\approx$ "Rome"）。**相似性基准**比较词对间的余弦相似度（第 01 章）与人类判断。常用数据集包括 WordSim-353、SimLex-999 和 Google 类比测试集。实际警告：在类比中表现优异的嵌入可能不是情感分类等下游任务的最佳。最佳评估通常是任务本身。

- 在第 06 章中，我们介绍 RNN、LSTM 和 GRU 作为序列数据架构。这里我们聚焦它们如何专门应用于语言任务。

- **语言模型 RNN** 一次读取一个令牌并在每步预测下一个令牌。隐藏状态 $h_t$ 将整个历史 $w_1, \ldots, w_t$ 压缩为固定大小向量，线性层加 softmax 将 $h_t$ 映射到词汇表上的分布。训练用针对真实下一个令牌的交叉熵损失，等同于最小化困惑度（文件 02）。关键限制：固定大小隐藏状态必须编码关于历史的一切，来自早期令牌的信息逐渐被覆盖。

- **双向 RNN** 在两个方向处理序列：一个 RNN 从左到右读，另一个从右到左读。在每个位置 $t$，前向隐藏状态 $\overrightarrow{h}_t$ 和后向隐藏状态 $\overleftarrow{h}_t$ 连接形成上下文感知表示 $h_t = [\overrightarrow{h}_t ; \overleftarrow{h}_t]$。这给模型访问过去和未来上下文，这对 POS 标注和 NER（文件 02）等任务强大，其中词的标签依赖于其前后词。双向 RNN 不能用于语言建模，因为预测未来令牌时不能偷看它们。

![双向 RNN：前向 RNN 从左到右读产生隐藏状态，后向 RNN 从右到左读，输出在每个位置连接](../images/bidirectional_rnn.svg)

- **深度堆叠 RNN** 将多个 RNN 层相互叠加。层 $l$ 在所有时间步的隐藏状态成为层 $l + 1$ 的输入序列。堆叠 2-4 层通常通过构建层次表示提高性能，类似于更深 CNN 如何构建特征层次（第 06 章）。超过 4 层，除非层间添加残差连接，否则梯度消失和过拟合成为问题。

- **序列到序列 (seq2seq)** 架构 (Sutskever et al., 2014) 将可变长度输入序列映射到可变长度输出序列。它由**编码器**RNN 组成，读取输入并将其压缩为上下文向量（最终隐藏状态），以及**解码器**RNN，以此上下文向量为条件一次生成一个输出令牌。

![Seq2seq 编码器 - 解码器：编码器 RNN 从左到右读输入令牌，最终隐藏状态作为初始状态传递给解码器 RNN，解码器 RNN 自回归生成输出令牌](../images/seq2seq_architecture.svg)

- Seq2seq 是机器翻译的突破架构。编码器读取法语句子，解码器生成英语翻译。解码器以特殊序列开始令牌开始并自回归生成令牌直到生成序列结束令牌。实用技巧：反转输入序列（馈送"chat le"而非"le chat"）改善结果，因为这在计算图中将第一个输入词放置在离第一个输出词更近的位置，缩短梯度路径。

- 瓶颈问题：整个输入必须压缩为单个固定大小向量。对长句子，该向量无法捕获所有信息，性能下降。这激发**注意力机制**。

- 第 06 章介绍注意力的现代 Q、K、V 公式。NLP 的原始注意力机制公式化为编码器和解码器状态间的对齐模型。

- **Bahdanau 注意力**（加性注意力，Bahdanau et al., 2015）用学习前馈网络计算解码器隐藏状态 $s_t$ 和每个编码器隐藏状态 $h_i$ 间的对齐分数：

$$e_{ti} = v^T \tanh(W_s s_{t-1} + W_h h_i)$$

- 分数通过 softmax 归一化为注意力权重，上下文向量是编码器状态的加权和：

$$\alpha_{ti} = \frac{\exp(e_{ti})}{\sum_j \exp(e_{tj})}, \quad c_t = \sum_i \alpha_{ti} h_i$$

- 解码器然后使用 $s_{t-1}$ 和 $c_t$ 生成下一个输出。关键洞察：不是对整个句子一个固定上下文向量，每个解码器步获得编码器状态的不同加权组合，允许模型"回看"输入的相关部分。

- **Luong 注意力**（乘性注意力，Luong et al., 2015）简化分数计算。**dot** 变体使用 $e_{ti} = s_t^T h_i$。**general** 变体使用 $e_{ti} = s_t^T W h_i$。这些比 Bahdanau 的加性分数更快，因为它们使用矩阵乘法而非前馈网络。Luong 注意力还从当前解码器状态 $s_t$（而非 $s_{t-1}$）计算上下文向量，这给它访问更多信息但使计算略有不同。

![源句子与其翻译间的注意力对齐热图，显示每个目标词注意力哪些源词，较亮单元格表示更高注意力权重](../images/attention_alignment.svg)

- 注意力权重经常可视化为热图，显示解码器生成每个输出令牌时关注哪些输入令牌。在翻译中，这些热图大致追踪源语言和目标语言间的词对齐，对角线模式被重排打破（例如，形容词 - 名词顺序在法语和英语间不同）。

- 在推理时，解码器必须在每步选择令牌。**贪心解码**在每步选择最高概率令牌，但这可能导致次优序列：局部好选择可能迫使模型进入全局坏句子。**束搜索**在每步维护前 $k$ 个（束宽）部分序列，通过所有可能下一个令牌扩展每个并保持总体最好 $k$ 个。

- 束宽 $k = 1$ 时，束搜索退化为贪心解码。典型值是 $k = 4$ 到 $k = 10$。更大束找到更好序列但按比例更慢。束搜索需要**长度归一化**以避免偏好更短序列，后者自然有更高总概率，因为它们乘更少项。归一化分数是：

$$\text{score}(y) = \frac{1}{|y|^\alpha} \sum_{t=1}^{|y|} \log P(y_t \mid y_{<t})$$

- 其中 $|y|$ 是序列长度，$\alpha$（通常 0.6-0.7）控制长度惩罚强度。$\alpha = 0$ 时，无长度归一化。$\alpha = 1$ 时，分数是每令牌对数概率（几何平均）。中间值平衡偏好简洁输出与不过早截断。

- 当 RNN 顺序处理文本时，**1D CNN** 通过在令牌序列上滑动滤波器并行处理它。每个滤波器检测局部模式（n-gram 特征）。

- **TextCNN** (Kim, 2014) 将多个不同宽度（例如，3、4、5 令牌）的 1D 卷积滤波器应用到输入嵌入矩阵。每个滤波器产生特征图，**时间上最大池化**从每个特征图取单个最大值，捕获模式是否在文本中任何地方检测到而无论位置。来自所有滤波器的池化特征连接并传递给分类器。

![TextCNN 架构：输入嵌入通过宽度 3、4 和 5 的并行卷积滤波器，每个后跟时间上最大池化，然后连接并馈送到全连接分类器](../images/textcnn_architecture.svg)

- TextCNN 快且对情感分析等文本分类任务惊人有效。它捕获局部 n-gram 模式但无法建模长程依赖：宽度 5 的滤波器仅见 5 个连续令牌。**扩张因果卷积**通过在滤波器元素间插入间隙（扩张）解决这个问题。堆叠带指数增加扩张率（1、2、4、8、...）的层指数增长感受野而不增加参数，允许模型捕获跨数百令牌的依赖。

- 至今讨论的所有嵌入（Word2Vec、GloVe、FastText）对每个词类型产生单个向量而无论上下文。"Bank"无论指金融机构还是河岸都获得相同嵌入。这是**上下文嵌入**解决的根本限制。

- **ELMo**（Embeddings from Language Models, Peters et al., 2018）通过在输入文本上运行深度双向 LSTM 语言模型生成上下文词表示。前向 LSTM 在每步预测下一个词；单独后向 LSTM 预测前一个词。两者都在大语料上作为语言模型训练。

- 在每个位置 $k$，ELMo 用任务特定学习权重组合所有 $L$ 层的隐藏状态：

$$\text{ELMo}_k = \gamma \sum_{j=0}^{L} s_j \, h_{k,j}$$

- 这里 $h_{k,j}$ 是位置 $k$ 和层 $j$ 的隐藏状态（层 0 是原始令牌嵌入），$s_j$ 是 softmax 归一化标量权重，$\gamma$ 是任务特定缩放因子。不同层捕获不同信息：下层捕获语法（POS 标签、词形态），上层捕获语义（词义、语义角色）。通过用学习权重混合所有层，ELMo 嵌入适应多样下游任务。

- ELMo 标志着**预训练然后微调**范式的开始：在大规模无标签文本上训练大语言模型，然后将其表示用于下游任务。ELMo 专门使用预训练表示作为固定或轻度微调特征，与任务特定输入连接。BERT 和 GPT（文件 04）通过端到端微调整个模型进一步推动这，证明 dramatically 更有效。

- 从 Word2Vec 到 ELMo 的进展说明 NLP 中反复出现的主题：从静态到动态表示、从局部到全局上下文、从浅到深模型。每步以计算成本换取更丰富表示。Transformers（文件 04）通过完全用注意力替换循环完成此进展，实现深度上下文化和并行计算。

1. 从头实现带负采样的 Word2Vec skip-gram。在小语料上训练并用 PCA 可视化学习嵌入。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 小语料
corpus = """the king ruled the kingdom . the queen ruled the kingdom .
the prince is the son of the king . the princess is the daughter of the queen .
a man worked in the castle . a woman worked in the castle .
the king and queen lived in the castle . the prince and princess played outside.""".lower().split()

vocab = sorted(set(corpus))
word2idx = {w: i for i, w in enumerate(vocab)}
idx2word = {i: w for w, i in word2idx.items()}
V = len(vocab)

# 生成带窗口大小 2 的 skip-gram 对
window = 2
pairs = []
for i, word in enumerate(corpus):
    for j in range(max(0, i - window), min(len(corpus), i + window + 1)):
        if i != j:
            pairs.append((word2idx[word], word2idx[corpus[j]]))

pairs = jnp.array(pairs)
print(f"词汇表：{V} 个词，训练对：{len(pairs)}")

# 模型参数
embed_dim = 16
key = jax.random.PRNGKey(42)
k1, k2 = jax.random.split(key)
W_in = jax.random.normal(k1, (V, embed_dim)) * 0.1    # 输入嵌入
W_out = jax.random.normal(k2, (V, embed_dim)) * 0.1   # 输出嵌入

# 一对的负采样损失
def neg_sampling_loss(W_in, W_out, target, context, neg_ids):
    v_in = W_in[target]      # (embed_dim,)
    v_out = W_out[context]   # (embed_dim,)
    v_neg = W_out[neg_ids]   # (k, embed_dim)

    pos_loss = -jax.nn.log_sigmoid(jnp.dot(v_in, v_out))
    neg_loss = -jnp.sum(jax.nn.log_sigmoid(-v_neg @ v_in))
    return pos_loss + neg_loss

# 训练循环
num_neg = 5
lr = 0.05

@jax.jit
def train_step(W_in, W_out, target, context, neg_ids):
    loss, (g_in, g_out) = jax.value_and_grad(neg_sampling_loss, argnums=(0, 1))(
        W_in, W_out, target, context, neg_ids)
    return loss, W_in - lr * g_in, W_out - lr * g_out

key = jax.random.PRNGKey(0)
for epoch in range(50):
    total_loss = 0.0
    for i in range(len(pairs)):
        key, subkey = jax.random.split(key)
        neg_ids = jax.random.randint(subkey, (num_neg,), 0, V)
        loss, W_in, W_out = train_step(W_in, W_out, pairs[i, 0], pairs[i, 1], neg_ids)
        total_loss += loss
    if (epoch + 1) % 10 == 0:
        print(f"第 {epoch+1} 轮：平均损失 = {total_loss / len(pairs):.4f}")

# 用 PCA 可视化（第 01 章）
embeddings = W_in
mean = embeddings.mean(axis=0)
centered = embeddings - mean
U, S, Vt = jnp.linalg.svd(centered, full_matrices=False)
coords = centered @ Vt[:2].T  # 投影到前 2 个主成分

plt.figure(figsize=(10, 8))
for i, word in idx2word.items():
    plt.scatter(coords[i, 0], coords[i, 1], c='#3498db', s=40)
    plt.annotate(word, (coords[i, 0] + 0.02, coords[i, 1] + 0.02), fontsize=9)
plt.title("Word2Vec Skip-gram 嵌入（PCA 投影）")
plt.grid(alpha=0.3); plt.show()
```

2. 构建字符级 RNN 语言模型，学习从小训练字符串生成文本。
```python
import jax
import jax.numpy as jnp

# 微小训练文本
text = "to be or not to be that is the question "
chars = sorted(set(text))
char2idx = {c: i for i, c in enumerate(chars)}
idx2char = {i: c for c, i in char2idx.items()}
V = len(chars)
data = jnp.array([char2idx[c] for c in text])

# RNN 参数
hidden_dim = 64
key = jax.random.PRNGKey(0)
k1, k2, k3, k4, k5 = jax.random.split(key, 5)

params = {
    'Wx': jax.random.normal(k1, (V, hidden_dim)) * 0.1,
    'Wh': jax.random.normal(k2, (hidden_dim, hidden_dim)) * 0.05,
    'bh': jnp.zeros(hidden_dim),
    'Wy': jax.random.normal(k3, (hidden_dim, V)) * 0.1,
    'by': jnp.zeros(V),
}

def rnn_step(params, h, x_idx):
    x = jnp.eye(V)[x_idx]  # one-hot
    h = jnp.tanh(x @ params['Wx'] + h @ params['Wh'] + params['bh'])
    logits = h @ params['Wy'] + params['by']
    return h, logits

def loss_fn(params, inputs, targets):
    h = jnp.zeros(hidden_dim)
    total_loss = 0.0
    for t in range(len(inputs)):
        h, logits = rnn_step(params, h, inputs[t])
        log_probs = jax.nn.log_softmax(logits)
        total_loss -= log_probs[targets[t]]
    return total_loss / len(inputs)

grad_fn = jax.jit(jax.grad(loss_fn))

# 训练
inputs = data[:-1]
targets = data[1:]
lr = 0.01

for step in range(500):
    grads = grad_fn(params, inputs, targets)
    params = {k: params[k] - lr * grads[k] for k in params}
    if (step + 1) % 100 == 0:
        l = loss_fn(params, inputs, targets)
        print(f"步 {step+1}: 损失 = {l:.4f}")

# 生成文本
def generate(params, seed_char, length=60):
    h = jnp.zeros(hidden_dim)
    idx = char2idx[seed_char]
    result = [seed_char]
    key = jax.random.PRNGKey(42)
    for _ in range(length):
        h, logits = rnn_step(params, h, idx)
        key, subkey = jax.random.split(key)
        idx = jax.random.categorical(subkey, logits)
        result.append(idx2char[int(idx)])
    return ''.join(result)

print(f"\n生成：{generate(params, 't')}")
```

3. 实现带 Bahdanau 注意力的玩具 seq2seq 模型用于序列反转。可视化注意力对齐矩阵。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 任务：反转数字序列（例如，[3, 1, 4] -> [4, 1, 3]）
vocab_size = 10  # 数字 0-9
SOS, EOS = 10, 11  # 特殊令牌
total_vocab = 12
embed_dim, hidden_dim = 16, 32
max_len = 5

key = jax.random.PRNGKey(42)
keys = jax.random.split(key, 8)

params = {
    'embed': jax.random.normal(keys[0], (total_vocab, embed_dim)) * 0.1,
    'enc_Wx': jax.random.normal(keys[1], (embed_dim, hidden_dim)) * 0.1,
    'enc_Wh': jax.random.normal(keys[2], (hidden_dim, hidden_dim)) * 0.05,
    'dec_Wx': jax.random.normal(keys[3], (embed_dim, hidden_dim)) * 0.1,
    'dec_Wh': jax.random.normal(keys[4], (hidden_dim, hidden_dim)) * 0.05,
    # Bahdanau 注意力
    'Ws': jax.random.normal(keys[5], (hidden_dim, hidden_dim)) * 0.1,
    'Wh_att': jax.random.normal(keys[6], (hidden_dim, hidden_dim)) * 0.1,
    'v_att': jax.random.normal(keys[7], (hidden_dim,)) * 0.1,
    # 输出投影（从 hidden + context 到 vocab）
    'Wo': jax.random.normal(keys[0], (hidden_dim * 2, total_vocab)) * 0.1,
}

def encode(params, seq):
    """编码输入序列，返回所有隐藏状态。"""
    h = jnp.zeros(hidden_dim)
    states = []
    for t in range(len(seq)):
        x = params['embed'][seq[t]]
        h = jnp.tanh(x @ params['enc_Wx'] + h @ params['enc_Wh'])
        states.append(h)
    return jnp.stack(states), h

def bahdanau_attention(params, dec_state, enc_states):
    """计算 Bahdanau 注意力权重和上下文向量。"""
    scores = jnp.tanh(enc_states @ params['Wh_att'] + dec_state @ params['Ws'])
    e = scores @ params['v_att']  # (src_len,)
    alpha = jax.nn.softmax(e)
    context = alpha @ enc_states
    return context, alpha

def decode_step(params, dec_h, prev_token, enc_states):
    x = params['embed'][prev_token]
    dec_h = jnp.tanh(x @ params['dec_Wx'] + dec_h @ params['dec_Wh'])
    context, alpha = bahdanau_attention(params, dec_h, enc_states)
    combined = jnp.concatenate([dec_h, context])
    logits = combined @ params['Wo']
    return dec_h, logits, alpha

def seq2seq_loss(params, src, tgt):
    enc_states, enc_final = encode(params, src)
    dec_h = enc_final
    loss = 0.0
    prev_token = SOS
    for t in range(len(tgt)):
        dec_h, logits, _ = decode_step(params, dec_h, prev_token, enc_states)
        log_probs = jax.nn.log_softmax(logits)
        loss -= log_probs[tgt[t]]
        prev_token = tgt[t]
    return loss / len(tgt)

# 生成训练数据：反转序列
key = jax.random.PRNGKey(0)
train_srcs, train_tgts = [], []
for _ in range(200):
    key, subkey = jax.random.split(key)
    length = jax.random.randint(subkey, (), 3, max_len + 1)
    key, subkey = jax.random.split(key)
    seq = jax.random.randint(subkey, (int(length),), 0, vocab_size)
    train_srcs.append(seq)
    train_tgts.append(seq[::-1])  # 反转

# 训练
grad_fn = jax.grad(seq2seq_loss)
lr = 0.01

for epoch in range(100):
    total_loss = 0.0
    for src, tgt in zip(train_srcs, train_tgts):
        grads = grad_fn(params, src, tgt)
        params = {k: params[k] - lr * grads[k] for k in params}
        total_loss += seq2seq_loss(params, src, tgt)
    if (epoch + 1) % 20 == 0:
        print(f"第 {epoch+1} 轮：平均损失 = {total_loss / len(train_srcs):.4f}")

# 可视化一个示例的注意力
test_src = jnp.array([3, 1, 4, 1, 5])
test_tgt = test_src[::-1]

enc_states, enc_final = encode(params, test_src)
dec_h = enc_final
attentions = []
prev_token = SOS
for t in range(len(test_tgt)):
    dec_h, logits, alpha = decode_step(params, dec_h, prev_token, enc_states)
    attentions.append(alpha)
    prev_token = test_tgt[t]

att_matrix = jnp.stack(attentions)
fig, ax = plt.subplots(figsize=(6, 5))
im = ax.imshow(att_matrix, cmap='Blues')
ax.set_xlabel("源位置"); ax.set_ylabel("目标位置")
src_labels = [str(int(x)) for x in test_src]
tgt_labels = [str(int(x)) for x in test_tgt]
ax.set_xticks(range(len(src_labels))); ax.set_xticklabels(src_labels)
ax.set_yticks(range(len(tgt_labels))); ax.set_yticklabels(tgt_labels)
for i in range(len(tgt_labels)):
    for j in range(len(src_labels)):
        ax.text(j, i, f"{att_matrix[i,j]:.2f}", ha='center', va='center', fontsize=9)
ax.set_title("Bahdanau 注意力对齐（序列反转）")
plt.colorbar(im); plt.tight_layout(); plt.show()
```
