# 微分学

*微分学捕捉瞬时变化率。本章涵盖极限、导数、微分法则、链式法则（反向传播的基础）以及机器学习中常见的导数。*

- 在前面的章节中，我们学习了如何将数据表示为向量并用矩阵变换它。但许多真实现象不是静态的。汽车加速、股票价格波动、神经网络的损失随权重更新而变化。**微积分**是变化背后的数学。

- 微积分问两个问题：某物此刻变化有多快？（微分学）以及它随时间累积了多少？（积分学）。本节解决"有多快"的问题。

- 想象你在开车并瞥了一眼速度表。它显示 60 公里/小时。这个数字不是你整个行程的平均速度；它是你在这个确切瞬间的速度。微分学给了我们计算这种瞬时变化率的工具。

- 但首先，让我们回顾一下直线方程：$y = mx + b$。

- 这是两个量之间最简单的关系。

    - $b$ 是**y 截距**，直线与 y 轴相交的地方（当 $x = 0$ 时的起始值）。
    - $m$ 是**斜率**，变化率：对于 $x$ 每增加 1 单位，$y$ 变化 $m$。
- 如果 $m = 3$，直线陡峭上升；如果 $m = 0$，直线是平的；如果 $m = -2$，直线下降。

- 斜率计算为 $m = \frac{\Delta y}{\Delta x} = \frac{y_2 - y_1}{x_2 - x_1}$，"y 变化了多少"与"x 变化了多少"的比率。

![直线方程：b 是 y 截距，m 是斜率（上升除以运行）](../images/line_equation.svg)

- 一旦你知道 $m$ 和 $b$，你就可以计算任何 $x$ 对应的 $y$。

- 例如，如果 $m = 2$ 且 $b = 3$，那么在 $x = 5$ 时：$y = 2(5) + 3 = 13$。

- 这两个参数完全确定了直线，预测任何输出只是代入而已。

- 对于直线，斜率在任何地方都相同。

- 这个想法推广到直线之外。任何函数都是将输入映射到输出的规则，一旦你知道它的公式（它的参数和形状），你就可以计算任何输入的输出并绘制结果。

- $y = x^2$ 给出抛物线，$y = \sin(x)$ 给出波浪，$y = e^x$ 给出指数增长。每个公式定义一个特定的曲线， comfortably 将函数读作形状对后续所有内容至关重要。

- 对于直线，斜率在任何地方都相同。但大多数有趣的函数是弯曲的，所以斜率从点到点变化。微积分给了我们找到曲线上任何单点斜率的方法。

- 我们还需要**极限**的概念。极限描述当输入越来越接近某个目标时函数接近什么值，而不必实际达到它。

$$\lim_{x \to a} f(x) = L$$

- 这读作："当 $x$ 接近 $a$ 时，$f(x)$ 接近 $L$。"函数不需要在 $x = a$ 处实际等于 $L$。它只需要任意接近。

- 例如，取 $f(x) = \frac{x^2 - 1}{x - 1}$。如果你直接代入 $x = 1$，你得到 $\frac{0}{0}$，这是未定义的。

- 但尝试接近 1 的值：$f(0.9) = 1.9$，$f(0.99) = 1.99$，$f(1.01) = 2.01$。输出显然趋向 2。

- 代数上，我们可以看到为什么：将分子因式分解为 $(x-1)(x+1)$，抵消 $(x-1)$ 项，我们得到 $f(x) = x + 1$ 对于所有 $x \neq 1$。所以当 $x \to 1$ 时，$f(x) \to 2$。

- 函数在 $x = 1$ 处有一个洞，但极限仍然存在。

- 极限是微积分中所有其他东西依赖的基础。

- 函数 $f(x)$ 在点 $x = a$ 处的**导数**测量瞬时变化率。几何上，它是该点处曲线切线的斜率。

![导数是曲线上某点切线的斜率](../images/tangent_line.svg)

- 为了计算这个斜率，我们从曲线上的两个点开始，计算穿过它们的直线的斜率（一条**割线**）。然后我们将第二个点越来越近地滑向第一个点，看看割线接近什么斜率。这是**差商**：

$$f'(a) = \lim_{h \to 0} \frac{f(a + h) - f(a)}{h}$$

![随着 h 缩小，割线接近切线](../images/difference_quotient.svg)

- 分子 $f(a+h) - f(a)$ 是输出的变化。分母 $h$ 是输入的变化。它们的比率是微小间隔上的平均变化率。当 $h \to 0$ 时，这个平均变成瞬时速率。

- 例如，让 $f(x) = x^2$。在 $x = 3$ 时：

$$f'(3) = \lim_{h \to 0} \frac{(3+h)^2 - 9}{h} = \lim_{h \to 0} \frac{9 + 6h + h^2 - 9}{h} = \lim_{h \to 0} (6 + h) = 6$$

- 所以在 $x = 3$ 时，函数 $x^2$ 以增加每单位输入 6 单位输出的速率增长。

- 函数在点处是**可微的**，如果这个极限存在。为此，函数必须连续（无跳跃）、平滑（无尖角），并且在点周围有定义。

- 如果你可以不用提起笔就画出曲线且没有任何扭结，它在那里可能是可微的。

- 每次从极限定义计算导数将是乏味的。幸运的是，少数法则让我们可以快速微分几乎任何函数。

- **常数法则**：常数的导数为零。如果 $f(x) = 5$，那么 $f'(x) = 0$。平线有零斜率。

- **幂法则**：微分的主力。将指数带下来并减一：

$$\frac{d}{dx} x^n = n x^{n-1}$$

- 例如：$\frac{d}{dx} x^3 = 3x^2$。立方变成二次。这适用于任何实指数，包括负数和分数：$\frac{d}{dx} x^{-1} = -x^{-2}$ 且 $\frac{d}{dx} \sqrt{x} = \frac{d}{dx} x^{1/2} = \frac{1}{2}x^{-1/2}$。

- **和/差法则**：逐项微分。

$$\frac{d}{dx}[f(x) \pm g(x)] = f'(x) \pm g'(x)$$

- **乘积法则**：当两个函数相乘时，导数不仅仅是导数的乘积。相反：

$$\frac{d}{dx}[f(x) \cdot g(x)] = f'(x)g(x) + f(x)g'(x)$$

- 这样想："第一个的变化率乘以第二个，加上第一个乘以第二个的变化率。"例如，$\frac{d}{dx}[x^2 \sin x] = 2x \sin x + x^2 \cos x$。

- **商法则**：对于函数的比率：

$$\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$$

- 一个有用的助记符："低乘高导减高乘低导，除以低的平方。"

- **链式法则**：机器学习最重要的法则。当函数复合时（一个在另一个里面），导数是沿链的导数的乘积：

$$\frac{d}{dx} f(g(x)) = f'(g(x)) \cdot g'(x)$$

- 这样想：像剥洋葱。微分外部函数（保持内部函数不变），然后乘以内部函数的导数。

![链式法则：微分外部，乘以内部的导数](../images/chain_rule.svg)

- 例如，$\frac{d}{dx} (3x + 1)^5 = 5(3x+1)^4 \cdot 3 = 15(3x+1)^4$。外部函数是 $(\cdot)^5$，内部是 $3x+1$。

- 链式法则是神经网络中**反向传播**的数学基础。深度网络是长链复合函数。为了计算损失如何随每个权重变化，我们从输出层反向到输入重复应用链式法则，在每步乘以局部导数。

- 以下是你将遇到的最常见导数。每个都可以从极限定义推导出来，但记住它们可以节省时间：

| 函数 | 导数 | 注释 |
|---|---|---|
| $e^x$ | $e^x$ | 唯一等于自身导数的函数 |
| $a^x$ | $a^x \ln a$ | 推广的指数 |
| $\ln x$ | $\frac{1}{x}$ | 自然对数 |
| $\log_a x$ | $\frac{1}{x \ln a}$ | 一般对数 |
| $\sin x$ | $\cos x$ | |
| $\cos x$ | $-\sin x$ | 注意负号 |
| $\tan x$ | $\sec^2 x$ | |

- 指数函数 $e^x$ 是非凡的：它是唯一等于自身导数的函数。这就是为什么 $e$ 在机器学习中无处不在，从 softmax 激活到概率分布。

- **洛必达法则**处理产生不定式（如 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$）的极限。当直接代入给出这些形式之一时，你可以分别对分子和分母求导并再次尝试极限：

$$\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}$$

- 条件：$f$ 和 $g$ 必须在 $a$ 附近可导，且 $g'(x) \neq 0$ 在 $a$ 附近（除了可能在 $a$ 本身）。原始极限必须给出不定式。

- 例如：$\lim_{x \to 0} \frac{\sin x}{x}$。直接代入给出 $\frac{0}{0}$。应用洛必达法则：$\lim_{x \to 0} \frac{\cos x}{1} = 1$。这个极限是基本的，它出现在信号处理和傅里叶分析中。

- 如果结果仍然是不定式，你可以重复应用该法则。例如，$\lim_{x \to 0} \frac{1 - \cos x}{x^2}$ 给出 $\frac{0}{0}$。第一次应用：$\lim_{x \to 0} \frac{\sin x}{2x}$，仍然是 $\frac{0}{0}$。第二次应用：$\lim_{x \to 0} \frac{\cos x}{2} = \frac{1}{2}$。

- 如果两个函数可微，它们的和、差、积、复合和商（分母非零处）也可微。这就是为什么我们可以自信地微分由简单部分构建的复杂表达式。

2. 使用 JAX 的自动微分计算 $f(x) = x^3 - 2x + 1$ 在几个点的导数。与分析导数 $f'(x) = 3x^2 - 2$ 比较。
```python
import jax
import jax.numpy as jnp

f = lambda x: x**3 - 2*x + 1
df = jax.grad(f)

for x in [0.0, 1.0, 2.0, -1.0]:
    print(f"x={x:5.1f}  autodiff: {df(x):.4f}  analytical: {3*x**2 - 2:.4f}")
```

3. 数值验证链式法则。定义 $f(x) = \sin(x^2)$，通过 `jax.grad` 计算其导数，并与分析结果 $2x\cos(x^2)$ 比较。
```python
import jax
import jax.numpy as jnp

f = lambda x: jnp.sin(x**2)
df = jax.grad(f)

for x in [0.5, 1.0, 1.5, 2.0]:
    numerical = df(x)
    analytical = 2 * x * jnp.cos(x**2)
    print(f"x={x:4.1f}  autodiff: {numerical:.6f}  analytical: {analytical:.6f}")
```
