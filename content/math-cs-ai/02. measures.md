# 统计度量

*统计度量用捕获离散、位置、形状和关联的单个数字总结数据。本文件涵盖方差、标准差、四分位数、偏度、峰度、协方差、相关性和 z 分数，ML 中探索性数据分析和特征工程的工具包。*

- 在前一文件中我们介绍了矩作为摘要统计家族。这里我们解构从中流出的实用工具：离散、位置、形状和关联的度量。

- **离散**回答问题：数据多分散？两个教室可以有相同平均测试分数，但非常不同的分布。

![两个带相同均值但不同分布的分布](../images/variance_spread.svg)

- 窄（蓝色）分布有低方差：大多数值紧密聚集在均值周围。宽（红色）分布有高方差：值更分散。

- **方差**是距均值的平均平方距离。我们平方避免正和负偏差相互抵消。

$$\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2$$

- 当使用样本（非完整总体）时，我们除以 $N - 1$ 而非 $N$。这个校正（称为 Bessel 校正）考虑样本倾向低估真实变异的事实：

$$s^2 = \frac{1}{N-1} \sum_{i=1}^{N} (x_i - \bar{x})^2$$

- **标准差**是方差的平方根：$\sigma = \sqrt{\sigma^2}$。它将度量带回原始单位。如果你的数据是厘米，方差是 cm$^2$，但标准差回厘米。

- **平均绝对偏差** (MAD) 是更简单替代。不是平方，取每个偏差的绝对值：

$$\text{MAD} = \frac{1}{N} \sum_{i=1}^{N} |x_i - \mu|$$

- MAD 比方差更鲁棒对异常值因为它不通过平方放大大的偏差。然而，方差更方便数学（它在证明和 ML 优化中分解好）。

- **位置**回答不同问题：特定值相对于数据其余部分位于哪里？

- **四分位数**将排序数据分成四相等部分。Q1（25 百分位数）是 25% 数据落于其下的值。Q2 是中位数（50 百分位数）。Q3 是 75 百分位数。

- **四分位距** (IQR) 是 $Q3 - Q1$。它捕获中间 50% 数据的分布，忽略极端值。

![箱线图显示 Q1、中位数、Q3、IQR、须和异常值](../images/quartiles_boxplot.svg)

- 箱线图是统计学中最有用的可视化之一。箱子从 Q1 跨到 Q3，内部线是中位数，须延伸到最极端非异常值，须外的点是异常值。

- **百分位数**推广四分位数。$p$-th 百分位数是 $p\%$ 观察落于其下的值。Q1 是 25 百分位数，中位数是 50，Q3 是 75。

- **z 分数**告诉你值距均值多少标准差：

$$z = \frac{x - \mu}{\sigma}$$

- z 分数 2 意味着值距均值 2 标准差上。z 分数 $-1.5$ 意味着它距均值 1.5 标准差下。这也称为**标准化**，广泛用于 ML 特征缩放，它将任何分布转换为均值 0 和标准差 1。

- **形状**描述分布的几何超出其中心和分布。

- **偏度**（来自前一文件的标准 3 阶矩）测量不对称。完美对称分布如正态曲线有零偏度。正偏度意味着更长右尾（例如，收入分布）。负偏度意味着更长左尾（例如，退休年龄）。

$$\text{Skewness} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^3$$

- **峰度**（标准化 4 阶矩）测量尾重量。正态分布有峰度 3。带更重尾的分布（更易异常值）有峰度大于 3。

$$\text{Kurtosis} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^4$$

- **相关性**测量两变量间关系的强度和方向。它回答：当一个变量上升，另一个倾向上升、下降或什么都不做？

![三个散点图显示正、无和负相关](../images/correlation_scatter.svg)

- **Pearson 相关** ($r$) 测量*线性*关联。范围从 $-1$（完美负）通过 $0$（无）到 $+1$（完美正）。

$$r = \frac{\sum_{i=1}^{N} (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2} \cdot \sqrt{\sum (y_i - \bar{y})^2}}$$

- 如果你回忆第 1 章的点积，Pearson 相关本质是 $\mathbf{x}$ 和 $\mathbf{y}$ 均值中心版本的余弦相似度。

- **Spearman 相关** ($\rho$) 测量*单调*关联。不是用原始值，它先排名它们然后计算排名的 Pearson 相关。这使它鲁棒对异常值并工作即使关系非线性，只要它一致递增或递减。

- **几何平均**是适当平均当值相乘时，像增长率。如果你的投资增长 10%、然后 20%、然后 30%，平均增长因子不是这些率的算术平均。而是：

$$\bar{x}_{\text{geo}} = \left(\prod_{i=1}^{N} x_i\right)^{1/N}$$

- 对增长率特别，先转换百分比为因子（1.10、1.20、1.30），计算几何平均，然后减 1。

- **指数移动平均** (EMA) 给更多权重到最近观察。不像简单移动平均其中窗口中所有点相等加权，EMA 指数衰减：

$$\text{EMA}_t = \alpha \cdot x_t + (1 - \alpha) \cdot \text{EMA}_{t-1}$$

- 平滑因子 $\alpha$（在 0 和 1 之间）控制旧观察多快失去影响。更高 $\alpha$ 意味着更响应最近变化，更低 $\alpha$ 意味着更平滑。在 ML 中，EMA 用于优化器如 Adam 和批归一化的运行统计。

- **异常值检测**识别异常远离其余的数据点。两种常用方法：
    - **IQR 方法**：点是异常值如果它落于 $Q1 - 1.5 \times \text{IQR}$ 下或 $Q3 + 1.5 \times \text{IQR}$ 上
    - **z 分数方法**：点是异常值如果 $|z| > 3$（距均值超过 3 标准差）

- IQR 方法更鲁棒因为它不假设正态分布。z 分数方法工作好在数据近似正态时但当分布重偏斜时可能失败。

1. 计算数据集的方差、标准差和 MAD 并比较它们。观察当你添加极端异常值时发生什么。
```python
import jax.numpy as jnp

data = jnp.array([4, 8, 6, 5, 3, 7, 9, 5, 6, 7], dtype=jnp.float32)

mean = jnp.mean(data)
variance = jnp.var(data)
std = jnp.std(data)
mad = jnp.mean(jnp.abs(data - mean))

print("原始数据：")
print(f"  方差：{variance:.3f}, 标准差：{std:.3f}, MAD: {mad:.3f}")

# 添加异常值并重新计算
data_outlier = jnp.append(data, 100.0)
mean2 = jnp.mean(data_outlier)
print(f"\n带异常值 (100):")
print(f"  方差：{jnp.var(data_outlier):.3f}, 标准差：{jnp.std(data_outlier):.3f}, MAD: {jnp.mean(jnp.abs(data_outlier - mean2)):.3f}")
```

2. 计算两变量间的 Pearson 和 Spearman 相关。实验不同关系。
```python
import jax
import jax.numpy as jnp

# 完美线性关系
x = jnp.array([1, 2, 3, 4, 5, 6, 7, 8], dtype=jnp.float32)
y = 2 * x + 1  # 尝试更改这个！

def pearson(a, b):
    a_c = a - jnp.mean(a)
    b_c = b - jnp.mean(b)
    return jnp.sum(a_c * b_c) / (jnp.sqrt(jnp.sum(a_c**2)) * jnp.sqrt(jnp.sum(b_c**2)))

def spearman(a, b):
    rank_a = jnp.argsort(jnp.argsort(a)).astype(jnp.float32)
    rank_b = jnp.argsort(jnp.argsort(b)).astype(jnp.float32)
    return pearson(rank_a, rank_b)

print(f"Pearson r:  {pearson(x, y):.4f}")
print(f"Spearman ρ: {spearman(x, y):.4f}")
```

3. 用 IQR 和 z 分数方法实现异常值检测，然后在偏斜数据上比较它们的结果。
```python
import jax.numpy as jnp

data = jnp.array([2, 3, 3, 4, 5, 5, 5, 6, 6, 7, 50], dtype=jnp.float32)

# IQR 方法
q1, q3 = jnp.percentile(data, 25), jnp.percentile(data, 75)
iqr = q3 - q1
lower, upper = q1 - 1.5 * iqr, q3 + 1.5 * iqr
iqr_outliers = data[(data < lower) | (data > upper)]
print(f"IQR 界限：[{lower:.1f}, {upper:.1f}]")
print(f"IQR 异常值：{iqr_outliers}")

# Z 分数方法
z_scores = (data - jnp.mean(data)) / jnp.std(data)
z_outliers = data[jnp.abs(z_scores) > 3]
print(f"\nZ 分数：{z_scores}")
print(f"Z 分数异常值 (|z| > 3): {z_outliers}")
```

4. 计算并绘制带不同平滑因子的指数移动平均在噪声数据上。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 生成噪声数据
key = __import__("jax").random.PRNGKey(0)
noise = __import__("jax").random.normal(key, shape=(50,))
signal = jnp.linspace(0, 5, 50) + noise

def ema(data, alpha):
    result = jnp.zeros_like(data)
    result = result.at[0].set(data[0])
    for t in range(1, len(data)):
        result = result.at[t].set(alpha * data[t] + (1 - alpha) * result[t - 1])
    return result

plt.figure(figsize=(10, 4))
plt.plot(signal, "o", alpha=0.3, label="原始数据", color="#999")
for alpha, color in [(0.1, "#e74c3c"), (0.3, "#3498db"), (0.7, "#27ae60")]:
    plt.plot(ema(signal, alpha), label=f"α={alpha}", color=color, linewidth=2)
plt.legend()
plt.title("带不同平滑因子的 EMA")
plt.show()
```
