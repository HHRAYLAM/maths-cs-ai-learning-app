## 📏 离散度量 - 数据有多分散

### 方差 - 平均平方距离

**方差**是距均值的平均平方距离：

$$\sigma^2 = \frac{1}{N} \sum_{i=1}^{N} (x_i - \mu)^2$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 为什么要平方？

- 避免正负偏差抵消
- 放大较大偏差的影响
- 数学上更方便（求导容易）

### 样本方差（Bessel 校正）

用样本估计时除以 $N-1$：

$$s^2 = \frac{1}{N-1} \sum_{i=1}^{N} (x_i - \bar{x})^2$$

**为什么**：样本倾向低估真实变异

### 例子计算

数据：$\{4, 8, 6, 5, 3\}$

1. 均值：$\mu = \frac{4+8+6+5+3}{5} = 5.2$
2. 方差：$\sigma^2 = \frac{(4-5.2)^2 + ... + (3-5.2)^2}{5} = 2.96$

</div>

```
方差可视化：

  A 批（低方差）    B 批（高方差）
     ↑                ↑
   ╱│╲              ╱ │ ╲
  ╱ │ ╲            ╱  │  ╲
 ╱  ●  ╲          ╱   ●   ╲
──────────        ────────────
  窄集中            宽分散

平方放大远距离偏差！
```

### 标准差 - 回到原始单位

**标准差**是方差的平方根：

$$\sigma = \sqrt{\sigma^2}$$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的理解</div>
  <div class="story-card-content">
    方差单位是<strong>平方</strong>（如 cm²）
    <br><br>
    标准差单位是<strong>原始</strong>（如 cm）
    <br><br>
    标准差更<strong>直观</strong>！
  </div>
</div>

### 例子

如果方差 $\sigma^2 = 4$ cm²，则标准差 $\sigma = 2$ cm

---

### 平均绝对偏差 (MAD) - 更鲁棒的替代

不是平方，取绝对值：

$$\text{MAD} = \frac{1}{N} \sum_{i=1}^{N} |x_i - \mu|$$

| 度量 | 优点 | 缺点 |
|------|------|------|
| 方差 | 数学方便，ML 常用 | 对异常值敏感 |
| MAD | 鲁棒 against 异常值 | 数学不便（不可导） |

## 📍 位置度量 - 值在哪里

### 四分位数 - 四等分点

**四分位数**将排序数据分成四等份：

```
数据排序：2, 3, 4, 5, 6, 7, 8, 9, 10
               ↑     ↑     ↑
              Q1    Q2    Q3
            (25%)  (50%) (75%)
```

- **Q1**（第 1 四分位数）= 25% 数据在其之下
- **Q2**（中位数）= 50% 数据在其之下
- **Q3**（第 3 四分位数）= 75% 数据在其之下

### 四分位距 (IQR)

$$\text{IQR} = Q3 - Q1$$

捕获中间 50% 数据的分布，忽略极端值！

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 箱线图

```
      须（最小值）
       │
  ┌────┴────┐  ← Q3
  │    ─    │  ← 中位数
  └────┬────┘  ← Q1
       │
      须（最大值）
       
       ●        ← 异常值
```

- 箱子：Q1 到 Q3（中间 50%）
- 中线：中位数
- 须：非异常值的极值
- 点：异常值

</div>
  <div class="lesson-visual-column">

```
IQR 的意义：

  IQR 小：       IQR 大：
  ┌─┐           ┌─────┐
  │ │           │     │
  └─┘           └─────┘
  数据集中       数据分散

中间 50% 的数据范围！
```

### z 分数 - 标准化位置

**z 分数**告诉你值距均值多少标准差：

$$z = \frac{x - \mu}{\sigma}$$

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的 z 分数</div>
  <div class="story-card-content">
    奶酪重量 $x = 105$g
    <br>
    均值 $\mu = 100$g
    <br>
    标准差 $\sigma = 2.5$g
    <br><br>
    $z = \frac{105 - 100}{2.5} = 2$
    <br><br>
    这块奶酪距均值<strong>2 个标准差</strong>！
  </div>
</div>

### 标准化（Standardization）

将任何分布转换为：
- 均值 = 0
- 标准差 = 1

$$x_{\text{new}} = \frac{x - \mu}{\sigma}$$

**ML 应用**：特征缩放，让不同特征有相同尺度

## 📐 形状度量 - 分布的几何

### 偏度 - 不对称程度

$$\text{Skewness} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^3$$

| 偏度 | 含义 | 例子 |
|------|------|------|
| = 0 | 对称 | 正态分布 |
| > 0 | 右尾长（正偏） | 收入分布 |
| < 0 | 左尾长（负偏） | 退休年龄 |

```
正偏：           负偏：
   ╱│╲             ╱│╲
  ╱ │ ╲           ╱ │ ╲
 ╱  │  ╲         ╱  │  ╲
╱   │   ╲       ╱   │   ╲
    └─→             ←─┘
  右尾长            左尾长
```

---

### 峰度 - 尾部重量

$$\text{Kurtosis} = \frac{1}{N} \sum_{i=1}^{N} \left(\frac{x_i - \mu}{\sigma}\right)^4$$

| 峰度 | 类型 | 尾部 |
|------|------|------|
| = 3 | 中峰（正态） | 标准 |
| > 3 | 尖峰 | 重尾（多异常值） |
| < 3 | 平峰 | 轻尾（少异常值） |

**超额峰度** = 峰度 - 3

## 🔗 相关性 - 变量间的关系

**相关性**测量两变量间关系的强度和方向：

<div class="lesson-two-column">
  <div class="lesson-text-column">

### Pearson 相关系数 ($r$)

测量**线性**关联：

$$r = \frac{\sum (x_i - \bar{x})(y_i - \bar{y})}{\sqrt{\sum (x_i - \bar{x})^2} \cdot \sqrt{\sum (y_i - \bar{y})^2}}$$

范围：$-1 \le r \le 1$

| r 值 | 含义 |
|------|------|
| +1 | 完美正相关 |
| 0 | 无线性相关 |
| -1 | 完美负相关 |

### Spearman 相关系数 ($\rho$)

测量**单调**关联（基于排名）

- 鲁棒 against 异常值
- 捕捉非线性但单调的关系

</div>
  <div class="lesson-visual-column">

```
相关性可视化：

正相关 (r≈1)    无相关 (r≈0)    负相关 (r≈-1)
  ↑                ↑                ↑
  │╲               │ ·              │╱
  │ ╲              │· ·             │ ╲
  │  ╲             │· · ·           │  ╲
  └────→           └────→           └────→
```

## 📊 其他重要度量

### 几何平均 - 相乘时的平均

适用于增长率：

$$\bar{x}_{\text{geo}} = \left(\prod_{i=1}^{N} x_i\right)^{1/N}$$

**例子**：投资增长 10%、20%、30%

$$\bar{x}_{\text{geo}} = (1.10 \times 1.20 \times 1.30)^{1/3} \approx 1.197$$

平均增长率 ≈ 19.7%（不是算术平均 20%）

---

### 指数移动平均 (EMA) - 重视最近数据

$$\text{EMA}_t = \alpha \cdot x_t + (1 - \alpha) \cdot \text{EMA}_{t-1}$$

- $\alpha$ = 平滑因子（0 到 1）
- $\alpha$ 大 → 更响应最近变化
- $\alpha$ 小 → 更平滑

**ML 应用**：Adam 优化器、批归一化的运行统计

```
EMA 可视化（α=0.3）：

  原始数据：╱╲╱╲╱╲╱╲
  EMA:      ╱──╲──╱──╲
  
  EMA 平滑噪声！
```

## 🚨 异常值检测

### IQR 方法（推荐）

异常值如果：
$$x < Q1 - 1.5 \times \text{IQR} \quad \text{或} \quad x > Q3 + 1.5 \times \text{IQR}$$

**优点**：不假设正态分布，鲁棒

### z 分数方法

异常值如果：
$$|z| > 3$$

**适用**：数据近似正态分布
