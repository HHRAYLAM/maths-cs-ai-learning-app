# 多元微积分

*多元微积分将导数和积分扩展到多变量函数，这对于具有数百万参数的 ML 模型至关重要。本章涵盖偏导数、梯度、雅可比矩阵、海森矩阵和使反向传播成为可能的多变量链式法则。*

- 到目前为止，我们的函数采用单个输入 $x$ 并产生单个输出 $f(x)$。但在 ML 中，我们几乎从不只处理一个变量。

- 考虑一个双变量函数，如 $f(x, y) = x^2 + y^2$。这在 3D 空间中定义了一个表面，一个碗形。我们想知道：如果我们在保持 $y$ 固定的同时轻微推动 $x$，$f$ 如何变化？这就是**偏导数**。

- $f$ 关于 $x$ 的**偏导数**，写作 $\frac{\partial f}{\partial x}$，将每个其他变量视为常数，并关于 $x$ 正常微分。

- 对于 $f(x, y) = x^2y + 3x - 2y$：

$$\frac{\partial f}{\partial x} = 2xy + 3 \qquad \frac{\partial f}{\partial y} = x^2 - 2$$

- 计算 $\frac{\partial f}{\partial x}$ 时，我们将 $y$ 视为常数，所以 $x^2y$ 微分为 $2xy$，$3x$ 微分为 $3$，$-2y$ 微分为 $0$。

- 计算 $\frac{\partial f}{\partial y}$ 时，我们将 $x$ 视为常数，所以 $x^2y$ 微分为 $x^2$，$3x$ 微分为 $0$，$-2y$ 微分为 $-2$。

- 从几何上讲，关于 $x$ 求偏导数就像用平行于 $xz$ 平面的平面（在固定 $y$ 值处）切片 3D 表面，并找到所得曲线的斜率。

![偏导数：通过固定一个变量切片表面](../images/partial_derivative.svg)

- **梯度** 将所有偏导数收集到一个向量中：

$$\nabla f = \left(\frac{\partial f}{\partial x_1}, \frac{\partial f}{\partial x_2}, \ldots, \frac{\partial f}{\partial x_n}\right)$$

- 对于 $f(x, y) = x^2 + y^2$：$\nabla f(x, y) = (2x, 2y)$。在点 $(1, 2)$：$\nabla f(1, 2) = (2, 4)$。

- 梯度有两个关键属性：

    - **方向**：它指向最陡增加的方向。想象一个在山上的徒步旅行者。他们位置的梯度指向上坡最陡的路径。

    - **大小**：$\|\nabla f\|$ 给出该最陡方向的增加率。大梯度意味着地形陡峭；小梯度意味着几乎平坦。

![梯度向量指向上坡，垂直于等高线](../images/gradient_contour.svg)

- 由于梯度指向上坡，沿相反方向移动（$-\nabla f$）会下坡，朝向较低值。这个简单的想法是**梯度下降**的基础，这是一种我们将在后续章节详细探讨的优化技术。目前，关键是梯度告诉你哪个方向是"上"以及攀登有多陡。

- **方向导数** 推广了偏导数。它不是问"$f$ 沿 $x$ 轴如何变化？"，而是问"$f$ 沿任意方向 $\mathbf{u}$ 如何变化？"它计算为梯度与单位向量的点积：

$$D_{\mathbf{u}} f = \nabla f \cdot \mathbf{u}$$

- 对于 $f(x, y) = x^2 + y^2$ 在 $(1, 2)$ 沿方向 $\mathbf{v} = (3, 4)$：首先归一化得到 $\mathbf{u} = (3/5, 4/5)$，然后 $D_{\mathbf{u}} f = (2, 4) \cdot (3/5, 4/5) = 6/5 + 16/5 = 22/5$。

- 偏导数是方向导数的特殊情况，其中方向沿坐标轴。如果方向导数在某个方向为零，函数在该点在该方向是平坦的。

- **等高线**（或水平曲线）连接函数具有相同值的点。对于 $f(x, y) = x^2 + y^2$，等高线是以原点为中心的圆：$x^2 + y^2 = c$，其中 $c$ 为不同值。

- 等高线永不相交（一个点不能有两个不同的函数值）。

- 梯度总是垂直于等高线，从较低值指向较高值。

- 紧密排列的等高线表示陡峭地形；稀疏排列的等高线表示缓坡。

- 到目前为止，我们的函数产生单个输出。但许多函数产生多个输出。函数 $\mathbf{F}: \mathbb{R}^n \to \mathbb{R}^m$ 接受 $n$ 个输入并产生 $m$ 个输出。**雅可比矩阵** 组织这样一个向量值函数的所有偏导数：

```math
J = \begin{bmatrix} \frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \\ \vdots & \ddots & \vdots \\ \frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n} \end{bmatrix}
```

- 雅可比的每一行是一个输出分量的梯度。对于具有 3 个输入和 2 个输出的函数，雅可比是 $2 \times 3$ 矩阵。

- 雅可比将导数推广到向量值函数。

- 就像标量函数的导数告诉你输出每单位输入变化多少一样，雅可比告诉你每个输出如何随每个输入变化。

- **雅可比的行列式** 度量变换在局部拉伸或压缩空间的程度。

- 如果行列式是 2，小区域面积翻倍。如果是 0，变换将空间坍缩到更低维度（从我们关于矩阵的章节回忆起，零行列式意味着奇异的、不可逆的变换）。

- 当组合多个变换（一个馈送到下一个）时，整体映射的雅可比是单个雅可比的乘积。我们将在后续章节看到这个想法变得至关重要。

- 梯度捕获一阶信息（斜率），而**海森矩阵** 捕获二阶信息（曲率）。

- 对于标量函数 $f(x_1, \ldots, x_n)$，海森矩阵是所有二阶偏导数的 $n \times n$ 矩阵：

```math
H = \begin{bmatrix} \frac{\partial^2 f}{\partial x_1^2} & \frac{\partial^2 f}{\partial x_1 \partial x_2} & \cdots \\ \frac{\partial^2 f}{\partial x_2 \partial x_1} & \frac{\partial^2 f}{\partial x_2^2} & \cdots \\ \vdots & \vdots & \ddots \end{bmatrix}
```

- 对于 $f(x, y) = x^3 + 2xy^2 - y^3$，梯度是 $(3x^2 + 2y^2,\; 4xy - 3y^2)$，海森矩阵是：

```math
H = \begin{bmatrix} 6x & 4y \\ 4y & 4x - 6y \end{bmatrix}
```

- 对角线条目（$6x$ 和 $4x - 6y$）告诉你 $x$ 方向的斜率如何随你在 $x$ 中移动而变化，$y$ 方向类似。

- 非对角线条目（$4y$）告诉你一个方向的斜率如何随你在另一个方向移动而变化。

- **克莱罗定理** 保证对于具有连续二阶导数的函数，混合偏导数相等：$\frac{\partial^2 f}{\partial x \partial y} = \frac{\partial^2 f}{\partial y \partial x}$。

- 这意味着海森矩阵是对称的，这（正如我们在矩阵章节看到的）保证实特征值和正交特征向量。

- 海森矩阵告诉我们函数在临界点（梯度为零的点）附近的形状：

    - 如果 $H$ 正定（所有特征值为正），该点是**局部最小值**，表面在每个方向向上弯曲，像一个碗。
    - 如果 $H$ 负定（所有特征值为负），该点是**局部最大值**，表面向下弯曲，像一个倒置的碗。
    - 如果 $H$ 同时具有正特征值和负特征值，该点是**鞍点**，表面在某些方向向上弯曲，在其他方向向下弯曲，像山口。

- **多变量链式法则** 将链式法则扩展到多个变量的函数。如果 $z = f(x, y)$，其中 $x = g(t)$ 且 $y = h(t)$，则：

$$\frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt}$$

- 从 $t$ 到 $z$ 的每条路径贡献一项：沿该路径的偏导数乘以中间变量关于 $t$ 的导数。

- 例如，如果 $z = x^2 y + 3x - y^2$，$x = \cos(t)$，$y = \sin(t)$：

$$\frac{dz}{dt} = (2xy + 3)(-\sin t) + (x^2 - 2y)(\cos t)$$

- 除了手动计算导数外，还有三种方法：

    - **数值微分**：近似 $f'(x) \approx \frac{f(x+h) - f(x-h)}{2h}$，对于小 $h$。简单但有噪声且不准确。
    - **符号微分**：代数地应用微分规则以产生精确公式。可能产生指数级增长的表达式。
    - **自动微分（autodiff）**：跟踪操作链并高效计算精确导数。这是 JAX、PyTorch 和 TensorFlow 使用的。它给出精确的数值（而非近似值），而无需产生臃肿的符号表达式。

1. 使用 `jax.grad` 计算 $f(x, y) = x^2 y + 3x - 2y$ 在点 $(1, 2)$ 的梯度。由于 $f$ 接受向量输入，使用 `jax.grad` 和 `argnums`。
```python
import jax
import jax.numpy as jnp

def f(x, y):
    return x**2 * y + 3*x - 2*y

df_dx = jax.grad(f, argnums=0)
df_dy = jax.grad(f, argnums=1)

x, y = 1.0, 2.0
print(f"∂f/∂x = {df_dx(x, y):.4f}  (预期：{2*x*y + 3:.4f})")
print(f"∂f/∂y = {df_dy(x, y):.4f}  (预期：{x**2 - 2:.4f})")
```

2. 使用 `jax.jacobian` 计算向量值函数的雅可比。与手动计算比较。
```python
import jax
import jax.numpy as jnp

def F(x):
    return jnp.array([x[0]**2 + x[1], x[0] * x[1]**2])

J = jax.jacobian(F)
x = jnp.array([1.0, 2.0])
print(f"(1,2) 处的雅可比:\n{J(x)}")
# 预期：[[2*x[0], 1], [x[1]**2, 2*x[0]*x[1]]] = [[2, 1], [4, 4]]
```

3. 使用 `jax.hessian` 计算 $f(x, y) = x^3 + 2xy^2 - y^3$ 的海森矩阵并验证它是对称的。
```python
import jax
import jax.numpy as jnp

def f(xy):
    x, y = xy[0], xy[1]
    return x**3 + 2*x*y**2 - y**3

H = jax.hessian(f)
point = jnp.array([1.0, 2.0])
hess = H(point)
print(f"海森矩阵:\n{hess}")
print(f"对称：{jnp.allclose(hess, hess.T)}")
# 预期：[[6x, 4y], [4y, 4x-6y]] = [[6, 8], [8, -8]]
```

4. 从头构建一个最小 autodiff 引擎。
    - 每个 `Var` 跟踪其值以及如何通过链式法则向后传播梯度。
    - 尝试用更多操作扩展它（除法、幂等）。
    - 这是 JAX、PyTorch 和 Numpy 设计的基础。
```python
class Var:
    def __init__(self, val, children=(), backward_fn=None):
        self.val = val
        self.grad = 0.0
        self.children = children
        self.backward_fn = backward_fn

    def __add__(self, other):
        out = Var(self.val + other.val, children=(self, other))
        def _backward():
            self.grad += out.grad    # d(a+b)/da = 1
            other.grad += out.grad   # d(a+b)/db = 1
        out.backward_fn = _backward
        return out

    def __mul__(self, other):
        out = Var(self.val * other.val, children=(self, other))
        def _backward():
            self.grad += other.val * out.grad  # d(a*b)/da = b
            other.grad += self.val * out.grad  # d(a*b)/db = a
        out.backward_fn = _backward
        return out

    def backward(self):
        # 拓扑排序然后传播梯度
        # 我们将在数据结构和算法中介绍这个
        order, visited = [], set()
        def topo(v):
            if v not in visited:
                visited.add(v)
                for c in v.children:
                    topo(c)
                order.append(v)
        topo(self)
        self.grad = 1.0
        for v in reversed(order):
            if v.backward_fn:
                v.backward_fn()

# f(x, y) = x*x*y + x 在 (3, 2)
x = Var(3.0)
y = Var(2.0)
f = x * x * y + x       # = 3*3*2 + 3 = 21

f.backward()
print(f"f = {f.val}")           # 21.0
print(f"df/dx = {x.grad}")     # 2*x*y + 1 = 13.0
print(f"df/dy = {y.grad}")     # x*x = 9.0
```
