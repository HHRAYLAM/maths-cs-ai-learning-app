# 卷积神经网络

*卷积神经网络直接从像素数据学习空间特征层次结构，用梯度优化的滤波器替换手工设计的滤波器。本文件涵盖卷积机制、池化、步幅、膨胀、感受野，以及定义图像分类的里程碑式架构（LeNet、AlexNet、VGG、ResNet、Inception、EfficientNet）。*

- 在文件 01 中，我们手工设计滤波器用于边缘检测、模糊和角点检测。自然的问题是：我们能从数据学习最优滤波器吗？这正是卷积神经网络 (CNN) 所做的。

- CNN 不是手工选择滤波器权重，而是通过梯度下降（第 06 章）学习它们，发现对手头任务直接有用的特征。

- 在第 06 章中，我们介绍了卷积操作、CNN 基础和滤波器学习的思想。这里我们更深入地探讨使 CNN 十多年来成为计算机视觉主导范式的架构创新。

- 回顾核心卷积操作：大小 $k \times k$ 的滤波器 $K$ 滑过输入特征图，在每个位置计算点积（第 06 章）。输出大小由三个超参数控制：

    - 步幅：滤波器在位置之间移动多少像素。步幅 1 表示滤波器一次移动一个像素。步幅 2 表示移动两个像素，将空间尺寸减半。跨步卷积是池化的替代下采样方案。
    - 填充：在输入边界周围添加零。"Same" 填充（$p = \lfloor k/2 \rfloor$）保持空间尺寸。"Valid" 填充（$p = 0$）减少它们。
    - 膨胀：在滤波器元素之间插入间隙。膨胀 2 的 3x3 滤波器仅用 9 个参数覆盖 5x5 感受野。膨胀卷积在不增加计算的情况下扩展感受野。

- 卷积后的输出空间大小：

$$\text{out} = \left\lfloor \frac{\text{in} - k + 2p}{s} \right\rfloor + 1$$

- 其中 $\text{in}$ 是输入大小，$k$ 是核大小，$p$ 是填充，$s$ 是步幅。该公式独立应用于高度和宽度。

- 神经元的感受野是原始输入中可以影响其值的区域。
    - 早期层有小的感受野（它们看到局部模式如边缘）。
    - 更深层有更大的感受野（它们看到更大的结构如物体部分）。
    
- 感受野随每层增长：大约每卷积层 $k - 1$ 像素（步幅或膨胀时更多）。

![感受野跨层增长：第 1 层神经元看到 3x3 补丁，第 2 层神经元看到 5x5 补丁，第 3 层神经元看到原始输入的 7x7 补丁](../images/receptive_field.svg)

- 池化层减少空间尺寸同时保留最重要信息。
    - 最大池化取每个窗口的最大值，保留最强激活（最突出的特征）。
    - 平均池化取均值，平滑特征图。带步幅 2 的 2x2 池将两个空间尺寸减半。

- 全局平均池化 (GAP) 将每个通道的整个空间范围平均为单个数字，产生长度等于通道数的向量。GAP 替换许多现代架构末端的全连接层，大幅减少参数数量并充当结构正则化器。

- 批归一化 (BatchNorm) 将每个小批量内的激活归一化为零均值和单位方差，然后应用可学习的缩放和平移（第 06 章）。在 CNN 中，BatchNorm 按通道应用：统计独立地对每个通道在批次和空间维度上计算。它稳定训练，允许更高学习率，并充当轻微正则化器。

- Dropout（第 06 章）在训练期间随机置零神经元。

- 在 CNN 中，空间 dropout (Dropout2D) 丢弃整个特征图通道而非单个像素，这更有效因为特征图中相邻像素高度相关。

- 数据增强通过在训练期间对每个图像应用随机转换人工扩展训练集：水平翻转、随机裁剪、旋转、颜色抖动（调整亮度、对比度、饱和度、色调）和 cutout（掩码随机矩形补丁）。网络以许多不同形式看到每个图像，强制它学习转换不变特征而非记忆特定像素模式。

- 高级增强策略包括Mixup（混合两个图像及其标签：$\tilde{x} = \lambda x_i + (1-\lambda) x_j$，$\tilde{y} = \lambda y_i + (1-\lambda) y_j$）、CutMix（将一个图像的矩形补丁粘贴到另一个图像上并按面积比例混合标签）和RandAugment（从固定集合中随机采样增强序列带单个强度参数）。

- CNN 架构的历史是逐步更深、更高效设计的故事，每个解决限制其前身的问题。

- LeNet-5 (LeCun et al., 1998) 是原始 CNN，为手写数字识别设计。两个卷积层后跟三个全连接层，带平均池化和 tanh 激活。它证明学习的滤波器优于手工设计特征，但按现代标准很小（60K 参数）。

- AlexNet (Krizhevsky et al., 2012) 以巨大优势赢得 ImageNet 竞赛，点燃深度学习革命。关键创新：ReLU 激活（替代 tanh，后者遭受梯度消失）、dropout 正则化、数据增强和在 GPU 上训练。五层卷积层，三层全连接层，6000 万参数。

- VGG (Simonyan and Zisserman, 2014) 表明仅使用 3x3 滤波器深层堆叠比大滤波器效果更好。两个堆叠的 3x3 滤波器有与一个 5x5 滤波器相同的感受野但更少参数（$2 \times 3^2 = 18$ vs $5^2 = 25$）和额外的非线性。VGG-16（16 层）和 VGG-19（19 层）仍广泛用作特征提取器。架构非常简单：卷积块带递增通道（64, 128, 256, 512），每个后跟最大池化。

![VGG 架构：堆叠 3x3 卷积块带递增通道深度（64→128→256→512），块间最大池化，以全连接层结束](../images/vgg_architecture.svg)

- GoogLeNet/Inception (Szegedy et al., 2014) 引入Inception 模块：不是选择单个滤波器大小，而是并行使用 1x1、3x3 和 5x5 卷积，连接它们的输出，让网络决定哪个尺度最有用。1x1 卷积在大滤波器前用作瓶颈减少计算。GoogLeNet 实现比 VGG 更好的精度但参数少 12 倍（6.8M vs 1.38 亿）。

![Inception 模块：四个并行分支（1×1、3×3、5×5 和池化）带 1×1 瓶颈，沿通道维度连接](../images/inception_module.svg)

- Inception 模块同时捕捉多尺度特征。1x1 滤波器捕捉逐点模式，3x3 捕捉局部纹理，5x5 捕捉更大结构。连接将所有视角组合成丰富表示。

- ResNet (He et al., 2016) 解决退化问题：更深的网络比较浅的网络表现更差，不是因为过拟合，因为它们更难优化。解决方案是跳跃连接（残差连接）：

$$\text{output} = F(x) + x$$

- 层学习残差 $F(x) = \text{output} - x$。如果最优变换接近恒等（这在深度网络中常见），学习接近零的残差比学习完整映射容易得多。跳跃连接还提供直接梯度通道，减少梯度消失。ResNet 训练 152 层网络，远深于以前任何东西。

![ResNet 块：输入 x 通过两个卷积层产生 F(x)，然后跳跃连接将 x 加回，给出输出 F(x) + x](../images/resnet_block.svg)

- 当输入和输出维度不同时（由于步幅或通道变化），投影捷径对 $x$ 应用 1x1 卷积匹配维度：$\text{output} = F(x) + W_s x$。

- 瓶颈块（用于 ResNet-50 和更深）使用三个卷积：1x1 减少通道，3x3 空间处理，1x1 扩展通道回来。这比两个 3x3 卷积便宜并允许更深网络。

- DenseNet (Huang et al., 2017) 将跳跃连接思想更进一步：密集块内每层连接到每个后续层。层 $l$ 接收所有前面层的特征图作为输入：$x_l = H_l([x_0, x_1, \ldots, x_{l-1}])$，其中 $[\cdot]$ 表示沿通道维度连接。这鼓励特征重用，增强梯度流，并减少总参数数量。

![DenseNet 密集块：每层通过连接接收所有前面层的特征图，创建密集连接实现最大特征重用](../images/densenet_block.svg)

- 高效架构针对移动设备和边缘硬件部署，其中计算、内存和能源受限。

- MobileNet (Howard et al., 2017) 用深度可分离卷积替换标准卷积，将操作分解为两步：
    1. 深度卷积：对每个输入通道应用单个 $k \times k$ 滤波器（无跨通道交互）
    2. 逐点卷积：应用 1x1 卷积跨通道组合信息

- 带 $C_{\text{in}}$ 输入通道和 $C_{\text{out}}$ 输出通道的标准 $k \times k$ 卷积每空间位置成本 $k^2 \cdot C_{\text{in}} \cdot C_{\text{out}}$ 次乘法。深度可分离卷积成本 $k^2 \cdot C_{\text{in}} + C_{\text{in}} \cdot C_{\text{out}}$，减少约 $k^2$ 倍。对于 3x3 滤波器，这约 9 倍便宜。

![深度可分离卷积：深度步对每通道应用一个 k×k 滤波器，然后逐点 1×1 卷积混合通道——相同输出形状，约 9 倍更少操作](../images/depthwise_separable_conv.svg)

- MobileNet-V2 引入逆残差块：用 1x1 卷积扩展通道，在扩展空间应用深度卷积，然后用 1x1 卷积投影回来。跳跃连接放在窄（瓶颈）层上，反转 ResNet 模式。扩展比通常 6。

- EfficientNet (Tan and Le, 2019) 引入复合缩放：不是独立缩放仅深度、仅宽度或仅分辨率，而是用固定比率一起缩放所有三个维度。给定缩放系数 $\phi$：

$$\text{depth}: d = \alpha^\phi, \quad \text{width}: w = \beta^\phi, \quad \text{resolution}: r = \gamma^\phi$$

- 满足 $\alpha \cdot \beta^2 \cdot \gamma^2 \approx 2$（所以 $\phi$ 每增加单位总计算大约翻倍）。网格搜索找到 $\alpha = 1.2$，$\beta = 1.1$，$\gamma = 1.15$ 作为基准比率。EfficientNet-B0 到 B7 逐步扩展，用远更少参数和 FLOPs 实现最先进精度。

![EfficientNet 复合缩放：单独缩放宽度、深度或分辨率 vs 用单个系数 φ 一起缩放所有三个](../images/efficientnet_scaling.svg)

- ShuffleNet 通过组卷积后跟通道打乱减少 1x1 卷积的成本（在 MobileNet 类架构中占主导）。组卷积将通道分成组并在每组内独立卷积，但这阻止跨组信息流。打乱操作在组间重排通道，以可忽略成本恢复信息混合。

- 迁移学习是将从一个任务训练的模型适应到不同任务的实践。在计算机视觉中，这几乎总是意味着从 ImageNet 预训练模型（140 万图像，1000 类）开始并适应领域特定数据集（医学图像、卫星图像、制造缺陷）。

- 特征提取：冻结所有卷积层，移除最终分类头，仅训练顶部新头。冻结层充当通用特征提取器。当目标域与 ImageNet 相似且目标数据集小时这效果好。

- 微调：解冻部分或全部卷积层并用小学习率训练。预训练权重作为起点而非固定特征。微调通常从仅解冻后面层开始（捕捉高级、任务特定特征）并可选地解冻更前面层。

- 迁移学习有效因为 CNN 的早期层学习通用特征（边缘、纹理、颜色）跨任务有用，而后面层学习任务特定特征。训练分类动物的网络仍有用于分类建筑的有用边缘检测器。

- 可视化 CNN 揭示网络学到了什么并帮助调试意外行为。

- 激活图（特征图）显示给定输入图像每个滤波器的输出。早期层激活看起来像边缘图；更深层产生越来越抽象、空间粗糙的激活。

- Grad-CAM（梯度加权类激活映射，Selvaraju et al., 2017）高亮输入图像中对模型预测最重要的区域。工作原理：
    1. 计算目标类分数对最后一层卷积层特征图的梯度（使用第 03 章的链式法则）
    2. 全局平均池化这些梯度获得每通道重要性权重
    3. 计算特征图的加权组合并应用 ReLU

$$L_{\text{Grad-CAM}} = \text{ReLU}\!\left(\sum_k \alpha_k A^k\right), \quad \alpha_k = \frac{1}{Z} \sum_i \sum_j \frac{\partial y^c}{\partial A^k_{ij}}$$

- 其中 $A^k$ 是第 $k$ 个特征图，$\alpha_k$ 是通道 $k$ 的重要性权重，$y^c$ 是类 $c$ 的分数。结果粗糙热图显示哪些区域驱动分类。应用 ReLU 因为我们对类有正面影响的特征感兴趣。

![Grad-CAM：狗的输入图像，最后一层卷积层的特征图，梯度加权组合，以及叠加在原始图像上的结果热图高亮狗的脸](../images/grad_cam.svg)

- 特征反演通过优化随机图像匹配目标特征从其特征表示重建输入图像（使用梯度下降在像素值上）。这揭示网络在每层保留什么信息。早期层重建近乎完美图像；更深层产生可识别但扭曲的图像，显示精细空间细节丢失而语义内容保留。

- Deep Dream 和神经风格迁移是特征可视化的创造性应用。Deep Dream 最大化选定层神经元的激活产生超现实、模式放大的图像。神经风格迁移优化目标图像匹配一个图像的内容特征（来自深层）和另一个图像的风格特征（滤波器激活的 Gram 矩阵，捕捉纹理统计）。

1. 用 JAX 从头实现简单 CNN 带两个卷积层、最大池化和分类头。在合成 2D 模式分类任务上训练它。
```python
import jax
import jax.numpy as jnp
import jax.lax as lax
import matplotlib.pyplot as plt

def conv2d(x, kernel, stride=1):
    """简单 2D 卷积用于单输入、单滤波器。"""
    return lax.conv(x[None, None], kernel[None, None], (stride, stride), 'SAME')[0, 0]

def max_pool(x, size=2):
    """2x2 最大池化。"""
    H, W = x.shape
    x = x[:H//size*size, :W//size*size]
    return x.reshape(H//size, size, W//size, size).max(axis=(1, 3))

def init_cnn(key):
    k1, k2, k3 = jax.random.split(key, 3)
    return {
        'conv1': jax.random.normal(k1, (5, 5)) * 0.3,
        'conv2': jax.random.normal(k2, (3, 3)) * 0.3,
        'fc_w': jax.random.normal(k3, (64, 1)) * 0.1,
        'fc_b': jnp.zeros(1),
    }

def forward_cnn(params, img):
    # Conv1 -> ReLU -> Pool
    h = jnp.maximum(0, conv2d(img, params['conv1']))
    h = max_pool(h)
    # Conv2 -> ReLU -> Pool
    h = jnp.maximum(0, conv2d(h, params['conv2']))
    h = max_pool(h)
    # 展平并分类
    flat = h.ravel()
    # 填充或截断到固定大小
    flat = jnp.pad(flat, (0, max(0, 64 - len(flat))))[:64]
    logit = (flat @ params['fc_w'] + params['fc_b']).squeeze()
    return jax.nn.sigmoid(logit)

# 生成合成数据：类别 0 = 低频模式，类别 1 = 高频
def make_data(key, n=200):
    images, labels = [], []
    for i in range(n):
        k1, key = jax.random.split(key)
        x, y = jnp.meshgrid(jnp.linspace(0, 4*jnp.pi, 32), jnp.linspace(0, 4*jnp.pi, 32))
        if i < n // 2:
            img = jnp.sin(x) + jax.random.normal(k1, (32, 32)) * 0.1
            labels.append(0)
        else:
            img = jnp.sin(4 * x) * jnp.sin(4 * y) + jax.random.normal(k1, (32, 32)) * 0.1
            labels.append(1)
        images.append(img)
    return images, jnp.array(labels, dtype=jnp.float32)

key = jax.random.PRNGKey(42)
images, labels = make_data(key)
params = init_cnn(jax.random.PRNGKey(0))

def loss_fn(params, img, label):
    pred = forward_cnn(params, img)
    return -(label * jnp.log(pred + 1e-7) + (1 - label) * jnp.log(1 - pred + 1e-7))

grad_fn = jax.grad(loss_fn)
lr = 0.01

for epoch in range(5):
    total_loss = 0.0
    for img, label in zip(images, labels):
        grads = grad_fn(params, img, label)
        params = {k: params[k] - lr * grads[k] for k in params}
        total_loss += loss_fn(params, img, label)
    print(f"Epoch {epoch}: loss = {total_loss / len(images):.4f}")

# 测试精度
preds = jnp.array([forward_cnn(params, img) > 0.5 for img in images])
acc = jnp.mean(preds == labels)
print(f"Accuracy: {acc:.2%}")
```

2. 可视化不同滤波器大小如何影响感受野。显示两个堆叠 3x3 滤波器覆盖与一个 5x5 滤波器相同的感受野但参数更少。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

def compute_receptive_field(layers):
    """从列表 (kernel_size, stride) 元组计算感受野大小。"""
    rf = 1  # 从 1 像素开始
    stride_product = 1
    for k, s in layers:
        rf += (k - 1) * stride_product
        stride_product *= s
    return rf

# 比较架构
configs = {
    '单个 5x5': [(5, 1)],
    '两个 3x3':    [(3, 1), (3, 1)],
    '三个 3x3':  [(3, 1), (3, 1), (3, 1)],
    '单个 7x7': [(7, 1)],
    '3x3 步幅 2 + 3x3': [(3, 2), (3, 1)],
}

print(f"{'配置':<25} {'RF':>4} {'参数 (每通道)':>20}")
print('-' * 55)
for name, layers in configs.items():
    rf = compute_receptive_field(layers)
    # 参数：每层 k^2 的总和（每输入 - 输出通道对）
    params = sum(k * k for k, s in layers)
    print(f"{name:<25} {rf:>4} {params:>20}")

# 可视化感受野
fig, axes = plt.subplots(1, 3, figsize=(14, 4))
for ax, (name, rf_size) in zip(axes, [('5x5 滤波器', 5), ('两个 3x3 滤波器', 5), ('三个 3x3 滤波器', 7)]):
    grid = jnp.zeros((9, 9))
    c = 4  # 中心
    half = rf_size // 2
    grid = grid.at[c-half:c+half+1, c-half:c+half+1].set(1.0)
    ax.imshow(grid, cmap='Blues', vmin=0, vmax=1)
    ax.set_title(f'{name}\nRF = {rf_size}x{rf_size}')
    ax.set_xticks(range(9)); ax.set_yticks(range(9))
    ax.grid(True, alpha=0.3)
plt.suptitle('感受野比较')
plt.tight_layout(); plt.show()
```

3. 从头实现 Grad-CAM。给定预构建简单 CNN，计算特定类的梯度加权激活图并可视化为热图。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

def simple_cnn(params, img):
    """返回预测和最后卷积激活的简单 CNN。"""
    # 卷积层（我们"最后卷积层"用于 Grad-CAM）
    H, W = img.shape
    k = params['conv'].shape[0]
    pad = k // 2
    img_pad = jnp.pad(img, pad, mode='edge')
    activation_map = jnp.zeros((H, W))
    for i in range(H):
        for j in range(W):
            activation_map = activation_map.at[i, j].set(
                jnp.sum(img_pad[i:i+k, j:j+k] * params['conv'])
            )
    activation_map = jnp.maximum(0, activation_map)  # ReLU

    # 全局平均池 -> 稠密 -> 输出
    pooled = activation_map.mean()
    logit = pooled * params['w'] + params['b']
    return jax.nn.sigmoid(logit), activation_map

# 创建测试图像：左侧亮区域（类别指示）
img = jnp.zeros((32, 32))

    'b': jnp.array(-0.5),

# 计算 Grad-CAM
def class_score(params, img):
    pred, _ = simple_cnn(params, img)
    return pred
```

4. 比较深度可分离卷积与标准卷积。计算两者的参数和 FLOPs 并显示它们以远更少计算产生相似输出。
```python
import jax
import jax.numpy as jnp

def standard_conv(x, kernel):
    """标准卷积：(H, W, C_in) * (k, k, C_in, C_out) -> (H, W, C_out)。"""
    H, W, C_in = x.shape
    k, _, _, C_out = kernel.shape
    pad = k // 2
    x_pad = jnp.pad(x, ((pad, pad), (pad, pad), (0, 0)), mode='constant')
    out = jnp.zeros((H, W, C_out))
    for i in range(H):
        for j in range(W):
            patch = x_pad[i:i+k, j:j+k, :]  # (k, k, C_in)
            for c in range(C_out):
                out = out.at[i, j, c].set(jnp.sum(patch * kernel[:, :, :, c]))
    return out

def depthwise_separable_conv(x, dw_kernel, pw_kernel):
    """深度可分离：深度 (k,k,C_in) 然后逐点 (C_in, C_out)。"""
    H, W, C_in = x.shape
    k = dw_kernel.shape[0]
    pad = k // 2
    x_pad = jnp.pad(x, ((pad, pad), (pad, pad), (0, 0)), mode='constant')

    # 深度：每通道一个滤波器
    dw_out = jnp.zeros((H, W, C_in))
    for i in range(H):
        for j in range(W):
            for c in range(C_in):
                patch = x_pad[i:i+k, j:j+k, c]
                dw_out = dw_out.at[i, j, c].set(jnp.sum(patch * dw_kernel[:, :, c]))

    # 逐点：跨通道 1x1 卷积
    out = dw_out @ pw_kernel
    return out

# 设置
H, W, C_in, C_out, k = 8, 8, 16, 32, 3
key = jax.random.PRNGKey(42)
k1, k2, k3, k4 = jax.random.split(key, 4)

x = jax.random.normal(k1, (H, W, C_in))
std_kernel = jax.random.normal(k2, (k, k, C_in, C_out)) * 0.1
dw_kernel = jax.random.normal(k3, (k, k, C_in)) * 0.1
pw_kernel = jax.random.normal(k4, (C_in, C_out)) * 0.1

# 比较
std_params = k * k * C_in * C_out
dw_params = k * k * C_in + C_in * C_out

std_flops = H * W * k * k * C_in * C_out
dw_flops = H * W * (k * k * C_in + C_in * C_out)

print(f"标准卷积：            {std_params:>8,} 参数，{std_flops:>10,} FLOPs")
print(f"深度可分离卷积：{dw_params:>8,} 参数，{dw_flops:>10,} FLOPs")
print(f"参数减少：      {std_params / dw_params:.1f}x")
print(f"FLOP 减少：           {std_flops / dw_flops:.1f}x")

std_out = standard_conv(x, std_kernel)
ds_out = depthwise_separable_conv(x, dw_kernel, pw_kernel)
print(f"\n标准输出形状：    {std_out.shape}")
print(f"深度可分离输出形状：{ds_out.shape}")
```
