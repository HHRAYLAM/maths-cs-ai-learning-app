## 📚 第一部分：什么是脑机接口？

### 核心概念

| 术语 | Tom 的比喻 | 技术定义 |
|------|-----------|---------|
| **脑机接口 (BCI)** | 🧠→🖥️ 用思想控制机器 | 大脑与外部设备的直接通信通道 |
| **神经信号** | 📡 大脑发出的电信号 | 神经元活动产生的电信号 |
| **解码算法** | 🔍 解读大脑想什么 | 将神经信号转换为控制指令 |
| **反馈回路** | 🔄 机器给大脑的回应 | 感觉反馈或视觉反馈 |

### BCI 的类型

```
┌─────────────────────────────────────────────────────────┐
│                    BCI 分类                             │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1️⃣ 侵入式 vs 非侵入式                                 │
│                                                         │
│     侵入式：🧠⚡ 电极直接植入大脑                        │
│            (信号清晰，但有手术风险)                     │
│                                                         │

│                                                         │
│  2️⃣ 主动式 vs 被动式                                   │

│     主动式：🤔"我想移动左手" → 控制假肢                 │

```

## 🔑 第二部分：BCI 工作原理

### 1. 信号采集（Signal Acquisition）

**Tom 的大脑信号捕捉器**：

```
🧠 大脑神经元活动
      ↓
📡 EEG 电极帽采集电信号
      ↓
📊 原始信号放大和数字化
      ↓
🔍 特征提取
      ↓
🤖 AI 解码算法
      ↓
🎮 控制指令输出
```

**常见信号类型**：

| 信号类型 | Tom 的比喻 | 特点 |
|---------|-----------|------|
| **EEG (脑电图)** | 📡 头皮上的电极帽 | 非侵入、时间分辨率高、空间分辨率低 |
| **ECoG (皮层脑电)** | 🧠📡 大脑表面的电极 | 侵入式、信号质量高 |
| **Spike signals** | ⚡ 单个神经元放电 | 最高精度、需要植入 |

**代码示例 - 简单的 EEG 分类器**：

```python
import numpy as np
from sklearn.discriminant_analysis import LDA

# Tom 的 BCI 训练数据
# X: EEG 信号特征 (时间戳 × 频率带)
# y: 意图标签 (0=静止，1=左手，2=右手，3=脚)

class SimpleBCIDecoder:
    def __init__(self):
        self.model = LDA()
    
    def train(self, X_train, y_train):
        """训练解码器"""
        # 提取特征：每个频带的功率
        features = self.extract_features(X_train)
        self.model.fit(features, y_train)
        print("🧠 BCI 解码器训练完成！")
    
    def predict(self, eeg_signal):
        """预测用户意图"""
        features = self.extract_features(eeg_signal)
        prediction = self.model.predict(features)
        return prediction
    
    def extract_features(self, eeg_data):
        """提取频带功率特征"""
        # 简化版：计算各频带的平均功率
        delta_power = np.mean(eeg_data[:, 0:4])
        theta_power = np.mean(eeg_data[:, 4:8])
        alpha_power = np.mean(eeg_data[:, 8:13])
        beta_power = np.mean(eeg_data[:, 13:30])
        gamma_power = np.mean(eeg_data[:, 30:100])
        
        return np.array([[delta_power, theta_power, alpha_power, 
                         beta_power, gamma_power]])
```
