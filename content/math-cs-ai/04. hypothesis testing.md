# 假设检验

*假设检验提供了一个严格的框架，用于判断观察到的效应是真实的还是由于偶然。本章涵盖零假设和备择假设、p 值、显著性水平、t 检验、卡方检验、ANOVA 以及 I/II 类错误——这些逻辑与 A/B 测试、模型比较和研究中使用的相同。*

- 统计学不仅仅是描述数据。通常你需要做出决策：新药是否有效？一个算法是否比另一个更快？平均值是否发生了变化？假设检验为你提供了一个使用数据回答这些问题的结构化框架。

- 想法很简单：假设没有发生变化（"零假设"），然后检查数据是否如此极端，以至于这个假设变得难以相信。

- **零假设**（$H_0$）是默认声明，通常是"无效应"或"无差异"的陈述。例如："平均交付时间仍然是 30 分钟"或"新模型不比旧模型好"。

- **备择假设**（$H_1$ 或 $H_a$）是你怀疑可能为真的内容："平均交付时间发生了变化"或"新模型更好"。

- 你永远不会直接证明 $H_1$。相反，你问：如果 $H_0$ 为真，我看到如此极端的数据的可能性有多大？如果非常不可能，你拒绝 $H_0$ 而支持 $H_1$。

- **检验统计量**是一个单一数字，总结你的样本结果与 $H_0$ 预测的内容相距多远。不同的检验使用不同的公式，但逻辑总是相同的：测量观察值与期望值之间的距离。

- **p 值**是在假设 $H_0$ 为真的情况下，观察到至少与你的一样极端的检验统计量的概率。小的 p 值意味着数据在 $H_0$ 下令人惊讶。

- **显著性水平**（$\alpha$）是你在查看数据之前设定的阈值。如果 $p \le \alpha$，你拒绝 $H_0$。常见选择是 $\alpha = 0.05$（5%）和 $\alpha = 0.01$（1%）。

![正态曲线，阴影显示拒绝域，标记检验统计量，高亮 p 值区域](../images/hypothesis_test.svg)

- 阴影尾部是拒绝域。如果你的检验统计量落在那里，数据在 $H_0$ 下足够令人惊讶，你拒绝它。绿色区域显示特定检验统计量的 p 值。

- 这是逐步过程：
    - **步骤 1**：陈述 $H_0$ 和 $H_1$
    - **步骤 2**：选择显著性水平 $\alpha$
    - **步骤 3**：收集数据并计算检验统计量
    - **步骤 4**：查找 p 值（或将检验统计量与临界值比较）
    - **步骤 5**：如果 $p \le \alpha$，拒绝 $H_0$。否则，无法拒绝 $H_0$

- **详细示例**：工厂声称他们的螺栓平均长度为 10 cm。你测量 36 个螺栓，发现样本均值为 10.3 cm。已知总体标准差为 0.9 cm。是否有证据表明平均值发生了变化？

- $H_0$：$\mu = 10$，$H_1$：$\mu \neq 10$，$\alpha = 0.05$

- 检验统计量（z 检验，因为 $\sigma$ 已知且 $n$ 大）：

$$z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}} = \frac{10.3 - 10}{0.9 / \sqrt{36}} = \frac{0.3}{0.15} = 2.0$$

- 对于 $\alpha = 0.05$ 的双尾检验，临界值是 $\pm 1.96$。我们的 $z = 2.0 > 1.96$，所以我们拒绝 $H_0$。p 值约为 0.046，小于 0.05。

- 结论：有统计学上显著的证据表明螺栓平均长度与 10 cm 不同。

- **单尾检验**检查特定方向的效应（$H_1$：$\mu > 10$ 或 $\mu < 10$）。整个 $\alpha$ 放入一个尾部，使其更容易在该方向拒绝 $H_0$，但不可能检测相反方向的效应。

- **双尾检验**检查任何差异（$H_1$：$\mu \neq 10$）。$\alpha$ 在两个尾部之间平分（每个 $\alpha/2$）。这更保守，但可以捕捉任一方向的效应。

- 即使有良好的过程，错误也会发生。恰好有两类错误：

![2x2 网格显示 I 类和 II 类错误：现实 vs 决策](../images/type_errors.svg)

- **I 类错误**（假阳性）：当 $H_0$ 实际为真时你拒绝了它。概率是 $\alpha$，你通过选择显著性水平来控制它。就像没有火灾时火警响起。

- **II 类错误**（假阴性）：当 $H_0$ 实际为假时你无法拒绝它。概率是 $\beta$。就像真实火灾时火警保持沉默。

- **功效**是 $1 - \beta$，正确拒绝虚假 $H_0$ 的概率。更高的功效意味着你更擅长检测真实效应。功效在以下情况增加：
    - 真实效应量更大（更大的差异更容易检测）
    - 样本量更大（更多数据 = 更精确）
    - 显著性水平 $\alpha$ 更大（但这增加了 I 类错误风险）
    - 变异性更低（更少噪声）

- I 类和 II 类错误之间存在张力。降低 $\alpha$（对假阳性更谨慎）会增加 $\beta$（更多假阴性）。在固定样本量下，你无法同时最小化两者。

- **参数检验**假设数据遵循特定分布（通常是正态分布）。当假设成立时，它们更强大。

- **Z 检验**：当 $\sigma$ 已知且 $n$ 大（$n \ge 30$）时，将样本均值与已知值比较。检验统计量：

$$z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}$$

- **T 检验**：与 z 检验类似，但用于 $\sigma$ 未知（从样本估计）或 $n$ 小的情况。使用 t 分布，它比正态分布有更重的尾部。更重的尾部解释了估计 $\sigma$ 的额外不确定性。

$$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$

- t 分布有一个称为**自由度**（$df = n - 1$）的参数。随着 $df$ 增加，t 分布接近正态分布。

- t 检验有几种变体：
    - **单样本 t 检验**：样本均值是否与特定值不同？
    - **独立双样本 t 检验**：两个独立组的均值是否不同？
    - **配对 t 检验**：两个相关测量的均值是否不同（例如同一受试者治疗前后）？

- **ANOVA（方差分析）**：检验三个或更多组均值是否相等。ANOVA 不是运行多个 t 检验（这会夸大 I 类错误率），而是通过比较组间方差与组内方差来进行单次检验。

$$F = \frac{\text{组间方差}}{\text{组内方差}}$$

- 大的 $F$ 比率意味着组间的差异超出了仅从随机变异中预期的程度。

- **非参数检验**对数据分布的假设更少。它们基于秩次而非原始值，使其对异常值和非正态性稳健。

- **卡方检验**（$\chi^2$）：检验观察频率是否与期望频率匹配。用于分类数据。例如：红色、蓝色和绿色汽车的比例是否与制造商声称的比例匹配？

$$\chi^2 = \sum \frac{(O_i - E_i)^2}{E_i}$$

- **Mann-Whitney U 检验**：独立双样本 t 检验的非参数替代。它通过比较秩次检验一组是否倾向于有更大的值。

- **Wilcoxon 符号秩检验**：配对 t 检验的非参数替代。通过查看差异的大小和方向来比较配对观察值。

- **Kruskal-Wallis 检验**：单因素 ANOVA 的非参数替代。通过比较所有组的秩次检验多个组是否来自同一分布。

- **拟合优度检验** 检查你的数据是否遵循特定的理论分布。卡方拟合优度检验将观察到的 bin 计数与假设分布下的期望计数进行比较。

- **正态性检验** 专门检查数据是否呈正态分布。常见的包括 Shapiro-Wilk 检验（对小样本强大）和 Kolmogorov-Smirnov 检验（将样本 CDF 与理论 CDF 比较）。

- 在机器学习中，假设检验出现在比较模型性能时。如果模型 A 达到 92% 的准确率，模型 B 达到 91%，差异是真实的还是只是噪声？对交叉验证分数的配对 t 检验可以回答这个问题。

## 编码任务（使用 CoLab 或 notebook）

1. 对螺栓工厂示例执行 z 检验。计算检验统计量、p 值并做出决策。
```python
import jax.numpy as jnp

x_bar = 10.3    # sample mean
mu_0 = 10.0     # null hypothesis value
sigma = 0.9     # known population std
n = 36           # sample size
alpha = 0.05

# Test statistic
z = (x_bar - mu_0) / (sigma / jnp.sqrt(n))
print(f"z = {z:.4f}")

# p-value (two-tailed) using the normal CDF approximation
# For |z| = 2.0, p ≈ 0.0456
from jax.scipy.stats import norm
p_value = 2 * (1 - norm.cdf(jnp.abs(z)))
print(f"p-value = {p_value:.4f}")
print(f"Reject H₀? {p_value <= alpha}")
```

2. 模拟 I 类错误：当 $H_0$ 为真时，我们多久错误地拒绝它一次？运行 10,000 次实验，检查拒绝率是否与 $\alpha$ 匹配。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(0)
mu_0 = 50.0
sigma = 10.0
n = 30
alpha = 0.05
n_experiments = 10_000

rejections = 0
for i in range(n_experiments):
    key, subkey = jax.random.split(key)
    sample = mu_0 + sigma * jax.random.normal(subkey, shape=(n,))
    z = (sample.mean() - mu_0) / (sigma / jnp.sqrt(n))
    p_value = 2 * (1 - __import__("jax").scipy.stats.norm.cdf(jnp.abs(z)))
    if p_value <= alpha:
        rejections += 1

print(f"Rejection rate: {rejections/n_experiments:.4f}")
print(f"Expected (α):   {alpha}")
```

3. 在两组上比较 t 检验和 Mann-Whitney U 检验。生成一组均值略高于另一组的数据，看看哪个检验检测到差异。
```python
import jax
import jax.numpy as jnp

key = jax.random.PRNGKey(99)
k1, k2 = jax.random.split(key)

group_a = jax.random.normal(k1, shape=(25,)) * 5 + 100
group_b = jax.random.normal(k2, shape=(25,)) * 5 + 103  # slightly higher mean

# Two-sample t-test (equal variance assumed)
n_a, n_b = len(group_a), len(group_b)
mean_a, mean_b = group_a.mean(), group_b.mean()
pooled_var = ((n_a - 1) * group_a.var() + (n_b - 1) * group_b.var()) / (n_a + n_b - 2)
se = jnp.sqrt(pooled_var * (1/n_a + 1/n_b))
t_stat = (mean_a - mean_b) / se
print(f"T-test statistic: {t_stat:.4f}")

# Mann-Whitney: count how often group_a values beat group_b values
u_stat = jnp.sum(group_a[:, None] < group_b[None, :])
print(f"Mann-Whitney U:   {u_stat}")
print(f"\nGroup A mean: {mean_a:.2f}, Group B mean: {mean_b:.2f}")
```
