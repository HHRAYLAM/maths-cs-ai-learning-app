# 强化学习

*强化学习通过试错最大化累积奖励来训练智能体做序列决策。本章涵盖 MDP、价值函数、贝尔曼方程、Q 学习、策略梯度、演员 - 评论家方法、PPO 和 RLHF——这些是游戏智能体和语言模型对齐背后的框架。*

- 监督学习需要标记数据。无监督学习在无标签数据中发现模式。**强化学习（RL）** 与两者都不同：智能体通过与环境交互、采取行动和接收奖励来学习。没有正确的标签；智能体必须通过试错发现良好行为。

- 想想教狗新把戏。你不会给它看正确行为的数据集。相反，它尝试各种动作，你对好动作给予奖励，随着时间的推移它弄明白你想要什么。RL 形式化了这个过程。

- RL 设置有五个核心组件。**智能体**是学习者和决策者。**环境**是智能体交互的外部一切。在每个时间步，智能体观察一个**状态**$s_t$，选择一个**动作**$a_t$，接收一个**奖励**$r_t$，并转移到新状态 $s_{t+1}$。智能体的目标是最大化随时间收集的总奖励。

![智能体 - 环境循环：智能体观察状态、采取行动、接收奖励、环境转移到新状态](../images/mdp_agent_loop.svg)

- **策略**$\pi$ 是智能体的策略：从状态到动作的映射。确定性策略每个状态给出一个动作：$a = \pi(s)$。随机策略给出动作上的概率分布：$\pi(a \mid s)$。RL 的目标是找到最优策略，即最大化期望累积奖励的策略。

- RL 的数学框架是**马尔可夫决策过程（MDP）**，由元组 $(S, A, P, R, \gamma)$ 定义：状态集 $S$、动作集 $A$、转移概率 $P(s' \mid s, a)$、奖励函数 $R(s, a)$ 和折扣因子 $\gamma$。

- **马尔可夫性质**（来自第 05 章）表示未来只依赖于当前状态，而不依赖于如何到达那里的历史：$P(s_{t+1} \mid s_t, a_t, s_{t-1}, \ldots) = P(s_{t+1} \mid s_t, a_t)$。这意味着状态包含做决策所需的所有信息。

- **折扣因子**$\gamma \in [0, 1)$ 决定智能体关心未来奖励与即时奖励的程度。从时间 $t$ 开始的折扣回报是：

$$G_t = r_t + \gamma r_{t+1} + \gamma^2 r_{t+2} + \cdots = \sum_{k=0}^{\infty} \gamma^k r_{t+k}$$

- 当 $\gamma = 0$ 时，智能体完全短视，只关心下一个奖励。当 $\gamma$ 接近 1 时，智能体有远见。折扣因子还确保总和收敛（如果奖励有界），这对数学上的良定义很重要。

- **价值函数**估计处于某个状态（或在某个状态采取某个动作）有多好。**状态价值函数**$V^\pi(s)$ 是从状态 $s$ 开始并遵循策略 $\pi$ 的期望回报：

$$V^\pi(s) = \mathbb{E}_\pi \left[ G_t \mid s_t = s \right]$$

- **动作价值函数**$Q^\pi(s, a)$ 是从状态 $s$ 开始、采取行动 $a$ 然后遵循 $\pi$ 的期望回报：

$$Q^\pi(s, a) = \mathbb{E}_\pi \left[ G_t \mid s_t = s, a_t = a \right]$$

- 关系：$V^\pi(s) = \sum_a \pi(a \mid s) \, Q^\pi(s, a)$。状态价值是动作价值的平均值，由策略加权。

- **贝尔曼方程**表达递归关系：状态的价值等于即时奖励加上下一个状态的折扣价值。对于状态价值函数：

$$V^\pi(s) = \sum_a \pi(a \mid s) \sum_{s'} P(s' \mid s, a) \left[ R(s, a) + \gamma \, V^\pi(s') \right]$$

- 对于最优价值函数 $V^{*}(s)$，智能体总是选择最佳动作：

$$V^{*}(s) = \max_a \sum_{s'} P(s' \mid s, a) \left[ R(s, a) + \gamma \, V^{*}(s') \right]$$

- 类似地，$Q^{*}$ 的**贝尔曼最优方程**：

$$Q^{*}(s, a) = \sum_{s'} P(s' \mid s, a) \left[ R(s, a) + \gamma \max_{a'} Q^{*}(s', a') \right]$$

- 一旦你有了 $Q^{*}$，最优策略是平凡的：总是选择具有最高 Q 值的动作：$\pi^{*}(s) = \arg\max_a Q^{*}(s, a)$。

- **动态规划**方法在你知道转移概率和奖励（完整模型）时求解 MDP。**策略评估**通过迭代应用贝尔曼方程直到收敛来计算给定策略的 $V^\pi$。**策略改进**获取价值函数并通过贪婪行为构建更好的策略：$\pi'(s) = \arg\max_a \sum_{s'} P(s' \mid s, a)[R(s,a) + \gamma V^\pi(s')]$。

- **策略迭代**在评估和改进之间交替，直到策略停止变化。保证收敛到最优策略。

- **价值迭代**将两个步骤合并为一个：重复应用贝尔曼最优方程直到 $V^{*}$ 收敛，然后提取策略。

$$V(s) \leftarrow \max_a \sum_{s'} P(s' \mid s, a) \left[ R(s, a) + \gamma \, V(s') \right]$$

- 动态规划需要知道 $P(s' \mid s, a)$，这通常不切实际。在大多数现实问题中，智能体不知道环境的动态；它只能与环境交互。这就是**无模型**方法发挥作用的地方。

- **时序差分（TD）学习**从经验中学习而无需知道模型。关键思想是**自举**：不是等到回合结束计算实际回报 $G_t$，而是使用当前价值函数估计它：

$$V(s_t) \leftarrow V(s_t) + \alpha \left[ r_t + \gamma \, V(s_{t+1}) - V(s_t) \right]$$

- 括号中的项是**TD 误差**：**TD 目标**（$r_t + \gamma V(s_{t+1})$）与当前估计 $V(s_t)$ 之间的差异。如果 TD 误差为正，状态比预期好，所以我们增加其价值。如果为负，我们减少它。

![状态转换显示 TD 目标：当前价值、奖励和自举的下一个价值，以及更新公式](../images/td_update.svg)

- TD 学习在每一步后更新（而不是在完整回合后），这使其比蒙特卡洛方法高效得多。它也适用于持续性（非回合制）环境。

- **SARSA**（状态 - 动作 - 奖励 - 状态 - 动作）是应用于 Q 值的 TD 学习。智能体在状态 $s$ 采取动作 $a$，观察奖励 $r$ 和下一个状态 $s'$，然后根据策略选择下一个动作 $a'$：

$$Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \, Q(s', a') - Q(s, a) \right]$$

- SARSA 是**on-policy**：它使用智能体实际采取的动作进行更新，包括探索。这使 SARSA 更保守；它学习的策略考虑了自己的探索噪声。

- **Q 学习**是最著名的 RL 算法。它像 SARSA，但它不使用智能体实际采取的动作，而是使用最佳可能动作：

$$Q(s, a) \leftarrow Q(s, a) + \alpha \left[ r + \gamma \max_{a'} Q(s', a') - Q(s, a) \right]$$

- Q 学习是**off-policy**：它学习最优 Q 值，无论遵循什么策略。智能体可以随机探索，同时仍然学习最优动作价值。这使 Q 学习更激进，通常收敛更快，但可能高估价值。

- **探索与利用**是基本困境：智能体应该利用已知的内容（选择估计价值最高的动作）还是探索未知动作（可能结果更好）？

- 最简单的策略是**epsilon-贪婪**：以概率 $\epsilon$ 采取随机动作（探索）；以概率 $1 - \epsilon$ 采取贪婪动作（利用）。常见调度从高 $\epsilon$（大量探索）开始，并随时间衰减。

- 表格方法（在表格中为每个状态 - 动作对存储一个价值）适用于小型离散状态空间。对于大型或连续状态空间，你需要函数逼近。**深度 Q 网络（DQN）** 使用神经网络逼近 $Q(s, a; \theta)$，其中 $\theta$ 是网络权重。

- DQN 引入了两种关键的稳定技术。**经验回放**：不是从连续转移中学习（高度相关），而是将转移存储在回放缓冲区中，并随机采样小批量进行训练。这打破了相关性并有效重用数据。

- **目标网络**：使用单独的、缓慢更新的网络副本来计算 TD 目标。没有这个，目标会在你每次更新网络时移动，造成"追逐自己尾巴"的不稳定性。目标网络定期更新（每 $N$ 步硬更新）或持续更新（软更新：$\theta^{-} \leftarrow \tau\theta + (1-\tau)\theta^{-}$）。

- DQN 损失只是预测 Q 值与 TD 目标之间的 MSE：

$$\mathcal{L}(\theta) = \mathbb{E} \left[ \left( r + \gamma \max_{a'} Q(s', a'; \theta^{-}) - Q(s, a; \theta) \right)^2 \right]$$

- 到目前为止的所有方法都学习价值函数并从中导出策略。**策略梯度**方法采用不同的方法：它们直接参数化策略 $\pi(a \mid s; \theta)$，并通过期望回报的梯度上升优化它。

- **策略梯度定理**给出期望回报关于策略参数的梯度：

$$\nabla_\theta J(\theta) = \mathbb{E}_\pi \left[ \nabla_\theta \log \pi(a \mid s; \theta) \cdot G_t \right]$$

- 这表明：增加导致高回报的动作的概率，减少导致低回报的动作的概率。对数概率梯度给出改变策略的方向，$G_t$ 缩放改变的量。

- **REINFORCE** 是最简单的策略梯度算法。运行一个回合，计算每一步的回报 $G_t$，然后更新：

$$\theta \leftarrow \theta + \alpha \, \nabla_\theta \log \pi(a_t \mid s_t; \theta) \cdot G_t$$

- REINFORCE 具有高方差，因为 $G_t$ 是期望回报的有噪声的单样本估计。一个常见的修复是减去**基线**（通常是平均回报或学习的价值函数），以减少方差而不引入偏差：

$$\theta \leftarrow \theta + \alpha \, \nabla_\theta \log \pi(a_t \mid s_t; \theta) \cdot (G_t - b)$$

- **演员 - 评论家**方法使用两个网络。**演员**是策略 $\pi(a \mid s; \theta)$。**评论家**是价值函数 $V(s; \phi)$，作为基线。优势 $A_t = r_t + \gamma V(s_{t+1}) - V(s_t)$ 替换 $G_t - b$：

$$\theta \leftarrow \theta + \alpha \, \nabla_\theta \log \pi(a_t \mid s_t; \theta) \cdot A_t$$

- 评论家通过最小化 TD 误差进行更新，就像基于价值的方法一样。演员使用策略梯度进行更新，评论家的优势估计减少方差。这是两全其美。

![双头架构：演员输出动作概率，评论家输出价值估计，优势信号指导演员更新](../images/actor_critic.svg)

- **PPO**（近端策略优化）是实践中使用最广泛的策略梯度算法。它解决了一个关键问题：如果策略更新过大，性能可能会灾难性地崩溃。

- PPO 使用**裁剪代理目标**。令 $r_t(\theta) = \frac{\pi(a_t | s_t; \theta)}{\pi(a_t | s_t; \theta_{\text{old}})}$ 为新策略与旧策略之间的概率比。损失是：

$$\mathcal{L}^{\text{CLIP}}(\theta) = \mathbb{E} \left[ \min\!\left( r_t(\theta) A_t, \; \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) A_t \right) \right]$$

- 裁剪（通常 $\epsilon = 0.2$）防止比率远离 1，使更新保持小而稳定。如果优势为正（动作好），比率上限为 $1 + \epsilon$。如果为负（动作差），比率上限为 $1 - \epsilon$。这比早期的信任域方法（TRPO）更简单、更稳定。

- PPO 正是通过 **RLHF**（人类反馈强化学习）训练 ChatGPT 风格模型所使用的算法。在 RLHF 中，奖励模型在人类偏好数据上训练（人类偏好两个输出中的哪一个？），然后 PPO 优化语言模型的策略以最大化这个学习到的奖励。

- **DPO**（直接偏好优化）通过完全消除奖励模型简化了 RLHF。DPO 不是训练奖励模型然后运行 RL，而是推导出一个闭式损失，直接从偏好数据优化策略：

$$\mathcal{L}_{\text{DPO}}(\theta) = -\mathbb{E} \left[ \log \sigma\!\left( \beta \log \frac{\pi_\theta(y_w \mid x)}{\pi_{\text{ref}}(y_w \mid x)} - \beta \log \frac{\pi_\theta(y_l \mid x)}{\pi_{\text{ref}}(y_l \mid x)} \right) \right]$$

- 这里 $y_w$ 是偏好（获胜）响应，$y_l$ 是不偏好（失败）响应。DPO 增加偏好输出的相对概率，实现比基于 PPO 的 RLHF 简单得多。

- RL 算法的两个重要区别。**On-policy 与 off-policy**：on-policy 方法（SARSA、PPO）从当前策略生成的数据中学习；off-policy 方法（Q 学习、DQN）可以从任何策略生成的数据中学习。Off-policy 方法样本效率更高（重用旧数据），但可能更不稳定。

- **基于模型与无模型**：无模型方法（到目前为止讨论的所有内容）直接从经验学习价值或策略。基于模型的方法学习环境模型（$P(s' \mid s, a)$ 和 $R(s, a)$）并用它进行规划（想象未来轨迹而不实际采取行动）。基于模型的方法样本效率更高，但增加了学习准确模型的复杂性。

- 总结 RL 格局：

| 方法 | 类型 | 关键思想 | 优势 |
|---|---|---|---|
| 价值迭代 | DP，基于模型 | 贝尔曼最优 | 精确解（小 MDP） |
| SARSA | TD，on-policy | on-policy 学习 Q | 保守，安全 |
| Q 学习 | TD，off-policy | 学习 Q*，贪婪目标 | 简单，有效 |
| DQN | 深度，off-policy | 神经 Q + 回放 + 目标网 | 扩展到状态高维 |
| REINFORCE | 策略梯度 | log 概率梯度 * 回报 | 简单策略优化 |
| 演员 - 评论家 | PG + 价值 | 演员 + 评论家低方差 | 实用灵活 |
| PPO | PG，裁剪 | 信任域般稳定 | 行业标准 |
| DPO | 直接偏好 | 跳过奖励模型 | 更简单 RLHF |

1. 为简单网格世界实现价值迭代。计算最优价值函数并提取最优策略。将两者可视化为热图和箭头图。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 4x4 gridworld: goal at (3,3), reward -1 per step, 0 at goal
grid_size = 4
gamma = 0.99
goal = (3, 3)

# Actions: up, down, left, right
actions = [(-1, 0), (1, 0), (0, -1), (0, 1)]
action_names = ['up', 'down', 'left', 'right']
action_arrows = ['\u2191', '\u2193', '\u2190', '\u2192']

def step(s, a):
    """Deterministic transition."""
    ns = (max(0, min(grid_size-1, s[0]+a[0])),
          max(0, min(grid_size-1, s[1]+a[1])))
    return ns

# Value iteration
V = jnp.zeros((grid_size, grid_size))
for iteration in range(100):
    V_new = jnp.array(V)
    for i in range(grid_size):
        for j in range(grid_size):
            if (i, j) == goal:
                continue
            values = []
            for a in actions:
                ns = step((i, j), a)
                values.append(-1 + gamma * float(V[ns[0], ns[1]]))
            V_new = V_new.at[i, j].set(max(values))
    if jnp.max(jnp.abs(V_new - V)) < 1e-6:
        print(f"Converged in {iteration+1} iterations")
        break
    V = V_new

# Extract policy
policy = [['' for _ in range(grid_size)] for _ in range(grid_size)]
for i in range(grid_size):
    for j in range(grid_size):
        if (i, j) == goal:
            policy[i][j] = 'G'
            continue
        best_a = max(range(4), key=lambda a: -1 + gamma * float(V[step((i,j), actions[a])[0], step((i,j), actions[a])[1]]))
        policy[i][j] = action_arrows[best_a]

fig, axes = plt.subplots(1, 2, figsize=(10, 4))
im = axes[0].imshow(V, cmap='YlOrRd_r')
axes[0].set_title("Optimal Value Function")
for i in range(grid_size):
    for j in range(grid_size):
        axes[0].text(j, i, f"{V[i,j]:.1f}", ha='center', va='center', fontsize=10)
plt.colorbar(im, ax=axes[0])

axes[1].imshow(jnp.ones((grid_size, grid_size)), cmap='Greys', vmin=0, vmax=2)
axes[1].set_title("Optimal Policy")
for i in range(grid_size):
    for j in range(grid_size):
        axes[1].text(j, i, policy[i][j], ha='center', va='center', fontsize=18)
plt.tight_layout(); plt.show()
```

2. 在简单网格世界上实现表格 Q 学习。训练智能体，绘制学习曲线，并显示学习到的 Q 值。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

grid_size = 5
goal = (4, 4)
actions = [(-1,0), (1,0), (0,-1), (0,1)]

# Q-table
Q = {}
for i in range(grid_size):
    for j in range(grid_size):
        Q[(i,j)] = [0.0] * 4

alpha = 0.1
gamma = 0.95
epsilon = 1.0
epsilon_decay = 0.995
min_epsilon = 0.01

def step(s, a_idx):
    a = actions[a_idx]
    ns = (max(0, min(grid_size-1, s[0]+a[0])),
          max(0, min(grid_size-1, s[1]+a[1])))
    r = 0.0 if ns == goal else -1.0
    done = ns == goal
    return ns, r, done

key = jax.random.PRNGKey(42)
rewards_per_episode = []

for ep in range(500):
    s = (0, 0)
    total_reward = 0
    for _ in range(100):
        key, subkey = jax.random.split(key)
        if float(jax.random.uniform(subkey)) < epsilon:
            key, subkey = jax.random.split(key)
            a = int(jax.random.randint(subkey, (), 0, 4))
        else:
            a = max(range(4), key=lambda i: Q[s][i])

        ns, r, done = step(s, a)
        total_reward += r
        # Q-learning update
        Q[s][a] += alpha * (r + gamma * max(Q[ns]) - Q[s][a])
        s = ns
        if done:
            break
    rewards_per_episode.append(total_reward)
    epsilon = max(min_epsilon, epsilon * epsilon_decay)

plt.figure(figsize=(8, 4))
# Smooth the curve
window = 20
smoothed = [sum(rewards_per_episode[max(0,i-window):i+1])/min(i+1, window)
            for i in range(len(rewards_per_episode))]
plt.plot(smoothed, color='#3498db', linewidth=1.5)
plt.xlabel("Episode"); plt.ylabel("Total Reward (smoothed)")
plt.title("Q-Learning on Gridworld")
plt.grid(alpha=0.3); plt.show()

# Show learned policy
arrow = ['\u2191', '\u2193', '\u2190', '\u2192']
print("Learned policy:")
for i in range(grid_size):
    row = ""
    for j in range(grid_size):
        if (i,j) == goal:
            row += " G "
        else:
            row += f" {arrow[max(range(4), key=lambda a: Q[(i,j)][a])]} "
    print(row)
```

3. 在多臂老虎机问题上实现 REINFORCE。展示策略如何在训练过程中演变以 favor 最佳臂。
```python
import jax
import jax.numpy as jnp
import matplotlib.pyplot as plt

# 5-armed bandit with different expected rewards
true_rewards = jnp.array([0.2, 0.5, 0.8, 0.3, 0.1])
n_arms = len(true_rewards)

# Policy: softmax over logits
logits = jnp.zeros(n_arms)
lr = 0.1
key = jax.random.PRNGKey(42)

policy_history = []
reward_history = []

for step in range(2000):
    probs = jax.nn.softmax(logits)
    policy_history.append(probs)

    # Sample action
    key, subkey = jax.random.split(key)
    action = jax.random.choice(subkey, n_arms, p=probs)

    # Get reward (Bernoulli)
    key, subkey = jax.random.split(key)
    reward = float(jax.random.uniform(subkey) < true_rewards[action])
    reward_history.append(reward)

    # REINFORCE update
    # grad log pi(a) = e_a - probs (for softmax parameterisation)
    grad_log_pi = -probs.at[action].add(1.0)  # one-hot(a) - probs
    logits = logits + lr * reward * grad_log_pi

policy_history = jnp.stack(policy_history)

fig, axes = plt.subplots(1, 2, figsize=(12, 4))
colors = ['#3498db', '#e74c3c', '#27ae60', '#9b59b6', '#f39c12']
for i in range(n_arms):
    axes[0].plot(policy_history[:, i], color=colors[i],
                 label=f'Arm {i} (true={true_rewards[i]:.1f})', linewidth=1.5)
axes[0].set_xlabel("Step"); axes[0].set_ylabel("P(arm)")
axes[0].set_title("Policy Evolution (REINFORCE)")
axes[0].legend(fontsize=8); axes[0].grid(alpha=0.3)

# Smoothed reward
window = 50
smoothed = [sum(reward_history[max(0,i-window):i+1])/min(i+1,window)
            for i in range(len(reward_history))]
axes[1].plot(smoothed, color='#27ae60', linewidth=1.5)
axes[1].axhline(y=0.8, color='#e74c3c', linestyle='--', alpha=0.5, label='Best arm')
axes[1].set_xlabel("Step"); axes[1].set_ylabel("Avg Reward")
axes[1].set_title("Reward Over Time"); axes[1].legend()
axes[1].grid(alpha=0.3)
plt.tight_layout(); plt.show()
```
