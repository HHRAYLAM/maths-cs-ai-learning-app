## 🔑 第二部分：分子表示学习

### 分子表示方法

Tom 的"分子语言"翻译器：

```
┌─────────────────────────────────────────────────────────┐
│              分子表示方法对比                           │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  1️⃣ SMILES 字符串 (线性表示)                            │
│     例子：CCO (乙醇)                                    │
│     优点：简洁、人类可读                                │
│     缺点：同一分子有多种 SMILES、结构信息丢失           │
│     Tom 比喻：分子的"拼音"写法                          │
│                                                         │
│  2️⃣ 分子图 (Graph 表示)                                 │
│     节点 = 原子，边 = 化学键                            │
│     优点：保留结构信息、适合 GNN                        │
│     缺点：计算复杂                                      │
│     Tom 比喻：分子"地图"                                │
│                                                         │
│  3️⃣ 分子指纹 (Fingerprint)                              │
│     固定长度二进制向量                                  │
│     优点：快速比较、适合相似性搜索                      │
│     缺点：信息有损                                      │
│     Tom 比喻：分子的"指纹 ID"                           │
│                                                         │
│  4️⃣ 3D 构象 (Conformer)                                 │
│     原子的三维坐标                                      │
│     优点：包含空间信息、对接必需                        │
│     缺点：构象多样、计算量大                            │
│     Tom 比喻：分子的"雕塑"                              │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

代码示例 - 分子特征提取：

```python
from rdkit import Chem
from rdkit.Chem import AllChem, Descriptors
import numpy as np

# Tom 的分子特征提取器
class MolecularFeatureExtractor:
    def __init__(self):
        pass
    
    def smiles_to_graph(self, smiles):
        """
        将 SMILES 转换为图结构
        
        Tom 的比喻：
        SMILES → 原子列表 (节点) + 键列表 (边)
        """
        mol = Chem.MolFromSmiles(smiles)
        
        # 节点特征 (原子)
        node_features = []
        for atom in mol.GetAtoms():
            features = [
                atom.GetAtomicNum(),      # 原子序数
                atom.GetDegree(),         # 连接数
                atom.GetFormalCharge(),   # 形式电荷
                atom.GetNumRadicalElectrons(),  # 未成对电子
            ]
            node_features.append(features)
        
        # 边特征 (化学键)
        edge_features = []
        for bond in mol.GetBonds():
            features = [
                bond.GetBondTypeAsDouble(),  # 键类型
                bond.GetIsConjugated(),      # 是否共轭
                bond.IsInRing(),             # 是否在环中
            ]
            edge_features.append([
                bond.GetBeginAtomIdx(),
                bond.GetEndAtomIdx(),
                *features
            ])
        
        return node_features, edge_features
    
    def compute_descriptors(self, smiles):
        """
        计算分子描述符
        
        Tom 的奶酪工厂质检单：
        - 分子量
        - LogP (亲脂性)
        - 氢键供体/受体
        - 可旋转键
        """
        mol = Chem.MolFromSmiles(smiles)
        
        return {
            'mw': Descriptors.MolWt(mol),           # 分子量
            'logp': Descriptors.MolLogP(mol),       # 亲脂性
            'hba': Descriptors.NumHAcceptors(mol),  # 氢键受体
            'hbd': Descriptors.NumHDonors(mol),     # 氢键供体
            'rotatable_bonds': Descriptors.NumRotatableBonds(mol),
            'tpsa': Descriptors.TPSA(mol),          # 极性表面积
        }
```

代码示例 - 简单的分子生成器：

```python
import torch
import torch.nn as nn
from rdkit import Chem

# Tom 的分子 VAE 生成器
class MolecularVAE(nn.Module):
    def __init__(self, vocab_size, hidden_dim=256, latent_dim=64):
        super().__init__()
        
        # 编码器：SMILES → 潜变量
        self.encoder = nn.GRU(
            input_size=vocab_size,
            hidden_size=hidden_dim,
            num_layers=3,
            batch_first=True
        )
        
        self.mu = nn.Linear(hidden_dim, latent_dim)
        self.logvar = nn.Linear(hidden_dim, latent_dim)
        
        # 解码器：潜变量 → SMILES
        self.decoder = nn.GRU(
            input_size=vocab_size,
            hidden_size=hidden_dim,
            num_layers=3,
            batch_first=True
        )
        
        self.output = nn.Linear(hidden_dim, vocab_size)
    
    def encode(self, x):
        """编码 SMILES 到潜空间"""
        _, hidden = self.encoder(x)
        hidden = hidden[-1]
        return self.mu(hidden), self.logvar(hidden)
    
    def reparameterize(self, mu, logvar):
        """重参数化技巧"""
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std
    
    def decode(self, z):
        """从潜空间解码生成 SMILES"""
        # 简化版，实际需要自回归生成
        _, hidden = self.decoder(z.unsqueeze(0))
        logits = self.output(hidden)
        return logits
    
    def forward(self, x):
        mu, logvar = self.encode(x)
        z = self.reparameterize(mu, logvar)
        output = self.decode(z)
        return output, mu, logvar
```

## 🧬 第五部分：逆合成预测

### 什么是逆合成？

Tom 的"拆解分子"游戏：

```
目标分子 (复杂)
     ↓ ❓ 如何合成？
     
逆合成分析：
目标分子 → 前体 1 + 前体 2 + ... → 更简单的原料

Tom 比喻：
把一道复杂菜肴拆解成原材料和烹饪步骤！
```

### AI 逆合成方法

| 方法 | Tom 比喻 | 特点 |
|------|---------|------|
| 模板法 | 📝 按食谱做菜 | 使用已知反应模板、准确但覆盖有限 |
| 无模板法 | 👨‍🍳 自由发挥 | 学习反应规则、更灵活但可能不合理 |
| 半模板法 | 📖 改良食谱 | 结合两者优点 |

代码示例 - 逆合成预测：

```python
# Tom 的逆合成预测器 (简化版)
class RetrosynthesisPredictor:
    def __init__(self, reaction_database):
        """
        初始化预测器
        
        Tom 的"反应食谱库"
        """
        self.reactions = reaction_database
    
    def predict_precursors(self, target_smiles, top_k=5):
        """
        预测目标分子的前体
        
        输入：目标分子 SMILES
        输出：Top K 个可能的前体组合
        """
        # 方法 1：基于模板匹配
        template_matches = self.match_templates(target_smiles)
        
        # 方法 2：基于模型预测
        model_predictions = self.model_predict(target_smiles)
        
        # 整合结果
        results = self.merge_and_rank(template_matches, model_predictions)
        
        return results[:top_k]
    
    def match_templates(self, smiles):
        """使用反应模板匹配"""
        # 简化版：查找已知的逆合成反应
        # 实际使用：ASKCOS, AiZynthFinder 等工具
        matches = []
        for reaction in self.reactions:
            if reaction.matches_product(smiles):
                matches.append(reaction.get_precursors())
        return matches
```
