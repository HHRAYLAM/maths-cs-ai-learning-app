# 🐱 第 2 课：向量属性 - Tom 的移动指令卡

> 本课故事：Tom 猫拿到了 Jerry 的逃跑路线情报！但情报上写着："Jerry 跑了'很长一段距离'，往'那个方向'"。Tom 很困惑：到底是多长？哪个方向？这就是向量属性要解决的问题！

## 🎬 今日故事

> Tom 看到 Jerry 留下了一串脚印。每个脚印都是一个向量，告诉 Tom Jerry 移动的信息。但 Tom 需要知道：
> - Jerry 跑了多远？（大小）
> - Jerry 往哪里跑的？（方向）
> - 两个脚印是不是同一个方向？（平行）
> - 还是完全不同的路线？（正交）

## 📏 向量的大小 - Jerry 跑了多远？

大小（Magnitude） 就是向量的长度，就像用尺子量一下 Jerry 的脚印有多长。

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 计算公式

对于向量 $\vec{v} = (x, y)$，大小是：

$$\|\vec{v}\| = \sqrt{x^2 + y^2}$$

这就是勾股定理！想象 Tom 从起点到终点的直线距离。

例子：
- $\vec{v} = (3, 4)$
- $\|\vec{v}\| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = \mathbf{5}$

Jerry 跑了 5 步远！

</div>
  <div class="lesson-visual-column">

```
         ↑
       4 |     Jerry 在 (3, 4)
         |       🐭
         |      /
         |     /  斜边 = 5
         |    /
         |   / 
       0 |——/————→
         0     3
         
    Tom 需要跑 5 步
    才能抓住 Jerry！
```

<div class="visual-caption">图 1：向量的大小 = 斜边长度</div>

</div>
</div>

<div class="key-point">
  <div class="key-point-title">🐾 大小的意义</div>
  <ul class="key-point-list">
    <li><strong>大小 = 0</strong>：Tom 原地不动</li>
    <li><strong>大小 = 5</strong>：Tom 需要跑 5 步</li>
    <li><strong>大小 = 10</strong>：Tom 需要跑 10 步，更远！</li>
  </ul>

## 🧭 向量的方向 - Tom 该往哪跑？

方向就是向量指向哪里。想象 Tom 拿着指南针，需要知道往哪个方向追。

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的困惑</div>
  <div class="story-card-content">
    有两个向量：
    <br><br>
    <strong>向量 A</strong>：大小 = 5，方向 = 东北
    <br>
    <strong>向量 B</strong>：大小 = 5，方向 = 西南
    <br><br>
    它们相等吗？<strong>不相等</strong>！虽然大小相同，但方向完全相反。
  </div>
</div>

### 方向的重要性

```
        北
         ↑
         |
    西 ←—┼—→ 东
         |
         ↓
        南

向量 A: (3, 4) → 东北方向
向量 B: (-3, -4) → 西南方向
```

关键点：
- 两个向量可以大小相同但方向不同
- 两个向量可以方向相同但大小不同
- 只有大小和方向都相同，才是相等的向量

---

## 📐 向量相等 - 完全相同的移动指令

当且仅当两个向量的所有对应分量都相等时，它们才是相等的：

$$\vec{a} = \vec{b} \iff a_x = b_x \text{ 且 } a_y = b_y$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 相等的例子

相等：
- $\vec{a} = (3, 2)$
- $\vec{b} = (3, 2)$
- $\vec{a} = \vec{b}$ ✓

不相等：
- $\vec{a} = (3, 2)$
- $\vec{b} = (2, 3)$
- $\vec{a} \neq \vec{b}$ ✗

位置无关：
- 从 (0, 0) 到 (3, 2) 的向量
- 从 (5, 5) 到 (8, 7) 的向量
- 这两个向量相等！都是 (3, 2)

</div>
  <div class="lesson-visual-column">

```
相等向量示例：

  🐱起点 1              🐱起点 2
   │→ (3,2)              │→ (3,2)
   ↓                    ↓
   🐭终点 1              🐭终点 2

两个向量完全相同！
都是向右 3，向上 2
```

<div class="visual-caption">图 2：位置不同，但向量相等</div>

## 🚂 平行向量 - 同一条路线

如果两个向量指向相同或相反的方向，它们就是平行的。

### 判断方法

$$\vec{a} \parallel \vec{b} \iff \vec{a} = k \cdot \vec{b} \text{（k 是某个标量）}$$

<div class="key-point">
  <div class="key-point-title">🔍 平行向量的特点</div>
  <ul class="key-point-list">
    <li><strong>k > 0</strong>：同方向平行（Tom 和 Jerry 往同一方向跑）</li>
    <li><strong>k < 0</strong>：反方向平行（Tom 和 Jerry 相向而行）</li>
    <li><strong>k = 2</strong>：Jerry 跑的距离是 Tom 的 2 倍</li>
  </ul>
</div>

### 例子

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的逃跑路线</div>
  <div class="story-card-content">
    <strong>Jerry 第一次跑</strong>：$\vec{v} = (2, 3)$
    <br><br>
    <strong>Jerry 第二次跑</strong>：$2\vec{v} = (4, 6)$
    <br><br>
    第二次是第一次的<strong>2 倍</strong>！它们平行，方向相同。
  </div>
</div>

```
平行向量可视化：

Jerry 第一次：  —→ (2, 3)
Jerry 第二次：  ——→ (4, 6)
                
方向相同，只是第二次更远！
```

## ⟂ 正交向量 - 垂直的路线

如果两个向量的夹角是 90°（直角），它们就是正交的（也叫垂直）。

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 正交的例子

- $\vec{a} = (3, 0)$ — 向右 3 步
- $\vec{b} = (0, 4)$ — 向上 4 步
- 它们互相垂直！

直观理解：
- 先向北走，再向东走
- 两个方向完全独立
- 向北走不会影响向东的距离

</div>
  <div class="lesson-visual-column">

```
    ↑
    | b = (0, 4)
    |
    |   90° 直角
    |
    +——————→
         a = (3, 0)

正交向量形成直角
```

<div class="visual-caption">图 3：正交向量互相垂直</div>

</div>
</div>

<div class="key-point">
  <div class="key-point-title">🧀 为什么正交很重要？</div>
  <ul class="key-point-list">
    <li><strong>独立信息</strong>：正交方向携带完全独立的信息</li>
    <li><strong>没有冗余</strong>：一个方向的变化不影响另一个</li>
    <li><strong>机器学习应用</strong>：正交特征 = 不相关的特征，这是好事！</li>
  </ul>
</div>

## 🔄 线性相关性 - 有没有冗余信息？

### 故事理解

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的情报分析</div>
  <div class="story-card-content">
    Tom 收集了三条 Jerry 的逃跑路线：
    <br><br>
    路线 A：<strong>(2, 1)</strong>
    <br>
    路线 B：<strong>(4, 2)</strong>
    <br>
    路线 C：<strong>(1, 3)</strong>
    <br><br>
    <strong>分析</strong>：路线 B = 2 × 路线 A，所以 B 是<strong>冗余</strong>的！
    <br>
    路线 C 是<strong>新方向</strong>，有用！
  </div>
</div>

### 定义

线性相关：如果一个向量可以用其他向量的组合表示出来，就是冗余的。

线性无关：每个向量都带来新的方向信息，没有冗余。

```
线性相关的例子：

A = (2, 1)
B = (4, 2) = 2 × A  ← B 是冗余的！
C = (1, 3)  ← C 是新的方向

结论：{A, B} 线性相关
      {A, C} 线性无关
```

<div class="key-point">
  <div class="key-point-title">🎯 关键理解</div>
  <ul class="key-point-list">
    <li><strong>平行向量总是线性相关</strong>：一个只是另一个的缩放版</li>
    <li><strong>正交向量总是线性无关</strong>：完全独立的方向</li>
    <li><strong>零向量与任何向量线性相关</strong>：它没有方向信息</li>
  </ul>
</div>

## 1️⃣ 单位向量 - 纯方向指示器

单位向量是大小正好为 1 的向量。它只表示方向，没有距离信息。

### 归一化：把任何向量变成单位向量

$$\hat{\vec{v}} = \frac{\vec{v}}{\|\vec{v}\|}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子

原始向量：$\vec{v} = (3, 4)$

1. 计算大小：$\|\vec{v}\| = \sqrt{3^2 + 4^2} = 5$

2. 归一化：$\hat{\vec{v}} = \frac{(3, 4)}{5} = (0.6, 0.8)$

3. 验证：$\|\hat{\vec{v}}\| = \sqrt{0.6^2 + 0.8^2} = \sqrt{0.36 + 0.64} = \sqrt{1} = 1$ ✓

结果：大小变成 1，方向保持不变！

</div>
  <div class="lesson-visual-column">

```
归一化过程：

原向量 (3, 4)
大小 = 5
     🐭
    /|
   / |
  /  | 4
 /   |
🐱———+
  3

缩小到 1/5 大小

单位向量 (0.6, 0.8)
大小 = 1
  🐭
 /
/ | 0.8
——+
0.6
```

<div class="visual-caption">图 4：归一化 = 缩放至单位长度</div>

</div>
</div>

### 标准单位向量

在二维和三维空间中，我们有标准的单位向量：

| 名称 | 二维 | 三维 | 含义 |
|------|------|------|------|
| $\hat{\mathbf{i}}$ | (1, 0) | (1, 0, 0) | x 轴方向 |
| $\hat{\mathbf{j}}$ | (0, 1) | (0, 1, 0) | y 轴方向 |
| $\hat{\mathbf{k}}$ | - | (0, 0, 1) | z 轴方向 |

任何向量都可以写成单位向量的组合：

$$(3, 2, 4) = 3\hat{\mathbf{i}} + 2\hat{\mathbf{j}} + 4\hat{\mathbf{k}}$$

## 📊 稀疏与稠密向量 - Jerry 的活动范围

### 稀疏向量

稀疏向量的大部分分量都是 0，只有少数几个位置有值。

例子：
$$\vec{s} = (0, 0, 3, 0, 0, 0, 1, 0, 0, 0)$$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的稀疏活动</div>
  <div class="story-card-content">
    房子里有 10 个房间，Jerry 只出现在第 3 间和第 7 间：
    <br><br>
    <strong>房间 1-2</strong>：没去过 (0)
    <br>
    <strong>房间 3</strong>：去过 3 次！
    <br>
    <strong>房间 4-6</strong>：没去过 (0)
    <br>
    <strong>房间 7</strong>：去过 1 次
    <br>
    <strong>房间 8-10</strong>：没去过 (0)
  </div>
</div>

### 稠密向量

稠密向量的大部分分量都非零，活动范围广。

例子：
$$\vec{d} = (3, 7, 2, 9, 1, 5, 8, 4, 6, 2)$$

<div class="key-point">
  <div class="key-point-title">💡 为什么稀疏性重要？</div>
  <ul class="key-point-list">
    <li><strong>节省存储</strong>：只需记录非零值的位置和大小</li>
    <li><strong>计算高效</strong>：0 的运算可以跳过</li>
    <li><strong>机器学习应用</strong>：稀疏特征 = 更高效的模型</li>
  </ul>
</div>

## 📝 本课总结

<div class="key-point">
  <div class="key-point-title">🎓 知识回顾</div>
  <ul class="key-point-list">
    <li><strong>大小</strong> = 向量的长度，用勾股定理计算</li>
    <li><strong>方向</strong> = 向量指向哪里</li>
    <li><strong>相等</strong> = 大小和方向都相同</li>
    <li><strong>平行</strong> = 同方向或反方向（一个 = k × 另一个）</li>
    <li><strong>正交</strong> = 垂直，夹角 90°，独立信息</li>
    <li><strong>线性无关</strong> = 没有冗余，每个向量都是新方向</li>
    <li><strong>单位向量</strong> = 大小为 1，纯方向指示器</li>
    <li><strong>稀疏向量</strong> = 大部分分量为 0</li>
  </ul>
</div>
