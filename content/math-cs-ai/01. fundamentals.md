## 📊 分布 - 数据的模样

**分布**描述值如何分散。其他一切——平均值、检验、预测——都来自理解分布。

### 频率分布 vs 概率分布

<div class="lesson-two-column">
  <div class="lesson-text-column">

**频率分布**：计算每个值出现的次数

```
重量范围    数量
95-97g     |||
98-100g    ||||| ||
101-103g   ||||| |
104-106g   ||
```

**概率分布**：用概率替换计数

```

```

## 🎲 随机变量 - 把结果变成数字

**随机变量**是将实验的每个结果映射到实数的函数：

$$X: \text{结果} \to \mathbb{R}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子：抛硬币

| 结果 | 随机变量 X |
|------|-----------|
| 正面 | $X = 1$ |
| 反面 | $X = 0$ |

现在可以做算术了！

### 两种类型

**离散随机变量**：可数的一组值
- 骰子的点数：1, 2, 3, 4, 5, 6
- 一小时内收到的邮件数：0, 1, 2, ...

**连续随机变量**：区间内的任何值
- 身高：1.75m, 1.751m, 1.7512m...
- 等待时间：可以是任意实数

</div>
  <div class="lesson-visual-column">

```
离散 vs 连续：

离散：   ●   ●   ●   ●   ●
        1   2   3   4   5
        （可数的点）

连续：   ───────────────
        0   1   2   3   4
        （连续的线）
```

## 📐 概率质量函数 (PMF) vs 概率密度函数 (PDF)

### 离散变量 → PMF

$$P(X = x) = p(x), \quad \sum_{x} p(x) = 1$$

**例子**：公平骰子
$$P(X = k) = \frac{1}{6}, \quad k = 1,2,3,4,5,6$$

### 连续变量 → PDF

$$P(a \le X \le b) = \int_a^b f(x)\, dx, \quad \int_{-\infty}^{\infty} f(x)\, dx = 1$$

**关键**：单个精确值的概率为 0，只有区间有正概率！

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的理解</div>
  <div class="story-card-content">
    <strong>离散</strong>：求和（加法）
    <br><br>
    <strong>连续</strong>：积分（第 3 章的知识！）
    <br><br>
    PDF 曲线下面积 = 概率！

## 🎯 期望值 - 平均能期望什么

**期望值**（期望值、均值）是所有可能值的加权平均：

$$E[X] = \sum_{x} x \cdot p(x) \quad \text{（离散）}$$

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\, dx \quad \text{（连续）}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子：公平骰子

$$E[X] = 1 \cdot \frac{1}{6} + 2 \cdot \frac{1}{6} + 3 \cdot \frac{1}{6} + 4 \cdot \frac{1}{6} + 5 \cdot \frac{1}{6} + 6 \cdot \frac{1}{6}$$

$$E[X] = \frac{1+2+3+4+5+6}{6} = \frac{21}{6} = 3.5$$

### 关键性质

**线性性**：$E[aX + b] = aE[X] + b$

这个性质在机器学习中非常有用！

</div>
  <div class="lesson-visual-column">

```
期望值 = 平衡点

     ╱│╲
    ╱ │ ╲
   ╱  ●  ╲  ← 均值 = 3.5
  ╱   │   ╲
 ╱────┴────╲
1    3.5    6

即使永远不会掷出 3.5，
但长期平均会收敛到 3.5！
```

## 📏 矩 - 描述分布的形状

**矩**是 $X$ 的幂的期望，描述分布的完整形状：

| 矩 | 名称 | 描述 |
|---|------|------|
| 第 1 原点矩 | **均值** $\mu$ | 中心在哪里 |
| 第 2 中心矩 | **方差** $\sigma^2$ | 分散程度 |
| 第 3 标准化矩 | **偏度** | 不对称程度 |
| 第 4 标准化矩 | **峰度** | 尾部重量 |

<div class="key-point">
  <div class="key-point-title">🎯 中心矩 vs 原点矩</div>
  <ul class="key-point-list">
    <li><strong>原点矩</strong>：$E[X^k]$（从零测量）</li>
    <li><strong>中心矩</strong>：$E[(X-\mu)^k]$（从均值测量）</li>
    <li>第 1 中心矩总是 0（正负抵消）</li>
    <li>第 2 中心矩 = 方差</li>
  </ul>
</div>

---

## 📊 四个矩的详细解释

### 第 1 矩：均值（中心）

$$\mu = E[X]$$

分布的"平衡点"，数据中心在哪里。

### 第 2 矩：方差（分散）

$$\sigma^2 = E[(X - \mu)^2]$$

值围绕均值分布有多分散。

**标准差**：$\sigma = \sqrt{\sigma^2}$（与原始数据同单位）

### 第 3 矩：偏度（不对称）

$$\text{Skewness} = \frac{E[(X - \mu)^3]}{\sigma^3}$$

- 偏度 > 0：右尾更长（正偏）
- 偏度 < 0：左尾更长（负偏）
- 偏度 = 0：对称

### 第 4 矩：峰度（尾部）

$$\text{Kurtosis} = \frac{E[(X - \mu)^4]}{\sigma^4}$$

- 峰度 = 3：正态分布（中峰）
- 峰度 > 3：重尾（尖峰）
- 峰度 < 3：轻尾（平峰）

**超额峰度** = 峰度 - 3

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子计算

数据集：$X = \{2, 4, 4, 4, 5, 5, 7, 9\}$

**步骤 1：均值**
$$\mu = \frac{2+4+4+4+5+5+7+9}{8} = \frac{40}{8} = 5$$

**步骤 2：方差**
$$\sigma^2 = \frac{(2-5)^2 + (4-5)^2 + ... + (9-5)^2}{8} = \frac{32}{8} = 4$$

**标准差**：$\sigma = \sqrt{4} = 2$

**步骤 3：偏度**
$$\text{Skewness} = \frac{1}{8} \cdot \frac{(-3)^3 + (-1)^3 + ... + 4^3}{2^3} = 0.656$$

正偏度 → 右尾更长（9 远高于均值）

**步骤 4：峰度**
$$\text{Kurtosis} = \frac{1}{8} \cdot \frac{(-3)^4 + (-1)^4 + ... + 4^4}{2^4} = 2.781$$

接近 3 → 尾部大致正态

</div>
