# 统计学基础

*统计学提供了描述数据和量化不确定性的语言。本章涵盖分布、随机变量、PMF、PDF、CDF、期望、方差、矩和中心极限定理，这些是支撑每个机器学习评估指标和损失函数的概念。*

- 统计学是从数据学习的科学。你收集观察结果，总结它们，并得出结论，通常是关于你无法直接测量的事物。

- 想象你想知道一个国家每个成年人的平均身高。你不能测量每个人，所以你测量一个样本并使用统计学对整个总体做出有根据的猜测。

- 有两个主要分支：
    - **描述统计**：总结你已有的数据（平均值、图表、表格）
    - **推断统计**：使用样本对更大群体做出断言

- 统计学的构建块是**分布**，描述值如何分布的描述。其他一切——平均值、检验、预测——都来自理解分布。

- **频率分布**计算每个值（或值范围）在你的数据中出现的频率。想象将考试分数分类到箱中并计算每个学生落入哪个箱。结果是直方图。

- **概率分布**用概率替换原始计数。它不说"12 名学生得分在 70 到 80 之间"，而是说"得分在 70 到 80 之间的概率是 0.24"。当数据连续时，直方图条变成平滑曲线。

![频率分布为直方图 vs 概率分布为平滑曲线](../images/distribution_types.svg)

- 左边的直方图是从你收集的实际数据构建的。右边的平滑曲线是描述数据背后模式的数学模型。一个是经验的，另一个是理论的。

- 为了用数学方式处理分布，我们需要一种将数字分配给结果的方法。这正是**随机变量**所做的。

- 随机变量是将实验的每个结果映射到实数的函数。抛硬币：结果是"正面"或"反面"，但随机变量 $X$ 将其转换为 $X(\text{正面}) = 1$ 和 $X(\text{反面}) = 0$。现在我们可以做算术了。

![随机变量将结果（硬币、骰子）映射到数轴](../images/random_variable.svg)

- **离散**随机变量采用可数的一组值：10 次抛掷中的正面次数、骰子的滚动、你一小时内收到的电子邮件数量。

- **连续**随机变量可以采用区间内的任何值：你的精确身高、直到下一辆公交车到达的时间、中午的温度。

- 这种区别很重要，因为它改变了我们计算概率的方式。对于离散变量，我们求和。对于连续变量，我们积分（回忆第 3 章的积分）。

- 对于离散随机变量，**概率质量函数（PMF）**给出每个特定值的概率：

$$P(X = x) = p(x), \quad \text{其中 } \sum_{x} p(x) = 1$$

- 对于连续随机变量，**概率密度函数（PDF）**给出落入范围内的概率。任何单个精确值的概率为零；只有区间有正概率：

$$P(a \le X \le b) = \int_a^b f(x)\, dx, \quad \text{其中 } \int_{-\infty}^{\infty} f(x)\, dx = 1$$

- 现在我们可以将数字分配给结果，最自然的问题是：我们平均期望什么值？

- **期望**（或期望值）是所有可能值的加权平均，其中权重是概率。把它想象成分布的"重心"。

- 如果你多次滚动公平的骰子，你的平均滚动值收敛到 3.5。这是期望值，即使你永远不会实际滚动出 3.5。

- 对于离散随机变量：

$$E[X] = \sum_{x} x \cdot p(x)$$

- 对于连续随机变量（使用第 3 章的积分）：

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x)\, dx$$

- 示例：公平的六面骰子对 $x = 1, 2, 3, 4, 5, 6$ 有 $p(x) = 1/6$。

$$E[X] = 1 \cdot \tfrac{1}{6} + 2 \cdot \tfrac{1}{6} + 3 \cdot \tfrac{1}{6} + 4 \cdot \tfrac{1}{6} + 5 \cdot \tfrac{1}{6} + 6 \cdot \tfrac{1}{6} = \frac{21}{6} = 3.5$$

- 期望是线性的，意味着 $E[aX + b] = aE[X] + b$。这个属性非常有用，在机器学习损失函数中不断出现。

- 期望告诉我们中心，但它没有说明值分布有多分散。为了描述分布的完整形状，我们需要**矩**。

- 矩是 $X$ 的幂的期望。第 $k$ 个**原点矩**是：

$$\mu_k' = E[X^k]$$

- 第一个原点矩（$k = 1$）就是均值：$\mu_1' = E[X] = \mu$。

- 原点矩是从零测量的。通常我们关心偏离均值。第 $k$ 个**中心矩**居中测量：

$$\mu_k = E[(X - \mu)^k]$$

- 第一个中心矩总是为零（均值上方和下方的偏离抵消）。第二个中心矩是**方差**。

- 为了在不同尺度上比较分布，我们通过除以标准差 $\sigma$ 的适当幂来**标准化**：

$$\tilde{\mu}_k = \frac{\mu_k}{\sigma^k}$$

- 每个矩捕获分布形状的不同方面：

![标注每个矩捕获什么的钟形曲线：均值（中心）、方差（分散）、偏度（不对称）、峰度（尾部权重）](../images/moments_shape.svg)

- **第 1 矩（均值）**：分布的中心在哪里。平衡点。
- **第 2 矩（方差）**：值围绕均值分布有多分散。方差越高意味着越宽。
- **第 3 矩（偏度）**：分布是向左还是向右倾斜。零偏度意味着对称。
- **第 4 矩（峰度）**：尾部有多重。峰度越高意味着极端异常值越多。

- 让我们为具体数据集计算所有四个矩：$X = \{2, 4, 4, 4, 5, 5, 7, 9\}$。

- **步骤 1：均值**（第 1 原点矩）

$$\mu = \frac{2 + 4 + 4 + 4 + 5 + 5 + 7 + 9}{8} = \frac{40}{8} = 5$$

- **步骤 2：方差**（第 2 中心矩）。从每个值减去均值，平方，然后平均：

$$\sigma^2 = \frac{(2{-}5)^2 + (4{-}5)^2 + (4{-}5)^2 + (4{-}5)^2 + (5{-}5)^2 + (5{-}5)^2 + (7{-}5)^2 + (9{-}5)^2}{8}$$

$$= \frac{9 + 1 + 1 + 1 + 0 + 0 + 4 + 16}{8} = \frac{32}{8} = 4$$

- **标准差**是 $\sigma = \sqrt{4} = 2$。

- **步骤 3：偏度**（标准化第 3 中心矩）。对偏离立方，平均，除以 $\sigma^3$：

$$\tilde{\mu}_3 = \frac{1}{8} \cdot \frac{(-3)^3 + (-1)^3 + (-1)^3 + (-1)^3 + 0^3 + 0^3 + 2^3 + 4^3}{2^3}$$

$$= \frac{1}{8} \cdot \frac{-27 -1 -1 -1 + 0 + 0 + 8 + 64}{8} = \frac{42}{64} = 0.656$$

- 正偏度意味着右尾更长，这是合理的，因为 9 远高于均值。

- **步骤 4：峰度**（标准化第 4 中心矩）。将偏离提升到第 4 次幂：

$$\tilde{\mu}_4 = \frac{1}{8} \cdot \frac{(-3)^4 + (-1)^4 + (-1)^4 + (-1)^4 + 0^4 + 0^4 + 2^4 + 4^4}{2^4}$$

$$= \frac{1}{8} \cdot \frac{81 + 1 + 1 + 1 + 0 + 0 + 16 + 256}{16} = \frac{356}{128} = 2.781$$

- 正态分布的峰度为 3（称为"中峰"）。我们的值 2.781 接近，表明尾部大致正常。高于 3 的值（"尖峰"）表示更重的尾部；低于 3（"平峰"）表示更轻的尾部。一些公式通过减去 3 报告**超额峰度**，所以我们的超额峰度将是 $-0.219$。

1. 计算_loaded_骰子的期望值，其中 6 面的概率为 0.3，所有其他面平均分配剩余概率。通过模拟 100,000 次滚动验证。
```python
import jax
import jax.numpy as jnp

# Loaded 骰子：6 面 p=0.3，其他平均分配 0.7
probs = jnp.array([0.14, 0.14, 0.14, 0.14, 0.14, 0.30])
faces = jnp.array([1, 2, 3, 4, 5, 6])

# 分析期望值
ev = jnp.sum(faces * probs)
print(f"期望值 (公式): {ev:.4f}")

# 模拟
key = jax.random.PRNGKey(42)
rolls = jax.random.choice(key, faces, shape=(100_000,), p=probs)
print(f"期望值 (模拟): {rolls.mean():.4f}")
```

2. 为工作示例中的数据集计算所有四个矩（均值、方差、偏度、峰度），然后修改数据并观察每个矩如何变化。
```python
import jax.numpy as jnp

x = jnp.array([2, 4, 4, 4, 5, 5, 7, 9], dtype=jnp.float32)

mean = jnp.mean(x)
variance = jnp.mean((x - mean) ** 2)
std = jnp.sqrt(variance)
skewness = jnp.mean(((x - mean) / std) ** 3)
kurtosis = jnp.mean(((x - mean) / std) ** 4)

print(f"均值：    {mean:.3f}")
print(f"方差：    {variance:.3f}")
print(f"标准差：  {std:.3f}")
print(f"偏度：    {skewness:.3f}")
print(f"峰度：    {kurtosis:.3f}")
print(f"超额峰度：{kurtosis - 3:.3f}")
```

3. 并排可视化公平骰子滚动的 PMF 和 CDF。尝试更改概率以观察形状如何变化。
```python
import jax.numpy as jnp
import matplotlib.pyplot as plt

faces = jnp.array([1, 2, 3, 4, 5, 6])
pmf = jnp.ones(6) / 6  # 公平骰子；尝试更改这些！
cdf = jnp.cumsum(pmf)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10, 4))

ax1.bar(faces, pmf, color="#3498db", alpha=0.8)
ax1.set_title("PMF")
ax1.set_xlabel("Face")
ax1.set_ylabel("P(X = x)")
ax1.set_ylim(0, 0.5)

ax2.step(faces, cdf, where="mid", color="#e74c3c", linewidth=2)
ax2.set_title("CDF")
ax2.set_xlabel("Face")
ax2.set_ylabel("P(X ≤ x)")
ax2.set_ylim(0, 1.05)
plt.tight_layout()
plt.show()
```
