# 优化

*优化是模型训练的数学核心，找到最小化损失函数的参数。本章涵盖临界点、凸性、梯度下降、牛顿法、拉格朗日乘子的约束优化，以及为现代深度学习提供动力的优化器（SGD、Adam）。*

- 训练神经网络、拟合回归线、调整超参数：在几乎每个 ML 算法的核心是一个**优化**问题。

- 我们有某个函数（损失、成本、目标），我们想找到使其尽可能小（或大）的输入。

- 在优化之前，我们需要理解函数的**零点**（或根）。$f(x)$ 的零点是 $f(x) = 0$ 的值 $x$。从图形上看，这些是 x 轴截距。

- 例如，$f(x) = x^2 - 3x + 2 = (x-1)(x-2)$ 在 $x = 1$ 和 $x = 2$ 处有零点。在零点之间，函数是负的（$f(1.5) = -0.25$）；在零点之外，它是正的。零点将数轴划分为函数具有恒定符号的区域。

- 零点的**重数**是对应因子出现的次数。

- 在单零点（重数 1）处，图形穿过 x 轴。在双零点（重数 2）处，图形接触 x 轴但不穿过，在该点显得"平坦"。

- 寻找零点很重要，因为导数 $f'(x)$ 的零点是 $f(x)$ 的**临界点**，是极大值和极小值的候选点。

- 在最大值或最小值处，切线是平坦的（斜率 = 0），所以 $f'(x) = 0$。

![临界点：导数等于零的地方，函数有峰值、谷值或鞍点](../images/critical_points.svg)

- 但并非每个临界点都是最大值或最小值。$f'(x) = 0$ 的点也可能是**拐点**（如 $f(x) = x^3$ 的 $x = 0$），函数在该点暂时变平但不改变方向。

- **二阶导数测试** 解决了这个问题。在临界点 $x = c$ 处，其中 $f'(c) = 0$：

    - 如果 $f''(c) > 0$：曲线向上凹（像碗），所以 $c$ 是**局部最小值**。
    - 如果 $f''(c) < 0$：曲线向下凹（像山丘），所以 $c$ 是**局部最大值**。
    - 如果 $f''(c) = 0$：测试不确定；需要更高阶导数或其他方法。

- 例如，$f(x) = x^3 - 3x$。导数是 $f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$，所以临界点在 $x = -1$ 和 $x = 1$。二阶导数是 $f''(x) = 6x$。在 $x = -1$：$f''(-1) = -6 < 0$（局部最大值）。在 $x = 1$：$f''(1) = 6 > 0$（局部最小值）。

- 如果函数图形上任意两点之间的线段位于图形上方（或在图形上），则函数是**凸的**。把它想象成碗形，处处向上弯曲。从数学上讲，如果对于所有 $x$ 有 $f''(x) \geq 0$，则 $f$ 是凸的。

![凸函数具有唯一的全局最小值；非凸函数可能有多个局部最小值](../images/convex_nonconvex.svg)

- 凸性很强大，因为凸函数有一个显著的属性：每个局部最小值也是**全局最小值**。没有欺骗性的局部山谷被困住。如果你把球滚进凸碗中，它总会到达底部。

- 如果 $-f$ 是凸的，则函数是**凹的**（向下弯曲）。函数在凹和凸之间过渡的点是**拐点**，出现在 $f''(x) = 0$ 处。

- **牛顿法** 使用切线找到函数的零点（以及扩展到其导数的临界点）。从初始猜测 $x_0$ 开始，迭代细化：

$$x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)}$$

![牛顿法：沿切线找到根的更好近似](../images/newtons_method.svg)

- 想法：在 $x_n$ 处，画切线并找到它与 x 轴的交叉点。该交叉点成为 $x_{n+1}$。对于行为良好且具有良好起点的函数，牛顿法收敛非常快（二次收敛，意味着正确数字的数量每步大约翻倍）。

- 例如，找到 $\sqrt{5}$（$f(x) = x^2 - 5$ 的零点）：$f'(x) = 2x$，所以 $x_{n+1} = x_n - \frac{x_n^2 - 5}{2x_n}$。从 $x_0 = 2$ 开始：$x_1 = 2.25$，$x_2 = 2.2361\ldots$，已经精确到小数点后四位。

- 如果初始猜测远离根、如果 $f'(x) = 0$ 在根附近、或如果函数附近有拐点，牛顿法可能失败。它还需要计算导数，这可能很昂贵。

- 对于优化（找到最小值而不是零点），我们将牛顿法应用于 $f'(x) = 0$，得到更新：

$$x_{n+1} = x_n - \frac{f'(x_n)}{f''(x_n)}$$

- 在多维情况下，这变为 $\mathbf{x}_{n+1} = \mathbf{x}_n - H^{-1} \nabla f(\mathbf{x}_n)$，其中 $H$ 是海森矩阵。这是前一章中二阶泰勒近似的实际应用：将函数近似为二次函数，跳到该二次函数的最小值，重复。

- **拉格朗日乘子** 解决**约束优化**：在约束 $g(x, y) = c$ 下找到 $f(x, y)$ 的最优值。我们不是在 $\mathbb{R}^n$ 中搜索全部，而是限制在约束成立的集合（曲线或表面）上。

- 关键洞察是几何的：在约束最优值处，$f$ 的梯度必须平行于 $g$ 的梯度。如果它们不平行，我们可以沿约束在仍然改善 $f$ 的方向上移动，所以我们还没有达到最优值。

- 我们引入新变量 $\lambda$（拉格朗日乘子）并定义**拉格朗日函数**：

$$\mathcal{L}(x, y, \lambda) = f(x, y) - \lambda(g(x, y) - c)$$

- 将所有偏导数设为零得到方程组，其解是约束最优值：

$$\frac{\partial \mathcal{L}}{\partial x} = 0, \quad \frac{\partial \mathcal{L}}{\partial y} = 0, \quad \frac{\partial \mathcal{L}}{\partial \lambda} = 0$$

![拉格朗日乘子：在最优值处，f 和 g 的梯度平行](../images/lagrange_multiplier.svg)

- 例如，在约束 $x^2 + y^2 = 1$ 下最大化 $f(x,y) = x^2 y$。拉格朗日函数是 $\mathcal{L} = x^2 y - \lambda(x^2 + y^2 - 1)$。求偏导：

$$2xy - 2\lambda x = 0, \quad x^2 - 2\lambda y = 0, \quad x^2 + y^2 = 1$$

- 从第一个方程（假设 $x \neq 0$）：$\lambda = y$。代入第二个：$x^2 = 2y^2$。结合约束：$2y^2 + y^2 = 1$，所以 $y = \frac{1}{\sqrt{3}}$。最大值是 $f = \frac{2}{3\sqrt{3}}$。

- 对于不等式约束（$g(x,y) \leq c$ 而不是 $= c$），**Karush-Kuhn-Tucker (KKT) 条件** 推广了拉格朗日乘子。约束要么是活跃的（绑定，视为等式），要么是非活跃的（解在内部，约束无关紧要）。

- 在实践中，我们很少手动优化。以下是主要算法家族：

    - **一阶方法**（仅使用梯度）：梯度下降、随机梯度下降（SGD）、Adam。这些每步成本低但可能收敛缓慢，特别是在病态问题上。

    - **二阶方法**（使用梯度和海森矩阵）：牛顿法收敛快但计算和求逆海森矩阵很昂贵（对于 $n$ 个参数是 $O(n^3)$）。**拟牛顿方法**（如 BFGS 和 L-BFGS）仅使用梯度信息近似海森矩阵，实现比一阶方法更快的收敛，而没有二阶方法的全部成本。

    - **共轭梯度**：对于大型稀疏系统高效，仅使用矩阵 - 向量积而不是存储完整海森矩阵。

    - **高斯 - 牛顿** 和 **Levenberg-Marquardt**：专门用于最小二乘问题（回归中常见），通过雅可比近似海森矩阵。

    - **自然梯度下降**：使用费雪信息矩阵考虑参数空间的几何形状，对于概率模型可能更有效。

- 优化器的选择取决于问题。对于深度学习，一阶方法（特别是 Adam）占主导地位，因为参数数量巨大（数百万到数十亿），使海森矩阵计算不切实际。对于具有平滑目标的较小问题，二阶方法可能快得多。

## 编码任务（使用 CoLab 或 notebook）

1. 实现牛顿法找到 $\sqrt{7}$（$f(x) = x^2 - 7$ 的零点）。观察快速收敛。
```python
import jax.numpy as jnp

f = lambda x: x**2 - 7
df = lambda x: 2*x

x = 3.0  # 初始猜测
for i in range(6):
    x = x - f(x) / df(x)
    print(f"步 {i+1}: x = {x:.10f}  (误差：{abs(x - jnp.sqrt(7.0)):.2e})")
```

2. 使用梯度下降最小化 $f(x, y) = (x - 3)^2 + (y + 1)^2$。最小值在 $(3, -1)$。尝试不同的学习率。
```python
import jax
import jax.numpy as jnp

def f(params):
    x, y = params
    return (x - 3)**2 + (y + 1)**2

grad_f = jax.grad(f)
params = jnp.array([0.0, 0.0])
lr = 0.1

for i in range(20):
    g = grad_f(params)
    params = params - lr * g
    if i % 5 == 0 or i == 19:
        print(f"步 {i:2d}: ({params[0]:.4f}, {params[1]:.4f})  loss={f(params):.6f}")
```

3. 数值求解约束优化问题。通过参数化 $y = 10 - x$ 并找到单变量函数的最优值，在约束 $x + y = 10$ 下最大化 $f(x,y) = xy$。
```python
import jax
import jax.numpy as jnp

# 代入约束：y = 10 - x，所以 f = x(10 - x) = 10x - x²
f = lambda x: x * (10 - x)
df = jax.grad(f)

# 梯度上升（我们想要最大值，所以加梯度）
x = 1.0
lr = 0.1
for i in range(20):
    x = x + lr * df(x)
print(f"x={x:.4f}, y={10-x:.4f}, f={f(x):.4f}")  # 应该是 x=5, y=5, f=25
```
