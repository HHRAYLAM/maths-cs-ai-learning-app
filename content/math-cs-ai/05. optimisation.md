## 🎯 零点与临界点

零点：$f(x) = 0$ 的点（x 轴截距）

临界点：$f'(x) = 0$ 的点（切线水平）

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 临界点的三种类型

| 类型 | 条件 | 形状 |
|------|------|------|
| 局部最大值 | $f'(c)=0$, $f''(c)<0$ | 🏔️ 山峰 |
| 局部最小值 | $f'(c)=0$, $f''(c)>0$ | 🥣 山谷 |
| 拐点/鞍点 | $f'(c)=0$, $f''(c)=0$ | ～ 波浪 |

### 例子

$f(x) = x^3 - 3x$

导数：$f'(x) = 3x^2 - 3 = 3(x-1)(x+1)$

临界点：$x = -1$ 和 $x = 1$

二阶导数：$f''(x) = 6x$

- $x = -1$：$f''(-1) = -6 < 0$ → 局部最大值
- $x = 1$：$f''(1) = 6 > 0$ → 局部最小值

</div>
  <div class="lesson-visual-column">

```
临界点可视化：

  f(x)
   ↑
   │  ╲      ╱
   │   ╲    ╱   ← 局部最大值 (f'<0)
   │    ╲  ╱
   │     ╲╱
   │      ╲
   │       ╲    ← 局部最小值 (f'>0)
   │        ╲╱
   └─────────→ x
   
   f'=0 的点 = 临界点
```

## 🥣 凸函数 - 唯一的全局最小值

凸函数：任意两点连线在函数图像上方（碗形）

数学定义：$f''(x) \geq 0$ 对所有 $x$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 凸函数的强大性质

每个局部最小值也是全局最小值！

```
凸函数：        非凸函数：
   ╱│╲            ╱│╲
  ╱ │ ╲          ╱ │ ╲
 ╱  │  ╲        ╱  │  ╲
╱___│___╲      ╱___│___╲
    ●            ●   ●
  唯一最小      多个局部最小
```

### 为什么重要？

- 凸优化：保证找到全局最优
- 非凸优化（如神经网络）：可能被困在局部最优

</div>
  <div class="lesson-visual-column">

```
凸 vs 非凸：

  凸函数 (碗)        非凸函数 (山谷)
     ╭───╮             ╭─╮ ╭─╮
    ╱     ╲           ╱ ╰─╯ ╲
   ╱       ╲         ╱       ╲
  ╱         ╲       ╱         ╲
 └───────────┘     └───────────┘
    唯一解            多个解
  
  凸函数：球总能滚到最低点
  非凸函数：球可能卡在小山谷
```

## 🚀 梯度下降 - 下坡算法

梯度下降：沿负梯度方向更新参数

$$w_{new} = w_{old} - \eta \cdot \nabla f(w_{old})$$

- $\eta$ = 学习率（步长）
- $\nabla f$ = 梯度（最陡上坡方向）
- $-\nabla f$ = 最陡下坡方向

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的梯度下降</div>
  <div class="story-card-content">
    Tom 下山步骤：
    <br><br>
    1. 看哪个方向最陡（计算梯度）
    <br>
    2. 往反方向走一小步（减去梯度）
    <br>
    3. 重复，直到到达谷底
    <br><br>
    <strong>学习率</strong>很重要：
    <br>
    - 太大 →  overshoot，错过最低点
    <br>
    - 太小 → 走太慢，好久才到
  </div>
</div>

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 例子

最小化 $f(x, y) = (x-3)^2 + (y+1)^2$

梯度：$\nabla f = (2(x-3), 2(y+1))$

从 $(0, 0)$ 开始，$\eta = 0.1$：

- 第 1 步：$\nabla f(0,0) = (-6, 2)$
  - $x = 0 - 0.1 \cdot (-6) = 0.6$
  - $y = 0 - 0.1 \cdot 2 = -0.2$

- 第 2 步：继续...

最终收敛到 $(3, -1)$（最小值点）

</div>
  <div class="lesson-visual-column">

```
梯度下降过程：

  损失
   ↑
   │╲
   │ ╲  步长太大
   │  ╲  → 来回震荡
   │   ╲_
   │    ╲___  步长适中 → 平滑收敛
   │     ╲______
   │      ╲_________
   └──────────────→ 步数
   
  学习率需要调优！
```

## 🎯 牛顿法 - 用曲率加速

牛顿法用二阶导数（曲率）信息更快收敛：

$$x_{n+1} = x_n - \frac{f'(x_n)}{f''(x_n)}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 与梯度下降对比

| 方法 | 更新公式 | 收敛速度 | 计算成本 |
|------|---------|---------|---------|
| 梯度下降 | $x - \eta f'$ | 慢（线性） | 低 |
| 牛顿法 | $x - \frac{f'}{f''}$ | 快（二次） | 高 |

### 例子：求 $\sqrt{5}$

找 $f(x) = x^2 - 5$ 的零点：

$f'(x) = 2x$，$f''(x) = 2$

牛顿更新：$x_{n+1} = x_n - \frac{x_n^2 - 5}{2x_n}$

从 $x_0 = 2$ 开始：
- $x_1 = 2.25$
- $x_2 = 2.2361...$
- $x_3 = 2.2360679...$（精确到 9 位小数！）

</div>
  <div class="lesson-visual-column">

```
牛顿法可视化：

  f(x)
   ↑
   │╲
   │ ╲      切线
   │  ╲    ╱
   │   ╲  ╱
   │    ╲╱
   │     ╲  切线与 x 轴交点
   │      ╲ ↓
   │       ╲___
   │        ╲  ╲
   │         ╲  ╲
   └──────────┴──┴─→ x
          x₀  x₁  x₂
  
  每步切线找根，收敛超快！
```

<div class="visual-caption">图 4：牛顿法用切线快速逼近根</div>

</div>
</div>

<div class="key-point">
  <div class="key-point-title">🎯 多变量牛顿法</div>
  <ul class="key-point-list">
    <li>公式：$\mathbf{x}_{n+1} = \mathbf{x}_n - H^{-1} \nabla f(\mathbf{x}_n)$</li>
    <li>$H$ = 海森矩阵（二阶导数矩阵）</li>
    <li><strong>优点</strong>：收敛极快</li>
    <li><strong>缺点</strong>：计算和求逆海森矩阵很贵（$O(n^3)$）</li>
    <li>适用于参数较少的问题</li>
  </ul>

## 🔗 拉格朗日乘数法 - 约束优化

约束优化：在约束 $g(x,y) = c$ 下优化 $f(x,y)$

拉格朗日函数：

$$\mathcal{L}(x, y, \lambda) = f(x, y) - \lambda(g(x, y) - c)$$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的约束理解</div>
  <div class="story-card-content">
    Tom 被限制在一条路径上走：
    <br><br>
    - 路径 = 约束 $g(x,y) = c$
    <br>
    - 目标 = 找到路径上的最低点
    <br><br>
    <strong>关键洞察</strong>：
    <br>
    在最优值处，$f$ 的梯度与 $g$ 的梯度<strong>平行</strong>！
    <br><br>
    否则，Tom 可以沿着路径继续走更低的地方。
  </div>
</div>

### 求解步骤

1. 构造拉格朗日函数 $\mathcal{L}$
2. 对所有变量求偏导并设为零：
   $$\frac{\partial \mathcal{L}}{\partial x} = 0, \quad \frac{\partial \mathcal{L}}{\partial y} = 0, \quad \frac{\partial \mathcal{L}}{\partial \lambda} = 0$$
3. 解方程组

### 例子

在约束 $x^2 + y^2 = 1$ 下最大化 $f(x,y) = x^2 y$

拉格朗日函数：$\mathcal{L} = x^2 y - \lambda(x^2 + y^2 - 1)$

求偏导：
- $\frac{\partial \mathcal{L}}{\partial x} = 2xy - 2\lambda x = 0$
- $\frac{\partial \mathcal{L}}{\partial y} = x^2 - 2\lambda y = 0$
- $\frac{\partial \mathcal{L}}{\partial \lambda} = -(x^2 + y^2 - 1) = 0$

解得：$x = \pm\sqrt{\frac{2}{3}}, y = \frac{1}{\sqrt{3}}$，最大值 $= \frac{2}{3\sqrt{3}}$

## 🤖 优化器在深度学习中的应用

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的优化器家族</div>
  <div class="story-card-content">
    <strong>一阶方法</strong>（只用梯度）：
    <br>
    - SGD（随机梯度下降）
    <br>
    - Adam（自适应学习率）⭐ 最流行
    <br><br>
    <strong>二阶方法</strong>（用海森矩阵）：
    <br>
    - 牛顿法（太快但太贵）
    <br>
    - BFGS/L-BFGS（近似海森）
    <br><br>
    <strong>深度学习为什么用 Adam？</strong>
    <br>
    - 参数太多（数百万），海森矩阵算不动
    <br>
    - Adam 自适应调整学习率，省心！
  </div>
</div>

### 常见优化器对比

| 优化器 | 特点 | 适用场景 |
|--------|------|---------|
| SGD | 简单，可能慢 | 小模型，凸问题 |
| SGD + Momentum | 有惯性，更快 | 一般深度学习 |
| Adam | 自适应学习率 | 大多数深度学习任务 ⭐ |
| AdamW | Adam + 权重衰减 | Transformer 训练 |
| L-BFGS | 近似牛顿法 | 小模型，科学计算 |
