## 🏗️ 多层感知机（MLP）- 最基础的深度网络

多层感知机是堆叠多个全连接层的网络：

$$h_1 = \sigma(W_1 x + b_1)$$
$$h_2 = \sigma(W_2 h_1 + b_2)$$
$$\cdots$$
$$y = W_{out} h_n + b_{out}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 为什么需要多层？

| 层数 | 学习能力 |
|------|---------|
| 1 层 | 线性边界（直线） |
| 2 层 | 简单曲线 |
| 3+ 层 | 复杂决策边界 |

### 没有非线性会怎样？

$$W_2(W_1 x) = (W_2 W_1) x$$

这只是另一个线性变换！多层坍缩成单层。

激活函数引入非线性，让多层有意义！

</div>
  <div class="lesson-visual-column">

```
深度网络的力量：

单层：只能画直线
●●●  ○○○
───────────  ← 线性边界

三层：可以画复杂形状
●●●  ○○○
 ╲  ╱
  ╲╱
  ╱╲
 ╱  ╲
●    ○  ← 非线性边界

层数越多 → 边界越复杂！
```

## 🔥 激活函数 - 引入非线性的魔法

激活函数决定神经元是否"激活"（输出信号）：

| 激活函数 | 公式 | 特点 | 适用场景 |
|---------|------|------|---------|
| ReLU | $\max(0, x)$ | 计算快，稀疏激活 | 默认选择 |
| Sigmoid | $\frac{1}{1+e^{-x}}$ | 输出 (0,1) | 二分类输出 |
| Tanh | $\frac{e^x-e^{-x}}{e^x+e^{-x}}$ | 输出 (-1,1)，零中心化 | RNN |
| GELU | $x \cdot \Phi(x)$ | 平滑，允许小负值 | Transformer |
| Swish | $x \cdot \sigma(x)$ | 平滑门控 | 类似 GELU |

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的理解</div>
  <div class="story-card-content">
    <strong>ReLU</strong>：像开关，负数关 (0)，正数开 (x)
    <br><br>
    <strong>Sigmoid</strong>：把任何数压缩到 0~1 之间
    <br><br>
    <strong>Tanh</strong>：压缩到 -1~1 之间，更居中
    <br><br>
    <strong>GELU/Swish</strong>：更平滑的 ReLU，现代模型首选
  </div>
</div>

---

### ReLU 的优缺点

```
ReLU 函数图：
     ↑
     │   ╱
     │  ╱
     │ ╱
─────●────→ x
    ╱│
   ╱ │
  ╱  │

x > 0：输出 = x（通过）
x < 0：输出 = 0（关闭）
```

优点：
- 计算超快（只需比较）
- 正输入不饱和（梯度始终为 1）
- 产生稀疏激活（很多神经元输出 0）

缺点：死亡 ReLU 问题
- 负输入永远输出 0
- 神经元可能"死亡"并停止学习

解决：Leaky ReLU 允许小负值通过

### 梯度消失问题

当网络很深时，梯度通过链式法则反向传播：

$$\frac{\partial J}{\partial w_1} = \frac{\partial J}{\partial y} \cdot \frac{\partial y}{\partial h_n} \cdot \ldots \cdot \frac{\partial h_2}{\partial h_1} \cdot \frac{\partial h_1}{\partial w_1}$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 梯度消失

如果每个偏导数 < 1：

$$0.5 \times 0.5 \times 0.5 \times \ldots \approx 0$$

梯度指数级缩小到零！

早期层几乎不学习

### 梯度爆炸

如果每个偏导数 > 1：

$$2 \times 2 \times 2 \times \ldots \to \infty$$

梯度爆炸，数值溢出！

</div>
  <div class="lesson-visual-column">

```
梯度流动可视化：

健康梯度：
层 1 → 层 2 → 层 3 → 输出
  ↑      ↑      ↑
  │      │      │
梯度正常流动

梯度消失：
层 1 → 层 2 → 层 3 → 输出
  ↑      ↑      ↑
  │      │      │
 小    中等    大
（早期层几乎无梯度）

梯度爆炸：
层 1 → 层 2 → 层 3 → 输出
  ↑      ↑      ↑
  │      │      │
超大 ← 大 ← 中等
（早期层梯度爆炸）
```

<div class="visual-caption">图 2：梯度消失和爆炸问题</div>

</div>
</div>

### 解决梯度问题的方法

| 方法 | 说明 |
|------|------|
| ReLU/GELU 激活 | 对正输入梯度为 1，不饱和 |
| 权重初始化 | He/Xavier 初始化控制梯度尺度 |
| 归一化层 | BatchNorm、LayerNorm 稳定梯度流 |
| 残差连接 | 跳跃连接让梯度直接流动 |
| 梯度裁剪 | 限制梯度最大值（防止爆炸） |

## 🎲 权重初始化 - 训练的起点

不好的初始化可能导致梯度消失或爆炸：

| 初始化方法 | 方差 | 适用场景 |
|-----------|------|---------|
| Xavier (Glorot) | $\frac{2}{d_{in} + d_{out}}$ | 线性、Tanh、Sigmoid |
| He (Kaiming) | $\frac{2}{d_{in}}$ | ReLU 及其变体 |

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的初始化理解</div>
  <div class="story-card-content">
    <strong>权重太大</strong>：激活值爆炸 → 梯度爆炸
    <br><br>
    <strong>权重太小</strong>：激活值消失 → 梯度消失
    <br><br>
    <strong>He 初始化</strong>：为 ReLU 设计，双倍方差补偿死亡神经元
    <br><br>
    <strong>Xavier 初始化</strong>：让激活方差在各层保持恒定
  </div>

## 📐 归一化层 - 稳定训练的魔法

归一化确保每层输入具有相似的分布（零均值、单位方差）：

$$\hat{x} = \frac{x - \mu}{\sqrt{\sigma^2 + \epsilon}}$$
$$y = \gamma \hat{x} + \beta$$

- $\gamma$（缩放）和 $\beta$（偏移）是可学习参数
- 允许网络在需要时撤销归一化

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 归一化的好处

1. 稳定梯度：防止梯度消失/爆炸
2. 加速收敛：更大的学习率
3. 减少正则化需求：本身有正则化效果
4. 对初始化不敏感：更容易训练

### 四种归一化

| 类型 | 归一化维度 | 适用场景 |
|------|-----------|---------|
| BatchNorm | 批量维度 | CNN |
| LayerNorm | 特征维度 | Transformer、RNN |
| InstanceNorm | 单样本单通道 | 风格迁移 |
| GroupNorm | 通道分组 | 小批量训练 |

</div>
  <div class="lesson-visual-column">

```
归一化维度可视化：

BatchNorm（按批量）：
[样本 1] ─┐
[样本 2] ─├→ 归一化
[样本 3] ─┘

LayerNorm（按特征）：
[特征 1, 特征 2, ..., 特征 n] → 归一化

InstanceNorm（按通道）：
[通道 1 的所有空间位置] → 归一化

GroupNorm（按组）：
[通道组 1] → 归一化
[通道组 2] → 归一化
```

## 🔀 残差连接 - 跳过层数的捷径

残差连接让输入绕过一层或多层：

$$\text{output} = F(x) + x$$

<div class="story-card jerry">
  <div class="story-card-title">🐭 Jerry 的理解</div>
  <div class="story-card-content">
    <strong>普通层</strong>：必须学习完整变换 H(x)
    <br><br>
    <strong>残差层</strong>：只学习残差 F(x) = H(x) - x
    <br><br>
    如果最优变换接近恒等（x → x），
    残差学习更容易！
    <br><br>
    <strong>ResNet</strong>：用残差连接堆叠 100+ 层！
  </div>
</div>

---

### 为什么残差连接有效？

```
普通深网络：
输入 → [层 1] → [层 2] → [层 3] → ... → [层 50] → 输出
        梯度要经过 50 层，容易消失！

残差网络：
输入 → [层 1] → [层 2] → [层 3] → ... → [层 50] → 输出
  │                                                      ↑
  └──────────────────────捷径─────────────────────────────┘
  梯度可以直接通过捷径流动！
```

效果：
- 梯度可以直接流向早期层
- 解决"网络越深效果越差"的退化问题
- 让训练 100+ 层成为可能

## 🧠 卷积神经网络（CNN）- 利用空间结构

CNN 用卷积代替全连接，大幅减少参数：

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 卷积运算

用滤波器（核）在输入上滑动，计算点积：

$$(\text{input} * K)[i,j] = \sum_{m,n} \text{input}[i+m, j+n] \cdot K[m, n]$$

### 关键超参数

| 参数 | 说明 | 效果 |
|------|------|------|
| 滤波器大小 | 如 3×3, 5×5 | 感受野大小 |
| 步幅 (Stride) | 每次移动多少像素 | 控制输出大小 |
| 填充 (Padding) | 边界补零 | 保留空间尺寸 |

### 输出大小公式

$$\text{out} = \left\lfloor \frac{\text{in} - k + 2p}{s} \right\rfloor + 1$$

- $k$ = 滤波器大小
- $p$ = 填充
- $s$ = 步幅

</div>
  <div class="lesson-visual-column">

```
卷积可视化：

输入 (5x5)         滤波器 (3x3)
┌─────────┐        ┌───────┐
│ ● ● ● ● ● │      │ 1  0 -1 │
│ ● ● ● ● ● │  *   │ 0  1  0 │
│ ● ● ● ● ● │      │-1  0  1 │
│ ● ● ● ● ● │      └───────┘
│ ● ● ● ● ● │
└─────────┘
      ↓
滑动计算点积 → 输出 (3x3)

相同滤波器在所有位置共享 → 参数少！
```

### 池化层 - 下采样

池化减少空间维度，保留最重要信息：

| 池化类型 | 操作 | 特点 |
|---------|------|------|
| 最大池化 | 取窗口内最大值 | 保留最显著特征 |
| 平均池化 | 取窗口内均值 | 平滑特征 |

```
最大池化 (2x2 窗口，步幅 2)：

输入 (4x4)              输出 (2x2)
┌───────────┐          ┌───────┐
│ 1  2  3  1 │   池化    │ 5  4  │
│ 3  5  1  2 │   →      │ 8  9  │
│ 2  1  4  3 │
│ 4  8  9  1 │
└───────────┘          └───────┘

2x2 窗口取最大值，尺寸减半！
```

## 🔄 循环神经网络（RNN）- 处理序列

RNN 按顺序处理序列，维护隐藏状态：

$$h_t = \tanh(W_h h_{t-1} + W_x x_t + b)$$

<div class="story-card">
  <div class="story-card-title">🐱 Tom 的序列处理</div>
  <div class="story-card-content">
    <strong>隐藏状态 h_t</strong>：到时间 t 为止的输入摘要
    <br><br>
    <strong>权重共享</strong>：相同 W_h、W_x 在所有时间步使用
    <br><br>
    <strong>应用</strong>：文本、语音、时间序列
  </div>
</div>

---

### LSTM - 长短期记忆网络

LSTM 解决 RNN 的梯度消失问题，引入记忆细胞：

```
LSTM 细胞结构：

          遗忘门 f_t
  h_{t-1} → [σ] ──────┐
  x_t     →           ⊙ ──→ c_t (记忆细胞)
                        ↑
          输入门 i_t    │
  h_{t-1} → [σ] ───→ ⊙ ←─ [tanh]
  x_t     →              ↑
                   新候选值 c̃_t
                        │
                        ↓
                  输出门 o_t
                  h_t (新隐藏状态)
```

三个门：
- 遗忘门：决定从记忆中擦除什么
- 输入门：决定写入什么新信息
- 输出门：决定暴露什么给下一层

---

### GRU - 简化版 LSTM

GRU（门控循环单元）简化 LSTM：

| 特性 | LSTM | GRU |
|------|------|-----|
| 门数量 | 3 个 | 2 个 |
| 状态 | 记忆细胞 + 隐藏状态 | 合并为一个 |
| 参数 | 多 | 少 |
| 效果 | 强 | 相当 |

GRU 更简单，通常效果与 LSTM 相当，是更轻量级的选择。

## 🎯 注意力机制 - 关注相关信息

注意力让模型决定输入中哪些部分最重要：

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

<div class="lesson-two-column">
  <div class="lesson-text-column">

### 图书馆比喻

| 概念 | 比喻 |
|------|------|
| Query (Q) | 你要找什么 |
| Key (K) | 每本书的标签 |
| Value (V) | 书的内容 |

### 计算步骤

1. $QK^T$：查询与所有键比较（相似度）
2. $\text{softmax}$：转为概率分布
3. 乘以 $V$：加权组合值

### 为什么除以$\sqrt{d_k}$？

防止点积太大导致 softmax 饱和！

</div>
  <div class="lesson-visual-column">

```
注意力可视化：

句子： "The cat sat on the mat"

当处理"sat"时，注意力关注：
         ╭────╮
         ↓    ↓
"The cat sat on the mat"
       ↑
    最高注意力（谁坐？）

当处理"the mat"时，注意力关注：
         ╭───╮
         ↓   ↓
"The cat sat on the mat"
                ↑
             高注意力（在哪？）

每个词关注相关的其他词！
```

Transformer 编码器块：

### 位置编码

注意力是排列等变的，需要位置信息：

原始 Transformer 使用正弦编码：

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d}}\right)$$
$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d}}\right)$$

现代模型 使用：
- 学习的位置嵌入
- RoPE（旋转位置编码）
- ALiBi（线性偏置）

## 🎨 生成模型 - 创造新内容

### 自动编码器

自动编码器学习压缩表示：

$$z = f_{enc}(x) \quad \text{(编码)}$$
$$\hat{x} = f_{dec}(z) \quad \text{(解码)}$$
$$\mathcal{L} = \|x - \hat{x}\|^2 \quad \text{(重建损失)}$$

瓶颈层迫使网络学习最重要的特征！

---

### 变分自动编码器（VAE）

VAE 添加概率：

$$z = \mu + \sigma \odot \epsilon, \quad \epsilon \sim \mathcal{N}(0, I)$$

损失函数：

$$\mathcal{L} = \underbrace{\|x - \hat{x}\|^2}_{\text{重建损失}} + \underbrace{D_{KL}(q(z|x) \| p(z))}_{\text{正则化}}$$

重参数化技巧 让采样可微，梯度可以流过！

<div class="key-point">
  <div class="key-point-title">🎯 VAE  vs  自动编码器</div>
  <ul class="key-point-list">
    <li><strong>自动编码器</strong>：点估计 z，只能重建</li>
    <li><strong>VAE</strong>：分布估计 z，可以从先验采样生成新数据</li>
    <li><strong>KL 散度</strong>：确保潜在空间平滑，可插值</li>
  </ul>
</div>

## 📝 本课总结

<div class="key-point">
  <div class="key-point-title">🎓 知识回顾</div>
  <ul class="key-point-list">
    <li><strong>MLP</strong>：堆叠全连接层 + 非线性激活</li>
    <li><strong>激活函数</strong>：ReLU（默认）、GELU（Transformer）、Sigmoid（输出层）</li>
    <li><strong>梯度问题</strong>：消失/爆炸，用 ReLU、归一化、残差解决</li>
    <li><strong>归一化</strong>：BatchNorm（CNN）、LayerNorm（Transformer）</li>
    <li><strong>残差连接</strong>：output = F(x) + x，训练超深网络</li>
    <li><strong>CNN</strong>：卷积提取空间特征，池化下采样</li>
    <li><strong>RNN/LSTM/GRU</strong>：处理序列，LSTM 解决长程依赖</li>
    <li><strong>注意力</strong>：QKV 机制，关注相关信息</li>
    <li><strong>Transformer</strong>：纯注意力，并行高效</li>
    <li><strong>VAE</strong>：概率生成模型，重参数化技巧</li>
  </ul>
