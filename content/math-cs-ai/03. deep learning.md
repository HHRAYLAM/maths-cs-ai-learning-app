# 深度学习

*深度学习堆叠非线性层以构建分层表示，将原始输入自动转换为有用特征。本章涵盖 MLP、激活函数、反向传播、CNN、RNN、LSTM、注意力、Transformer、GAN、VAE、扩散模型和归一化技术。*

- 什么使网络"深"？浅层网络只有一个隐藏层；深层网络有很多层。深度让网络构建分层表示，早期层学习简单特征（边缘、色调），后期层将它们组合成复杂概念（面孔、句子）。这种组合性是深度学习强大的原因。

- 最简单的深度网络是**多层感知机（MLP）**，也称为全连接或稠密网络。每层计算：

$$h = \sigma(Wx + b)$$

- 这里 $W$ 是权重矩阵（第 2 章），$b$ 是偏置向量，$\sigma$ 是非线性激活函数。一层的输出成为下一层的输入。没有非线性，堆叠层将毫无意义：$W_2(W_1 x) = (W_2 W_1)x$，这只是另一个线性变换。这正是第 2 章的矩阵乘法坍缩。

- **激活函数**引入使深度有意义的非线性。

- **ReLU**（修正线性单元）：$\text{ReLU}(x) = \max(0, x)$。它是最广泛使用的激活函数。它计算快速，对正输入不饱和，并产生稀疏激活（许多神经元输出恰好为零）。缺点：具有负输入神经元总是输出零，如果它们永久困在那里，它们会"死亡"并停止学习。

- **Sigmoid**：$\sigma(x) = \frac{1}{1+e^{-x}}$，将输入压缩到 $(0, 1)$。适用于二分类的输出层，但在隐藏层中有问题，因为当输入远离零时梯度消失（曲线几乎平坦）。

- **Tanh**：$\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}$，压缩到 $(-1, 1)$。零中心化（不像 sigmoid），有助于梯度流动，但在极端情况下仍然遭受梯度消失。

- **GELU**（高斯误差线性单元）：$\text{GELU}(x) = x \cdot \Phi(x)$，其中 $\Phi$ 是标准正态 CDF。它是对 ReLU 的平滑近似，允许小负值通过。GELU 是 GPT 和 BERT 的默认选择。

- **Swish**：$\text{Swish}(x) = x \cdot \sigma(x)$，另一个平滑门控。在实践中与 GELU 相似。

![ReLU、Sigmoid、Tanh 和 GELU 的并排图及其关键属性](../images/activation_functions.svg)

- 具有 $d_{\text{in}}$ 输入和 $d_{\text{out}}$ 输出的稠密层有 $d_{\text{in}} \times d_{\text{out}} + d_{\text{out}}$ 个参数（权重加偏置）。矩阵乘法 $Wx$ 只是来自第 2 章的矩阵 - 向量乘法。在批量设置中，输入是形状为 $(B, d_{\text{in}})$ 的矩阵 $X$，输出是形状为 $(B, d_{\text{out}})$ 的 $XW^T + b$。

- **通用近似定理**指出，具有足够神经元的单个隐藏层可以在紧致域上以任意精度近似任何连续函数。这听起来深度不应该重要，但关键是"足够神经元"。在实践中，深度网络可以用比浅层网络指数级更少的参数表示相同的函数。深度给你效率，而不仅仅是表达能力。

- 随着网络变深，出现两种梯度病理。**梯度消失**：当梯度通过许多层（通过链式法则，第 3 章）时，它们乘以许多因子。如果这些因子始终小于 1（如 sigmoid 和 tanh 饱和时发生），梯度指数级缩小到零。早期层几乎不学习。**梯度爆炸**：如果因子始终大于 1，梯度指数级增长，导致数值溢出和不稳定训练。

- 梯度消失/爆炸的解决方案：
  - 使用 ReLU 或 GELU 激活（对正输入梯度为 1，无饱和）
  - 仔细的权重初始化
  - 归一化层
  - 残差连接（跳跃连接）
  - 梯度裁剪（对于梯度爆炸）：将梯度范数限制在最大值

- **权重初始化**很重要，因为它决定了训练开始时激活和梯度的尺度。如果权重太大，激活爆炸；太小，它们消失。

- **Xavier（Glorot）初始化**从方差为 $\frac{2}{d_{\text{in}} + d_{\text{out}}}$ 的分布设置权重。这使激活的方差在各层之间保持大致恒定，假设线性或 tanh 激活。

- **He（Kaiming）初始化**使用方差 $\frac{2}{d_{\text{in}}}$，这是为 ReLU 激活校准的（因为 ReLU 将一半激活归零，你需要双倍方差来补偿）。

- **归一化层**通过确保每层的输入具有一致的统计量（大致零均值、单位方差）来稳定训练。

- **批量归一化（BatchNorm）** 在批量维度上归一化：对于每个通道/特征，计算小批量中所有样本的均值和方差，然后归一化。它添加可学习的缩放（$\gamma$）和偏移（$\beta$）参数，以便网络可以在需要时撤销归一化：

$$\hat{x} = \frac{x - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}, \quad y = \gamma \hat{x} + \beta$$

- BatchNorm 有一个问题：它依赖于批量大小。对于非常小的批量，统计量有噪声。在推理时，你使用运行平均而不是批量统计，这造成了训练/测试差异。

- **层归一化（LayerNorm）** 在每个单独样本的特征维度上归一化。它不依赖于批量中的其他样本，使其成为 Transformer 和循环网络的标准选择。

- **实例归一化**在每个样本和每个通道独立地在空间维度上归一化。它在风格迁移中很流行。

- **组归一化**将通道分成组并在每组内归一化。它是 LayerNorm 和 InstanceNorm 之间的折衷。

![3D 张量带有彩色切片，显示 BatchNorm、LayerNorm 和 InstanceNorm 归一化的维度](../images/normalization_types.svg)

- **Dropout** 是一种正则化技术，在训练期间随机将一部分 $p$ 的神经元归零。这迫使网络不依赖任何单个神经元，鼓励冗余表示。在测试时，所有神经元都是活跃的。**反转 dropout** 在训练期间按 $\frac{1}{1-p}$ 缩放激活，以便测试时不需要缩放。这是标准实现。

- **卷积神经网络（CNN）** 利用空间结构。卷积层不是将每个输入连接到每个输出（如稠密层中那样），而是在输入上滑动一个小滤波器（核），在每个位置计算点积。相同的滤波器权重在所有位置共享，这大幅减少了参数并建立了平移不变性。

- 对于具有大小为 $k \times k$ 的滤波器 $K$ 的 2D 输入，**卷积运算**为：

$$(\text{input} * K)[i,j] = \sum_{m=0}^{k-1} \sum_{n=0}^{k-1} \text{input}[i+m, j+n] \cdot K[m, n]$$

![输入网格带有在其上滑动的 3x3 滤波器，在每个位置产生逐元素乘法和求和的输出特征图](../images/cnn_convolution.svg)

- 输出大小取决于三个超参数。**步幅**控制滤波器在位置之间移动多少像素（步幅 2 将空间尺寸减半）。**填充**在输入边界周围添加零（"same"填充保留空间大小，"valid"填充不）。输出大小公式：$\text{out} = \lfloor (\text{in} - k + 2p) / s \rfloor + 1$。

- **池化**层对特征图进行下采样。最大池化在每个窗口中取最大值；平均池化取均值。池化在保留最重要信息的同时减少空间维度。

- **膨胀卷积**在滤波器元素之间插入间隙，增加感受野而不增加参数。膨胀率为 2 意味着 3x3 滤波器覆盖 5x5 区域。

- **1x1 卷积**是使用 1x1 滤波器的卷积。它们不看空间邻居；相反，它们在通道之间混合信息。将它们视为在每个空间位置应用稠密层。它们用于廉价地改变通道数。

- **跳跃连接**（残差连接）让输入绕过一层或多层：$\text{output} = F(x) + x$。层只需要学习残差 $F(x) = \text{output} - x$，当最优变换接近恒等时这更容易。ResNets（残差网络）使用这个技巧堆叠超过 100 层，解决了更深网络比更浅网络表现更差的退化问题。

- CNN 构建**特征层次结构**。早期层检测边缘和纹理。中间层将这些组合成部件（眼睛、车轮）。后期层识别整个物体。每层的感受野（它可以"看到"的输入区域）随深度增长。

- **嵌入**将离散 token（单词、字符、项目 ID）映射到稠密向量。嵌入层只是查找表：形状为（词汇量大小，嵌入维度）的矩阵 $E$。查找 token $i$ 意味着选择 $E$ 的第 $i$ 行。这等效于乘以 one-hot 向量，这只是矩阵 - 向量乘法的一个特例（第 2 章）。嵌入在训练期间学习，因此相似的 token 最终具有相似的向量。

- **Token 化**是将原始文本转换为 token 序列的过程。词级 token 化在空格处分割，但无法处理未见过的单词。**子词 token 化**（BPE、WordPiece、SentencePiece）将文本分解为频繁的子词单元，平衡词汇量大小和覆盖率。单词"unhappiness"可能变成 ["un", "happiness"] 或 ["un", "happ", "iness"]。

- **循环神经网络（RNN）** 一次处理一个序列元素，维护一个携带信息向前的隐藏状态：

$$h_t = \tanh(W_h h_{t-1} + W_x x_t + b)$$

- 隐藏状态 $h_t$ 是网络直到时间 $t$ 所见的一切的压缩摘要。相同的权重 $W_h$ 和 $W_x$ 在所有时间步共享（权重共享，就像 CNN 共享空间权重）。

- 普通 RNN 难以处理长序列，因为梯度消失：从步骤 $t$ 到步骤 $t-k$ 的梯度信号通过 $k$ 次乘以 $W_h$，它指数级缩小（或爆炸）。

- **LSTM**（长短期记忆）通过引入单独的记忆细胞状态 $c_t$ 来解决这个问题，该状态随时间流动且干扰最小。三个门控制什么信息进入、离开和持续：

- **遗忘门**决定从记忆细胞状态中擦除什么：$f_t = \sigma(W_f [h_{t-1}, x_t] + b_f)$
- **输入门**决定写入什么新信息：$i_t = \sigma(W_i [h_{t-1}, x_t] + b_i)$，具有候选值 $\tilde{c}_t = \tanh(W_c [h_{t-1}, x_t] + b_c)$
- 记忆细胞状态更新：$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$
- **输出门**决定暴露什么：$o_t = \sigma(W_o [h_{t-1}, x_t] + b_o)$，以及 $h_t = o_t \odot \tanh(c_t)$

![LSTM 细胞显示遗忘门、输入门、输出门、记忆细胞状态高速通道和数据流连接](../images/rnn_lstm_cell.svg)

- 记忆细胞状态像传送带：信息可以在许多时间步内不变地流动（遗忘门保持在接近 1），这解决了长程依赖的梯度消失问题。

- **GRU**（门控循环单元）通过将记忆细胞状态和隐藏状态合并为一个，并使用两个门而不是三个来简化 LSTM：更新门（组合遗忘和输入）和重置门。GRU 具有更少的参数，通常表现与 LSTM 相当。

- RNN（包括 LSTM）的根本限制是顺序处理：你必须在处理 token 2 之前处理 token 1，在处理 token 3 之前处理 token 2。这阻止了并行化并造成信息瓶颈，因为所有上下文必须通过固定大小的隐藏状态挤压。

- **注意力**解决了这两个问题。注意力不让模型将整个输入压缩成固定向量，而是让它回顾所有输入位置并决定哪些位置与当前输出相关。

- 现代表述使用**查询、键和值（Q、K、V）**。把它想象成图书馆搜索：你有一个查询（你在找什么）、键（每本书上的标签）和值（实际的书内容）。你将查询与所有键进行比较，以确定要检索哪些值。

- **缩放点积注意力**：

$$\text{Attention}(Q, K, V) = \text{softmax}\!\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

- $QK^T$ 计算每个查询与每个键之间的相似度。这是矩阵乘法（第 2 章），条目是点积，测量余弦相似度（第 1 章）。除以 $\sqrt{d_k}$ 防止点积变得太大（这会使 softmax 饱和并产生近 one-hot 分布和梯度消失）。softmax 将相似度转换为概率分布。乘以 $V$ 产生值的加权组合。

- **多头注意力**运行 $h$ 个并行注意力操作，每个操作具有 Q、K 和 V 的不同学习投影。这让模型同时关注来自不同表示子空间的信息。一个头可能关注句法关系，而另一个关注语义关系。输出被拼接并投影：

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, \ldots, \text{head}_h) W^O$$

- **Transformer** 架构（Vaswani 等人，2017）完全由注意力和前馈层构建，没有循环。编码器块重复：多头自注意力、加和层归一化、前馈网络、加和层归一化。解码器块添加掩码自注意力（防止模型看到未来 token）和关注编码器输出的交叉注意力层。

![Transformer 编码器块：多头注意力、加和层归一化、前馈网络、加和层归一化，带有残差连接](../images/transformer_block.svg)

- **位置编码**是必要的，因为注意力是排列等变的，这意味着它将输入视为集合而非序列。没有位置信息，"the cat sat on the mat"和"the mat sat on the cat"将是相同的。原始 Transformer 使用正弦位置编码：

$$PE_{(pos, 2i)} = \sin\!\left(\frac{pos}{10000^{2i/d}}\right), \quad PE_{(pos, 2i+1)} = \cos\!\left(\frac{pos}{10000^{2i/d}}\right)$$

- 每个位置获得一个唯一的向量，模型可以使用它来区分位置。现代模型通常使用学习的位置嵌入或相对位置编码（RoPE、ALiBi）。

- Transformer 并行处理所有 token（自注意力矩阵 $QK^T$ 在一次矩阵乘法中计算），这使得它们在现代硬件上比 RNN 快得多地训练。权衡是自注意力在序列长度上是 $O(n^2)$（每个 token 关注所有其他 token），而 RNN 是 $O(n)$。这就是为什么长上下文模型需要特殊注意力变体（稀疏注意力、线性注意力、flash 注意力）。

- **视觉 Transformer（ViT）** 通过将图像分割成固定大小的块（如 16x16）、将每个块扁平化为向量、并将块视为 token 序列，将 Transformer 应用于图像。一个可学习的 [CLS] token 被前置，其最终表示用于分类。尽管没有任何卷积归纳偏置，ViT 在足够数据上训练时匹配或超越 CNN。

- **MLP-Mixer** 是更简单的架构，用 MLP 替换注意力和卷积。它在"token 混合"MLP（应用于空间位置）和"通道混合"MLP（应用于特征）之间交替。它具有竞争力，表明现代架构的关键洞察不是注意力本身，而是跨 token 和特征高效混合信息。

- **自动编码器**通过训练网络重建其自己的输入来学习压缩表示。编码器将输入映射到较低维的瓶颈（潜在代码），解码器将其映射回来：

$$z = f_{\text{enc}}(x), \quad \hat{x} = f_{\text{dec}}(z), \quad \mathcal{L} = \|x - \hat{x}\|^2$$

- 瓶颈迫使网络学习最重要的特征。自动编码器用于降维、去噪（在噪声输入上训练，重建清晰输出）和异常检测（高重建误差信号不寻常的输入）。

- **变分自动编码器（VAE）** 添加了概率转折。编码器不编码到单一点 $z$，而是输出分布的参数（高斯的均值 $\mu$ 和方差 $\sigma^2$）。潜在代码从该分布采样：$z = \mu + \sigma \odot \epsilon$，其中 $\epsilon \sim \mathcal{N}(0, I)$。这个**重参数化技巧**使采样可微，因此梯度可以流过。

- VAE 损失有两项：

$$\mathcal{L} = \underbrace{\|x - \hat{x}\|^2}_{\text{重建}} + \underbrace{D_{\text{KL}}(q(z|x) \| p(z))}_{\text{正则化}}$$

- KL 散度项（来自第 5 章）将学习到的后验 $q(z|x)$ 推向先验 $p(z) = \mathcal{N}(0, I)$，确保潜在空间平滑且结构良好。然后你可以从先验采样并解码以生成新数据。这使 VAE 成为生成模型。
