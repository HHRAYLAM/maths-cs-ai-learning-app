# 基与对偶

*基定义了向量空间的坐标系，对偶揭示了线性函数如何作用于向量。本章涵盖线性无关性、生成集、基变换、对偶空间和余向量，这些是机器学习中 PCA、特征变换和注意力查询背后的概念。*

- 我们已经看到向量生活在具有一定数量维度的空间中。但是什么定义了这些维度？这就是**基向量**发挥作用的地方。

- **基**是一组向量，可以通过缩放和加法（线性组合）构建空间中的每个其他向量，没有冗余。它们是空间的构建块。

- 基必须满足两个条件：

    - **线性无关**：没有基向量可以由其他向量构建。每个向量都贡献一个真正新的方向。

    - **生成**：空间中的每个向量都可以表示为基向量的组合。没有遗漏。

- 基中向量的数量等于空间的**维度**。在 $\mathbb{R}^2$ 中你需要 2 个，在 $\mathbb{R}^3$ 中你需要 3 个，依此类推。

- 最自然的基是**标准基**，沿每个坐标轴的单位向量：

    - 在 $\mathbb{R}^2$ 中：$\hat{\mathbf{i}} = (1, 0)$ 和 $\hat{\mathbf{j}} = (0, 1)$
    - 在 $\mathbb{R}^3$ 中：$\hat{\mathbf{i}} = (1, 0, 0)$，$\hat{\mathbf{j}} = (0, 1, 0)$，$\hat{\mathbf{k}} = (0, 0, 1)$

- 任何向量都只是这些基向量的加权和。向量 $(3, 2)$ 实际上是 $3\hat{\mathbf{i}} + 2\hat{\mathbf{j}}$。权重（3 和 2）是该向量在该基中的**坐标**。

- 但标准基不是唯一有效的基。在 $\mathbb{R}^2$ 中，向量 $(1, 1)$ 和 $(-1, 1)$ 也构成基。它们是线性无关的，可以到达平面中的任何点。同一个向量在这个新基中将只有不同的坐标。

- **基变换**使用不同的构建块重新表达相同的向量。向量没有移动，我们只是从不同的角度描述它。

- 这是通过乘以**基变换矩阵** $P$ 来完成的，其列是用旧坐标写入的新基向量。要返回，乘以 $P^{-1}$。

- 在机器学习中，基变换经常出现。例如，PCA 找到一个新的基（主成分），数据更容易理解，坐标轴与最大变化方向对齐。

- 现在，这里隐藏着一个更深刻的想法。当我们写 $\mathbf{v} = (3, 2)$ 时，坐标 3 和 2 实际上是沿每个基方向"测量" $\mathbf{v}$ 的结果。第一个坐标问"$\hat{\mathbf{i}}$ 中有多少在 $\mathbf{v}$ 中？"，第二个问"$\hat{\mathbf{j}}$ 中有多少？"

- 这些测量中的每一个都是**线性泛函**，一个接受向量并返回单个数字的函数。所有此类线性泛函的集合形成**对偶空间** $V^\ast$。

- 可以这样想：向量是被测量的对象，线性泛函是测量它们的*尺子*。对偶空间是所有可能尺子的集合。

- 对于每个基 $\{\mathbf{e}_1, \mathbf{e}_2, \ldots, \mathbf{e}_n\}$，有一个相应的**对偶基** $\{\mathbf{e}_1^\ast, \mathbf{e}_2^\ast, \ldots, \mathbf{e}_n^\ast\}$。每个对偶基向量精确提取一个坐标：

```math
\mathbf{e}_i^\ast(\mathbf{e}_j) = \delta_{ij} = \begin{cases} 1 & \text{如果 } i = j \\ 0 & \text{如果 } i \neq j \end{cases}
```

- $\mathbf{e}_1^\ast$ 应用于 $\mathbf{e}_1$ 时返回 1，对于其他向量返回 0。它完美地隔离了第一个坐标。

- 点积连接了这两个世界。当你计算 $\mathbf{u} \cdot \mathbf{v}$ 时，你可以认为一个向量作为"尺子"测量另一个向量。点积 $\mathbf{u} \cdot \mathbf{v}$ 与应用由 $\mathbf{u}$ 定义的线性泛函到向量 $\mathbf{v}$ 是相同的。

- 这意味着每个向量秘密地定义了一个线性泛函，每个线性泛函可以用一个向量表示。在有限维度中，对偶空间本质上是原始空间的镜像。

- 对偶现在可能看起来抽象，但它是许多实用想法的基础：坐标是对偶基求值，点积是对偶配对，像神经网络中的注意力这样的变换通过让一组向量"查询"另一组来操作，这就是对偶的实际应用。

## 编码任务（使用 CoLab 或 notebook）

1. 在两个不同的基中表达一个向量并验证它们表示相同的点。尝试创建自己的基并看看向量获得什么坐标。
```python
import jax.numpy as jnp

v = jnp.array([3.0, 2.0])

# 标准基：坐标只是分量
print(f"Standard basis coords: {v}")

# 新基：(1,1) 和 (-1,1)
P = jnp.array([[1.0, -1.0],
               [1.0,  1.0]])
new_coords = jnp.linalg.solve(P, v)
print(f"New basis coords: {new_coords}")

# 验证：从新坐标重建
reconstructed = new_coords[0] * P[:, 0] + new_coords[1] * P[:, 1]
print(f"Reconstructed: {reconstructed}")
```

2. 验证对偶基性质：每个对偶基向量精确提取一个坐标并为其他坐标返回零。
```python
import jax.numpy as jnp

# R3 中的标准基
e1 = jnp.array([1.0, 0.0, 0.0])
e2 = jnp.array([0.0, 1.0, 0.0])
e3 = jnp.array([0.0, 0.0, 1.0])

v = jnp.array([5.0, 3.0, 7.0])

# 每个点积提取一个坐标
print(f"e1 · v = {jnp.dot(e1, v)}")
print(f"e2 · v = {jnp.dot(e2, v)}")
print(f"e3 · v = {jnp.dot(e3, v)}")
```
