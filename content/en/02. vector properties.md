# 向量属性

*向量属性描述了定义向量行为的几何和代数特征。本文件涵盖大小、方向、单位向量、相等性、平行性、正交性和线性无关性，这些都是每个机器学习特征空间的基本构建块。*

- 向量的**大小**（或长度）告诉你它*延伸多远*。把它想象成箭头的长度。对于向量 $\mathbf{a} = (a_1, a_2, a_3)$，其大小为：

$$\|\mathbf{a}\| = \sqrt{a_1^2 + a_2^2 + a_3^2}$$

- 这只是扩展到更高维度的勾股定理，测量从原点到该点的直线距离。

- 向量的**方向**告诉你它*指向哪里*；只需想象一条从原点到坐标点的直线。

- 当原点未明确指定时，我们通常默认为 (0,0,...0)，即中心点，至少出于可视化目的。

- 位置无关紧要，它始终关于位移：从原点绘制的向量 $(3, 2)$ 和从另一点绘制的相同向量 $(3, 2)$ 仍然是相等的。

![向量相等性：从两个不同起点绘制的相同 (3,2) 向量](../images/vector_equality.svg)

- 两个向量可以有相同的长度但指向完全不同的方向，或者指向相同的方向但长度不同。

![相同方向，不同大小（v 和 2v）vs 相同大小，不同方向](../images/magnitude_direction.svg)

- 当且仅当两个向量的所有对应分量都匹配时，它们才**相等**；相同的长度、相同的方向，完全相同的箭头。

$$\mathbf{a} = \mathbf{b} \iff a_i = b_i \text{ for all } i$$

- 如果一个向量是另一个向量的标量倍数，则两个向量是**平行**的。它们沿着同一直线指向，要么方向相同，要么完全相反。

$$\mathbf{a} \parallel \mathbf{b} \iff \mathbf{a} = k\mathbf{b} \text{ for some scalar } k \neq 0$$

![平行向量：a 和 b 指向相同方向，a 和 -b 指向相反方向](../images/parallel_vectors.svg)

- 如果 $k > 0$，它们指向相同方向。如果 $k < 0$，它们指向相反方向。无论哪种情况，它们都在通过原点的同一直线上。

- 直观地说，平行向量不携带"新"的方向信息。一个只是另一个的拉伸或翻转版本。

- 如果两个向量指向完全独立的方向，则它们是**正交**（垂直）的。沿着一个向量移动对另一个方向没有进展。

![正交向量：u 和 v 以直角相交](../images/orthogonal_vectors.svg)

- 想象一下先向北走再向东走，这些是正交的方向，无论向北走多少都不会让你向东移动。我们将会经常遇到正交性。

- 正交性在机器学习中至关重要：正交的特征携带完全独立的信息，这对于表示来说是理想的。

- 更一般地，任何两个向量之间都有一个**角度** $\theta$，范围从 $0°$ 到 $180°$。

- 这个角度捕捉了两个方向之间的整个关系：$0°$ 意味着平行（相同方向），$180°$ 意味着平行（相反方向），而 $90°$ 意味着正交。之间的所有值都是混合状态。

- 机器学习中的大多数向量关系都位于这个范围内的某处。稍后，我们将看到精确的工具（点积、余弦相似度）来计算这个角度。

- 如果一组向量中至少有一个可以通过缩放和加法从其他向量构建出来，则该组向量是**线性相关**的。它没有为集合带来新的信息。

- 例如，如果 $\mathbf{c} = 2\mathbf{a} + 3\mathbf{b}$，那么 $\mathbf{c}$ 是冗余的，你已经通过 $\mathbf{a}$ 和 $\mathbf{b}$ 获得了 $\mathbf{c}$ 所提供的所有内容。

- 平行向量总是线性相关的，因为一个只是另一个的缩放副本。任何包含零向量的集合也是线性相关的。

- 如果向量中没有一个可以由其他向量构建出来，则这些向量是**线性无关**的。每个向量都贡献一个真正新的方向。正交向量总是线性无关的。

- 在二维中，两个线性无关的向量可以到达平面中的任何点。在三维中，你需要三个。这种"需要多少个独立向量"的想法直接关联到维度。

- 当向量的大多数分量为零时，该向量是**稀疏**的。相反，大多数分量非零的情况称为**稠密**。

$$\mathbf{s} = [0, 0, 3, 0, 0, 0, 1, 0, 0, 0]$$

- 稀疏性很重要，因为它影响存储和计算。稀疏向量可以通过仅跟踪非零项来更高效地存储和处理。

- **单位向量**是大小正好为 1 的向量。它纯粹表示方向，没有长度信息。你可以通过除以其大小将任何向量转换为单位向量：

$$\hat{\mathbf{a}} = \frac{\mathbf{a}}{\|\mathbf{a}\|}$$

- 这个过程称为**归一化**。它去掉了"多远"的信息，只保留"哪个方向"。

- 标准单位向量沿着每个坐标轴：$\hat{\mathbf{i}} = (1, 0, 0)$，$\hat{\mathbf{j}} = (0, 1, 0)$，$\hat{\mathbf{k}} = (0, 0, 1)$。任何向量都可以写成这些的组合，例如 $(3, 2, 4) = 3\hat{\mathbf{i}} + 2\hat{\mathbf{j}} + 4\hat{\mathbf{k}}$。

## 编码任务（使用 CoLab 或 notebook）

1. 计算向量的大小并验证其与勾股定理一致，然后修改为计算单位向量。
```python
import jax.numpy as jnp

a = jnp.array([3.0, 4.0])

magnitude = jnp.sqrt(jnp.sum(a ** 2))
print(f"Magnitude of a: {magnitude}") 
```

2. 通过测试一个向量是否是另一个向量的标量倍数来检查两个向量是否平行。
```python
import jax.numpy as jnp

a = jnp.array([2, 4, 6])
b = jnp.array([1, 2, 3])

ratios = a / b
print(f"Ratios: {ratios}")
print(f"Parallel: {jnp.allclose(ratios, ratios[0])}")
```
