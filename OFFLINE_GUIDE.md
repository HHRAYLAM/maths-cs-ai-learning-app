# 离线学习指南

## 如何在飞机上离线学习

### 方法 1：PWA 离线模式（推荐）

1. **出发前准备（必须在家完成）：**
   - 打开 https://hhraylam.github.io/maths-cs-ai-learning-app/
   - 等待页面完全加载（所有课程内容会自动缓存）
   - 刷新页面 1-2 次，确保所有内容加载完成
   - 点击底部导航栏的每个标签页，确保都能正常显示

2. **验证离线模式：**
   - 打开浏览器开发者工具（F12）
   - 进入 Network（网络）标签
   - 勾选"Offline"模拟离线
   - 刷新页面，如果仍能正常显示说明缓存成功

3. **飞行中离线使用：**
   - 打开应用（无需网络连接）
   - 所有已访问过的内容都可以离线学习
   - 学习进度会保存在本地
   - 恢复网络后，进度会自动同步

### 方法 2：下载整个项目到本地

如果 PWA 缓存不够用，可以下载整个项目：

```bash
# 克隆仓库到本地
git clone https://github.com/HHRAYLAM/maths-cs-ai-learning-app.git

# 使用任意 HTTP 服务器启动
# 方式 1：使用 Python
cd maths-cs-ai-learning-app
python -m http.server 8000

# 方式 2：使用 Node.js
npx serve .

# 方式 3：使用 VS Code Live Server 插件
```

然后在浏览器打开 `http://localhost:8000` 即可离线学习。

## 注意事项

1. **首次加载需要网络**：PWA 缓存需要在有网络时先访问一次
2. **缓存大小**：完整内容约几 MB，确保设备有足够空间
3. **浏览器兼容性**：
   - Chrome/Edge：完全支持
   - Safari：部分支持（iOS 11.3+）
   - Firefox：完全支持
4. **清除缓存**：如需重新缓存，在浏览器设置中清除网站数据
